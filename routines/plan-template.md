# Plan Template — Claude Code UI のPlan機能用テンプレート

> 大井が「これプランにして」と言ったらこのテンプレでプランを書く。
> Claude Code UI の Plan 機能に貼り付けると、Claudeが長時間自律実行する。

## 標準テンプレ

```markdown
# Plan: <目標タイトル>

## 完了基準（測定可能）
- ...
- ...

## 期限
- 開始: YYYY-MM-DD
- 締切: YYYY-MM-DD

## フェーズ分解
### Phase 1: <タイトル> (見積X時間)
**目的**: ...
**手順**:
- [ ] ステップ1
- [ ] ステップ2
**OpenClaw委任**: あり/なし（あれば dispatch.ps1 コマンド）
**完了確認**: <ファイル / 数値 / 外部反応>

### Phase 2: ...
（同上）

## 中間レポート
- 各Phase完了時にDiscord通知（Claude🧠）
- フェーズ完了でwiki/_routines/plans/{plan-slug}/ に記録
- 大井に判断仰ぐ事項あれば即停止して確認

## 関連リンク
- [[entities/<product>]]
- [[20_areas/<area>]]
- 進捗トラッカー: wiki/_routines/plans/{plan-slug}/progress.md

## 失敗時の対処
- OpenClaw失敗3連続 → Discord red通知 → 大井判断仰ぐ
- フェーズで予定時間2倍超 → 再計画
```

## 即使えるPlan例（参考）

### Plan A: AIpa Web モニター3社獲得（1-2週間）

```markdown
# Plan: AIpa Web モニター3社獲得

## 完了基準
- モニター価格15万円で3社契約
- 事例掲載許可取得

## 期限
- 開始: 2026-05-19
- 締切: 2026-06-02

## Phase 1: ターゲット業界選定 (1時間)
**目的**: 採用難易度が高く、Web投資余力ある地方中小を絞る
**手順**:
- [ ] [[entities/aipa-web]] の Deep Research を再読
- [ ] 候補業界3つ選定（建設・物流・製造）
**OpenClaw委任**: なし（Claude判断）

## Phase 2: 各業界10社リサーチ (1日・OpenClaw並列30本)
**目的**: アプローチ可能な実在企業30社のリスト作成
**手順**:
- [ ] 業界1（建設）10社→OpenClaw 10本dispatch
- [ ] 業界2（物流）10社→OpenClaw 10本dispatch
- [ ] 業界3（製造）10社→OpenClaw 10本dispatch
**OpenClaw委任**: あり
**完了確認**: wiki/research/aipa-web-targets/{業界}-{社}.md × 30本

## Phase 3: アプローチメール草稿 (1日・OpenClaw並列30本)
**目的**: 各社向けカスタマイズメール
**手順**:
- [ ] 30本のテンプレ生成（OpenClaw）
- [ ] Claude校正・大井レビュー

## Phase 4: 大井送信 (大井のタスク・複数日)
**目的**: 30社にメール送信、返信待ち

## Phase 5: フォロー&クロージング (返信ベース)
**目的**: 商談→契約3件
```

## 大井が言うトリガーフレーズと対応

| 大井の言葉 | Claudeの動き |
|---|---|
| 「これプランにして」 | このテンプレで起こす + Plan UI 用テキスト出す |
| 「○○を1週間で達成したい」 | 期限付きプラン化 |
| 「長期で見たい」 | プラン化 + 週次レビューで進捗追跡 |
| 「分解して」 | フェーズ分解だけして即実行（プラン化はオプション） |
| 「一旦保留」 | プランをwiki/_routines/plans/に保存して中断 |
