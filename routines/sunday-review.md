# 日曜の棚卸し 21:00（30分）

## あなたの役割
大井湧瑛の右腕として、週次で「先週の動き + 来週の打ち手」を棚卸す。

> 月次レビューは廃止。週次だけでいい。

## 実行手順（30分以内）

### 1. 先週の素材を集める（10分）

**朝/夜のメモ（軽量版）:**
- `wiki\_routines\morning-{月}.md` 〜 `morning-{土}.md`
- `wiki\_routines\evening-{月}.md` 〜 `evening-{土}.md`

**事実データ:**
- `queue\done\` の今週完了タスク（OpenClaw成果物）
- `~/.claude/projects/` の今週セッション数（時刻のみ）
- gh CLI で主要リポジトリの今週commit数
- Stripe MCP（あれば）で今週の入金合計

### 2. 棚卸し（Sonnet判定・15分）

#### A. 動いた事業 / 動かなかった事業
15事業について：
- 今週動きあり: <プロジェクト名>
- 1週間以上動きなし: <プロジェクト名>（捨てるか・寝かすか判断）

#### B. 今週の数字
- 売上（Stripe MCP）
- 新規顧客 / 解約
- LP/SNS反応
- API/SaaSコスト

数字とれないなら「不明」でOK。

#### C. 大井の脳汁マップ更新
- 今週脳汁が出た事業: <Tier A候補>
- 今週冷めた事業: <Tier C・捨て候補>
- 新しく出てきたアイデア: <記録>

### 3. OpenClaw品質振り返り（5分・PDCA改善）

`wiki\_inbox\` と `wiki\<正式パス>` のうち今週分を巡回し、OpenClaw出力品質を評価：

**評価軸:**
- 良品質（昇格できた）: X件
- 微妙（再dispatch必要だった）: Y件
- ダメ（削除した）: Z件

**改善発見:**
- ハルシネーションが多かった部署: ___
- フォーマット崩れが多かったテンプレ: ___
- prompt-prefix.md に追加すべきガードレール: ___

**改善アクション:**
- prompt-prefix.md の修正提案（あれば直接編集）
- 特定テンプレ（scripts\templates\<部署>\<name>.md）の修正提案

### 4. 来週の打ち手3個（10分）

**判定軸：**
1. 売上に最も近い動き
2. 期限がある案件（ビジコン応募・商談・面談）
3. 脳汁駆動できる事業
4. 「やめる事業」の判断（重要）

**3個構成：**
- 攻める1個（脳汁出る・売上近い）
- 種まき1個（中期育成・コンテンツ・関係構築）
- やめる/寝かす1個（Tier C下げ or Killの判断）

### 5. /idea-judge skill 利用（任意）
週次なら軽め、月初の日曜なら全プロジェクトの7軸採点再判定。

### 6. /grill-me skill 利用（迷ったら）
来週の最優先1個に絞れないなら `/grill-me` で大井への問い生成。

### 9. 終了
