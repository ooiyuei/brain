# Agents of Flag (AOF)

> **一言**: Agents of Flagは、光線銃を使った新しいリアルFPSスポーツ。
> サバゲーの興奮を、痛み・装備差・自己申告なしで再現し、FPSのランク・戦績・チーム・大会・観戦の中毒構造をリアル空間に持ち込む。
> 目指すのは、**世界を熱狂させるリアルeスポーツ**。

## コンセプト

### 何が新しいか
- 光線銃 + センサー + A/Bフラグ制の5vs5リアルFPS
- 既存の問題を全部解決:
  - サバゲー → 痛い・装備差・自己申告が嫌
  - レーザータグ → データなし・eスポーツ感なし・子供向け
  - アーケードシューター → 対人戦なし・緊張感なし
- FPS（Valorant/CS的）のゲーム性をリアル空間で実現

### ターゲット
- 20代〜30代男女
- FPSゲーマー（リアル体験したい層）
- サバゲー興味あるが痛みが怖い層
- eスポーツ観戦者
- 法人（社内イベント・懇親会・採用）

## ルール設計

### 基本ルール
- **5vs5**、A/Bフラグ制（Counter-Strike的）
- 攻撃チームがフラグ奪取・運搬・設置
- 防衛チームがフラグを守る
- **3点先取**（3ラウンド先取）

### ルール種別
| ルール | 内容 |
|--------|------|
| Casual | 復活あり、時間制、初心者向け |
| Swift | 短時間・テンポ重視 |
| Formal | 復活なし、競技向け |

### Defense Lock
攻撃側が3ラウンド連続で負けると発動 → 次ラウンドで防衛有利ルール適用

### HP設計
- HP2（通常）/ 頭ショット: 2ダメ（即ダウン）/ 胴体: 1ダメ
- ダウン後は復活地点へ

## ハードウェア設計

### 銃（ライトガン）
- AOF専用3D筐体、赤外線送受信
- 音・振動・LED搭載、**非軍事デザイン**

### センサー（装備）
- ヘッドバンド: チーム色LED / ヘッドショット時赤フラッシュ
- ベスト: チーム色LED / 胴体被弾時赤 / ダウン時赤

### フラグドック
A/Bフラグ物理設置 / 磁石極性判定 / 台座スイッチ / LEDリング / 観戦UI反映

### LED演出
**銃**: チームカラー / 発射フラッシュ / 被弾赤 / ダウン赤 / 復活時チーム色 / エラー黄 / バッテリー低下黄点滅
**装備**: ヘッドバンド（チーム色/ヘッドショット赤）/ ベスト（チーム色/被弾赤/ダウン赤）
**店内**: 入口ロゴ / フィールド照明 / カウントダウン / フラッグ状態 / ラウンド勝利演出 / MVP演出 / 観戦モニター連動 / スポンサー表示

## 観戦UI

フィールドにカメラ設置 → 大画面TVで観戦

**表示**: チーム名 / HP / 生存状況 / A/Bポイント / フラッグ状況 / キルログ / 残り時間 / MVP候補

**脳汁演出**: HEADSHOT / ACE / CLUTCH / LAST AGENT STANDING / FLAG CAPTURED / MATCH POINT / DEFENSE LOCK / COMEBACK CHANCE / HERO MOMENT

> 観戦UIの目的: 待機時間を退屈ではなく、次の試合への熱狂を高める時間に変える。

## ランク・戦績システム

### ロール別スコア（K/Dだけで評価しない）
| スコア | 内容 |
|--------|------|
| Combat Score | 撃ち合い |
| Flag Score | 取得・運搬・刺し込み |
| Defense Score | 防衛・リーチ阻止 |
| Support Score | カバー・味方支援 |
| Clutch Score | 重要局面での活躍 |

初心者でも、撃ち合い以外で貢献できる設計。

### 継続ループ
