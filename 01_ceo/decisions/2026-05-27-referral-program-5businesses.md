ご提示いただいた「主力5事業横断ルール」に基づく、紹介プログラム（リファラルプログラム）の設計案を提案します。

この案は、**「紹介者へのインセンティブ（報酬率）」**、**「被紹介者への誘因（特典）」**、**「法的リスクの回避（契約書テンプレ）」**の3点に焦点を当て、5つの異なる事業領域で共通して適用できる汎用性の高いフレームワークとして構築しています。

# 主力5事業横断型 リファラルプログラム設計書

## 1. プログラムの基本概念
*   **目的:** 既存顧客・パートナーのネットワークを活用し、新規顧客の獲得コスト（CAC）を抑制しつつ、信頼性の高いリードを獲得する。
*   **対象事業:** 主力5事業（例：SaaS事業、コンサルティング事業、ハードウェア販売、サブスクリプションサービス、教育研修事業など※仮称）
*   **横断ルール:** 事業間での紹介も可能とし、クロスセルを促進する。

---

## 2. 紹介者報酬率（インセンティブ設計）

紹介者の動機付けを最大化するため、**「成功報酬型（成果連動）」**を基本とし、事業種別に応じた統一基準を設けます。

### 基本ルール
*   **報酬形態:** 現金（銀行振込）または 社内で使えるクレジット（ポイント制）
*   **支払いタイミング:** 被紹介者の契約成立かつ**初回決済完了後**、かつ**クーリングオフ期間終了後**
*   **有効期間:** 紹介リンク生成から30日以内の契約が対象（Cookie保持期間）

### 事業別 報酬率・上限額 一覧表

| 事業カテゴリ | 報酬基準 | 単価上限 | 備考 |
| :--- | :--- | :--- | :--- |
| **A. SaaS/プラットフォーム** | 初年度契約額の **10%** | 50,000円/件 | 継続契約（2年目以降）は **5%** の継続報酬あり |
| **B. 高額コンサルティング** | 契約額の **5%** | 200,000円/件 | 分割払いの場合は、入金に応じて按比例支払 |
| **C. ハードウェア/製品** | 販売額の **3%** | 10,000円/件 | 返品・返金があった場合は報酬を差し引く |
| **D. サブスクリプション** | 月額料金の **1ヶ月分** | 15,000円/件 | 継続月数に応じてボーナス加算あり |
| **E. 教育/研修サービス** | 受講料の **15%** | 30,000円/件 | 団体紹介（3名以上）は **20%** にアップ |

> **横断特典:** 異なる事業間での紹介（例：SaaS顧客からコンサルティング顧客へ）の場合、報酬率を **1.2倍** にインセンティブを追加。

---

## 3. 被紹介者特典（新規顧客誘因）

紹介された側（被紹介者）にも明確な価値を提供し、成約率を向上させます。

### 共通特典
*   **初回割引:** 契約金額の **10% OFF** または **固定額割引（例: 10,000円OFF）**
*   **無料トライアル/相談:** 通常有料の機能やコンサルティングを **1ヶ月間無料** で提供

### 事業別 追加特典

| 事業カテゴリ | 被紹介者特典内容 |
| :--- | :--- |
| **A. SaaS/プラットフォーム** | 導入サポートキット（初期設定代行）無料 + 延長トライアル（1ヶ月追加） |
| **B. 高額コンサルティング** | 戦略診断セッション（30分）無料 + 提案書作成手数料免除 |
| **C. ハードウェア/製品** | 延長保証（1年間無料） + 専用アクセサリー1点プレゼント |
| **D. サブスクリプション** | 年会費契約時の **2ヶ月分無料** |
| **E. 教育/研修サービス** | 受講資料のデジタル版プレゼント + 修了証発行手数料免除 |

## 4. 契約書テンプレート（紹介プログラム規約）

紹介者（リファラー）と当社（プラットフォーム側）の間で締結する、基本的な規約の骨子です。
※実際の運用では、弁護士による法的審査を必ず行ってください。

---

### **紹介プログラム参加規約**

**第1条（目的）**
本規約は、[会社名]（以下「当社」という）が実施する紹介プログラム（以下「本プログラム」という）への参加条件、紹介者の権利・義務、報酬の支払い条件などを定めることを目的とします。

**第2条（定義）**
1. 「紹介者」とは、本プログラムに登録し、自らのネットワークを通じて新規顧客（以下「被紹介者」という）を当社に紹介する個人または法人を指します。
2. 「被紹介者」とは、紹介者の紹介により、当社のサービスまたは商品（以下「対象サービス」という）を新規契約した顧客を指します。
3. 「紹介成功」とは、被紹介者が対象サービスとの契約を結び、かつ初回決済が完了し、クーリングオフ期間等による解約がない状態を指します。

**第3条（紹介方法）**
1. 紹介者は、当社が提供する専用紹介リンクまたは紹介コードを使用して、被紹介者を当社へ誘導しなければなりません。
2. 紹介者は、虚偽の情報や不当な勧誘を行ってはなりません。

**第4条（報酬）**
1. 紹介成功した場合、当社は紹介者に対し、別紙「報酬基準表」に記載された金額（以下「紹介報酬」という）を支払います。
2. 紹介報酬の支払いは、被紹介者の契約成立後、かつ所定の審査期間（通常30日）を経てから、月次で一括して行います。
3. 紹介報酬は現金または当社指定のクレジットとして支払われます。

**第5条（除外事項）**
以下の場合は、紹介報酬を支払わない場合があります。
1. 紹介者自身、またはその親族・従業員が被紹介者となった場合。
2. 被紹介者が詐欺、不正行為等により契約が無効となった場合。
3. 被紹介者が紹介報酬を得ることを目的として、自らのアカウントから偽装紹介を行った場合。

**第6条（プライバシーとデータ保護）**
1. 紹介者は、被紹介者の個人情報（氏名、連絡先等）を収集・保存・第三者提供してはなりません。
2. 紹介者は、当社のプライバシーポリシーおよび関連法令を遵守し、被紹介者の同意を得ずに個人情報を扱うことは禁止します。

**第7条（規約の変更）**
当社は、必要に応じて本規約および報酬基準を変更できるものとします。変更後は、当社ウェブサイト等で公告し、効果を生じます。

**第8条（準拠法）**
本規約は日本法に準拠し、解釈されます。

## 5. 運用上の注意点・リスク管理

1.  **重複紹介の防止:**
    *   Cookieや referral ID を使用し、最後にクリックした紹介者（Last Click Attribution）を優先するか、または事業部ごとに優先順位を定義する。
2.  **不正防止:**
    *   同一IPアドレスからの複数登録、ボットによる自動登録の検知システムを導入する。
    *   紹介者と被紹介者の関係性（親族・従業員）を自己申告させる。
3.  **税務処理:**
    *   個人への報酬支払いは「一時所得」または「雑所得」となり、源泉徴収が必要になる場合がある。税理士と連携し、適正な処理を行う。
4.  **事業横断の課題:**
    *   5事業間でCRM（顧客管理システム）を統合し、紹介元の事業を正確にトラッキングできる体制を整備する。
