# ⚠ 重要 — この決議書の意味

**法人登記後 3か月以内** に役員報酬を **書面で決定** しなければ、税務上「定期同額給与」と認められず **損金不算入** = 法人税が跳ね上がる。

株式会社 Almeo (ALMEO) の場合:
- 登記完了予定: 2026-06-?
- 期限: 登記日 + 3か月 = 2026-09-?
- 書面決議 → 取締役会議事録 → 保管 (永久)

# 取締役決議書

## 第1号議案: 代表取締役の月額役員報酬の決定

株式会社 Almeo (以下「当社」) は、会社法第361条及び当社定款の規定に従い、下記のとおり代表取締役 大井湧瑛 の月額役員報酬を決定する。

### 決定内容

| 項目 | 内容 |
|---|---|
| 対象役員 | 代表取締役 大井湧瑛 |
| 報酬区分 | 定期同額給与 |
| 月額報酬 (税込) | **【金額を記入】** 円 |
| 支給方法 | 毎月末日締・翌月25日払 |
| 支給開始月 | 【YYYY年MM月】より |
| 支給期間 | 翌定時株主総会まで (毎年見直し) |
| 適用税法 | 法人税法 第34条 第1項 第1号 (定期同額給与) |

### 決議の根拠

1. 当社定款第○条 (役員報酬) に基づく
2. 会社法第361条 (取締役の報酬等)
3. 法人税法第34条第1項第1号 (定期同額給与)

### 補足

- 本決議は当社の経営状況・業績の見通しを勘案した上で決定
- 月額報酬は税込支給額とし、当社負担の社会保険料・源泉所得税等は別途算定
- 退職慰労金については別途決議事項とする
- 業績連動報酬・賞与は本決議の対象外

## 使用上の注意

1. **登記後 3か月以内に書面化** ← 絶対遵守
2. **月額報酬の金額** は税理士と相談して決定
   - 安すぎる: 個人税負担増 (給与所得控除を活かせない)
   - 高すぎる: 法人税は減るが個人税負担増・社会保険料増
   - 推奨: 法人課税所得が出ない or 過小な水準で個人課税最小化
3. **支給開始月** は登記月 or 翌月から (定期同額の起算点)
4. 1年間は **金額変更不可** (変更すると定期同額崩れる)
5. 業績連動・賞与は別決議で対応
6. 株主総会議事録 (1人株主なら自己整理) と取締役会議事録の両方が必要な場合あり (定款規定確認)
7. **電子契約 OK** (クラウドサイン等) だが、印影付き PDF が安全
8. 保管期間: **永久** (`05_corp/legal/governance/YYYY/`)
