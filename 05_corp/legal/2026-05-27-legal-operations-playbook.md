# 法律回し 運用ルールブック v1

> 大井方針: 「法律回し頼みたい」
> = **コンプライアンス・法務対応を仕組み化** = 1人運営でも漏れない・後で困らない体制を構築。

---

## 🎯 大原則 3つ

### 原則1: 「気づいた時にやる」を捨てる → カレンダー駆動
- 法務は **「いつ・何を・どこに」が決まってる** ものほぼ全部
- 気合や記憶で回さない、Google Calendar に **全部入れる**
- リマインド = 1週間前 + 当日朝

### 原則2: 法務 = 弁護士に任せきりにせず「社内の最低ライン」を引く
- 月3-5万の顧問弁護士は「相談用」、書類作成・初期チェックは自社
- 各書類に「弁護士確認必須マーク (⚠)」「自社で完結可能 (✅)」 を分類
- 月1回・30分の弁護士相談タイムを確保

### 原則3: 「あとで分かる」を全部記録
- 議事録・契約書・領収書は **PDF + Obsidian 索引** で永久保管
- メール・Slack の重要やり取りも スクショして `05_corp/legal/correspondence/` に格納
- 全ファイル命名規則: `YYYY-MM-DD-種別-相手方-件名.pdf`

## 📅 月次ルーティン (毎月1日・60分・Calendar 固定)

### 経理 (15分)
- [ ] 前月の請求書発行 (顧客全社・freee 自動化)
- [ ] 前月の支払い実行 (光熱費・SaaS 等・自動引落含む)
- [ ] 前月の領収書 PDF 保管 (Drive `経理/YYYY/MM/`)
- [ ] freee/マネーフォワード で前月確定 (税理士共有)

### 法務 (20分)
- [ ] 前月の契約締結を `05_corp/legal/contracts/` に PDF 保管
- [ ] 期限切れ近い契約 (3ヶ月以内) を確認・更新交渉
- [ ] 取締役決議書 (役員報酬・重要契約等) を作成・押印・保管
- [ ] 株主名簿の更新 (異動あれば)

### コンプラ (15分)
- [ ] アカウント棚卸し (鍵管理・不要 SaaS 解約)
- [ ] 個人情報の取扱状況確認 (新規顧客追加・退会対応)
- [ ] セキュリティアラート確認 (Supabase RLS・Vercel deploy 失敗・GitHub の脆弱性警告等)
- [ ] バックアップ確認 (brain/_secrets backup・Drive backup・GitHub )

### 営業・対外 (10分)
- [ ] 顧客への月次報告書送付 (ALMEO 契約 5社等)
- [ ] 商工会議所への活動報告 (会員企業の場合)
- [ ] X/note の発信実績まとめ → 来月の発信計画

→ **Calendar に毎月1日 09:00-10:00 「法務月次ルーティン」固定登録 (Tomatoカラー)**

## 🚨 法務イベント発生時の判断フロー

### 取引先から契約書を渡された
```
[1] 自社業務委託契約と比較 → 不利条項チェック
   - 知財帰属が相手側になってないか
   - 損害賠償が無制限になってないか (上限キャップ要求)
   - 競業避止が広すぎないか
   - 解除条件が一方的でないか
   - 反社条項あるか
   ↓
[2] 修正したい箇所を赤入れ → 相手に返送
   ↓
[3] 高額・長期契約 → 弁護士に有料レビュー (¥3-5万/件)
   ↓
[4] 締結 → 電子契約 (クラウドサイン / 電子印鑑GMOサイン)
   ↓
[5] PDF 保管 → `05_corp/legal/contracts/YYYY/`
```

### 顧客から個人情報削除要求
```
[1] 本人確認 (なりすまし防止)
   ↓
[2] 削除実行 (Supabase 該当レコード論理削除)
   ↓
[3] 関連バックアップ・ログも削除確認
   ↓
[4] 14日以内に「削除完了」通知 (個情法65条)
   ↓
[5] 対応記録を `05_corp/legal/privacy-requests/YYYY/` に保管
```

### 退職者・元外注先からクレーム
```
[1] 即座に SOS (顧問弁護士に連絡・¥0)
   ↓
[2] やり取り全部メールに切替 (口頭禁止)
   ↓
[3] 返答は弁護士確認後 (即答しない)
   ↓
[4] 必要なら和解 or 提訴判断
   ↓
[5] 全記録を保全 (削除厳禁)
```

### SNS で炎上・誹謗中傷
```
[1] 反応しない (24時間ルール)
   ↓
[2] スクショ保全 (削除前提に証拠化)
   ↓
[3] 度合いに応じて: 黙殺 / 訂正発表 / 弁護士相談 / 刑事告訴
   ↓
[4] 自社 SNS の発信内容を見直し
```

### 補助金・税務調査
```
[1] 通知文書を即読み (期限確認)
   ↓
[2] 税理士に即共有 (24時間以内)
   ↓
[3] 求められた書類を準備 (期限内)
   ↓
[4] 立会日は弁護士・税理士同席
   ↓
[5] 後日結果に応じて修正申告 or 異議申立
```
