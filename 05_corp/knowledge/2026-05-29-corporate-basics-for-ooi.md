# 法人としての立ち回り 基礎知識

> 相方からの宿題: 「会社としての基本的な立ち回りを勉強して」
> 大井 (17歳・Almeo 法人化目前) が「分かってる」と言えるレベルまで。
> 比喩: **会社 = RPG のギルド** で説明する。

---

## §0. 全体像 (まず地図)

```
[株主] = オーナー (株を持つ人・ギルドの持ち分を持つ人)
   ↓ お金を出して株をもらう / 経営を委任する
[取締役] = 経営を任された人 (ギルマス・大井はここも兼ねる)
   ↓ 経営する
[会社 (法人)] = 事業をやる箱
   ↓ 関係を結ぶ
[他社] ← 業務提携 / 資本提携 / 子会社化 / M&A
```

**今の大井 = Almeo の「100%株主」かつ「代表取締役」** (一人で両方やってる)。

---

## §1. 株式会社の基本構造 (最重要)

### 株式 (株) とは
- **会社の所有権を細切れにしたもの**
- ギルドの「持ち分」。株を持つ = 会社の一部を所有してる
- 例: 1000株発行して大井が全部持つ = 大井が100%オーナー

### 株主 (かぶぬし) とは
- **株を持ってる人 = 会社のオーナー**
- 株主の権利:
  - 配当をもらう (利益の分配)
  - 株主総会で議決権 (経営方針に投票)
  - 会社が売れたら株を売って儲ける (キャピタルゲイン)

### 取締役 (とりしまりやく) とは
- **株主から経営を任された人**
- 「代表取締役」= 取締役のトップ = 社長 (大井)
- 株主 ≠ 取締役 のこともある (出資者と経営者が別)
- 大井は今「株主 兼 取締役」= オーナー社長

### 資本金 (しほんきん) とは
- **会社の元手 (スタート資金)**
- 株主が出したお金
- 1円でも設立できるが、信用のため 100万〜1000万が一般的
- 大井の Almeo は [要確認] 円で設立

### 株主総会 (かぶぬしそうかい)
- **会社の最高意思決定機関**
- 株主が集まって重要事項を決める
- 一人株主 (大井) なら「一人で全部決められる」

---

## §2. 持株比率の意味 (資金調達で命を握る数字)

**株を何%持ってるかで、何ができるかが変わる。** ここ超重要。

| 持株比率 | できること | 通称 |
|---|---|---|
| **100%** | 全部一人で決定 | 完全支配 (今の大井) |
| **2/3超 (66.7%)** | 定款変更・合併・解散・増資など **重要事項を単独決定** | 特別決議ライン |
| **50%超 (過半数)** | 取締役の選任・解任・配当を決定 = **実質支配** | 普通決議ライン |
| **1/3超 (33.4%)** | 重要事項を **拒否できる** (拒否権) | 拒否権ライン |
| **3%以上** | 帳簿閲覧・株主総会招集請求 | 少数株主権 |
| **1%以上** | 議題提案権 | - |

### 大井が絶対に覚えるべきこと
- **資金調達 (出資を受ける) = 株を渡す = 大井の持株比率が下がる**
- これを「希薄化 (ダイリューション)」と言う
- **50% を切ると会社の支配権を失う** (取締役を解任される可能性)
- **2/3 を切ると重要事項を単独で決められなくなる**

→ 投資を受ける時、「いくら欲しいか」だけじゃなく「何%渡すか」を死守する。
→ 相方と組むなら「**大井と相方の持株比率を最初に明文化**」が最重要。

### 共同創業の落とし穴 (相方がいる大井に直結)
- **50:50 は最悪** (意見割れたら何も決まらない・デッドロック)
- 推奨: **大井が過半数 (51%以上) を持つ** or 役割で明確に分ける
- 創業株主間契約 (Founders' Agreement) を必ず結ぶ
  - 誰が何%か
  - 辞めたら株はどうなるか (ベスティング)
  - 揉めた時の決め方

## §3. 業務提携 vs 資本提携 (他社と組む2パターン)

### 業務提携 (ぎょうむ ていけい)
- **お金 (株) は動かさず、協力だけする関係**
- 例: 「Almeo が A社と共同で AI 導入サービスを売る」
- 軽い・解消しやすい・リスク低い
- ギルドで言えば「**同盟**」(対等な協力)

種類:
- 販売提携 (お互いの商品を売り合う)
- 技術提携 (技術を共有・共同開発)
- 生産提携 (作るのを協力)

### 資本提携 (しほん ていけい)
- **株を持ち合う or 出資する = お金が動く・関係が強い**
- 例: 「A社が Almeo に出資して株を持つ」
- 重い・解消しにくい・関係が深い
- ギルドで言えば「**血の同盟 (婚姻同盟)**」(後戻りしにくい)

### 業務資本提携
- 業務提携 + 資本提携 を両方やる (よくある形)

→ **大井の判断軸**: 軽く組むなら業務提携、本気で一蓮托生なら資本提携。
→ 17歳・スタートアップ段階では **まず業務提携から** (株を簡単に渡さない)。

## §4. 子会社・親会社・グループ会社

### 親会社・子会社
- **子会社 = 親会社が 50%超の株を持って支配してる会社**
- **親会社 = 支配する側**
- 例: 「Almeo HD が Testall社の株を100%持つ」→ Testall社は Almeo HD の子会社

| 用語 | 持株比率 | 関係 |
|---|---|---|
| 完全子会社 | 100% | 完全支配 |
| 子会社 | 50%超 | 支配 |
| 関連会社 | 20〜50% | 影響力あり・支配せず |
| グループ会社 | - | 親子・兄弟会社の総称 |

### なぜ子会社を作るのか (大井の将来に直結)
1. **事業ごとにリスク分離** (Testall が失敗しても Almeo 本体は無傷)
2. **事業ごとに別の投資家を入れられる** (Testall だけに出資受ける)
3. **事業ごとに売却できる** (大井塾の出資先が Testall を買う等)
4. **税務メリット** (グループ通算等・要税理士)

### 大井の将来像 (推測)
```
Almeo HD (持株会社・大井が支配)
   ├── Almeo社 (AIコンサル)
   ├── Testall社 (EdTech)
   ├── AIpaX School社 (教育)
   └── 大井塾社 (アクセラ)
```
→ 今は1社でいい。事業が大きくなったら分社化を検討。

## §5. ホールディングス (持株会社)

- **事業を自分でやらず、子会社の株だけ持つ会社**
- グループ全体を統括する「本部」
- ギルドで言えば「**連合の本部**」(各分隊を束ねる)
- 例: ソフトバンクグループ、楽天グループ
- メリット: グループ経営の効率化・事業売却がしやすい
- → 大井が複数事業を整理する時の **最終形**

---

## §6. 出資・資金調達 (お金を集める)

### 調達の段階 (ラウンド)
```
自己資金 (大井の金)
   ↓
エンジェル投資 (個人投資家・数百万〜数千万)
   ↓
シード (VC等・数千万〜1億)
   ↓
シリーズA (1〜数億)
   ↓
シリーズB, C... (数億〜数十億)
   ↓
IPO (上場) or M&A (売却) = EXIT
```

### 投資家の種類
- **エンジェル投資家** = 個人の金持ち (元起業家等)
- **VC (ベンチャーキャピタル)** = 投資ファンド (他人の金を運用)
- **CVC** = 事業会社の投資部門 (例: トヨタの投資部門)

### 出資を受ける = 株を渡す
- お金をもらう代わりに株を渡す
- **大井の持株比率が下がる (希薄化)**
- バリュエーション (会社の値段) が高いほど、少ない株で多くの金を取れる
- 例: 1億円の価値の会社に1000万出資 → 約9%の株を渡す

→ **大井の判断**: 調達は「いくら欲しいか」より「**会社をいくらと評価させるか**」が勝負。

---

## §7. M&A (合併・買収) と EXIT

### M&A とは
- **M = Merger (合併)、A = Acquisition (買収)**
- 買収 = 他社を買う or 自社が買われる
- 合併 = 2社が1つになる

### EXIT (出口)
- **創業者が株を売って利益を確定すること**
- 2パターン:
  1. **IPO (上場)** = 株式市場に公開・誰でも株を買える
  2. **M&A (売却)** = 大企業等に会社を売る
- → 大井の3年ゴール「**大井塾 第1期生の EXIT**」= これのこと

### スタートアップの基本ストーリー
```
起業 → 成長 → 資金調達 → さらに成長 → EXIT (IPO or 売却) → 創業者が大金を得る
```
→ これが「会社を作って売る」ゲームの全体像。大井塾で教える内容そのもの。

---

## §8. ストックオプション (人を雇う武器)

- **社員に「将来、株を安く買える権利」を渡す**
- 例: 「今 1株1万円だけど、君は3年後に1000円で買える権利あげる」
- 会社が成長して株価が上がれば、社員は差額で儲かる
- **給料が安くても優秀な人を採れる** = スタートアップの採用武器
- → 大井が将来 社員を雇う時、現金の代わりにこれで釣れる
