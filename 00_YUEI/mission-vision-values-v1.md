## 🎯 MISSION (なぜ俺がやるか)

> **「諦めを強要された人が、自分の力で立ち上がれる時代を作る。」**

### なぜこれか
- 俺自身が「環境で諦めるしかなかった子ども」だった
  - 父親の暴力で這って交番に辿り着いた
  - 通信制高校 (HR高等学院) でレールから外れた
- でも里親に拾われて起業した = **諦めずに立ち上がった**
- だから事業も全部、この構造を持つ:
  - **Almeo** → 大企業に負ける中小企業を AI で武装
  - **Testall** → 塾に行けない高校生を AI で武装
  - **大井塾** → 起業家になりたいけど方法が分からない人を実践で武装
  - **AIpaX School** → AI を触れない若者を実装可能な人材へ
- 一貫して「諦めの強要を、AI と仕組みで破る」

## 🌅 VISION (世界をどう変えるか・3層)

### 短期 (2029・20歳)
- 月収 **1,500万円** 安定化
- **SFC 3年生**
- **3本柱事業**: AIpaX School / Testall / Agents of Flag
- **Almeo** を現金牛として維持 (月収の半分)
- 大井塾 第1期生 EXIT 達成
- スタートアップ業界での認知獲得 (タイミー小川嶺・dinii 山田真央級)

### 中期 (2034・25歳)
- **IPO or M&A 1社** 達成
- 大井塾 100期生輩出
- 日本のスタートアップエコシステムの中核プレイヤー
- 「学歴・出身・年齢で諦めなくていい」証拠を量産

### 長期 (2044・35歳)
- 「諦めの強要」を **構造的に終わらせる制度** の設計者
- 日本の教育とビジネスの境界を再定義
- アジアの中小企業に AI を届ける
- 若手起業家輩出の世界一アクセラレーターを建てる

## ⚔ VALUES (どう動くか・5つ)

### 1. 走り続ける
止まることが死、動き続けることが生存。
ただし、長距離走。**寝ることも走ることの一部**。

### 2. 圧倒的差を作る
「妥協」「中庸」「ぼちぼち」を否定。
「圧倒的」「ZONE」「王者」を肯定。
僅差で勝つくらいなら参戦しない。

### 3. 地に足を着く
1人ずつ、目の前のお客に、現場で勝つ。
「絶対に買う1人」を解像度マックスで見据える。
全国・世界は、その積み重ねの先。

### 4. 主人公感を貫く
敵を見せる。物語化する。**英雄として生きる**。
ただの起業家じゃなく、「物語の主役」として動く。

### 5. 弱さも見せる
一人で戦わない。共闘者を持つ。外部臓器 (Claude・相方・母親) に流す。
**強さも弱さも、同じ一つの体に共存する**。

## 🔥 「俺がやる必然性」 (一行)

> **「学校に行けなかった17歳が、AI と仕組みで、自分と同じ境遇の人を救う。」**

これが俺の事業全部を貫く軸。
- 小川嶺 (タイミー) = 自分の労働経験 → 全国の労働市場
- 山田真央 (dinii) = 自分の外食経験 → 全国の飲食店
- 大井湧瑛 (Almeo + Testall + 大井塾) = 自分の被諦め経験 → 全国の諦めた人

**Founder-Market Fit が桁違いに強い**。
17歳という事実は弱みじゃなく、「物語の必然性の証明」。
