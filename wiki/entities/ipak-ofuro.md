# おふろかふぇIPAK

街中でお湯に浸かりながら1人時間を満喫できる都市型おひとり様温浴コンセプト。

## コンセプト

「街中のお風呂カフェ」。完全おひとり様向け。都市部での孤独な充電スポット。  
→ IPAKシリーズ共通テーマ：人と話す必要なし・QR入店・サブスク

## ポジション

| 競合 | 特徴 | IPAKとの違い |
|------|------|------------|
| **おふろcafé**（温泉道場） | 長時間滞在・コーヒー/漫画無料・大型施設 | IPAK：小型・都市型・「おひとり様専用」 |
| **スパジャポ** | 関東最大級・岩盤浴・にぎやか | IPAK：静かな完全個室 |
| **銭湯・スーパー銭湯** | 都度払い・大衆向け | IPAK：サブスク・ソロ専用 |
| **chocoZAP** | 無人型ジム・ダーツ等も展開中 | IPAK：お風呂特化・よりニッチ |

## 市場背景

- 温浴施設業界市場規模：約**1兆1,000億円**（スーパー銭湯等含む）
- 2024年：温浴施設の営業許可件数が廃止件数を上回る（2007年以来初）→ 回復基調
- 「おふろcafé」はFC展開で全国10店舗超に拡大中（若者・女性層に人気）
- **ひとり温浴需要**: 「ひとり焼肉」「ひとりカラオケ」文化の延長線上

## ビジネスモデル案

| プラン | 月額（案） |
|--------|----------|
| ライト（月8回） | 3,980円 |
| スタンダード（通い放題） | 5,980円 |
| ビジター（都度） | 1,200円/回 |

- 無人運営（QR入退室・自動課金）
- 完全個室ブース型（4〜8ブース / 1店舗）
- 都市型小型物件（20〜50坪）で初期投資を抑える

## ユニットエコノミクス（試算）

| 項目 | 金額 |
|------|------|
| 家賃（都心20坪想定） | 20〜40万円/月 |
| 設備（湯船ユニット×6） | 初期500〜1,000万円 |
| 光熱費（温水コスト大） | 15〜25万円/月 |
| BEP（スタンダード5,980円想定） | 約100名 |

**課題**: 設備コスト（温水・排水・ユニット）が大きい。ダーツと比較して初期投資が重い。

## IPAKシリーズ内の位置づけ

IPAKシリーズの中で最も**初期投資が重い**業態。ダーツ（170万）・カフェ（低コスト）・釣りの実績を積んでから展開する「後発型」が現実的。
