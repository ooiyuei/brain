# AIpa Web

> 採用サイト制作を「打ち合わせだらけの受託」から「フォーム入力で進むAI制作オペレーション」に変える。

## 一文サマリー

**Aipa WEBは、Web制作の面倒を「フォームとAIで消す」サービス。**  
「AIで作る」ではなく、フォーム→AI制作初稿→人間確認→1週間公開→月額更新というオペレーション全体を商品化する。

## 事業の本質

- 売るのは「Webサイト」ではない
- 売るのは「中小企業がWebサイトを作るときの面倒を全部減らすこと」
- ポジションは「AI Web制作会社」ではなく **「社外Web担当者」**
- 打ち出しは「AIを活用したWeb制作・更新代行」

## なぜ採用サイト特化が強いか

1. **痛みが強い**：建設・物流・製造・介護・飲食・美容・地方中小（20〜300名）の採用課題は売上/人手不足/現場崩壊に直結
2. **型化しやすい**：FV/採用コピー/会社紹介/数字で知る/仕事内容/求める人物像/1日の流れ/社長メッセージ/社員インタビュー/福利厚生/募集要項/FAQ/応募導線 が定型
3. **大井の経験と噛み合う**：採用/人材/営業/CS/AI/Web制作/顧客課題整理を横断できる

## 競合と差別化

| 競合 | 弱点 | AIpa Webの優位 |
|------|------|--------------|
| AIサイトビルダー（Wix/STUDIO/Framer/Durable） | 顧客が自分で考える必要が残る | AIを裏側で使い、制作と運用を代行 |
| ノーコード制作代行 | 採用特化・運用代行が弱い | 採用特化＋月額運用込み |
| Web制作会社 | 高い・遅い・やりとり多い | 安い・早い・フォームで楽 |
| 採用支援会社 | Web制作が高い・遅い | 受け皿を安く早く |

差別化の本質：「AIで作る」ではなく、**フォームでヒアリングを標準化→AIが構成・コピー・文章→人間が事実確認→月額運用** のオペレーション全体。

## Deep Research 競合MAP（2026-05-14）

主戦場は制作会社ではなく **無料採用ツール + AIビルダー**。

| 類型 | 代表サービス | 価格/納期 | 強み/弱み |
|------|------------|---------|----------|
| 無料採用 | Airワーク（リクルート）/ engage（エン・ジャパン） | 0円 / 10〜15分 | 無料・簡単／差別化弱 |
| AI制作 | STUDIO / Wix / Framer | 590円〜・$17〜・$10〜 / 即日 | 速い／**セルフ前提** |
| 制作会社 | ONE / ベイジ | ONE 40〜100万 / 1〜5ヶ月、ベイジ 運用10万〜 | 品質／高コスト |

**重要警報**: STUDIOは2026年時点で**AI機能+Figma取込**を備える。「AIで速いだけ」では埋没する。

**勝ち筋**: 「AIで安い」ではなく、**採用LPを7日で公開し、求人票・応募導線・更新まで代行** すること。

## 制作フロー

```
Step1: フォーム入力（顧客）
Step2: フォーム回答→AI用プロンプトに変換
Step3: Claudeで構成・コピー・文章作成
Step4: GPTimage2でデザイン方向性
Step5: Claudeで実装（Tailwind/Next.js/Astro/shadcn/ui/Vercel）
Step6: 人間が確認（事実関係/給与/権利/スマホ表示/応募導線/SEO）
Step7: 顧客確認（修正1回まで）
Step8: 公開（ドメイン/SSL/GA/応募通知）
Step9: 月額運用
```

## 売り文句案

- フォーム入力だけで、採用サイトを最短7日公開。
- 求人広告の前に、応募者の受け皿を整えませんか？
- 採用LPを最短1週間で。原稿作成から公開後の更新まで一括対応。
- 入力するだけ。あとは、採用サイトの形にします。

### 営業時のNG/OK

× 「AIでWebサイトを作ります」（安っぽく聞こえる）

## 初期ターゲット

- 地方中小企業・30〜300名規模・採用に困っている

## 営業先の見つけ方

1. Indeed/engage/AirWORK/求人ボックスで求人企業を見つけ、採用サイトが弱い会社へアプローチ
2. 「静岡 建設 採用」等で地方企業を検索
3. 知り合い経由（地元企業/経営者/採用担当/人材会社/士業/商工会議所/起業家コミュニティ）
4. 求人広告代理店と組む（「求人媒体に出す前に、採用サイトを整えましょう」）

## 補助金活用（営業の武器）

- **IT導入補助金2026**：上限450万円（ウェブ制作・デジタルツール対象）
- **小規模事業者持続化補助金**：ウェブ制作費最大50万円（補助率2/3）

## リスクと対策

| リスク | 対策 |
|--------|------|
| 安売りしすぎる | 最低価格決定・修正回数制限・オプション明確化 |
| 修正地獄 | 修正1回・大幅変更は追加料金・初稿前に方向性確認 |
| AI生成の品質事故 | 給与/待遇/条件は必ず顧客確認・AI生成文は人間確認 |
| 写真・ロゴの権利問題 | フォームで使用許可確認 |
| 競合に真似される | 採用特化×業種別テンプレ×フォーム設計×月額運用×地方営業力 |
