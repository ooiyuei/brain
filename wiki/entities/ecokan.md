# EcoKan（エコカン / eco canteen）

> オフィス向けマイボトル専用自販機。洗浄・補充を一台で。福利厚生 × SDGs。

## コンセプト

**マイボトル専用自販機** をオフィスに設置する。

- **洗浄機能 + 飲料補充を1台で完結**
- 企業福利厚生として導入（会社負担）
- 環境価値（ペットボトル削減量）を社員に見える化

## 解決する課題

| 課題 | 内容 |
|------|------|
| マイボトルが続かない | 職場での洗浄・補充が面倒で諦める |
| オフィスゴミが増えている | 缶・ペットボトルゴミが毎日大量発生 |
| SDGs対応が形だけ | 掲げているが具体的な行動ができていない企業 |

## ターゲット

- 従業員50名以上のオフィス
- SDGs・健康経営に取り組む企業
- オフィスビルのテナント（複数社に1台）

## ビジネスモデル

| 収益モデル | 内容 |
|-----------|------|
| **月額サブスク** | 設置費 + 飲料補充費込み（数万円/台） |
| 従量課金 | 1杯 60〜100円（ペットボトル購入より安い） |
| レベニューシェア | 飲料メーカーとの協業 |

## 価格感

- **1杯**: 60〜100円（自販機より安い・ペットボトルより環境に良い）
- **月額設置**: 数万円/台（規模・飲料種別によって変動）

## 差別化ポイント

| 比較対象 | 違い |
|---------|------|
| 既存ウォーターサーバー | マイボトル洗浄機能あり |
| 普通の自販機 | ペットボトル廃棄ゼロ |
| 給茶器 | 飲料バリエーション・CO2見える化機能 |

- 「使うたびにCO2削減量が見える」環境価値の可視化
- 健康経営・SDGs双方に訴求できる（CSR報告書に載せられる）

## 市場背景

| 指標 | 数値 |
|------|------|
| 再利用可能な水筒グローバル市場（2025年） | **110億2,000万USD** |
| 日本市場規模（2026年予測） | **約890億円** |
| 日本のペットボトル市場（2024年） | 4,195億円（前年比増） |
| IT導入補助金2026 | SDGs関連設備も対象、上限450万円 |

→ 脱プラ規制強化・政府補助の追い風あり。企業のESG対応需要は確実に増えている。

## フェーズ設計

| Phase | 内容 |
|-------|------|
| Phase 1 | PoC：1〜3社に設置・課題検証 |
| Phase 2 | プロダクト改良・導入事例の作成 |
| Phase 3 | 法人営業スケールアップ |
| Phase 4 | 飲料メーカーとの提携（飲料ラインの充実） |
| Phase 5 | 全国展開 / ビルオーナー向けモデル |

## リスク

| リスク | 対策 |
|--------|------|
| ハードウェア開発コスト | 既存自販機メーカーへのOEM提案から入る |
| 衛生・メンテナンス | 洗浄機構の設計と保守体制の構築 |
| 企業の意思決定が遅い | 担当者へのCSR文脈・補助金文脈での提案 |
