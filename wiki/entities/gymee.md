# Gymee（会議管理SaaS）

> 中小企業の「会議後の実行管理不足」をAIで解決。月額19,800円のサブスクSaaS。docomo STARTUP PBL参加。

## ポジション

**「文字起こしAI」でも「大企業向け会議管理」でもない。中小企業向けの実行追跡AI。**

- 文字起こし → ツール代替（月数千円止まり）
- 実行管理/次アクション提示 → コンサル代替（月2万円台受容）

## 解決する課題

ターゲット：20〜200名規模の中小企業の経営者・管理者

| 課題 | 詳細 |
|------|------|
| 決めたことが実行されない | 会議はあるのに誰も動かない |
| 同じ議論を何度も繰り返す | 前回の議論が活かされない |
| 責任があいまい | 誰が何をやるのか不明確 |
| コンサルに払えない | 月30〜100万円は中小企業には払えない |

→ Gymeeなら月19,800円で解決

## ソリューション

議事録テキスト/Zoom文字起こし/手打ちメモを入力 → AIが自動整理：

1. **決定事項** / **未決事項** / **担当者** / **期限**
2. 過去会議と突き合わせ → 停滞タスク検知 / 同じ議論の繰り返し検知
3. 経営者向けアウトプット：
   - 期限超過一覧
   - 次回会議で優先すべき論点
   - 週次サマリー

## CPF検証結果（docomo STARTUP PBL）

| 検証内容 | 結果 |
|---------|------|
| 課題認識 | 「強い・深刻」と認識した企業が多数 |
| 支払い意欲 | 具体的な支払い意欲を表明した企業が複数 |
| MVPで最も反応良かった機能 | **期限超過の一覧表示**（「ぱっと見でわかるのがいい」） |
| コンサル的示唆への反応 | 「次に何をすべきかがわかる」に価値を感じた |

### 仮説の変化

- **Before**: 会議の質を上げたい
- **After**: 「会議で決めたことが放置されていないか確認する手間をなくしたい」

→ 課題の本質は「記録不足」ではなく **実行の追跡**

### 学び

1. 顧客が払う額は「何の代替か」で決まる → コンサル代替ポジで月2万円台
2. 欲しいのは分析ではなく「次の一手」 → 精緻なレポートより具体的示唆
3. 中小企業には「軽すぎず重すぎない支援」が必要
