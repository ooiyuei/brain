# EEMUS（エームス）

> 静岡の高校生と地元企業をつなぐキャリア共創プロジェクト。人材流出・採用ミスマッチ・早期離職の負のサイクルを断ち切る。

## 解決する構造的課題

| 指標 | データ |
|------|--------|
| 静岡県外大学進学率 | 71%（県内高卒・大学進学者約1.7万人のうち） |
| Uターン就職率 | 33% |
| 県内正社員不足企業 | 50.8%（2025年1月時点） |
| 建設業の人手不足 | 79.5% |
| 高卒3年以内離職率 | 38.4%（全国） |

## ミッション

**静岡の高校生が自分らしい未来を描けるように、地域企業と共に進路の選択肢を広げること**

## 提供価値（企業向け）

1. **早期採用候補との接点** → 将来の採用母集団を高校時代から育成
2. **ミスマッチと早期離職の低減** → リアルな職場体験で入社後ギャップをなくす
3. **採用プロセスの効率化** → 学生が企業情報を把握した状態で面談に臨む
4. **ブランド発信と地域貢献** → CSR活動としての認知向上
5. **社員の成長** → 若者への指導がリーダーシップ・プレゼン力を磨く場に

## 提供サービス

| サービス | 内容 |
|---------|------|
| 企業見学・ジョブシャドウイング | 実際の職場を訪れ仕事の雰囲気を体感 |
| 社員との座談会・講話 | 現場社員がキャリアのリアルを語る |
| 適性診断・キャリア相談 | 自分の興味・強みを理解するための支援 |
| 地元企業マッチング | インターン・就職に発展するサポート |

## 参加の流れ

1. お問い合わせ・初回相談 → 課題ヒアリング
2. プログラム設計（対象学年・人数・実施時期）
3. 実施準備と告知（学校・学生調整）
4. プログラム実施（EEMUSスタッフが同行）
5. 振り返りと次のステップ

## ビジネスモデル

- 標準プログラム（座談会・見学）: **無料**
- 特別設備・長期受け入れ: 要相談
- 収益化方針: 企業への人材紹介フィー or 有料プログラム（今後設計）

## 原体験・背景

- 大井が「静岡で高校生と地元企業をつなぐ人材事業をやりたい」という問題意識
- PM Agentでの人材紹介経験が背景
- 旧名OASIS → 最優秀賞受賞

## 差別化

- パンフレット・会社説明会 → EEMUSは「仕事のリアル」体験
- 採用ピッチ資料的な透明性ある情報開示でモチベーション向上（SmartHR事例：応募数5.3倍）
- 静岡ローカルに特化 → 全国展開サービスが手薄な領域

## 次の一手

- 出店候補企業の選定（建設業・製造業など人手不足が深刻な業種）
- 県内高校1〜2校との連携話
- 有料化モデルの設計
