# Testall

> テスト結果を、次の45分でやるべき勉強に変える。受験戦略OS。  
> 「勉強は才能よりマネジメント」

## 事業サマリー

| 項目 | 内容 |
|------|------|
| ターゲット | 高校生・浪人生（スマホメイン） |
| コアバリュー | テスト返却 → 弱点特定 → **今日の45分タスクに自動変換** |
| スタック | Next.js + Supabase + Claude AI |
| JUDGE評価 | **75点 / Tier A** |

## 解決する課題

- テストを受けても何を復習すべきか分からない
- 勉強計画が続かない
- やる気ではなく「仕組み」で成績を上げたい層にリーチできていない

→ 既存の学習アプリは「何を勉強するか」を自分で決める必要がある  
→ Testallはテスト結果入力→AI診断→実行まで一貫して自動化

## 機能設計

| 機能 | 内容 |
|------|------|
| テスト追加 | 科目・点数・分野別ミスを入力 |
| AI診断 | Claude APIが弱点分析・優先度付け |
| 集中モード | 今日やるべき45分タスクを提示 + タイマー |
| 週計画 | 週単位の学習スケジュール管理 |
| マイページ | 成長記録・連続学習日数 |
