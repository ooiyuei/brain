# OMNI

次世代のUNOになるカードゲーム。エンタメ・アナログゲーム領域への参入。

## コンセプト

UNOのポジション（誰でも知っている・どこでも遊べる・ルールがシンプル）を狙う新カードゲーム。  
「既存のカードゲームより面白いものを作る」より「文化になるゲームを作る」が本質。

## 市場データ

| 指標 | 数値 |
|------|------|
| TCG（トレーディングカード）国内市場（2023年） | **2,774億円**（前年比大幅増） |
| TCG市場のピーク成長 | 2020年1,222億円→2023年2,774億円（2.3倍） |
| アナログゲーム市場 | 成長継続中（矢野経済研究所 2025版） |
| UNO世界販売 | 1億以上のゲームが販売されるメガヒット商品 |

→ アナログゲーム市場は好調。「カードゲームの9割は5年以内に終了する」という課題もあり、継続性設計が重要。

## ポジション

| 比較軸 | UNO | ポケカ/遊戯王 | OMNI（目指す姿） |
|--------|-----|------------|----------------|
| ルール | 超シンプル | 複雑 | シンプル＋深み |
| コスト | 安い | 高い（希少性） | 中程度 |
| 対象年齢 | 全年齢 | 主に10〜30代 | 全年齢 |
| コレクション性 | なし | 高い | 中程度（セット完結型） |
| 対戦の深さ | 浅い | 深い | 中程度 |

## ゲームデザイン方向性（仮）

- セット完結型（追加課金なし）でコスパよく楽しめる
- 場を盛り上げる「観戦・応援」要素を内包
- スマホアプリとの連動（スコア管理・オンライン対戦）
- イベント・大会設計を最初から組み込む

## ビジネスモデル案

| 収益源 | 内容 |
|--------|------|
| ゲーム本体販売 | B2C（Amazon・玩具店）、B2B（企業研修・景品） |
| 大会・イベント | 参加費・観戦チケット |
| スポンサー | ゲーム内ロゴ・大会協賛 |
| ライセンス | 海外展開・IP利用料 |

## 開発ステータス

- 企画段階。ゲームルール・カードデザイン未公開
- **最初の検証**: 手書きプロトタイプ → 身近な人10人でテスト → フィードバック収集

## 参入障壁と対策

| 障壁 | 対策 |
|------|------|
| 大手ゲームメーカーとの競合 | SNSでバズらせて先に認知を取る |
