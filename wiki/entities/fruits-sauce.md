# フルーツソースプロジェクト

静岡の捨てられている果物を「日本フルーツソース」に持ち込んでおいしいソースにして瓶詰め・ブランディング。

## コンセプト

- フードロス問題（静岡の農業廃棄物）を資源に変える
- 日本フルーツソースとの協業（既存の製造ラインを活用）
- 瓶に入れてブランディングすることで付加価値を生む

## 連携先：日本フルーツソース株式会社

**所在地**: 〒421-0122 静岡市駿河区用宗2-8-16  
**公式**: [fruitsauce.co.jp](https://fruitsauce.co.jp/)

| 製品カテゴリ | 内容 |
|------------|------|
| フルーツソース | 果実を使った各種ソース |
| クラフトコーラ | 独自シロップ |
| チャイシロップ | スパイス系 |
| ハーブシロップ | ハーブ系 |
| ストレートジュース | 搾汁製品 |

**OEM・製造委託**: 最小ロット1個から対応（加工費2万円以上）  
**特徴技術**: 「常温常圧濃縮技術」 → 市場にない高濃度シロップを添加物なしで製造可能  
**サポート範囲**: 製法・レシピ開発→容器選定→充填→殺菌→仕様書作成まで一気通貫

## 副連携候補：梅ヶ島くらぶ

**所在地**: 静岡市梅ヶ島（標高約750m、昼夜気温差大・霧環境 → 渋みの少ない茶葉）  
**公式**: [umegashima.love](https://umegashima.love/)

| 製品 | 特徴 |
|------|------|
| 梅ヶ島紅茶 | 華やかな香りと黒糖のようなコク |
| チャイティーバッグ | 紅茶/ほうじ茶+自然農法スパイス、1包で4〜5杯分 |
| チャイシロップ | 濃縮タイプ、ホテル・レストラン卸実績あり |

→ 梅ヶ島紅茶×静岡フルーツのコラボ商品ライン設計が可能

## ポジショニング

地域課題 × フードビジネス × ブランディング。静岡らしさを活かしたプロダクト。

## ビジネスモデル案

| フェーズ | 内容 |
|---------|------|
| 1. 試作 | 廃棄果物を日本FSに持ち込み、1個ロットで試作（〜2万円） |
| 2. 販路開拓 | 都内のカフェ・セレクトショップへのルート営業 |
| 3. ECブランディング | オンラインストア立ち上げ・クラファン |
| 4. OEM拡大 | 他カフェへのB2B卸・梅ヶ島くらぶとのコラボ商品化 |

## 次の一手

- 日本フルーツソースへのコンタクト（試作依頼）
- 廃棄果物の調達先農家リストアップ（静岡市周辺）
- ブランド名・パッケージデザインの方向性を決める
