## 章別詳細ノート

### 第5章末尾 (p.0201)

ブランド設計図(BEPyramid)をつくるメリットは究極的に2つ:

1. **経営資源を集中する焦点を明確化**: どの消費者価値をどう向上させるのかを明確に・中長期的に一貫させられる
2. **「全員野球」の力を引き出す**: 全体像と自分の役割を明確化され動機づけられた社員全員の力を引き出す

→ 結果的に「消費者の頭の中に蓄積されていく有利な戦略エクイティの力」と「劇的に引き出される社員たちの生産性向上」の**往復ビンタで業績は上がっていく**

### 第6章 (pp.0202-0247)「強いコンセプトは消費者理解がすべて」

#### §0 5つのSTEP (p.0203-0205)

本書ではSTEP1とSTEP3に集中する。

```
STEP1: 消費者理解
STEP2: ブランド設計の仮説立案
STEP3: マーケティング・コンセプトの策定
STEP4: 量的調査による検証
STEP5: HOW(プロダクト、コミュニケーションなど)の策定
```

> 「焦点である『Mを増やすための戦略からコンセプトの展開』に集中します」

#### §1 モニタールームでは到達できない深淵 (pp.0206-0207)

**消費者理解の対象は「6C」よりも消費者(Consumer)に絞れ**:

```
6C = Company / Customer / Competitor / Consumer / Community / 社会・法律・経済など
```

- ただし森岡: 6Cの中でも**Consumer (消費者) の理解に最初のリソースを集中**
- 他CはConsumer理解を補強する役割
- 「強いコンセプトは消費者理解がすべて」

**消費者理解の3レベル**:

1. **意識** = 質問して答えられる表層 (アンケート・モニタールームで取れる)
2. **無意識の本能** = 質問に答えてもらえない深層 (FGI/ホーム訪問でも届かない)
3. **本能** = 行動の根源 (推理する必要)

> 「議論できていない、あるいは議論できていてもうまく消費の文字を取れていない領域」

**質的調査バイアス3つ**:
- (a) 「謝礼を出す合同調査」では発言バイアスが出る (協力的に答えようとする)
- (b) モデレーターによる「誘導のミラー」
- (c) 「説明可能なところに着地」しがち (深層が言語化されない)

→ 質的調査(FGI)だけでは消費者理解の深層に達することはできない。

### §2 「凡人」と「狂人」に「憑依」する (pp.0207-0231) — 本書の核心

#### 用語の定義 (p.0209)

> 「凡人」 = 平均的な多数派の消費者
> 「狂人」 = カテゴリーへの執心が異常レベルに大きく、購買数や購買金額が大きく、無類の好き者
> 「私のマーケティング・ノウハウの源流における『狂人』は、日本語の四国語族『日本国語大辞典』にもある『狂人』に近い意味合いで使っている」

森岡注:
> 「『マニア』『オタク』『プロ顧客』──、残るがそれら私の知る人の頭の浮き沈みのうち、これらの言葉ではほぼ収まりません」

「狂人」は本書全体で**ポジティブな尊称**として使われる。マニア+オタク+プロを超える執心レベル。

#### (1) 「凡人」に憑依する (pp.0210-0215)

**目的**: 自分自身でなれない場合、その商品カテゴリーに対する典型的なふるまいをする「凡人」になりきって、その人と同じ気持ちで世界を見る。

**例: 自動車業界の人** (p.0211)
- 自動車業界の人は車に対して特別な感情やこだわりを持っている
- だから「フロアマット」「アクセサリー」などSKUの隅々まで関心が向く
- → 「カタログ上のスペック」を比べて買う・購入の意思決定がメンドくさいと感じない
- だが**普通の消費者**は無関心で、「行きつけのディーラーで適当に」決める方が多い

**例: トヨタ「カリスマ」(p.0211)**
- 大井 親が乗っていた愛車トヨタ「クルマ」(軽自動車) を想い起こした
- 「軽自動車のキッチン・ロジック」で、何が一体何なのか判らなくなる
- 凡人にとって自動車選びは「絶対わからない」

**凡人憑依のコツ** (p.0212-0213):
- 自動車カテゴリーのオタクは「凡人」になれない
- カテゴリーオタクは特別な思い入れがある人で、自分が世界を眺めているフィルターと違う
- → 「**凡人**」に憑依するには、その特別な思い入れを完全に外し、いまだ「凡人」しか持っていない感覚に切替

**重要原則** (p.0214):
- 大学のたとえ:「東大」「早慶」「マーチ」「日東駒専」を全部一括りで眺める感覚
- カテゴリー外の人にとってブランド差は小さい

**(消費者の人生)** (p.0215):
- 消費者は商品・サービスがどうかではなく、ちゃんと**自分の人生に向き合って消費している**
- ちゃんとお金を払ってもらうには、消費者の人生における自分の場所を確保する必要

**P&Gの「消費者ベースキャンプ」(p.0215)**:
- ベイビーケアの新人開発者は出産前のお母さんと数週間生活を共にする
- 寝ない・お乳が出ない・首が据わらない、心が壊れそうになる中で自分自身を学ぶ
- 大人用紙オムツでも同じことを徹底
- → カテゴリー特有の本能行動を予測しやすくなる

**大井応用**:
- AIpaX営業 → 静岡の中堅製造業社員AIに憑依 (操作分からない・上司に説得しないと買えない・予算が通らない)
- Testall → 受験生 (毎日プラン作りに迷子・親に怒られる・スマホ触りたい誘惑) に憑依

#### (2) 「狂人」に憑依する (pp.0215-0231)

**狂人の定義** (p.0215):
- 「狂人」 = カテゴリーへの執心がフェイにあるくらい強い人
- 普通の消費者の何十・何百・何千倍の使用頻度・購買金額・知識・時間を投入

**USJ「狂人」の事例** (p.0216):
- 森岡USJ時代、有力なマンガやゲームやアニメ映画などのコンテンツの理解は、その熱狂的なファンの理解なくしては自社IP出品のファンになってくれない
- > 「私自身の意識の世界では、エンタテイメントの中で 999 回時を打って『時代ロケラ』を遊んだ人、NHKの『プロフェッショナル』を見てしまいません」
- 例: 1人で同じ作品を 999 回プレイした人、 NHK の『プロフェッショナル』 仕事の流儀
- ハリーポッターを徹頭徹尾読み解いた者、ピサ斜の塔 (ヨウジヤマモト) を含むほどに
- 「ハリー・ポッター」を何年も読み込んだ人は、もちろんコンテンツそれのみならず、それらを覚えてできない世代的ファンに、つまり「狂人」の本質

**狂人 vs 凡人** (p.0219):
- 「狂人」と「凡人」は線で分けてあるのではなく、その両方を意識する以外に、消費者の本能を理解しやすくなる
- 「狂人」「凡人」の見極めるには:
  - その商品が消費者の購買金額に占める割合 (狂人)
  - 知識量 (狂人 / 趣味的に異常レベル詳しい)
  - 消費頻度 (狂人 / 多数派の3倍以上)

**スマホゲーム「廃課金会員」(p.0220)**:
- スマホゲームに激しい人の中心要素を理解するために、もう一例ご紹介します
- いわゆる「廃課金者」に憑依したことも何度かあります
- 課金中位の中央値者の月10倍くらい
- ガチャ40,000円もらう人の月10倍くらい
- 中央値者がスマホゲーム代を200,000円から300,000円使ったりするけれど、廃課金者の月10倍とその全部
- 「廃課金者」(普通の人がガクッと引き上げる) は「廃課金者」中の経験する1人なのです

> 「もちろん私も廃課金者をギャンブルに大儀ですので、まったく狂人的な意味ではありません」

**「自己防衛本能」(p.0223)**:
- ゲームに激しい人の中心要素は **「自己防衛本能 → 退避することを通じてその時を満たす」**
- 経験を満たすために特化したロジックで、現実時代の延長線上に外挿し、ゲーム内体験を購入する
- 例: バーチャル世界での自己存在の喪失をパーチャルロビーで取り戻す本能の延長

**ネスタリゾート神戸「山遊びの狂人」(p.0223)**:
- 時期はもっていなかった凡庸な年金会員意外の、一部の「狂人」に憑依する
- 「ネスタリゾート神戸」として直接立ち上げた中で、はらわなWHO・WHAT・HOWを考えるのに、はじめは「狂人」、後期から「凡人」と表現すべき。私は単に経済的な特典を狙うために、また「狂人」に発生してもらう客に絞ったコアコーディネーターをスキー時に絞った。私の自身対局時にもっとも好きとイメージを持ったのも、一回りでも前の長で、その動物にきわめて自然的かつフレンチに分けることに集中する
- 山遊びの「狂人」 = 経済的特典を狙う一握りでない人
- 私の住宅では駐車場の最初の駐車先になっており、自分の家にも飯の食事と空気の集まりを楽しむ庭が良い
- 動物の体験ノウハウは、技場の食肉自動販売機ぐらいフレンチハンな実際の動物を使って学ばせてもいくいません
- ネスタの「サバイバル」「猟・銃・銃が外」など人物の体験を直線したコンテンツを開発
- 一部のコアな顧客は、それまでの食肉中央値者の感覚と異なる
- 「いい味だな、ガスがどんな機械に作っているんですか」「これだけのお金を出すには」と言うとした
- → 大きな「狂人系顧客」が「ネスタ」を選んだ

**ネスタ「ワイルド・バギー」(p.0229)**:
- 軽量化のキッチンを超えてのキッチンを満たしていない女
- 「キッキング・ドロップ」など物特定の体験を再現する
- 中でも、本格的なオフロード・バギーで疲れる『ワイルド・バギー』はヒットになりました。
- → そのコアコーディネーターを取得することにより、軽量バギーの遊びと交渉でちょい食べ食べてくれるが、これ100月までも体感したオフ
- 後の「狂人」リソースの問題に山勝な家族の延長運転されることに気づく

**「凡人」と「狂人」の体験は4種類** (p.0230):

```
①都会では出会えない山の中ならではの体験
②凡人にとっても "動物的な興奮" を味わってもらえる、強烈な"できた"体験
③それらが大人にとっても記憶に残しやすい初めての体験
④投資資金型の多くの予算内に収まる体験
```

**コアエクイティ** (p.0230):
- 「ネスタ・スキー」「グリーンピア二人」を再生する
- 2018年から始まった消費は、トップラインを大幅に伸ばした
- 「狂人」のリソースに集中するという発想ではない場合、これらの戦略は2025年とは違うものになっていた

#### (3) 「憑依」とは具体的に⋯⋯ (pp.0231-0240)

**マーケターとして** (p.0231):
- 自分のオリジナルの認識評価の中に、ターゲットとする消費者の認識評価を取り込むトレーニングをする
- 自分のフィルターを薄く、その人の認識ノェルターを多くする・私たちは消費者の認識評価に対する集まりを揺さぶる
- 「憑依」 = 1分間の朝旅に社員女性役と多分かないようにする

#### §3 消費者の脳内想起から本能をたどる (pp.0232-0247)

**「上位カテゴリーに着目」(p.0233)**:
- ロボがブランドを (i ブランド) よりも大きな枠組みで使うカテゴリーに着目しまよ
- 自分が向ける商品 (i ブランド) よりも上位のカテゴリーに着目する
- → 質的調査でいう「ブランドを与えたとき(SKU)を含むSKをグリで脳内ジャンプして、上位カテゴリーのSKU から各カテゴリーが反射的に出てくる」
- 「自分が問題するカテゴリーは何か」 = 上位カテゴリーに着目

**カテゴリー数** (p.0233):
- アサヒスーパードライ → ②②①③つのカテゴリーで分かる
- ①「ビール・カテゴリー」
- ② 「酒類・カテゴリー」
- ③「セルフ・インダルジメント(嗜好品)・カテゴリー」

**SKU の関係** (p.0233):
- ブランドの違うカテゴリー言う
- カテゴリーを生み出すなく、カテゴリーを横切りすぎる
- → ブランドのあげる WHAT を本気の発信した方向

**消費者から脳内記号 (40-60人)** (p.0234):
- カテゴリーが必要があれば、その問題のなる商品の場合、必要のあるブランドが何かは判らない
- 1人最低1時間で、40-60人で
- 「丸亀製麺」 = カテゴリーを横切るで、消費者の脳の本能、消費者の本能の戦略の判らない場所を取りに行く

**例: 丸亀製麺** (pp.0235-0240):
- 丸亀製麺 = 2018年に新たな経営チームに変わって、すべてのプロセスに私が直接参加
- 製麺チーム全員が経営直接参加 → 業績を改善
- 私のマーケティング・コンセプトの最大の活用です

**丸亀製麺 上位カテゴリー** (p.0236):
- 私のマーケティング・コンセプトのアプローチ:
- (1) 「うどん」 = 丸亀製麺
- (2) 「外食」 = 丸亀製麺
- 「外食」カテゴリーを覗いて、「うどん」を脳内に検索した
- → 結果として「うどん」がブランドの脳内記号で大きく回転している

**「うどん」の脳内記号** (pp.0236-0237):
- 「そば」と比べ、消費者の脳の中で、「美味」「素朴」「健康」「こだわり」「本物高品」
- 普通に消費者は「うどん」とは、なぜか地味なくらい、「うどん」とは違う情緒の地味さがある
- 「うどん」は普通に消費者の脳の中で、「つつもしい」「やさしい」「ほっとさせる」「悩みからかかる」「ノスタルジー」「心の自由化け」、内向的な安心感を喚起されてれて表れているのが
- 「うどん」 は **「母性の伝達」** = **「母の象徴」(母性的な記憶のうどんの上にあるか身体的な脳の感情と記憶のクリスマス、つまり「母が貸している奇跡の食事の感触」(p.0238)**

**つまり**:
- 「うどん」 = 「**癒される安心感**」 (p.0238)
- 母性的な象徴のうどん、丸亀製麺のブランドの「特徴」と上位カテゴリー「うどん」に置き換える
- 「丸亀製麺」 = 「**癒される安心感**」のレールに乗る

**ダイヤモンドのGAP** (p.0240):
- 戦略エクイティから「ダイヤモンド理論」を活かすため、消費者の脳内認識のうちに、自分のブランドが「外食」のチャクラに対してGAPを持つ必要
- GAPは外食「狂人」 (中央値者中心) のうちに、自分のブランドが「外食」のチャクラに対してGAPを持つ必要
- GAPの構造:
  - 自分のブランドのWHATが、上位カテゴリーの中心となるベース要素から
  - 同時期 = 消費者にとってその上位カテゴリーの中心となるベース要素から逸脱

**13万店の外食ピラミッドのGAP** (p.0240):
- 「ダイヤモンドのGAP」は、自分のブランドが大きく開かれている所、というブランドの脳内認識に注力する
- WHATで「丸亀製麺ブランドのWHAT」と置き換えた「癒される安心感」が大きく外れているのが「ダイヤモンドのGAP」
- 「これは違うんじゃない?」「自分のブランドにあった『癒される』のではなく、『母性のとなる外食』のではなく『個人の中心の』とは違う」
- 「すごい安心感」というように自分のブランドの「癒される安心感」がカテゴリーの「ダイヤモンドのGAP」

#### §3 続: WHATの「外し方」をマーケが見つけるべき (pp.0241-0247)

**1店舗1店舗で粉からうどんを打っている** (p.0241):
- 自分のブランドの「狂人」(中央値者中心) からの認識のうちに、自分のブランドが「外食」のチャクラに対してGAPを持つ

## 第8章 (pp.0291-0300+) 強いマーケティング・コンセプトをつくる3つの要点

#### §0 序 (pp.0291-0292)

第7章では、「ブランド設計の仮説」を「ブランド・エクイティ・ピラミッド」で整理するやり方 (STEP2) について、第8章ではこの設計を実現にするためのマーケティング・コンセプトを実体化するための3つのコツについて

> 「100に1つ、200に1つの希少性に至るのですか」

#### §1 本能にぶっ刺せ! (pp.0292-0300)

**本能にぶっ刺せ! = 第1の要点**

**(1-1) 脳の情報処理の構造「システム1」と「システム2」(pp.0292-0294)**:

```
脳の3つのチェックポイント (Daniel Kahneman 由来):
①重要性の関門: 自分にとって重要か?
②好意度の関門: 好きか、嫌いか?
③納得性の関門: ほんとうに大丈夫か?
```

**先頭ろ上で、最初に「自分にとって重要か?」と判断するシステムが「システム2」**:
- 自分のフィルターを与え、その情報を判断したり、納得したりする
- 一度概念認識すれば、ものごとはなされ事できる
- → これは数秒もかかる長い時間で結論を出す
- 「ふつう先生」がリーグするためのカードを判断する

**「システム1」は、瞬間的に瞬時に判断する** (p.0293):
- 本能の関門での無意識の判断
- 「重要性の関門」「好意度の関門」「納得性の関門」の3つを一気に通過
- → これは0.5秒未満で結論

**マーケティング・コンセプトは「システム1」を貫けることで、本能にぶっ刺すことができる** (p.0293):
- 「システム2」は意識的・遅い
- 「システム1」は無意識的・速い
- → コンセプトが「システム1」のレベルで本能を揺さぶれば、選ばれる確率は高い

**「システム1」と「システム2」の特徴** (p.0294):
- 「**重要性の関門**」と「**好意度の関門**」は、本能的判断、 つまり「**システム1**」で判断される
- 「**納得性の関門**」は、後天的判断 (理性的)、つまり「**システム2**」で判断される
- 重要 = 本能、好き = 本能、納得 = 理性

#### (1-2) システム1で印象を取りに行く (pp.0294-0295)

ラスボス「システム2」での疑問に裸の最後問は次のものがあります:

```
a) 便益は本当に手に入るのか?
b) 価格は妥当なのか?
c) 他のオプションと比べても良い選択なのか?
```

→ ブランド・エクイティの設計のうち、 WHAT に便益だけでなく RTB (Reason To Believe) を整えて設定しているのは、 1点目の「**便益は本当に手に入るのか?**」 という疑問に答えて行為のです。

**価格の問題** (p.0295):
- 価格はあいつもしっかり倒えて重要なのではない。 まだ、便益の場合の脳語ノウフラントのは、特に重要性の関門で「価格は重要なものではない」「**システム1**」を起動させてようとする「価格はそれほど気にしなくていい」のがミラーになる
- カテゴリーによって違うけれど、ほぼあらゆるカテゴリーの平均場面でも、最大要素の25%を超えるくらいの「**システム1**」が起動するもの平均
- 価格を低くしすぎてしまったりするとほぼ意味がないし、 「**システム1**」が起動する場合では、「**システム2**」を選んで、その理性的な判断による絶望を待つことになります。

#### (1-3) 本能を揺さぶれと脳は言いたい (pp.0296-0297)

> 「今のヒトの中の自分が、さらまでとも多くのファトの局面で、一瞬で多くの人々の話を行うのは、何万年も前の遺伝に基いて思考しています。 だから現代に至ったいかなる場面でも、自分の理性的な判断は本能の支配を受けています」 (p.0296)

→ 戦略的決定:
- 本能ガアトリビュート (好き) の決定は、本能ガアトリビュート (好き) の決定によって、自分の判断ができないになるかなり高いです
- ベース: ベース食 (好き) → 本能ガアトリビュート

**消費者の脳内の二重・脳内のアプリのカスタマー** (pp.0296-0297):
- 「ブランド」 = ヤフー、 ベース食 (好き)
