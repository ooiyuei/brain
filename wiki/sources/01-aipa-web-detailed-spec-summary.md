# Aipa WEB 事業構想まとめ

## 一文サマリー
採用特化のAI制作オペレーションで、中小企業のWeb制作を爆速・低コスト・月額サブスク型へ転換。

## キーポイント
- 採用サイト制作を「受託」から「フォーム入力→AI初稿→人間確認→公開」のオペレーション型商品へ転換。
- 競合との差別化は「AI単体」ではなく、標準化されたヒアリングと人間による品質担保を含む全体フロー。
- 採用は痛みが強く、かつ構成が型化しやすいため、建設・物流・製造など採用難易度の高い地方中小企業をターゲット。
- 初期費用はモニター価格15万円（通常30〜50万円の半額）、継続収益は月額サブスク（社外Web担当者）を柱に据える。
- 安売りリスクを回避するため、最低価格設定、修正回数制限、および「事実確認・権利関係」は必ず人間が行う体制を構築。
- 営業は「AIでサイトを作る」ではなく「Web制作の面倒を減らす」を訴求し、求人媒体の弱点を突くアプローチを取る。
- 最終目標は、業種別テンプレや自動見積もりを含む「AI Web制作オペレーションSaaS」への進化。

## アクションアイテム
- Googleフォームを最終版として完成させ、必須項目と分岐ロジックを確定させる。
- 架空企業（静岡建設・物流・美容など）のデータを用い、Claudeで初稿を生成する実証テストを実施する。
- 検証結果を基に、1つの完全なサイト制作を完了させ、所要時間と課題点を記録する。
- 事例掲載条件付きで、モニター企業3社を15万円の価格で獲得する。

## 関連事業
- 地方中小企業向け「社外Web担当者プラン」（月額運用代行）
- 求人媒体代理店との連携による「採用サイト整備」パッケージ販売
- 将来的な「AI Web制作オペレーションSaaS」の開発・提供

## オリジナルからの抜粋(最重要な数字や引用)
- 「価格と速さで勝負する牛丼みたいな戦略」
- 「売るのは『Webサイト』ではなく『中小企業がWebサイトを作るときの面倒を全部減らすこと』」
- 「AI一撃の80点を、人間が事実関係/給与/権利/スマホ表示/応募フォーム/SEO確認で100点に」
- 「モニター価格15万円（事例掲載条件）」
- 「フォーム入力だけで、採用サイトを最短7日公開」
