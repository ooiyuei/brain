# FishOps Agent構想

## 一文サマリー
釣り現場の「勘」をデータ化し、AI運営者で改善するB2B事業。

## キーポイント
- 釣具サブスクではなく「釣れる体験」を提供するAI運営エージェント
- 管理釣り場・釣具屋・漁協の現場課題（高齢化・データ不足）を解決
- 「釣れなかったデータ」の収集と分析で運営最適化を行う
- MVPはLINE/フォーム等既存ツールで低コスト検証
- 管理釣り場→釣具屋→漁協・観光へ段階的に事業拡大

## アクションアイテム
- 管理釣り場10店舗へのヒアリング（放流・釣果記録の現状）
- 港前釣具屋10店舗へのヒアリング（初心者対応・SNS課題）
- 初心者向け釣り体験イベント（10名）の実施と課題抽出
- FishOps Agent v0（管理釣り場・釣具屋用）のプロトタイプ制作

## 関連事業
- 地域密着型手ぶら釣り体験プラットフォーム
- 漁協・自治体向けDX支援サービス
- 小規模養殖場のAI副操縦士システム

## オリジナルからの抜粋(最重要な数字や引用)
- 「釣れない釣りを、データとAIで"釣れる体験"に変える。」
- 「月額でAIスタッフを雇ってください」
- 「釣れなかったデータを取る（改善点は失敗にある）」
- MVP1：管理釣り場AIレポート（Googleフォーム/LINE/スプレッドシート）
- 料金例：管理釣り場Lite 9,800〜、Standard 29,800〜
