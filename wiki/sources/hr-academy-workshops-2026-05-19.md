## 1. ストーリーテリング 3軸フレーム

### 中核フレーム
```
普通の思考 × 武器・素材 × 提案した思考
```

### 事例1: ジャパネット高田のカメラ
| 普通の思考 | 武器・素材 | 提案した思考 |
|---|---|---|
| 高性能カメラ?普段使ってスマホで困らない | ①スマホより高画質 ②写真プリントせず忘れがち | 一眼レフなら大きくプリント→長く思い出に遺してプレゼントできる |

### 事例2: JR シンデレラエクスプレス
| 普通の思考 | 武器・素材 | 提案した思考 |
|---|---|---|
| 新幹線はビジネスマン用。日曜最終便は中途半端 | 「その日一番最後」遅くまでいられる機能性+エモさ | 遠距離恋愛カップルが週末を少しでも長く一緒に過ごせる乗り物 |

### 事例3: 北条政子の演説 (1221年承久の乱)
| 普通の思考 | 武器・素材 | 提案した思考 |
|---|---|---|
| 朝廷激おこで怖い。寝返ったほうがいいかも | ①幕府を守る側に大義名分 ②守らないデメリット大 | 既に大恩義あり。使命。地位も財産も失うかも |

### 設計のコツ
1. **いきなり提案から考えるな** → まず「普通はどう考えてる?」を言語化する
2. **武器・素材を大量に書き出す** — どれが効くか最初は分からない
3. **逆張り練習を日常的に**: 「常識やみんなとは逆の結論を出すとしたら?」
4. **筋トレ例**: 「筋トレはいいぞ」と先に結論 → 「努力が報われるとは限らない人生で、正しい方法で努力さえすれば必ず結果が出るのが筋トレ」というストーリーで成立させる

### ストーリーテリングの2つの効果
- **顧客のハートを動かす** (営業・コンテンツ)
- **チームメイトのハートを動かす** (リーダーシップ・特にピンチの時)

## 2. 合意形成「見つける」の4ステップ

```
1. 意見を分解
2. 共通点を探す
3. 共通点から再考する
4. アイデアを出し合う
```

### 使用例: 卒業旅行 (1人20万x3-4人 = 60-80万)
- メンバーごとの希望をまず分解 (USJ行きたい/自然がいい/海外/食/etc)
- 共通点 (思い出/予算内/移動少なめ/etc) を探す
- 共通点から再考 → 新案を出し合う
- 結果: 大阪USJ/北海道スキー/長野紅葉/関西食巡り等の多様な合意形成パターン

## 3. 個人テーマ探究 11項目ロジックモデル

### 構造 (これがHR高等学院の探究の中核)
```
①テーマ概要 — 何を目指してどんなことに取り組む?
②Why you? — 私がこのテーマに取り組む理由・原動力
③価値(自分) / 価値(社会)
④現状 — 今どうなっている?
⑤ありたい姿 — どんな状態になれば理想?
⑥問題 — 現状とありたい姿のギャップ
⑦原因 — 問題の原因は何? 根拠は?
⑧課題 — ギャップを埋めるために必要なこと
⑨解決方法 — 課題解決に何が考えられる?
⑩仮説検証/プロトタイピング — 何に取り組む?
⑪検証結果 — 何が得られた? 新しく分かったことは?
```

### 大井の事業設計にそのまま使える
- AIpa Web → ①〜⑪まで埋められる事業
- AIpaX → 既に4社契約済 = ⑪検証結果がある
- AIpaX school → ①〜⑨まで埋まっている、⑩⑪が第1期生体験会で得る

### 生徒の探究テーマ例 (大井の同期)
- コスプレYouTuber / 壱岐の地域活性 / 若者の海外体験 / 探究の改善 / esports / 質の良い睡眠 / 幸せ / 自己決定できない件 / 演説と支持率 / 作曲

## 4. CHINTAI型 サービス企画 5項目

```
①誰のためのもの? — 具体的ペルソナ1人を描く
②どんな暮らしの課題・ニーズに注目した?
③どんなサービス・仕組みで叶える?
④アイデア実現に必要なモノ・コト
⑤ビジネスモデル — 支払う主体は?
```

### 審査基準 (大井の事業評価に使える)
- **自分が使いたい・友達に勧めたいものか?** → N=1の視点
- **その会社がやるべきと言える?** → 「どの会社でもできる案」じゃないか
- **N=1の視点 × 社会潮流** → 説得力ある事業に育つ

### 学生の出したアイデア例
- 美大生向け「アトリエ共用部分」シェア物件 (はつみん)
- ペットの譲渡サイトに広告→飼うタイミングで引っ越しきっかけ (玉越)
- 部屋マッチングAI/部屋の模型/口コミサイト (はる/はやと等)
- 一軒家防音シェアハウス+10ギガLAN完備 (ゲーマー向け)

## 6. 産業課題 × AI解決 (キャリア探究)

### 拠点別の課題
| 拠点 | 産業 | 課題 |
|---|---|---|
| 横浜 | 水産加工業 | 高齢化・後継者不足 |
| 渋谷 | 都市型農業 | 土地・コスト問題 |
| 代々木 | 食品製造業 | フードロス・原材料価格高騰 |
| オンライン | 畜産業 | 環境負荷 (CO2・におい) |

### 産業分析フレーム (4項目)
```
①この産業の今 (現状・規模感)
②課題は何?
③解決策 (AIや技術革新含めて)
④笑顔になるのは誰?
```

### 大井の事業に当てはめる
- AIpa Web → 採用業 × AI (採用課題・人手不足解決)
- AIpaX → 中小企業のあらゆる業務 × AI (5%導入率問題解決)
- 自社事例として横展開可

## 7. 熱海観光復活 (データ読解+論理説明)

### 入試問題形式 (静岡大・情報学部 2022改題)
- 図表6つ (再生取組/宿泊客推移/年代別構成/ロケ誘致/Google検索/SNS年代調査)
- 観光がどう回復したか + メディアの関わりを論理的に説明

### 学んだフレーム
- **データを読み取る→意図を推測→論理的に説明**
- 官民連携・ブランド再構築・年代別ターゲット・SNS活用 が観光復活の鍵
- 若者はテレビを見ない → SNS媒体シフト

### 大井の事業に転用
- 自社事業の「データドリブン改善」フレームそのまま使える
- 営業時の「業界データ→課題提示」プレゼンに応用
