# サブスクボックス 1日10万円 (個人開発バズ実例)

## 実績
- ファイナンスランキング 1位
- 全体ランキング 4位 (一瞬・Googleアプリの間)
- 3万1000 ダウンロード
- 17-18万円売上 (本動画時)
- 最初の1日で10万円
- 月の売上 30万円弱予測

## 技術スタック詳細
- **フロントエンド**: Expo (React Native ベース・Windows iOS出せる)
- **バックエンド**: なし (ローカル保存のみ)
- **データベース**: Expo SQLite (ローカル)
- **課金**: RevenueCat (Apple 30%手数料・15%申請可)
- **分析**: PostHog (MCP対応・ファネル分析・離脱検知)
- **AI コーディング**: クロードコード ほぼ全機能

## バックエンド代替案
- データベース必要時: Turso / Convex / Firebase
- スパース は RLS設定難しい
- Cloudflare R2 (画像 10GB無料)
- Stripe Web経由 = 3.6% (Apple 30%回避戦略)

## 逆算思考フレーム (詳細)
```
月20万 → 1日6,300円
→ 1,000円買切なら 1日6個販売
→ CVR 1%なら 1日600ダウンロード
→ X バズ or ASO or SNS集客
```
