# ソロプレナー入門 月3万円稼ぐロードマップ完全解説

## 📊 サマリー
- サムネAI = 月3万円 (MRR 1万9800円・準売上3万3000円)
- ストライプダッシュボード公開実例
- 個人開発 9ステップ詳細
- 「不快を解消するアプリ作れ」原則

## 🔥 9ステップ ロードマップ

### Step 1: アイデア発掘
**「自分の不快を解消するアプリ作って」**
- 自分の痛みから = サムアルトマン・Netflix もそう
- 日常メモ → 重複 = 本物のペイン
- AI は ペイン発掘 を代替できない (人間の経験則)
- 海外バズの日本ローカライズ も有効
- 競合いる ≠ 諦める理由 (差別化点・1機能特化)

### Step 2: 市場・競合調査
- Google検索 / Apple Store検索
- ディープリサーチ (ChatGPT/クロード) 10-20分
- **競合 = 需要証明** + 1機能特化で勝つ
- 大手はニッチに参入しない → 個人の勝ち目
- 「肌感覚で OK」(時間かけても不確実性高い)
- 心理状況 = モメンタム チェック (やる気あれば突っ走る)

### Step 3: MVP開発
**「1つの痛みを解決できたらすぐ出す」**
- 最小限の機能・全機能実装するな
- メアド認証不要・Google認証だけでOK
- ダーク・素材ページ・テンプレート全部後回し
- バイブコーディング (クロードコード / カーソル どちらでも)
- スピード重視 = スタートアップ思考

### 技術スタック (Web)
- フロント+バック: NEXT.JS App Router
- DB: Turso (100枠+10GB無料・MCP対応) or Supabase
- 認証: Supabase OAuth / Better Auth / Clerk
- 決済: Stripe (3.6%) ← ポーラより安い
- 画像ストレージ: Cloudflare R2 (10GB無料)
- メール: Resend
- インフラ: Vercel ($20/月) or Netlify or Cloudflare Workers

### 技術スタック (Mobile)
- フロント: Expo (Windows iOSビルド可)
- ネイティブ必須なら Mac + Swift / Flutter
- 決済: RevenueCat (Apple 30%・15%申請可)
- 分析: PostHog (MCP対応)

### Step 4: リリース
- 完璧主義 = 現実逃避
- Google Analytics 必須
- SNS告知 (X / TikTok)
- Google Search Console インデックス申請

### Step 5: 分析
- **課金されたか** これだけでいい
- CVR 1% 合格・2-5% 優秀・0% ピボット
- MCP サーバー (Supabase / Turso) で分析
- GA / GSC で SEO分析
- AI に分析と仮説立案させる (クロード 4.5 Opus)

### Step 6: ボトルネック発見+改善 (PDCA)
**バケツの穴 = ファネル分析**
- LP訪問→新規登録: Google認証 / オンボーディングフロー
- 課金転換率低い: マイクロコピー修正 (1.3倍効果)
- CTA/価格プラン見直し

### Step 7: PMF達成判定
**3軸シグナル全クリア**
- 定量: 継続15%+/30日・CVR 3%+・オーガニック成長
- 訂正: 「なくなったら残念」40%+ / 紹介自然発生
- 経済: LTV ≧ CAC × 3
- 課金率 4-6% = 業界標準
- 8割ピボット覚悟

### Step 8: ユニットエコノミクス計算
- LTV (生涯顧客価値) - AIコスト = 粗利LTV
- CAC = 1人顧客獲得コスト
- LTV/CAC ≧ 1 (最低限) ・3 (理想)
- 赤字でない事チェックしてからスケール

### Step 9: スケール
- PMF達成後・乾いた燃料で焚き火
- SEO/ASO (CPA 0円)
- X (ビルドインパブリック)
- TikTok (バズ起点・フック3秒)
- リファラル (クレジット付与・指数関数拡散)
- 広告 (ROAS 計算・AB テスト)
- インフルエンサー (PMF後)
