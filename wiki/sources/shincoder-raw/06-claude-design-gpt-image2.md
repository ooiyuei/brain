# クロードデザイン + GPT Image 2 でLP作成

## 📊 サマリー
- AI臭ないLPを2-3時間で量産
- 「素材がデザインの良さを決める」原則
- 起業ホームページ・受託制作ポートフォリオ・大井5事業LP応用可

## 🔥 核心ノウハウ

### 素材重視の発想転換
**「クロードデザインだけでは限界がある」**
- ポンと頼んでも AI臭が抜けない
- 素材 (背景図形・アイコン・人物画像) を別途生成して投入
- 本来のLP制作 = デザイナーが素材作る → エンジニアがCSS配置
- AIで同じプロセスを再現

### 5ステップ制作フロー
1. **ChatGPT Image 2** で LP モックを1枚生成 (構造化プロンプト)
2. **GPT Image 2** で 素材 (背景図形・アイコン・ロゴ等) を抽出 (グリッド形式)
3. **クロード Web** で 素材分割 + 投化
4. **クロードデザイン** に LP + 素材を渡して実装
5. **クロードコード** に export + 微調整 + デプロイ

### プロンプト構造化テンプレ
```
目的: __
業種: __
ターゲット: __
トンマナ: __ (ポップ・遊び心・ノーション風・ミニマル・等)
デザイン要件:
- 不規則な図系
- 柔らかい抽象形状
- (詳細...)
```

### 重要原則
- 「ベストプラクティスは確立されない・自分で構築」
- 「スキルは育てる・1発で完璧は無理」
- 「業務ワークフローを考えて自分のスキル化」
