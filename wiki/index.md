## 📐 ブレイン運用ルール（2026-05-19更新）

### 書く場所のルール
| 内容 | 書く場所 |
|------|---------|
| 構想・仕様 | entities/{事業} / concepts/ |
| 行動した事実 | **meta/execution-log** |
| 売上・契約・支出 | **meta/financial-log** |
| 新しい人との接点 | **meta/people** |
| 事業の凍結判断 | **meta/graveyard** |
| 思いつき | _tasks/inbox |
| 今日のタスク | _tasks/today |
| 日常メモ | _routines/(時間) |
| ソース取り込み | sources/ → 該当 entities/ |

### 増やすルール
- 新しい事業案 → [[meta/business-portfolio]] で分類してから entities/ に追加
- 「次の一手」を3行で書けない事業は [[meta/graveyard]] へ

### 減らすルール
- 6ヶ月以上動かない事業 → 自動的に [[meta/graveyard]] へ
- 大井が「冷めた」と発言した事業 → 即 [[meta/graveyard]]
