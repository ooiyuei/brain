# 次の90日 — 月収100万への割り算

> AI経済地図と大井15事業の答え合わせから抽出した、5/19〜8/19の実行プラン。
> 全体は3つの主軸 × それぞれの「ザ・モデル」設計。

---

## 主軸（3本のみ・他は寝かす）

### A. AIpa Web — 月15万円 × モニター3社 = 月45万円
**Deep Skill層 × 中小企業 × AI**
- 地方30-300名・建設/製造/運輸/サービス業
- 16セクションフォーム → AI初稿 → モニター契約

### B. AIpaX — β顧客1社 × 50〜80万円 = 月50万円相当
**既存業界 × AI / 中小企業の知識ギャップ（77% vs 5%）**
- 「会社別AI保存庫コア + リアルタイム電話/営業回答支援」
- 初回事例価格でβ確保

### C. AIpaX school — 中高生スクール構築・月謝モデル仕込み
**教育 × AI / 1年カリキュラム5モジュール**
- 第1期生5名 × 月3万円 = 月15万円（90日後ローンチ目標）

合計潜在月商: **45 + 50 + 15 = 月110万円**（達成可能ライン）

## 各事業のザ・モデル化（センターピン強化）

### AIpa Web のザ・モデル
```
[Marketing] → [IS] → [FS] → [CS]
SNS投稿     LP接触    商談     納品+運用
                              ↓
                          月10万定額
```
- **IS = 大井自身（最初）** → AIエージェントで自動化（Month 3〜）
- KPI: 商談数 + 成約率（質を担保）
- フィードバックループ: CSから「次の事例ネタ」をMarketing側へ

### AIpaX のザ・モデル
```
[マーケ] → [IS] → [FS] → [CS]
記事     LINE    商談     導入後の保存庫運用
```
- IS段階で **AIエージェント = 大井のクローン** で対応自動化
- 過去の良い対話を学習させる

### AIpaX school のザ・モデル
```
[マーケ] → [IS] → [FS] → [CS]
親向けLP  体験会  契約    講師AI + 中高生コミュニティ
```
- 講師業をAI化（カリキュラム自動配信）→ 1人で20名指導可
- CS = 中高生がリリースしたアプリの紹介 → 次期Marketing素材
