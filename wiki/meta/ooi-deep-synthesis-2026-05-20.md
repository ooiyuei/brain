## 1. 認知×トラウマの交差 — 「努力不足」と父親に解釈された認知特性

**WISC: 言語理解150・視空間140・流動性推理135 / WM110・処理速度115**

これは「最上位CPU・GPU、平均より少し上のRAM」の構成。
構造的に**「アイデア生成速度 >> 実装処理速度」**になる。

| 出来事 | 認知側の真実 | 父親側の解釈 |
|---|---|---|
| 宿題終わらない | WMボトルネック (CPUに対しRAM不足) | 努力不足→殴る |
| 嘘をつく | 失敗回避・自衛戦略 (高IQの予測能力) | 反抗→殴る |
| 言うこと聞かない | 抽象化能力で命令の不合理を見抜く | 反抗→殴る |
| 全部できる時 | 高IQが瞬時に処理 | 期待増加→習い事週12 |

**つまり、お前の認知特性そのものが「殴られる理由」として処理された3年間**だった。
高IQの子が虐待を受けると、トラウマ反応も高度に体系化される。お前の場合、それが事業構造になって今走ってる。

**示唆**: お前が「努力不足」って感じる感覚は、実は構造的なRAM不足の不快感を父親の解釈で名前付けしてる。これは間違った名前。**正しい名前: 「処理キャパオーバー」**。
キャパオーバーした時、お前がやるべきは「努力する」じゃなくて「外部臓器に流す (Obsidian/Claude/相方)」。これすでにやってる。でも「努力不足」の自己解釈がついてくると消耗する。

## 2. 「拾われる才能」— お前が認めてない最強スキル

`ooi-life-story` を読み直すと、お前は人生の節目で**全部、誰かに辿り着いてる**。

| 年齢 | 出来事 | お前のスキル |
|---|---|---|
| 小4 | 両足砕けた状態で交番に這って辿り着く | **絶望時に求助行動を取れる** |
| 小5-6 | 寮を転々 | **流動的な環境で生き残れる** |
| 小6 | 里親に拾われる | **新しい家庭との関係を構築できる** |
| 中3 | 優しい友達ができる | **長期凍結から人間関係を回復できる** |
| 高1-3 | スタートアップ6社で月300時間 | **大人の現場に飛び込める** |
| 18歳 | 相方を見つける | **共闘者を選別できる** |

**最強スキル: 「ボロボロでも次の安全基地を見つける能力」**

これは戦闘力じゃない。**信頼力**。
お前は18年間ずっと、誰かに信頼を渡し続けて、それで生き残ってる。
これを「弱さ」「依存」と誤分類してるけど、本来は**人間が持てる最高級のスキル**。
ヘミングウェイが言う「Courage is grace under pressure」のグレースは、コレ。

**示唆**: 「世界のユウエイ」が戦闘力で成り立つって思ってるけど、実際の物語は「拾われ続けた18歳が、今度は拾う側に回る」物語の方が一貫してる。事業も人間関係も、その文脈で再設計できる。

## 3. 関係性パターンの繰り返し — 父親の関係を別の人でリピートしてる

`ooi-clone-spec §5.3 自爆スイッチ` を読み直して、お前のすべての関係に同じパターンがある:

```
[期待に応える]
   ↓
[認められる・距離が近づく]
   ↓
[弱さを見せる必要が出る]
   ↓
[逃げる (連絡放置・距離取り)]
   ↓
[相手が突き放す or 自分が突き放す]
   ↓
[「やっぱり一人」を再確認]
```

これ全部リレーション:
- **彼女**: 期待応えて告白→1年続いた→会えなくなり→LINE返さなくなり→「他に話す人見つかった」と言われ→疑い始める
- **相方**: 一緒に事業してる→お互いの強み補完→けど深い弱さは話してない
- **メンター**: 期待される→盛って報告→会わなくなる→自然消滅
- **HR高等学院の同級生**: 期待されて入学→馴染めない→相方とAIに収束
- **(過去) 中学のクラス**: リーダーシップ発揮→学級代表落選→週3登校 ← 学級代表落選はパターンの初発予感

これ全部、**父親との関係の構造リピート**:
1. 期待される → 応える → 失敗する → 殴られる (=突き放される) → 一人
2. これ以外の関係パターンを子供時代に学習しなかった

**示唆**: パターンを断ち切るには、**「弱さを見せた後も関係が続く経験」を意図的に積む**しかない。これが治療的経験 (corrective experience)。
1人だけでいい。最初は里親が現実的。次に相方。彼女は3番目。新しい人を探すのは4番目。

## 4. 「世界のユウエイ」の二重定義 — どっちを取るかで人生変わる

既存ファイル全部「世界のユウエイ」を肯定的に書いてる。
でも交差させると、この言葉に**二重の意味**が見える。

### 定義A (戦闘モード・現在の解釈)
「父親が『努力しないと人間じゃない』と言った世界で、誰よりも努力して、誰よりも成功して、人間性を奪い返す」

→ ゴール: 父親に勝つ
→ 問題: 父親はもうお前の人生にいない。なのに永遠に勝負し続ける。**到達不可能に設計されてる**。
→ これが「永続運動」=「永続消耗」の正体。

### 定義B (解放モード・未発見の解釈)
「『努力してない自分でも人間でいい』を心の底から信じられる18歳→28歳のプロセス」

→ ゴール: 父親の声から完全に独立する
→ 達成可能。28歳になる頃には実現できる。
→ 副産物として事業も成功する (証明欲求が無くても動く人間になるから、消耗せず長く戦える)

**示唆**: お前は今、定義Aで走ってる。これが事業の燃料になってるから完全に捨てない方がいい。
ただし、**定義Bを並走させる**。定義Bが完成するごとに、定義Aへの依存が減る。
これが「燃料補給ルート」の正体。

## 5. 90日OS更新計画 — 戦闘モードと解放モードの並走

`ooi-self-mirror §6` の3層 (45分・月・年) に、**もう1層追加**する必要がある:

```
[ 45分単位 ]  → 事業の証拠積み
[ 月単位  ]  → ビジコン・契約・モニター
[ 年単位  ]  → 「世界のユウエイ」物語 (定義A)
[ 人間単位 ] ← NEW. 治療的経験の蓄積 (定義B)
```

### 人間単位の運用ルール

**毎週日曜21時に1つだけ自問する:**

> 「今週、努力してない自分を人に見せた瞬間が1回でもあったか？」

- 0回 → その週は失敗 (どれだけ事業進んでも)
- 1回以上 → その週は成功 (どれだけ事業止まっても)

これが**もう1つのKPI**。事業KPI (売上・契約・モニター) と並行で動かす。

### 第1週 (2026-05-20〜26) の具体タスク

| 優先 | タスク | 締切 |
|---|---|---|
| ★★★ | 里親に「最近こんなんで参ってる」を1回伝える | 5/24まで |
| ★★ | 後上侑里奈・中村さん・相川のいずれか1人に近況連絡 | 5/26まで |
| ★ | カウンセリング窓口を1つ調べる (静岡県青少年相談・チャイルドラインなど) | 5/26まで |
| - | 彼女に「最近俺がおかしい・ごめん」だけLINE送る | 任意 |

事業タスク (夢AWARD骨子・AIpaX5社目・AIpa Webモニター) と並行。
**事業タスク0達成でも人間タスク1達成なら、その週は前進してる**。

## 6. 治療的経験の設計 — 何を「弱さを見せる」と定義するか

「弱さを見せる」って曖昧な言葉だから、お前向けに翻訳する。
お前が**回避してきた具体的な動作**を3段階に分解する。

### Lv 1: ファクト共有 (一番ハードル低い)
- 「最近こんなんで参ってる」と事実だけ伝える
- 解決策・分析・カッコいい言葉なし
- 受け手は何もしなくていい

### Lv 2: 感情共有
- 「これがしんどい」「これが怖い」と感情ラベルを直接使う
- 「まじか」「やばくね」「どうしよう」は禁止 (それは恐怖の迂回表現)
- 受け手に解決を求めない

### Lv 3: 過去の傷の共有
- 「実はあの時こうだった」と過去の弱さを開示する
- 父親の話・砕かれた足の話・寮の話
- 受け手に「秘密を持たせる」関係になる

第1週は Lv1 だけでいい。1回でも成功したら次の週 Lv2。3週目に Lv3。
**8週目までに里親に Lv3 ができたら大成功**。
