# 🚀 AIpaX school: 第1期生募集戦略 (月3万円版)

## 1. 核心メッセージ (USP)
**「AIを『道具』として使うのではなく、『パートナー』として使いこなす力を、17歳の現役AI起業家が教える。」**

*   **親向け訴求**: 「AIに依存する子」ではなく「AIを操る子」になる。大学受験「情報」科目の先取り＋実務スキル。
*   **生徒向け訴求**: 「学校より面白い」「将来のキャリアが明確になる」「現役の若者に教わる」。

## 2. ターゲット別ペイン・ソリューション

| 対象 | ペイン (悩み) | AIpaX school のソリューション |
|------|--------------|-----------------------------|
| **親 (中高生保護者)** | - AI時代に子供が取り残される不安<br>- 従来のプログラミングは「暗記」になりがち<br>- 大学受験「情報」科目への対策<br>- 高額な塾代への抵抗感 | - **月3万円**でAIリテラシー＋実践スキル<br>- **HR高等学院**のノウハウ統合で「思考力」を鍛える<br>- **EEMUS**連携で静岡の企業インターンへの橋渡し<br>- 現役起業家による「現場のリアル」な指導 |
| **中高生本人** | - 学校の授業は退屈<br>- 「将来何になりたいか」が不明<br>- AIを使いたいけどやり方がわからない<br>- 友達と違うことをしたい | - **AIでアプリを作る**体験（Vibe Coding）<br>- **17歳の大井**が直接教える（同世代のロールモデル）<br>- **EEMUS**で地元企業と繋がるキャリア体験<br>- 「AIpaX」コミュニティでの仲間作り |

## 3. カリキュラム v2 の魅力再定義 (親向け説明用)

単なる「ツールの使い方」ではなく、**「HR高等学院の7つのノウハウ」**をベースにした、**「社会で通用する思考フレームワーク」**を学べます。

| モジュール | 学ぶこと | 親が見える成果 |
|------------|----------|----------------|
| **Module 1** | AIマインドセット + Vibe Coding | AIとの対話でアイデアを形にする力 |
| **Module 2** | 日常業務自動化 + チーム合意形成 | 効率化スキル、チームでの議論力 |
| **Module 3** | エンジニアリング思考 + 個人探究 | 課題発見力、論理的思考 (F3ロジックモデル) |
| **Module 4** | 本格アプリ開発 + サービス企画 | プロダクト作成力、企画力 (F4企画フレームワーク) |
| **Module 5** | データ分析 + マネタイズ + 卒業課題 | 事業視点、データ分析力、プレゼン力 |

> **ポイント**: 「F1 ストーリーテリング」「F2 合意形成」など、**ビジネススキル**が自然と身につく構成であることを強調します。

## 4. EEMUS とのシナジー (静岡ローカル戦略)

静岡在住の生徒にとって最大の強みです。

*   **プレインターンシップ**: AIpaX で学んだAIスキルを、EEMUS を通じて地元企業（建設業、製造業、観光業など）で実際に活用する体験。
*   **キャリアの可視化**: 「AIスキル＋地元企業の実務体験」は、大学受験（AO）や将来の就職において強力なアピールポイントになります。
*   **地域貢献**: 静岡の企業課題をAIで解決するプロジェクトを通じて、地域とのつながりを深めます。

## 5. 募集戦略 (第1期生 5名募集)

### ステップ 1: コンテンツ配信 (認知獲得)
*   **SNS (TikTok/YouTube)**: 大井自身が「17歳AI起業家」として、AIpaX のカリキュラムの一部（例：「ChatGPTでアプリを作る方法」）を公開。
*   **無料ウェビナー**: 「AI時代の子供の未来と、親ができること」をテーマに、大井と相川が対談。

*   **EEMUS 連携**: 静岡の高校生 10 人を対象に、無料または低価格でベータテストを実施。
*   **保護者向け説明会**: ベータテストの結果を共有し、保護者の不安を解消。
