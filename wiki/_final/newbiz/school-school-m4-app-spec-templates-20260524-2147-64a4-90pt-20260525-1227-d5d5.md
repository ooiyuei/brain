# 🚀 AIpaX school: 戦略統合メモ

## 1. プロジェクトの核心 (Core Identity)

*   **名前**: AIpaX school
*   **ターゲット**: 中高生（特に高校生）＋ その親（不安解消）
*   **価格**: 月額 3万円（プレミアム/実践重視）
*   **最大の特徴 (USP)**:
    1.  **講師のリアルさ**: 「17歳現役AI起業家・大井」が直接教える。机上の空論ではなく、実際に稼いでいる現場の知見。
    2.  **確立されたメソッド**: HR高等学院で検証された「思考フレームワーク（F1-F5）」を統合したカリキュラムv2。
    3.  **EEMUSとの連動**: 静岡ローカルでは「AIスキル習得 → EEMUS経由で地元企業インターン → 就職/起業」の最短ルートを提供。

## 2. カリキュラム v2 の強み (Why this works?)

既存のスクール（QUREO等）との決定的な違いは、**「AIを道具として使う」だけでなく、「AIを使ってどう考え、どう伝えるか」まで含めた総合的なリテラシー**を教える点です。

| モジュール | 期間 | 内容 | 習得される「本質的な力」 |
| :--- | :--- | :--- | :--- |
| **M1** | M1-2 | AIマインドセット + バイブコーディング | **F1 ストーリーテリング**: 思考の構造化 |
| **M2** | M3-4 | 業務自動化 + チーム合意形成 | **F2 合意形成**: 他者との協働・調整力 |
| **M3** | M5-6 | エンジニアリング思考 + 個人探究 | **F3 ロジックモデル**: 問題解決の論理構築 |
| **M4** | M7-8 | 本格アプリ開発 + サービス企画 | **F4 CHINTAI型企画**: 市場価値のある企画力 |
| **M5** | M9-12 | データ分析 + マネタイズ + 卒業課題 | **F5 産業課題×AI**: 業界視点と収益化 |

> **親への訴求ポイント**: 「プログラミングの書き方」だけでなく、「社会で通用する問題解決能力・プレゼン力・論理的思考」が身につく。

## 3. EEMUS とのシナジー (The Ecosystem)

AIpaX school は、**EEMUS（静岡の高校生×地元企業マッチング）** の「プレ教育・差別化要素」として機能します。

*   **現状の課題**: 高校生が地元企業に興味を持っても、「何ができるか」が不明確で応募に繋がらない。
*   **AIpaX × EEMUS の解決策**:
    1.  **AIpaX でスキル習得**: 高校生が AI を使って実際にプロダクトや企画を作る。
    2.  **ポートフォリオの完成**: 「AIを使って〇〇を改善した」という具体的な実績ができる。
    3.  **EEMUS で企業と出会う**: その実績を持って地元企業（建設業、製造業等）と対等な立場で対話できる。
    4.  **結果**: 企業側も「即戦力になりそうな若者」として評価し、早期採用やインターンに発展。

> **戦略的意義**: AIpaX school は、EEMUS の「需要（企業）」と「供給（学生）」を繋ぐ**「品質保証（スキル）」**の役割を担う。

## 4. 競合優位性と市場ポジショニング

| 比較軸 | 既存スクール (QUREO等) | AIpaX school |
| :--- | :--- | :--- |
| **価格** | 月額 1〜3万円 | **月額 3万円** (高付加価値) |
| **講師** | 契約講師・動画 | **大井湧瑛（現役起業家）** |
| **カリキュラム** | プログラミング言語中心 | **思考フレームワーク + AI活用 + 起業体験** |
| **ターゲット** | 小学生〜中学生中心 | **高校生〜大学生（キャリア直結）** |
| **成果** | 作品作成 | **ポートフォリオ + 起業家マインド + 就職/起業への道** |

*   **市場の隙間**: 「プログラミング教育」は飽和しつつあるが、「AIリテラシー × キャリア形成」を高校生向けに実践的に教えるスクールはほぼ存在しない。
*   **価格設定の根拠**: 月額3万円は高額に見えるが、「EEMUS経由での優良企業インターン/就職」へのアクセス権や、「現役起業家からの個別指導」という価値があれば納得感がある。

## 5. 直近のアクションプラン (Next Steps)

### Phase 1: 第1期生の募集と検証 (今すぐ)
1.  **募集要項の確定**:
    *   対象: 静岡県内の高校生（優先）、中学生（上級者）
    *   人数: 5名（第1期生）
    *   価格: 月額 3万円（早期割引等あり）
    *   特典: EEMUS経由での地元企業インターン紹介保証（または優先枠）
2.  **ターゲットへのアプローチ**:
    *   **親向け**: 「AI時代の子どもの未来不安」を解決するメッセージ。HR高等学院での実績、大井の経歴（虐待からの脱出、起業家としての実績）を信頼の証として提示。
    *   **生徒向け**: 「学校では学べない、現役17歳起業家からのリアルなAI活用術」。EEMUSとの連動で「地元企業での活躍」を約束。
3.  **カリキュラムの微調整**:
    *   最初の1ヶ月（M1）を特に充実させ、即効性のある「AIツールで何かを作る」体験を提供し、継続率を高める。

### Phase 2: コンテンツ配信と認知拡大 (1ヶ月以内)
1.  **SNSでの発信**:
    *   TikTok/YouTubeで「17歳起業家が教えるAI活用術」を配信。
    *   具体的な事例（例: 「AIで地元企業の売上を10%上げた方法」）を公開。
2.  **EEMUSとの連携強化**:
    *   EEMUS参加企業に対して、「AIスキル習得済みの高校生」を推薦する仕組みを作る。
    *   企業側にも「AIpaX school 修了生」の価値を説明し、採用の質向上を訴求。

### Phase 3: スケールアップ (3ヶ月後)
1.  **カリキュラムの標準化**:
    *   大井以外の講師（HR高等学院のネットワーク等）でも教えられるよう、教材を体系化。
2.  **他地域への展開**:
    *   静岡だけでなく、他の地域でも「AIpaX school × 地域マッチング」のモデルを展開。
3.  **法人向け研修**:
    *   新卒採用前のAI研修として、企業にAIpaX school のコンテンツを提供。

## 6. リスクと対策

*   **リスク**: 大井個人の依存度过高。
    *   **対策**: カリキュラムv2を明確に文書化し、他の講師（HR高等学院の卒業生等）が補完できる体制を作る。
*   **リスク**: 価格の高さによる応募者の少なさ。
    *   **対策**: EEMUSとの連動（就職/インターンへの近道）という明確なベネフィットを提示。また、分割払いや成績優秀者への奨学金制度を検討。
*   **リスク**: AIツールの急速な変化。
    *   **対策**: カリキュラムを「ツールの使い方」ではなく「思考フレームワーク」に焦点を当て、変化に強い構造にする。
