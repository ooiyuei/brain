# 🚀 AIpaX school: 戦略的統合と次の一手

## 1. プロジェクトの核心：なぜ「月3万円」で勝てるのか

既存のプログラミングスクール（Tech Kids, QUREO等）との決定的な違いは、**「AIを「道具」として使いこなす「思考の外部化」スキル**」を提供することです。

*   **親の不安**: 「AIに依存して考える力がなくなるのでは？」→ **回答**: 「AIを手足のように使い、人間は『問い』と『判断』に集中する力を鍛える」
*   **生徒の欲求**: 「ただの勉強」ではなく、「実際に何かを作り、世の中に出す体験」
*   **価格の合理性**:
    *   月3万円は、個別指導塾（月5〜8万円）より安く、オンライン教材（月数千円）より「人間との対話とフィードバック」がある。
    *   **HR高等学院のノウハウ統合**により、高品質なカリキュラムを低コストで提供可能。

## 2. ターゲットと訴求ポイントの再定義

| ターゲット | 親への訴求ポイント | 生徒への訴求ポイント |
|------------|-------------------|---------------------|
| **高校生** | 「大学受験『情報』科目対策」＋「EEMUS連携で地元企業インターンへのアピール力向上」 | 「学校より現場で学ぶ」「17歳起業家が直接教える」「AIでアプリを作る」 |
| **中学生** | 「中学プログラミング必修化の上位版」「未来のスキルを先取り」 | 「ChatGPTで遊べる」「自分だけのアプリが作れる」 |
| **小学生（高学年）** | 「AIリテラシーの早期習得」 | 「AIと会話して絵や文章が作れる」 |

**★ 最大強み**: **大井（17歳現役AI起業家）のリアルな体験**
「学校より現場で学ぶ」スタイルをそのままカリキュラム化。これは既存の講師陣（大学生・社会人）にはない「共感力」と「リアルな成功体験」を提供します。

## 3. カリキュラムv2の強化ポイント（HR高等学院ノウハウ統合）

HR高等学院で得た**「合意形成」「ストーリーテリング」「ロジックモデル」**などのフレームワークを、AIスキルと融合させます。

### Module 1: AIマインドセット + バイブコーディング体験 (Month 1-2)
*   **内容**: ChatGPT/Claudeの基礎 + プロンプト設計 + Replit/Cursorでのノーコード体験
*   **HRフレームワーク**: F1 ストーリーテリング3軸（普通の思考×武器×提案）
*   **成果**: 「AIに指示を出して、動くプロトタイプを作る」体験

### Module 2: 日常業務自動化 + チーム合意形成 (Month 3-4)
*   **内容**: 資料作成AI (Gamma/Notion AI) + チームでのアイデア出し
*   **HRフレームワーク**: F2 合意形成4ステップ
*   **成果**: 「AIを使って、チームで合意形成し、資料を作る」体験

### Module 3: エンジニアリング思考 + 個人探究テーマ設定 (Month 5-6)
*   **内容**: 論理的思考の訓練 + 卒業課題のテーマ設定
*   **HRフレームワーク**: F3 11項目ロジックモデル
*   **成果**: 「自分の興味を、論理的なプロジェクトテーマに落とし込む」

### Module 4: 本格アプリ開発 + サービス企画 (Month 7-8)
*   **内容**: AI活用アプリの開発 + ビジネスモデルの設計
*   **HRフレームワーク**: F4 CHINTAI型5項目企画
*   **成果**: 「AIを使ったサービス企画書とプロトタイプ」

### Module 5: データ分析 + マネタイズ + 卒業課題発表 (Month 9-12)
*   **内容**: 市場分析 + 収益化の仕組み + 最終プレゼン
*   **HRフレームワーク**: F5 産業課題×AI4項目
*   **成果**: 「卒業課題の発表＋EEMUS連携企業へのプレゼン」

## 4. EEMUSとのシナジー：最強の組み合わせ

**EEMUS（静岡の高校生×地元企業マッチング）** と **AIpaX school** を連動させます。

1.  **プレスクールとしての位置づけ**:
    *   EEMUSのインターン参加前に、AIpaXで「AIリテラシー」と「プレゼン力」を身につける。
    *   「AIを使える高校生」は、企業からの評価が大幅にアップする。
2.  **卒業課題のリアルな場**:
    *   AIpaXの卒業課題を、EEMUSの企業課題に紐付ける。
    *   優秀な生徒は、インターンや就職に直結する「実績」として企業に提出可能。
3.  **マーケティングの効率化**:
    *   EEMUSのネットワークを通じて、信頼性の高い生徒募集が可能。
    *   企業側も「AI教育を受けた生徒」を採用したいというニーズに応えられる。
