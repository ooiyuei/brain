## パターンA：採用ペイン直球（建設業向け）

**件名**：採用サイト、3年放置になってませんか？

```
{社名} {担当者}さま

突然のメールすみません、大井湧瑛と言います。
高校生で会社やってる、ちょっと変わったやつです。

ぶっちゃけ、御社のHPの採用ページ見て、「もったいない」って思って書いてます。
工事の現場写真もあるし、社員の声も載せられる素材ある会社なのに、テンプレ通りの求人ボックスで終わってる。

うちは「採用サイト + AIで応募を増やす仕組み」を月15万でやってます。
モニター3社限定で、普通の制作会社が100万取るやつを15万で作って、運用も含めます。

普通のWeb制作と違うのは、AIが応募者の質問に24h答える設定までやること。
「未経験OKって書いてあるけど本当に大丈夫？」みたいな、現場の人が答えるの面倒な質問を全部AIに任せられます。

返信なくても気にしないでください。
気が向いたら、月15万のモニター枠が残ってるか聞いてみてください。

大井湧瑛
{連絡先}
```

**狙い**：高校生＝舐められるリスクを「ちょっと変わったやつ」で先回り回収。価格・差別化を1メールで言い切る。

## パターンB：質問型（製造業向け）

**件名**：採用にいくらかけてますか？

```
{社名} {担当者}さま

大井です、初めまして。
ひとつ聞きたくてメールしました。

御社、ハローワークと求人サイトで、月いくらかかってますか？
体感、月10〜20万くらいの会社が多いんですが、応募の質ってどうですか？

「数は来るけど、面接に来ない」「来ても続かない」
このパターン、製造業だと8割の会社が言います（うちで聞いた範囲）。

原因は、応募する前の段階で「ここで働くイメージ」が湧かないから。
求人サイトはどこも同じテンプレで、応募者は条件しか見てない。

うちは、AIで応募者の質問に24h答える採用サイトを月15万で作ります。
求人サイト代を半分にして、その分を採用サイトに回すと、応募の質が上がる、っていう構造です。

興味あれば、御社の今の採用フローだけ聞かせてください。
売り込みは2通目以降で考えます。

大井
```

**狙い**：質問で考えさせて、相手の頭の中に「採用フロー」を再現させる。売り込み感を2通目以降に分散。

## パターンC：数字なし・原体験（運輸業向け）

**件名**：dinii時代に学んだこと

```
{社名} {担当者}さま

大井湧瑛と申します。
高校3年で、自分の会社を立ち上げて2年目です。

前職、飲食店向けのSaaSでカスタマーサクセスをやってました。
そこで気づいたのが、中小企業の経営者がほんとに困ってるのは「ツールの使い方」じゃなくて、「自分の業界を分かってない外注先に説明する時間」だってことです。

御社の業界（運輸）も、同じだと思ってます。
「うちの仕事は荷物運ぶだけじゃない」って何度説明しても伝わらない外注先に、また同じ説明する。これに時間取られるのが一番きつい。

うちは、業界特化の採用サイトを月15万で作ってます。
「うちの仕事を分かってる人が作る」前提で、最初の打ち合わせで現場見せてもらって、社員2人くらいにヒアリングします。

成果物は、求人サイト3つ分の効果を1つの自社採用サイトで出すこと。
3ヶ月で測定します。

もし話聞いてもいいかな、と思ったら返信ください。
無料で1時間、御社の採用課題だけ聞きます。提案は次の回で。

大井
```

**狙い**：原体験（dinii）で信頼を作る。数字を持ち出さない代わりに「測る」と約束する。

## パターンD：他社事例（サービス業向け）

**件名**：介護業界の採用、変えました（途中経過）

```
{社名} {担当者}さま

突然失礼します、大井です。

最近、某介護施設の採用サイトを作り直してます。
今2ヶ月目で、応募者数は2倍、面接来訪率も7割→9割に上がりました。
推測ですが、求人サイトの「○○手当あり」だけじゃなく、現場のリアル（夜勤シフトの実態・キャリアパス）を載せたのが効いてます。

御社（サービス業）も、同じ構造の課題があるかなと思ってメールしました。
「条件で選ばれる」と、合わない人が来る。「中身で選ばれる」ようにすると、定着する。

うちのやり方：
1. 経営者と現場社員2人にヒアリング（90分）
2. AIで応募者の質問に答える採用ページを作る
3. 求人サイトと連動させて、応募の流れを設計する

月15万、3社限定モニター枠です。
気になったら、現状の応募人数だけ教えてください。比較データを作って返します。

大井
```

**狙い**：他社事例＝同業の安心感。数字に「推測ですが」を付けて誠実さを担保。

## パターンE：時間制限なし・知的好奇心（経営者向け）

**件名**：採用の話、3年後どうしたいですか？

```
{社名} {担当者}さま

大井です。
今日は売り込みじゃなく、聞きたいことがあって書いてます。

御社、3年後の採用、どうしたいと思ってますか？
たぶん、今の体制では未来は描けない、って感じてる経営者は多いです。

求人サイトに月10万、紹介会社に1人入るたびに50万、それでも辞める。
このループから抜けるには、「自社の人を集める仕組み」を持つしかない。

僕がやってるAIpa Webは、そのための装置です。
採用サイト + AIで応募者対応 + 運用までで月15万。
ただ作って終わりじゃなく、毎月数値を見て改善します。

提案するつもりはありません。
ただ、「3年後の採用」というテーマで30分話せる人を探してます。

もし御社で考えてることがあれば、返信ください。
何もなくても、お時間取らせました。

大井湧瑛
```

**狙い**：売らない姿勢で、決定権者（経営者）の知的好奇心を取りに行く。
