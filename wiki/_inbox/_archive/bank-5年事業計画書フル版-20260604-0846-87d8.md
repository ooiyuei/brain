# IPAK-ダーツ 5年事業計画書

## 結論
IPAK-ダーツは「1人でダーツバーに行けない人」200万人をターゲットに、月額4,980円でサブスク型ダーツ施設を展開する。2026年6月オープン予定で、12ヶ月目標は300会員・月売上186万円。5年後には全国100店舗展開を目指す。

## 事業概要

| 事業名 | IPAK-ダーツ（サブスク型ダーツ施設） |
|--------|--------------------------------------|
| ターゲット | 1人でダーツバーに行けない人（推測: 200万人） |
| 月額料金 | 4,980円（スタンダード） |
| 初期投資 | 170万円 |
| 営業利益 | 70万円超（12ヶ月目標） |
| 5年後目標 | 全国100店舗展開 |

## 市場分析

### ダーツ市場規模
- 全国ダーツ人口: 693万人（2025年厚労省調査）
- 定期的にプレイする層: 100万人
- 潜在層: 593万人（推測: 1人で行けない・酒苦手・若年層）

### 競合比較

| 競合 | 月額/1回 | サブスク | お酒なし | QR入店 |
|------|---------|---------|---------|--------|
| **IPAK** | **4,980円/月** | ✓ | ✓ | ✓ |
| ダーツバー | 2,000〜3,000円/回 | ✗ | ✗ | ✗ |
| ゲームセンター | 500〜1,000円/回 | ✗ | ✓ | ✗ |
| chocoZAP | 3,278円/月 | ✓ | ✓ | ✓（ダーツなし） |

## 解決する課題

| ペイン | 内容 |
|--------|------|
| お酒あり | ダーツバーは飲酒必須→未成年不可・敷居高い |
| 対人ハードル | 常連グループの雰囲気で1人では入りにくい |
| 高コスト | 都度払い1回2,000〜3,000円 |
| 環境 | ゲームセンターは騒音・狭い・マシンが古い |

## ソリューション

| | Before | After |
|--|--------|--------|
| 心理的ハードル | バーの雰囲気 | QR入店・人と話す必要なし |
| コスト | 1回2,000〜3,000円 | 月4,980円で通い放題 |
| 年齢制限 | 20歳以上 | 全年齢OK |
| 練習環境 | ゲーセン | 専用施設・照明・マシン完備 |

## 収益モデル

| プラン | 月額 |
|--------|------|
| 学割 | 1,480円 |
| スタンダード | 4,980円 |
| プレミアム（マイダーツ保管+優先レーン） | 7,980円 |
| ビジター | 500円/時 |

- **ARPU**: 6,200円
- **LTV**: 4,980円 × 8ヶ月 = 39,840円
- **CAC**: 5,000円（初月50%OFF）
- **LTV/CAC**: 8.0x

## 月次固定費

| 項目 | 月額 |
|------|------|
| 家賃 | 150,000円 |
| 光熱費 | 30,000円 |
| マシンリース（DARTSLIVE） | 50,000円 |
| アルバイト | 20,000円 |
| 保険+雑費 | 15,000円 |
| **合計** | **265,000円** |

## オペレーション

自動化率80%（入退店・課金は完全自動）

```python
会員登録（アプリ） → QR入店 → 自由プレイ → QR退店 → 月額自動課金
↓
月次イベント（DARTSLIVE検定/大会）→ コミュニティ形成 → 継続
```

| 業務 | 担当 | 頻度 |
|------|------|------|
| 施設巡回+清掃 | アルバイト | 週2回 |
| マシンメンテ | DARTSLIVE保守 | 月1回 |
| イベント企画 | 大井+相川 | 月1回 |
