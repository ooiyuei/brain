# 中小企業の採用サイト「自社制作」が失敗する3つの理由と、AIpa Webが解決する「7日公開」の正体

「自社でWebサイトを作りたい」。
そう思っても、実際に手を動かすのは容易ではない。特に中小企業の採用担当者は、本業の合間に「求人票の作成」「応募サイトの構築」「SNS投稿」をこなさなければならない。

結果として、多くの企業が以下のパターンに陥る。

1. **WixやSTUDIOでテンプレをいじるが、コピーが書けず放置**
2. **Web制作会社に依頼すると、見積もり50万円〜、納期1ヶ月〜**
3. **求人媒体（Airワークやengage）の無料ページだけで妥協し、応募が来ない**

「AIpa Web」は、この「自社制作のハードル」と「制作会社のコスト・期間」の狭間にある課題を、**「フォーム入力→AI初稿→最短7日公開」**というオペレーションで解決する。

## なぜ「自社制作」は難しいのか？

中小企業（従業員30〜300名規模）が採用サイトを作る際にあるのは、技術不足ではなく**「情報整理の手間」**と**「文章作成の負荷」**だ。

### 1. 構成の迷い
「会社紹介」「仕事内容」「福利厚生」「応募資格」……何をどの順番で載せれば、応募意欲が高まるか。
Webの専門知識がないと、ただの「会社案内パンフレット」のWeb版になりがちだ。

### 2. コピーライティングの壁
「当社について」の文章を、自社の視点で書いてしまうと、応募者には「自分事」として響かない。
「応募者が何を求めているか」を逆算した文章構成が必要だが、これをゼロから考えるのは時間がかかる。

### 3. 更新の手間
公開して終わりではない。募集職種の変更、新卒・中途の切り替え、イベント告知……
一度作ったサイトを手動で更新するのは、ITリテラシーが低い担当者にとって大きな負担になる。

## AIpa Webの解決策：AIは「代わり」ではなく「裏方の担当者」

AIpa Webは、単に「AIでサイトを作る」サービスではない。
**「社外Web担当者」を月額制で提供する**サービスだ。

### フローの比較

| 項目 | 一般的なWeb制作会社 | AIpa Web |
|------|-------------------|----------|
| **ヒアリング** | 対面・Zoomでの長時間ミーティング | Googleフォーム（16セクション） |
| **構成・文章** | 担当者がゼロから作成 | AIがフォーム回答から自動生成 |
| **デザイン** | 一からデザイン | GPTimage2による方向性提案＋実装 |
| **修正** | 何回もやりとり（追加料金あり） | 2回まで無料（それ以上はオプション） |
| **公開まで** | 2週間〜1ヶ月 | **最短7日** |
| **月額運用** | 別契約（10万円〜） | **標準に含まれる** |

### 具体的なプロセス

1. **フォーム入力**
   担当者はGoogleフォームに、自社の情報（会社概要、仕事内容、給与、福利厚生など）を入力する。
   「未定」や「相談したい」も選択可能。迷ったらAIが提案する。

2. **AIによる初稿生成**
   フォームの回答が、AI（Claude）によってサイト構成・コピー・デザイン案に変換される。
   ここで重要なのは、**「採用特化」のテンプレート**が使われている点だ。
   建設業、製造業、サービス業など、業種別のベストプラクティスが適用される。

3. **人間による事実確認**
   AIが生成した文章や構成を、大井（または担当者）が事実確認する。
   給与、待遇、会社概要の正確性は、AIではなく人間が担保する。

4. **公開と月額運用**
   Vercelへのデプロイ後、公開。
   以降は月額プランに応じて、募集要項の変更、写真の差し替え、アクセスレポートの提供などを行う。

## 誰が「明日」使うのか？

このサービスが最も必要としているのは、以下の人物だ。

> **地方の中小企業、採用担当者（1名体制）**
>
> *   本業（営業や現場管理）の合間に採用対応をしている
> *   Web制作会社に50万円も出せない
> *   Wixなどで作ろうとしたが、文章が書けず挫折した経験がある
> *   「求人票」は作れるが、「採用サイト」の運用は面倒

彼らが求めているのは、**「完璧なサイト」ではなく、「応募者が集まる受け皿の最低ライン」**だ。
AIpa Webは、その「最低ライン」を、最短7日、初期費用15万円（モニター価格）で提供することで、彼らの負担を劇的に減らす。

## 競合との違い：なぜ「AIpa Web」なのか？

### STUDIOやWixとの違い
STUDIOやWixは「ツール」だ。ユーザーが自分で考え、自分で組み立てる必要がある。
AIpa Webは「サービス」だ。ユーザーは「何を載せたいか」を伝えるだけで、AIが裏側で構成・文章・デザインを処理し、人間が確認する。
**「AIで速いだけ」ではなく、「運用まで含めた代行」**が差別化の核だ。

### 制作会社との違い
制作会社は「品質」は高いが、コストと期間がかかる。
AIpa Webは「標準化されたフォーム」と「AIの活用」により、コストを1/3以下、期間を1/4以下に圧縮する。
品質は「採用特化のテンプレート」と「人間の最終確認」で担保する。
