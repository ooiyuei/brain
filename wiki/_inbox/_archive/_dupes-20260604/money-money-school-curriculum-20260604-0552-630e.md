# AIpaX school 1年カリキュラム

## 5モジュール構造

### モジュール1: AIマインドセット + バイブ体験 (Month 1-2)
**学習目標**: AIを手足のように使えるスキルを体験的に学ぶ。  
**週次トピック**:
1. AIとは何か？AIの社会への影響
2. AIツールの基礎（ChatGPT / Claude / Canva AI）
3. プロンプト設計の基本
4. バイブコーディング体験（Replit / Cursor）
5. AIと倫理の関係
6. AIで何ができるか？実例紹介
7. AIと人間の協働
8. AIマインドセットのまとめ

**課題と発表**:  
- AIツールで簡単なプレゼンを作成  
- AIと人間の協働の例を1つ提示  

**必要ツール**:  
- Antigravity / Cursor / Claude / Figma / Replit  

**評価方法**:  
- プレゼンの完成度  
- プロンプト設計の適切性  
- AIと人間の協働の理解度  

---

### モジュール2: 日常業務自動化 (Month 3-4)
**学習目標**: 日常業務をAIで自動化するスキルを習得。  
**週次トピック**:
1. 日常業務の自動化とは？
2. AIで自動化できる業務の種類
3. データ整理とAIの活用
4. チームで合意形成（F2）
5. 自動化の実践例
6. チームで課題を設定
7. 自動化の実装
8. 自動化の成果発表

**課題と発表**:  
- 自分の日常業務をAIで自動化する  
- チームで合意形成し、自動化の実装  

**必要ツール**:  
- Antigry / Cursor / Claude / Notion / Google Sheets  

**評価方法**:  
- 自動化の実装の完成度  
- チームでの合意形成の質  
- 自動化の成果の提示  

---

**週次トピック**:
1. エンジニアリング思考とは？
2. プロジェクトの構築方法
3. 外部との連携の重要性
4. チームでのアイデア共有
5. 外部との連携の実践
6. チームでの課題設定
7. 外部との連携の実装
8. 外部との連携の成果発表

**課題と発表**:  
- 自分のプロジェクトを構築し、外部と連携  
- チームでのアイデア共有と実装  

**必要ツール**:  
- Antigravity / Cursor / Claude / Figma / Notion  

**評価方法**:  
- プロジェクトの構築の完成度  
- 外部との連携の実装の質  
- チームでのアイデア共有の成果

8. アプリの成果発表

**評価方法**:  
- アプリの完成度  
- チームでの企画立案の質  
- アプリのリリースの成果
