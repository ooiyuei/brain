## アプリ名・1行説明  
**学習スケジュールメーカー**  
高校生が1日30分で学習計画を自動作成するアプリ

## ターゲット  
**明日使う1人**  
17歳の公立高校2年生・英語が苦手な男子  
毎日1時間の学習時間があるが、どの教科を優先すべきか迷っている

## 功能リスト v0.1 (MVP・3機能のみ)  
1. **学習時間割作成**  
   - 教科・時間・曜日を入力  
   - 1日30分の学習時間に合わせた計画を自動生成  
   - 未習得単語数を基に優先順位を付ける  

2. **英語単語学習機能**  
   - 1日5単語の学習を自動スケジュール  
   - 単語の意味・例文・画像を表示  
   - テストで正解率を記録  

3. **学習履歴管理**  
   - 今日の学習内容を記録  
   - 未習得単語をリスト化  
   - 進捗をグラフで可視化  

## 画面構成 (3画面)  
1. **ホーム画面**  
   - 今日の学習計画表示  
   - 英語単語学習ボタン  
   - 学習履歴ボタン  

2. **学習計画作成画面**  
   - 教科選択（英語・数学・国語など）  
   - 時間・曜日入力  
   - 生成ボタン  

3. **学習履歴画面**  
   - 今日の学習内容一覧  
   - 未習得単語リスト  
   - 進捗グラフ  

## データモデル  
- **User**  
  - id (UUID)  
  - name (文字列)  
  - created_at (datetime)  
- **StudyPlan**  
  - id (UUID)  
  - user_id (UUID)  
  - subject (文字列)  
  - time (時間)  
  - day_of_week (文字列)  
  - created_at (datetime)  
- **WordStudy**  
  - id (UUID)  
  - user_id (UUID)  
  - word (文字列)  
  - meaning (文字列)  
  - example (文字列)  
  - image_url (文字列)  
  - correct_rate (数値)  
  - created_at (datetime)  

## 技術スタック  
- **Frontend**: React Native + Expo  
- **Backend**: Supabase (PostgreSQL)  
- **認証**: Supabase Auth  
- **画像取得**: Unsplash API  
- **広告**: AdMob  
- **決済**: Stripe  

## 月3万円達成のロジック  
- **DAU (日均利用者数)**: 100人  
- **広告単価**: 10円/クリック  
- **広告クリック率**: 5%  
- **広告収益**: 100人 × 5% × 10円 = 500円/日  
- **月収益**: 500円 × 30日 = 15,000円  
- **課金率**: 1%（月1人）  
- **課金収益**: 1人 × 3,000円 = 3,000円/月  
- **合計収益**: 18,000円（月）
