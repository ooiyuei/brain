## アプリ名・1行説明  
**学習スケジュールメーカー**  
高校生が1日30分で学習計画を自動作成するアプリ

## ターゲット  
**明日使う1人**  
17歳の公立高校2年生・数学が苦手な男子  
偏差値55〜60、受験勉強を始めたばかりの層  
「毎日1時間勉強するけど成果がでない」  
「何から始めたらいいか分からない」

## 功能リスト v0.1 (MVP・3機能のみ)  
1. **学習科目選択**  
   - 数学・英語・国語・理科・社会の5科目から選択  
   - 1科目を「主な勉強科目」として指定  

2. **学習時間割作成**  
   - 1日30分の学習時間を指定  
   - 1週間のスケジュールを自動作成  
   - 毎日同じ時間帯に学習を推奨  

3. **学習内容提案**  
   - 毎日の学習内容を自動提案（例: 数学の「二次関数」）  
   - 毎日1問の問題を出題（無料版は10問まで）  
   - 解説付きで正解・不正解を判定  

## 画面構成 (3画面)  
1. **ホーム画面**  
   - 学習科目選択画面  
   - 1週間スケジュール表示（日付別）  
   - 今日の学習内容表示  

2. **学習画面**  
   - 問題表示（例: 数学の二次関数の問題）  
   - 回答入力欄  
   - 解説表示（正解・不正解判定）  

3. **統計画面**  
   - 1週間の学習時間合計  
   - 正解率表示  
   - おすすめの学習科目表示  

## データモデル  
- **ユーザー**  
  - ID (UUID)  
  - 名前 (匿名可)  
  - 学年 (1〜3年)  
  - 偏差値 (推測: 55〜60)  
  - 学習科目 (5科目中1科目)  

- **学習履歴**  
  - ユーザーID  
  - 日付  
  - 学習時間 (分)  
  - 学習内容 (科目名)  
  - 正解数 / 総問数  

- **問題データ**  
  - ID (UUID)  
  - 科目名  
  - 問題文  
  - 選択肢  
  - 正解肢  
  - 解説  

## 技術スタック  
- **Frontend**: React Native + Expo  
- **Backend**: Supabase (PostgreSQL)  
- **広告**: AdMob (Rewarded Ads)  
- **課金**: Stripe (課金率1%)  
- **UI**: Figmaでデザイン、Expoで実装  

## 月3万円達成のロジック  
- **DAU x 広告単価**  
  - DAU (日間アクティブユーザー): 100人  
  - 広告単価: 300円  
  - 月収: 100人 × 300円 × 30日 = 90,000円  
  - 課金率1%: 90,000円 × 1% = 900円（月収合計: 9,900円）  
- **課金ロジック**:  
  - 有料版は「100問分の問題」を購入（1,000円）  
  - 月1回の課金を推奨（1人あたり1,000円 × 30人 = 30,000円）
