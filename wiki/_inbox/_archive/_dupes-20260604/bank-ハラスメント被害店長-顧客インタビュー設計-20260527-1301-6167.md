# ハラスメント被害店長 顧客インタビュー設計

## 1. 目的と前提

**目的**: `[[entities/kasunote]]` の価格改定（980→2,980円）と「2026年10月義務化」訴求が、**実際にハラスメント被害を受けた店長**に刺さるかを検証し、成約までの摩擦を削ぐ。

**前提**:
- 対象は「被害者」だが、購買意欲は「将来のリスク回避（法的証拠）」と「現在のストレス軽減（記録の手間）」にある。
- 単なる感想聴取ではなく、「明日から使うか」の判断材料を得る。
- 大井の「脳汁駆動」を抑制し、**証拠と数字**で語る。

## 2. ターゲット選定基準（明日使う1人）

以下の条件を**全て満たす**人物を1名確保。

| 基準 | 詳細 | 理由 |
|------|------|------|
| **実被害者** | 過去に客や社員からハラスメント（暴言・無視・不当解雇脅し等）を受けた経験がある | 課題の切実さが異なる。他人事ではない層のみが動く。 |
| **現場責任者** | 店長・副店長・責任者格（従業員30名以下の小規模店舗） | 労務対応の最終責任者であり、かつ意思決定権がある。 |
| **ITリテラシー** | スマホ操作に抵抗がない（LINE・SNS日常使い） | MVPのUX障壁を排除。PC必須だと離脱率100%。 |
| **業界** | 飲食・小売・サービス業（B2C） | `[[entities/kasunote]]` のターゲット。製造業は対象外。 |
| **アクセス経路** | 大井の知人・HR高等学院OB・商工会議所・またはXでの直接DM | 信頼性の高い接触経路。 |

**[要確認]**: 現在、この条件に合う人物が何名いるか。0名なら、X広告やコミュニティでの募集を並行して開始。

## 3. インタビュー設計（30分）

**形式**: 1on1（オンラインまたは対面）。録音OK（後でテキスト化）。
**報酬**: 無料体験（3ヶ月）＋ 感謝のコーヒー券（1,000円）。

### フェーズ1: 共感と課題の可視化（10分）

> 「今日はカスハラノートの話ではなく、店長としての『辛い経験』をお聞きしたいんです。強制ではありません。」

1. **過去の実例**
   - 「過去に、客や社員から理不尽な扱い（暴言・不当なクレーム・パワハラ等）を受けたことはありますか？」
   - 「その時、どう対応しましたか？（我慢した・上司に相談した・記録した等）」
   - 「その対応で、満足しましたか？ 後悔や不安は残っていますか？」

2. **現在のリスク**
   - 「もし今、同じようなことが起きたら、どう証明しますか？（メモ・録音・目撃者）」
   - 「法的な証拠として残すのは、現実的に難しいと感じますか？（時間・手間・技術）」

### フェーズ2: ソリューションの提示と反応（10分）

> 「そこで、30秒で記録できて、改ざん不可能な法的証拠になるツールを作りました。使ってみてどう思うか、率直な感想を聞かせてください。」

1. **デモ（5分）**
   - `[[entities/kasunote]]` の画面を見せながら、実際に「架空のクレーム」を30秒で記録する。
   - 「このタイムスタンプと記録は、後から変更できません。裁判でも証拠になります。」

2. **価格提示（5分）**
   - 「月額2,980円です。1日100円以下で、万が一の時に店長を守れます。」
   - 「もし、この価格で『明日から使いたい』と思える理由と、その理由で『使いたくない』理由を教えてください。」

### フェーズ3: 成約への摩擦の特定（10分）

1. **抵抗感の掘り下げ**
   - 「2,980円という価格に対して、何が一番高いと感じますか？（コスト・手間・必要性）」
   - 「もし、1,000円なら使いますか？ 5,000円なら使いますか？ どの価格帯で『コスパが良い』と感じますか？」

2. **行動への誘導**
   - 「もし、明日から使えるとすると、一番最初に記録したいシチュエーションはありますか？」
   - 「今日、無料で3ヶ月体験できます。使ってみますか？（その場で登録）」

## 4. 観察ポイント（大井のメモ用）

| 項目 | 観察点 |
|------|--------|
| **感情の動き** | 話を聞いた時、眉をひそめるか、うなずくか。共感の深さ。 |
| **言語化の質** | 「大変だった」という抽象的な言葉か、「客が電話を切った瞬間、手が震えた」という具体像か。 |
| **価格反応** | 2,980円に対して、即座に「高い」と言うか、「検討する」と言うか。 |
| **行動の勢い** | 「使ってみます」と即答するか、「家族と相談します」と言うか。 |
| **機能の優先度** | 「記録」か「共有」か「証拠」か。どの機能に価値を感じるか。 |

## 5. 次のアクション（明日の朝9時）

1. **候補者リストアップ**: 条件に合う人物を3名リストアップ。
2. **連絡**: 1名にDMまたは電話。
3. **インタビュー**: 1名と30分実施。
4. **記録**: 録音をテキスト化し、`[[entities/kasunote]]` の改善点として反映。
