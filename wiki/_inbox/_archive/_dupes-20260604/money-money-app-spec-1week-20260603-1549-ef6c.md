## アプリ名・1行説明  
**「スケジュールラボ」**  
高校生が毎日の学習計画を立て、実行状況を可視化するアプリ

## ターゲット  
**「明日から使える高校生」**  
特に受験生・進学先を決める時期の生徒。学習計画を立てられない、実行できないという悩みを持つ層

## 功能リスト v0.1 (MVP・3機能のみ)  
1. **学習計画作成**  
   - 日ごとの学習時間・科目・タスクを自由に設定  
   - タスクを「必須」「選択」に分類  
   - 通知設定（開始時間にアラーム）  

2. **実行状況可視化**  
   - 毎日の学習時間・達成率をグラフで表示  
   - 週ごとの進捗をスライド形式で確認  
   - 未達成タスクをピンクで強調  

3. **学習環境の最適化**  
   - 学習時間に合わせたBGM再生（無料曲のみ）  
   - モード切り替え（集中モード/リラックスモード）  
   - モバイルの通知をオフにできる「静かな学習モード」

## 画面構成 (3画面)  
1. **ホーム画面**  
   - 今日の学習計画一覧  
   - 進捗グラフ（日・週）  
   - 通知設定ボタン  

2. **タスク作成画面**  
   - 科目・時間・タスク内容入力  
   - タスクの優先度設定（必須/選択）  
   - 通知時間設定  

3. **進捗確認画面**  
   - 今日の達成率・未達成タスク一覧  
   - 週ごとの進捗スライド  
   - モード切り替えボタン（集中/リラックス）

## データモデル  
| テーブル名 | フィールド | 説明 |
|------------|----------|------|
| users | id, name, email, created_at | ユーザー情報 |
| study_plans | id, user_id, date, plan, completed | 学習計画 |
| tasks | id, study_plan_id, task_name, priority, completed | 個別タスク |
| settings | id, user_id, mode, notification_time | モード・通知設定 |

## 技術スタック  
- **Frontend**: React Native + Expo  
- **Backend**: Supabase（リアルタイムデータベース）  
- **通知**: Firebase Cloud Messaging  
- **音楽**: SoundCloud API（無料曲のみ）  
- **広告**: AdMob（バナー広告）  
- **課金**: Stripe（有料モードの購入）  

## 月3万円達成のロジック  
- **DAU（日均利用者数）**: 100人  
- **広告単価**: 1人あたり10円（AdMobバナー広告）  
- **課金率**: 10%（有料モードの購入）  
- **月収**: 100人 × 10円 = 1,000円（広告） + 100人 × 10% × 3,000円 = 30,000円（課金） = **31,000円**
