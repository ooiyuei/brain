# AIpa Web 静岡県商工会議所経由営業文

---

## 📌 件名  
AIpa Web 〜中小企業の採用サイト制作を「フォームで進むAIオペレーション」に変える〜  
静岡県商工会議所様へ

---

## 📝 本文

静岡県商工会議所 拝啓

いつもお世話になっております。  
株式会社AIpa（以下「AIpa」といいます）は、中小企業向けの採用サイト制作と運用を「AIで進める」サービス「AIpa Web」を展開しております。  
本日は、静岡県商工会議所様の皆様に、AIpa Webのサービスをご提案させていただきます。

---

## 🎯 なぜ「AIpa Web」が必要なのか？

静岡県の中小企業様には、以下のような課題がございます。

- **採用サイトの制作が「打ち合わせだらけの受託」**  
- **Web制作にかかるコスト・時間が膨大**  
- **応募者の質問対応が手間で、採用に直結する課題**  
- **地方の中小企業では、人材不足が深刻**  

AIpa Webは、これらの課題を「**フォームで進むAI制作オペレーション**」で解決します。

---

## 🚀 AIpa Web は「AIで作る」ではなく「AIで進める」オペレーション

**AIpa Webの特長は以下の通りです。**

| 項目 | 内容 |
|------|------|
| **制作方法** | フォーム入力でヒアリング → AIが初稿作成 → 人間が事実確認 → 1週間で公開 |
| **価格** | モニター限定：月15万円 → 通常：月10万円（※月額更新あり） |
| **対応範囲** | 採用LP、応募導線、求人票、FAQ、社長メッセージ、社員インタビューなど |
| **AI機能** | AIが応募者の質問に24時間対応（採用業界初） |
| **運用サポート** | 月額で更新・運用を代行 |

---

## 🧠 なぜ「AIpa Web」が静岡県の中小企業に適しているのか？

- **地方の中小企業の採用課題が深刻**  
  建設・製造・運輸・サービス業など、従業員30〜300名の企業様に特化。  
- **AIが応募者の質問に24時間対応**  
  採用業界初の機能で、応募者の満足度を高め、採用率を向上。  
- **月額で更新・運用を代行**  
  一時的な制作ではなく、継続的な運用をサポート。  
- **地域貢献の観点からも、中小企業の採用支援が重要**  
  静岡県の地域経済を支えるため、AIpa Webは中小企業の採用支援を強力にサポートします。

---

## 📈 今後の展開と静岡県への貢献

AIpa Webは、静岡県の中小企業様に向けた「**採用支援の新たな基盤**」として、以下のような展開を予定しています。

- **2026年6月〜**：静岡県商工会議所様をはじめとする地域の中小企業様への導入開始  
- **2026年12月〜**：月額更新・運用サポートを開始  
- **2027年3月〜**：地域の採用支援イベントやセミナーの開催（静岡県商工会議所様との連携）
