## アプリ名・1行説明  
**「RhythmNote」**  
高校生が毎日の学習リズムを管理し、集中力を維持するためのアプリ

## ターゲット  
**明日使う1人**  
17歳の高校2年生・女子・東京都在住  
偏差値55〜60の学力レベル  
毎日塾に通い、家では自習を頑張っているが、集中力が続かない  
「勉強のリズムが崩れると頭が働かない」と感じている

## 功能リスト v0.1 (MVP・3機能のみ)  
1. **学習スケジュール作成**  
   - 毎日の勉強時間・休憩時間・学習内容を記録  
   - タイマー機能で集中時間管理  
   - 通知でスケジュールを確認  

2. **学習リズム分析**  
   - 1週間の学習時間・集中時間のグラフ表示  
   - 休憩時間の長さを分析し、改善提案  
   - 週ごとの学習効率を比較  

3. **学習モチベーションサポート**  
   - 1日1回の「学習目標」を設定  
   - 完成後に達成感を表示し、モチベーションアップ  
   - 他のユーザーと学習成果を比較（匿名）

## 画面構成 (3画面)  
1. **ホーム画面**  
   - 今日のスケジュール一覧  
   - 今日の学習リズム分析結果  
   - 通知アイコン  

2. **学習スケジュール画面**  
   - 日付選択で過去/今後のスケジュールを確認  
   - タイマーを起動・停止  
   - 学習内容を記録  

3. **学習リズム分析画面**  
   - 1週間のグラフ表示  
   - 休憩時間の分析結果  
   - モチベーションサポート画面へのリンク

## データモデル  
| テーブル名 | フィールド | 説明 |
|------------|----------|------|
| users | id, name, email, created_at | ユーザー情報 |
| study_sessions | id, user_id, start_time, end_time, content, focus_time | 学習セッション情報 |
| analysis_data | id, user_id, week, study_time, rest_time, efficiency | リズム分析データ |
| goals | id, user_id, goal, completed, date | 学習目標情報 |

## 技術スタック  
- **Frontend**: React Native + Expo  
- **Backend**: Supabase（リアルタイムデータベース）  
- **広告**: AdMob（インアプリ広告）  
- **課金**: Stripe（課金機能）  
- **通知**: Firebase Cloud Messaging（FCM）  

## 月3万円達成のロジック  
- **DAU（日均利用者数）**: 100人  
- **広告単価**: 1人あたり10円（AdMob平均）  
- **課金率**: 10%（月1回の課金）  
- **月収**: 100人 × 10円 = 1,000円（広告収入）  
- **課金収入**: 100人 × 10% × 3,000円 = 30,000円  
- **合計収入**: 31,000円（月収達成）
