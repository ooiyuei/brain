## アプリ名・1行説明  
**「RhythmNote」**  
高校生が毎日の学習リズムを管理し、効率的に勉強するためのアプリ

## ターゲット  
**明日使う1人: 高校2年生の女子（A子）**  
A子は毎日10時間以上勉強するが、集中力が続かず、勉強の効率が落ちている。  
「どうやって勉強時間を有効に使えるか？」が悩みの種。

## 功能リスト v0.1 (MVP・3機能のみ)  
1. **学習スケジュール作成**  
   - 1日3回の学習セッションを設定（朝・昼・夜）  
   - 毎回の学習時間と科目を指定  
   - タイマー機能で集中モードを実現  

2. **学習リズム可視化**  
   - 毎日の学習時間と集中度をグラフで可視化  
   - 週ごとの学習リズムを比較  
   - 通知で「今週の学習リズムが落ちている」とアラート  

3. **学習習慣の習慣化サポート**  
   - 毎日学習を続けると、ポイントが貯まり「学習マスター」称号がもらえる  
   - ポイントは広告収益に変換可能（課金は1%）  
   - 連続学習日数が10日以上で、特別な通知を送る  

## 画面構成 (3画面)  
1. **ホーム画面**  
   - 今日の学習スケジュールが表示  
   - 今日の学習リズムグラフ  
   - 通知アイコン（学習リズムが落ちている場合）  

2. **スケジュール作成画面**  
   - 3つの学習セッションを設定  
   - 毎回の学習時間と科目を入力  
   - タイマーを設定  

3. **学習リズム画面**  
   - 週ごとの学習リズムグラフ  
   - 通知が来た場合、アラートが表示  
   - ポイントの残りと称号の表示  

## データモデル  
| データ名 | 説明 | データ型 |
|---------|------|---------|
| ユーザーID | ユーザーの識別子 | UUID |
| 学習スケジュール | 毎日の学習スケジュール | JSON |
| 学習リズム | 毎日の学習リズムデータ | JSON |
| ポイント | 学習習慣化のポイント | Integer |
| 通知履歴 | 通知の履歴 | JSON |
| タイマー履歴 | タイマーの使用履歴 | JSON |

## 技術スタック  
- **Frontend**: React Native + Expo  
- **Backend**: Supabase（リアルタイムデータベース）  
- **広告**: AdMob（通知広告）  
- **課金**: Stripe（1%課金）  
- **通知**: FCM（Firebase Cloud Messaging）  

## 月3万円達成のロジック  
- **DAU（日均利用者数）**: 100人  
- **広告単価**: 1人あたり10円（AdMobの平均）  
- **課金率**: 1%（月100人 × 10円 = 1,000円）  
- **合計収益**: 10,000円 + 1,000円 = 11,000円（月）  
- **達成条件**: DAUを100人以上に維持し、広告を効果的に表示
