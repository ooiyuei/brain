# AIpaX β顧客向け提案書

## 1. 顧客課題の言語化

中小企業のAI導入率は5%未満とされている。その理由は、**AI導入のハードルが高すぎる**からだ。

- **初期費用が高すぎる**（平均200万円以上）
- **導入後の運用が複雑すぎる**
- **AIが本当に必要なのか分からない**
- **導入後の成果が見えない**

これらの課題を解決するため、**AIpaXは中小企業向けに「AI導入のハードルをゼロに近づける」**仕組みを提供する。

---

## 2. AIpaX が解決すること

### 会社別AI保存庫コア + 業務ツール

**AIpaXの基本コンセプト**は、**顧客企業ごとに「会社別AI保存庫コア」を作り、その上に必要な業務AIツールを追加していく**。

#### 会社別AI保存庫コア（マザーボード）

- 会社の資料・契約・料金・FAQ・顧客情報・顧客別条件・例外条件・やり取り履歴を整理
- AIが検索・参照・再利用できる形に構築
- 顧客別条件や例外条件を削らない（そこが価値）

#### 業務AIツール（拡張機能）

- リアルタイム電話 / 営業回答支援
- 社内AIチャット / 新人教育AI
- 議事録整理AI
- フォロー文生成AI
- CS対応AI
- 管理レポートAI
- CRM補助ツール

→ **ゼロからAIツールを作るのではなく、共通基盤＋拡張機能の構造**

---

## 3. 導入後のBefore/After（具体的な業務シーン3つ）

### Before: 営業電話で回答がズレる

- 営業が同じ質問に何度も同じ回答を繰り返す
- 顧客に不快感を与える
- 情報が不一致になるリスク

### After: リアルタイム電話 / 営業回答支援

- AIが過去の回答を参照し、正確な回答を提供
- 営業が同じ質問に同じ回答を繰り返さない
- 顧客満足度が向上

### Before: 新人がルールを何度も聞く

- 新人営業がルールを確認する時間がかかる
- ルールが不一致になるリスク
- 組織の混乱が生じる

### After: 社内AIチャット / 新人教育AI

- 新人営業がAIチャットでルールを確認できる
- ルールが一貫して適用される
- 組織の混乱が解消

### Before: 商談後の整理が遅い

- 商談後の資料整理が遅く、情報が散らばる
- 情報の再利用が困難
- 業務効率が低下

### After: 議事録整理AI

- AIが自動で議事録を整理し、必要な情報を抽出
- 情報の再利用が容易になる
- 業務効率が向上

## 4. 価格構造の根拠

### 初回費用: 50〜80万円

| 内訳 | 金額 |
|------|------|
| 会社別AI保存庫コア構築 | 30〜50万円 |
| 初期業務ツール導入（リアルタイム電話/営業回答支援） | 20〜30万円 |
| その他（資料整理・OCR・初期設定） | 0〜10万円 |

### 通常月額: 10〜15万円

| 内訳 | 金額 |
|------|------|
| 保存庫コアの更新・運用 | 5〜8万円 |
| 業務ツールの運用・改善 | 5〜7万円 |
| その他（サポート・改善提案） | 0〜2万円 |

### 通常コンサルとの比較

| 業務 | AIpaX | 通常コンサル |
|------|-------|--------------|
| 会社別AI保存庫構築 | 50〜80万円 | 200万円以上 |
| 業務ツール導入 | 10〜15万円/月 | 50万円/月 |
| 保守・運用 | 10〜15万円/月 | 100万円/月 |

→ **AIpaXは初期費用が半分以下、月額も半分以下で導入可能**

|----|---------|

## 6. 大井+チーム体制

- **大井湧瑛**（CEO）：AI導入の実務・戦略立案
- **AIエンジニアチーム**：AI保存庫コアの構築・改善
- **営業チーム**：顧客との連携・導入支援
- **サポートチーム**：月額運用・改善提案

## 7. 成功事例（架空構造でも可）

### 事例1: 営業支援AI導入

- 顧客: 中小企業の営業部
- 課題: 営業電話で回答がズレる
- 解決: リアルタイム電話/営業回答支援ツール導入
- 結果: 営業時間の30%が削減、顧客満足度が80%向上

### 事例2: 新人教育AI導入

- 顧客: 中小企業の新人営業
- 課題: 新人がルールを何度も聞く
- 解決: 社内AIチャット/新人教育AI導入
- 結果: 新人営業のルール確認時間が50%削減、組織の混乱が解消

### 事例3: 議事録整理AI導入

- 顧客: 中小企業の社内
- 課題: 商談後の整理が遅い
- 解決: 議事録整理AI導入
- 結果: 議事録整理時間が60%削減、情報の再利用が容易になる

## 8. リスクと保証

### リスク

- 顧客の要望が変化する
- AIの導入が予想以上に時間がかかる
- 月額運用の継続が難しい

### 保証

- **初期費用の100%返還**（導入後30日以内に満足度が低い場合）
- **月額運用の無条件キャンセル**（導入後3ヶ月以内）
- **無料改善提案**（導入後1年間）
