# [[entities/eemus]] の連携候補高校20校リスト（静岡県内優先）

---

## 1. 静岡県立静岡高等学校（静岡市）

- **校名**：静岡県立静岡高等学校  
- **特徴**：進学校 / 高度な学習環境 / 高い進学実績  
- **連携メリット仮説**：進学校の生徒は学力が高い傾向にあるため、企業の高専門性の業務にも適応しやすい。また、進学率が高いが、就職希望者も一定数おり、キャリア教育のニーズが高まっている。  
- **接点**：進路指導主任 / 校長 / PTA  
- **提案案**：  
  - 進学希望者と就職希望者を分けてプログラムを実施  
  - 進学希望者向けには「大学進学と就職の両方を視野に入れたキャリア教育」を提案  
  - 就職希望者向けには「地元企業との連携によるキャリア選択の実践」を提案  
  - 進路指導主任と連携し、進学・就職の選択肢を広げる支援を実施

## 2. 静岡県立富士高等学校（静岡市）

- **校名**：静岡県立富士高等学校  
- **特徴**：実業系 / 工業・商業系の教育 / 地域との連携が強い  
- **連携メリット仮説**：実業系の高校生は、地元企業との連携に適している。また、地域との関わりが深いことから、企業との連携がスムーズに進む可能性が高い。  
- **接点**：校長 / 進路指導 / 地域連携担当  
- **提案案**：  
  - 工業・商業系の生徒を対象に、地元企業との連携を強化  
  - 企業見学やジョブシャドウイングを実施し、実際の職場体験を提供  
  - 地域連携担当と連携し、企業との連携を推進  
  - 進路指導主任と連携し、キャリア教育の支援を実施
