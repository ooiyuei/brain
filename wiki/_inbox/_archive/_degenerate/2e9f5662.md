# EEMUS（エームス）連携候補高校20校リスト（静岡県内優先）

---

## 1. 静岡県立静岡高等学校（静岡市）

- **校名**：静岡県立静岡高等学校  
- **特徴**：進学校 / 高度な学習環境 / 多くの進学実績  
- **連携メリット仮説**：進学校の生徒はキャリア選択に深い関心を持ち、企業との接点が広がる可能性が高い。  
- **接点**：進路指導教諭 / 校長 / PTA  
- **提案案**：  
  - 進路指導教諭と連携し、キャリア教育の一環としてEEMUSのプログラムを導入。  
  - 校長と連携し、学校の教育ビジョンとEEMUSの理念を一致させ、全校的に推進。  
  - PTAとの連携で、保護者層にもEEMUSの価値を周知し、地域社会の協力を得る。  
  - 進学実績が高いことから、企業側も「将来の採用候補」として期待できる。

## 2. 静岡県立御殿場高等学校（御殿場市）

- **校名**：静岡県立御殿場高等学校  
- **特徴**：実業系 / 地域密着 / 地域の産業との連携が強い  
- **連携メリット仮説**：実業系の高校生は、地元企業との連携に強い意欲を持ち、実践的なキャリア教育に興味がある。  
- **接点**：校長 / 産業教育担当教諭 / PTA  
- **提案案**：  
  - 産業教育担当教諭と連携し、地元企業との連携を深める。  
  - 校長と連携し、地域の産業と学校教育の連携を強化。  
  - PTAとの連携で、地域の企業が学校の教育に積極的に関わるよう促す。  
  - 地域密着型の高校であるため、EEMUSのローカル展開に適している。
