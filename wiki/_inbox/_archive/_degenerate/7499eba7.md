## 1. AIpa Web

### 1.1 商品設定  
AIpa Webは、AIを活用したWebアプリケーションの提供サービスです。主に、企業向けのAIチャットボットやカスタマーサポートツールを提供しており、顧客は月額または年額のサブスクリプションで利用できます。  
- **商品名**：AIpa Web Standard / AIpa Web Pro / AIpa Web Enterprise  
- **商品説明**：AIチャットボットの構築・運用、カスタマーサポートの自動化、カスタマイズ機能の有無  
- **商品カテゴリ**：SaaS / AIツール / カスタマーサポート  

### 1.2 価格モデル  
- **定期型**：月額 / 年額（月額は年額の12倍）  
- **単発型**：初期導入費用（1回の支払い）  
- **価格帯**：  
  - Standard：10,000円/月  
  - Pro：20,000円/月  
  - Enterprise：カスタマイズ価格  

### 1.3 支払い方法  
- **Stripeの支払い方法**：  
  - クレジットカード（Visa / MasterCard / JCB / American Express）  
  - クレジットカードの自動更新（サブスクリプション）  
  - クレジットカードのキャンセル・変更機能  
  - クレジットカードのセキュリティ対策（PCI DSS準拠）  

### 1.4 法的考慮  
- **契約書**：サブスクリプション契約書（利用規約、プライバシーポリシー、サービスレベルアグリーメント）  
- **データ保護法**：GDPR / JIS Q 15001 / PIA（個人情報保護法）  
- **税務処理**：消費税の計算、課税対象の明確化  
- **支払いの透明性**：明細の明確化、支払いのタイミングの指定（例：月末支払い）  

### 1.5 導入工数  
- **初期設定**：  
  - 商品設定：2人日  
  - 価格モデル設定：1人日  
  - 支払い方法設定：1人日  
  - 法的考慮の確認：1人日  
- **導入後**：  
  - サポート体制の整備：1人日  
  - テスト環境の構築：1人日  
- **合計**：約6人日
