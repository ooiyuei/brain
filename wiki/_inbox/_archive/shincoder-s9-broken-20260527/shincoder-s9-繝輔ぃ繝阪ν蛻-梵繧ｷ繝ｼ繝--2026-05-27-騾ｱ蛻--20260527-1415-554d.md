# AIpaX school 戦略提言：17歳起業家・大井の「実力証明」としての最適化

提供された情報（AIpaX school, EEMUS, FishOps Agent, 大井のプロフィール等）を統合し、**「月収100万円（2026年6月期限）」**という目標と、**SFC AO志望**という将来の目標を両立させるための具体的なアクションプランを提案します。

現在のAIpaX schoolは「カリキュラムv2完成・第1期生5名募集」という状態ですが、単なるスクール運営にとどまらず、**「大井湧瑛という17歳起業家の実力（思考力・実行力・AI活用能力）を可視化するメディア」**として位置づけることが重要です。

## 1. 核心戦略：AIpaX schoolを「大井の実績」のエンジンにする

### なぜか？
- **SFC AO志望**: 「AIによる思考の外部化」を志望理由にするなら、実際に**「AIを使って教育を再構築し、成果を出した」**という実績が最強の裏付けになります。
- **月収100万円**: スクール単体では月3万円×50名=150万円が目標ですが、集客コストがかかります。しかし、**「大井が教える」プレミアム感**と**EEMUS連携**を使えば、高単価・低コストで集客可能です。
- **競合優位**: QUREOやTech Kidsは「プログラミングの技術」を教えますが、大井は**「17歳で起業し、現場でAIを使いこなすマインドセットと実践」**を教えます。これは既存スクールにはない**「リアルなロールモデル」**としての差別化になります。

### 位置づけの再定義
| 従来のイメージ | 再定義后的イメージ |
|--------------|------------------|
| 子供向けAI講座 | **「未来の起業家育成ラボ」** |
| プログラミング学習 | **「AIを手足のように使うビジネス実践」** |
| 講師は外部委託 | **17歳現役起業家・大井が直接メンタリング** |
| 受講生は生徒 | **大井のビジネスパートナー候補 / EEMUS連携人材** |

## 2. 収益化ロードマップ（月収100万円達成シミュレーション）

### フェーズ1：信頼構築と初期収益（2026年5月〜6月）
**目標**: 月額売上30万円（第1期生20名 / 月額1.5万円相当の価値提供）

- **価格戦略**:
  - **ライトプラン（5,980円）**: 動画教材のみ。認知獲得用。
  - **プレミアムプラン（29,800円/3ヶ月 or 10,000円/月）**: **大井の直接メンタリング + EEMUSインターン紹介権**。
    - *理由*: 親御さんは「子供がAIで稼ぐスキルを身につける」ことに高い価値を感じます。特に「地元企業（EEMUS）への就職/インターン」に直結する点は静岡ローカルで強力です。
- **集客チャネル**:
  1. **EEMUS連携**: 静岡の高校3年生（大学受験で情報科目対策が必要）と高校1-2年生（インターン準備）にアプローチ。「AIリテラシー + キャリア設計」のセットで提案。
  2. **SNS（TikTok/YouTube）**: 「17歳起業家が教える、AIで月100万円稼ぐための思考法」を配信。カリキュラムの断片を公開し、信頼を構築。
  3. **HR高等学院ネットワーク**: 大井の母校・知人を活用した口コミ。

### フェーズ2：スケールと法人化（2026年7月〜9月）
**目標**: 月額売上70万円追加（合計100万円）

- **法人研修パッケージ**:
  - EEMUSでつながった地元企業（建設業、製造業など）向けに、「若手社員向けAI活用研修」を提供。
  - 大井の「現場で学ぶ」スタイルを企業向けにアレンジ。
  - 単価：1社あたり50万円〜100万円（3〜5社契約で目標達成）。
- **スクール拡大**:
  - 静岡県外へのオンライン展開。
  - 「AIpaX認定」のブランド化。

## 3. カリキュラムv2の強化ポイント（SFC AO・起業家としての成長を兼ねて）

カリキュラムv2は既にHR高等学院のノウハウを統合していますが、**「大井の思考プロセス」を可視化**することが重要です。

### Module 1: AIマインドセット + バイブコーディング体験
- **内容**: ChatGPT/Claudeの活用だけでなく、**「なぜそのプロンプトでその答えが出るのか」**を大井の実例（PM Agentでの営業、AIpaXでの開発）で解説。
- **AO連携**: 「AIとの対話を通じて、自分の思考をどう外部化したか」を記録し、志望理由書に反映。

### Module 2: 日常業務自動化 + チーム合意形成
- **内容**: F2合意形成4ステップを実践。チームでプロジェクトを回す体験。
- **EEMUS連携**: 地元企業の実務課題（例：資料作成、データ整理）を題材に、AIで解決策を提案する。

### Module 3: エンジニアリング思考 + 個人探究テーマ
- **内容**: F3ロジックモデルを用いた課題設定。
- **FishOps Agentとの連動**: 水産業界の課題をAIで解決するアイデアを練る。大井の他の事業（FishOps）と連携することで、**「実社会の問題解決」**を実感できる。

### Module 4: 本格アプリ開発 + サービス企画
- **内容**: F4 CHINTAI型企画。Replit/Cursorでの開発。
- **AO連携**: 開発プロセスを記録し、「技術的課題をどう解決したか」をエビデンスとして蓄積。

### Module 5: データ分析 + マネタイズ + 卒業課題発表
- **内容**: F5産業課題×AI分析。卒業課題として、実際にサービス化またはビジネスプラン化。
- **コンテスト連携**: 卒業課題を「高校生みんなの夢AWARD」や「コムロコンサルティング」などのコンテストに応募。受賞実績がAO・起業家としての信用になる。

## 5. リスクと対策

- **リスク**: 大井の時間が不足し、SFC AO対策や他の事業（FishOps等）に影響が出る。
  - **対策**: スクール運営は相川やメンターに委譲。大井は「カリキュラム監修」と「プレミアムプランのメンタリング」に集中。
- **リスク**: 競合（QUREO等）との価格競争。
  - **対策**: 価格競争ではなく、「大井というロールモデル」と「EEMUS連携によるキャリア保証」で差別化。
- **リスク**: カリキュラムの質が保てない。
  - **対策**: HR高等学院のノウハウをベースにしているため、ある程度の質は担保されている。ただし、大井のリアルな失敗談や成功談を混ぜることで、唯一無二のコンテンツにする。
