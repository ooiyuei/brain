## 📌 タスク概要：紹介プログラム 主力5事業横断ルール

### 1. 紹介者報酬率（例）

| 紹介内容 | 紹介者報酬率 | 備考 |
|----------|--------------|------|
| 新規契約（1事業） | 10%（契約金額の10%） | 1事業ごとに発生 |
| 新規契約（複数事業） | 15%（契約金額の15%） | 5事業すべてを紹介した場合 |
| 再契約（1事業） | 5%（契約金額の5%） | 既存顧客の再契約 |
| 再契約（複数事業） | 10%（契約金額の10%） | 5事業すべてを再契約した場合 |
| 既存顧客の紹介 | 3%（紹介対象の契約金額の3%） | 既存顧客から紹介された場合 |

> **※注意事項**  
> - 紹介報酬は、紹介者が紹介した顧客が契約を完了した時点で発生します。  
> - 紹介報酬は、紹介者が所属する事業部またはチームに振り分けられます。  
> - 紹介報酬は、紹介者本人が直接受け取る場合も、所属チームに振り分けられる場合も可能です。

### 2. 被紹介者特典（例）

| 特典内容 | 内容 | 備考 |
|----------|------|------|
| 紹介割引 | 紹介者を介して契約した場合、10%の割引 | 1事業ごとに適用 |
| 紹介特典 | 紹介者を介して契約した場合、初期費用無料 | 1事業ごとに適用 |
| 紹介ポイント | 紹介者を介して契約した場合、ポイントを付与 | 1ポイント = 100円相当 |
| 紹介特典（複数事業） | 5事業すべてを紹介した場合、特別特典を付与 | 例：年間利用料無料、特別サポートなど |
| 紹入者特典 | 紹介者を介して契約した場合、特典を付与 | 例：無料セミナー参加権、資料プレゼントなど |

> **※注意事項**  
> - 特典は、紹介者が所属する事業部またはチームに連携して提供されます。  
> - 特典の内容は、事業ごとに調整可能。  
> - 特典は、紹介者が紹介した顧客が契約を完了した時点で適用されます。

### 4. 紹介プログラムの運用ルール（例）

| 項目 | 内容 |
|------|------|
| 紹介者登録 | 事業部またはチームに所属する社員が紹介者として登録 |
| 紹介対象 | 事業部またはチームに所属する社員以外の顧客 |
| 紹介報酬の支払い | 紹介者が所属するチームまたは事業部に振り分け |
| 紹介報酬の支払いタイミング | 紹介者が紹介した顧客が契約を完了した時点で支払い |
| 紹介報酬の上限 | 1事業あたりの紹介報酬は、契約金額の10%まで |
| 紹介報酬の下限 | 紹介報酬は、契約金額の3%以上 |
| 紹介報酬の支払い方法 | 紹介者本人への支払い、または所属チームへの振り分け |

---

### 5. 紹介プログラムの運用フロー（例）

1. **紹介者登録**  
   - 事業部またはチームに所属する社員が紹介者として登録

2. **紹介対象の選定**  
   - 紹介者が紹介したい顧客を選定

3. **紹介の実施**  
   - 紹介者が紹介対象を紹介し、契約を成立させる

4. **紹介報酬の発生**  
   - 紹介者が紹介した顧客が契約を完了した時点で紹介報酬が発生

5. **紹介報酬の支払い**  
   - 紹介報酬は、紹介者が所属するチームまたは事業部に振り分け

6. **特典の適用**  
   - 被紹介者に特典を適用

7. **契約書の作成**  
   - 紹介者と被紹介者との契約書を作成

---

### 6. 紹介プログラムの運用マニュアル（要件）

- 紹介者と被紹介者の関係を明確に
- 紹介報酬の発生条件を明確に

- 特典の内容を明確に
- 紹介プログラムの運用フローを明確に
- 紹介プログラムの運用ルールを明確に
- 紹介プログラムの運用マニュアルを定期的に更新
