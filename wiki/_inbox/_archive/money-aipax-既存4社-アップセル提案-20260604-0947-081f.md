# 📄 AIpaX 既存4社 アップセル提案3パターン

以下は、**AIpaX 既存4社向けのアップセル提案3パターン**です。それぞれの提案には、**追加ツールの導入・年契約の提案・紹介報酬の提示**が含まれており、顧客の継続的価値向上とリテンションを促進する戦略を組み込んでいます。

---

## ✅ アップセル提案1: **「保存庫コア + 新しい業務ツール」の追加**

### 📌 概要
既存の「保存庫コア + リアルタイム電話/営業回答支援」に、**新たな業務AIツール**を追加することで、業務の効率化をさらに高めます。

### 📌 推奨ツール
- **社内AIチャット / 新人教育AI**
- **フォロー文生成AI**
- **CS対応AI**

### 📌 価格
- **初期追加価格**: 30〜50万円
- **月額追加**: 3〜5万円

### 📌 プロセス
1. 既存保存庫コアの状態を確認
2. 新しいツールの導入に必要なデータ整理を実施
3. ツールの導入と運用サポートを提供

### 📌 紹介報酬
- 既存顧客が新規顧客を紹介した場合、**紹介報酬として10万円**を支給

---

## ✅ アップセル提案2: **年契約プランへの移行**

### 📌 概要
現在の**月額契約**から、**年契約プラン**に移行することで、**コスト削減**と**継続的なサポート**を提供します。

### 📌 年契約プランの特徴
| 項目 | 内容 |
|------|------|
| **初期費用** | 100〜150万円（月額の12ヶ月分） |
| **月額費用** | 10〜15万円（月額のまま） |
| **特典** | 1年間の無料サポート / ツールの更新 / データのバックアップ |
| **メリット** | 月額の固定費を1年間で節約 / リスク軽減 / リテンション向上 |

### 📌 紹介報酬
- 既存顧客が新規顧客を紹介した場合、**紹介報酬として15万円**を支給

---

## ✅ アップセル提案3: **「保存庫コア + 多機能AIツールパッケージ」の導入**

### 📌 概要
**保存庫コア**に、**複数のAIツールを一括導入**することで、**業務の全領域をカバー**するパッケージプランを提案します。

### 📌 推奨ツールパッケージ
- リアルタイム電話/営業回答支援
- 社内AIチャット / 新人教育AI
- 議事録整理AI
- CS対応AI
- フォロー文生成AI

### 📌 価格
- **初期費用**: 150〜200万円
- **月額費用**: 15〜20万円
- **特典**: 1年間の無料サポート / ツールの更新 / データのバックアップ

### 📌 紹介報酬
- 既存顧客が新規顧客を紹介した場合、**紹介報酬として20万円**を支給

---

## 📌 紹介報酬の実施要件

| 紹介報酬 | 条件 |
|----------|------|
| 10万円 | 新規顧客が既存顧客の紹介で契約 |
| 15万円 | 新規顧客が既存顧客の紹介で年契約プランに移行 |
| 20万円 | 新規顧客が既存顧客の紹介でパッケージプランに移行 |
