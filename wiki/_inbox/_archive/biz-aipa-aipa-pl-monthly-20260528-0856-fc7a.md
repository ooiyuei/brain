# AIpa Web 月次PL試算書（月売上45万円想定）

## 1. 前提条件とビジネスモデルの再定義

本試算は、[[entities/aipa-web]] の「モニター契約3社」による初期売上月45万円（単価15万円×3社）をベースとした月次損益計算書（PL）である。

**重要な前提の整理：**
文書内の「Deep Research KPI試算」では月額顧客25社で売上315万円とするシナリオが提示されているが、本質問は「月15万x3社=月45万売上」に特化している。
ここで注意すべきは、AIpa Webの商品構造が**「初期制作費（一次性）」**と**「月額運用費（継続性）」**の2層構造を持っている点である。

*   **シナリオA（純粋な新規受注3社分のみ）：** 当月のキャッシュフローは初期費用のみに依存。
*   **シナリオB（既存顧客の継続＋新規3社）：** 通常、SaaS/代行業務では「既存顧客の月額運用費」が固定費をカバーする基盤となる。

本試算では、**「月間新規受注3社（初期費用45万円）＋ 既存顧客からの月額運用収入（仮に0円、または極小）」** という、最も保守的かつ「新規開拓フェーズ」の純粋なPLとして構成する。もし既存顧客が1社もいない状態でのPLであれば、この数字は「赤字確定」に近いが、これは「投資段階」であることを示唆する。

しかし、文脈として「月次PL」を問われている以上、**「月間継続的な売上構造」**として捉えるのが一般的である。
そこで、以下の2つの解釈を提示する。

1.  **解釈1：月間新規受注3社分の「初期制作費」のみを売上とする場合**
    *   これは「当月の現金流入」であり、PL上の「売上高」にはなるが、原価（工数）も当月に発生する。
2.  **解釈2：月間継続的な「月額運用費」の売上構造を問うている場合**
    *   この場合、月45万円の売上を得るためには、月額1万円の顧客が45社必要、または月額5万円の顧客が9社必要となる。

**結論：** 文脈「月15万x3社=月45万売上」は、**「初期制作費（一次性売上）」**を指していると解釈するのが自然である。したがって、本PLは**「月間新規受注3社による初期制作費収入と、それに伴う直接原価（人件費・ツール費）」**を軸に構成する。

## 3. 分析と改善ポイント

### 3.1. 現状の問題点
*   **売上原価率の高さ（65.3%）：** 人件費が売上に対して65%を占めている。これは「AIpa Web」の核心价值である「AIによる効率化」が、PL上では「人件費として計上」されているため。
*   **営業利益率21%：** 一般的なWeb制作業界（30〜40%）と比較すると低め。しかし、**「初期費用15万円」という低価格戦略**を維持するなら、この利益率は許容範囲かもしれない。
*   **既存顧客の不在：** 月額運用費（純利益率90%以上）が0円のため、固定費（61,500円）をカバーする余力が薄い。

### 3.2. 改善シナリオ：月額運用費の追加効果

もし、既存顧客が**10社**いて、平均月額運用費が**15,000円**の場合：

| 項目 | 金額（円） | 備考 |
|------|------------|------|
| **初期制作費売上** | **450,000** | 新規3社 |
| **月額運用費売上** | **150,000** | 10社 × 15,000円 |
| **合計売上高** | **600,000** | |
| **売上原価（初期分）** | **(294,000)** | 新規3社分 |
| **売上原価（運用分）** | **(15,000)** | 10社 × 1,500円（AI自動更新＋月1回確認） |
| **決済手数料** | **(12,000)** | 60万円の2% |
| **合計売上原価** | **(321,000)** | |
| **総利益** | **279,000** | 46.5% |
| **販管費** | **(61,500)** | 固定費は変わらない |
| **営業利益** | **217,500** | **36.3%** |

> **結論：** 月額運用費（継続売上）が加わることで、**営業利益率が36.3%まで向上**し、ビジネスとして成立する。

### 3.3. 戦略的提言

1.  **月額運用費の獲得を最優先：**
    *   初期制作費15万円は「獲客コスト（CAC）」と捉え、**月額運用費の獲得**を主目的とする。
    *   最初の3社は「モニター」として低価格で提供し、**月額運用契約**を強く推奨する。
2.  **人件費のAI自動化：**
    *   「フォーム入力→サイト初稿生成」を完全自動化し、人間の工数を「校正・確認」のみに集中させる。
    *   1社あたりの人件費を**90,000円→30,000円**に削減できれば、営業利益率は**50%以上**に跳ね上がる。
3.  **IT導入補助金の活用：**
    *   顧客に「IT導入補助金」を申請させることで、実質的な顧客負担額を半減させ、成約率を向上させる。
    *   これにより、**「月額運用費」の継続率**が向上する。
