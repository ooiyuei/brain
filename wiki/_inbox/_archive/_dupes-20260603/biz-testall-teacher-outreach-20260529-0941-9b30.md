# Testall 教育者連携プログラム：塾講師・予備校・高校教師向け連携案

Testallは、受験生向けのAI学習マネージャーとして、受験戦略を自動化するアプリケーションです。このプロジェクトは、高校1〜3年生の独学受験生（塾なし層70万人）をターゲットに、テスト結果をもとに弱点を特定し、45分のタスクに自動変換する仕組みを提供しています。Testallは、Next.js + Supabase + Claude AIをスタックとしており、JUDGE評価は75点（Tier A）とされています。現在はMVP実装済みで、教育者連携準備中です。

## 教育者連携プログラムの概要

Testallは、教育者（塾講師・予備校・高校教師）との連携を強化し、学習効果を高めるためのプログラムを設計しています。この連携プログラムは、教育者にメリットを提供し、Testallの利用を促進するための戦略を含みます。以下に、連携メリット、初期5名募集案、連携契約書草案、インセンティブ設計を説明します。

## 連携メリット

Testallの教育者連携プログラムには、以下のメリットがあります：

1. **学習効果の向上**：Testallは、受験生のテスト結果をもとに弱点を特定し、45分のタスクに自動変換する仕組みを提供します。これにより、受験生の学習効果が向上し、教育者も指導の質を高めることができます。
2. **指導の効率化**：Testallは、受験生の学習進捗を自動的に追跡し、指導計画をサポートします。これにより、教育者の指導効率が向上し、より多くの生徒に効果的な指導が可能になります。
3. **生徒のモチベーション向上**：Testallは、受験生の学習進捗を可視化し、達成感を高める仕組みを提供します。これにより、生徒の学習意欲が向上し、教育者の指導効果が高まります。
4. **教育者との連携強化**：Testallは、教育者との連携を強化し、学習データを共有する仕組みを提供します。これにより、教育者も生徒の学習状況を把握し、より効果的な指導が可能になります。

## 初期5名募集案

Testallは、教育者との連携を強化するため、初期5名の教育者を募集します。募集条件は以下の通りです：

1. **教育者経験**：塾講師・予備校・高校教師の経験があること。
2. **指導実績**：受験生の指導経験があること。
3. **学習支援の意欲**：Testallの学習支援を活用し、生徒の学習効果を高めたい意欲があること。
4. **連携意欲**：Testallとの連携を積極的に行いたい意欲があること。

募集は、Testallの公式サイトやSNSを通じて行います。応募者は、Testallのサービス内容や連携プログラムについて理解した上で、応募書類を提出します。

## 連携契約書草案

Testallと教育者の連携契約書は、以下の内容を含みます：

1. **連携目的**：Testallと教育者の連携を通じて、受験生の学習効果を高めること。
2. **連携内容**：Testallの学習支援を活用し、生徒の学習進捗を把握し、指導計画をサポートすること。
3. **連携条件**：Testallのサービス利用条件に従い、教育者が生徒の学習状況を把握し、指導計画をサポートすること。
4. **連携期間**：Testallと教育者の連携は、Testallのサービス利用期間内に限り有効です。
5. **連携責任**：Testallと教育者は、連携を通じて生徒の学習効果を高める責任を負います。

## インセンティブ設計

Testallは、教育者との連携を促進するため、インセンティブを設計しています。インセンティブは、以下の通りです：

1. **報酬制度**：Testallは、教育者に報酬を支払います。報酬は、生徒の学習進捗に応じて支払われます。
2. **指導料の割引**：Testallは、教育者に指導料の割引を提供します。これにより、教育者がTestallのサービスを活用しやすくなります。
3. **指導資料の提供**：Testallは、教育者に指導資料を提供します。これにより、教育者が生徒の学習状況を把握しやすくなります。
4. **指導の支援**：Testallは、教育者に指導の支援を提供します。これにより、教育者が生徒の学習効果を高めやすくなります。

Testallは、教育者との連携を通じて、受験生の学習効果を高め、教育者の指導効果を向上させるためのプログラムを設計しています。このプログラムは、Testallのサービス利用者に新たな価値を提供し、教育者との連携を促進します。
