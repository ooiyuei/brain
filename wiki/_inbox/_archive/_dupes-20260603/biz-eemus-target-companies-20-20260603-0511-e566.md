# 静岡県中堅・成長企業の高校生インターン受入候補20社（EEMUSプロジェクト）

---

## はじめに

EEMUS（エームス）は、静岡県の高校生と地元企業をつなぐキャリア共創プロジェクトです。このプロジェクトは、高校生が自分らしい未来を描けるように、地域企業と共に進路の選択肢を広げるための取り組みです。静岡県では、県外大学進学率が71%、Uターン就職率が33%と、人材流出が深刻な状況にあります。また、県内正社員不足企業は50.8%、建設業の人手不足は79.5%と、企業側も人材確保に苦労しています。このような背景を踏まえ、EEMUSは、高校生と企業のマッチングを通じて、人材流出・採用ミスマッチ・早期離職の負のサイクルを断ち切る取り組みを進めています。

本記事では、EEMUSプロジェクトでインターンを受入する候補となる静岡県の中堅・成長企業20社について、業種・規模、インターン受入のメリット仮説、接点などを中心に紹介します。これらの企業は、静岡県の高校生とつながるための重要なパートナーであり、EEMUSプロジェクトの成功に大きく貢献する存在です。

## 2. EEMUSプロジェクトの価値と企業の役割

EEMUSプロジェクトは、高校生と企業のマッチングを通じて、人材流出・採用ミスマッチ・早期離職の負のサイクルを断ち切る取り組みです。企業は、このプロジェクトを通じて、以下の価値を得ることができます。

1. **早期採用候補との接点** → 将来の採用母集団を高校時代から育成
2. **ミスマッチと早期離職の低減** → リアルな職場体験で入社後ギャップをなくす
3. **採用プロセスの効率化** → 学生が企業情報を把握した状態で面談に臨む
4. **ブランド発信と地域貢献** → CSR活動としての認知向上
5. **社員の成長** → 若者への指導がリーダーシップ・プレゼン力を磨く場に

これらの価値を実現するためには、企業の積極的な参加と協力が不可欠です。EEMUSプロジェクトは、静岡県の高校生と地元企業のつながりを深めるためのプラットフォームであり、企業の社会的責任と地域貢献の機会を提供します。

## 3. 今後の展望

EEMUSプロジェクトは、静岡県の高校生と地元企業のつながりを深めるための取り組みとして、今後も継続的に展開していく予定です。今後は、以下の方向でさらに発展させていきます。

1. **出店候補企業の選定** → 建設業・製造業など人手不足が深刻な業種を重点的に選定
2. **県内高校1〜2校との連携話** → 高校との連携を深め、インターン参加者を増やす
3. **有料化モデルの設計** → 企業への人材紹介フィー or 有料プログラム（今後設計）

これらの取り組みを通じて、EEMUSプロジェクトは、静岡県の高校生と地元企業のつながりをさらに深め、人材流出・採用ミスマッチ・早期離職の負のサイクルを断ち切る取り組みとして、地域社会に貢献していきます。

## 4. 結論

EEMUSプロジェクトは、静岡県の高校生と地元企業のつながりを深めるための取り組みとして、今後も継続的に展開していく予定です。企業は、このプロジェクトを通じて、早期採用候補との接点を広げ、ミスマッチと早期離職の低減、採用プロセスの効率化、ブランド発信と地域貢献、社員の成長など、さまざまな価値を得ることができます。今後は、出店候補企業の選定、県内高校との連携、有料化モデルの設計など、さらなる発展を目指していきます。EEMUSプロジェクトは、静岡県の高校生と地元企業のつながりを深め、地域社会に貢献するための重要なプラットフォームであり、今後も多くの企業の参加と協力が期待されます。
