主力5事業を横断する「紹介プログラム（リファラルプログラム）」の設計案を提案します。

このプログラムは、**「紹介者のインセンティブ最大化」「被紹介者のハードル低下」「法的リスクの最小化」**の3点を軸に設計されています。

---

## 1. 概要：横断型紹介プログラムの基本方針

*   **対象事業:** 主力5事業（例：SaaS販売、コンサルティング、クラウドインフラ、サポートサービス、教育研修 など※実際の事業内容に合わせて調整）
*   **基本ルール:** どの事業で契約が成立しても、**「紹介者」には共通の報酬体系を適用**し、**「被紹介者」には事業固有の特典＋共通特典**を提供する。
*   **目的:** 顧客基盤の相互展開（クロスセル）の促進、新規顧客獲得コストの削減、既存顧客のロイヤルティ向上。

---

## 2. 紹介者報酬率（インセンティブ設計）

紹介者は、紹介した契約の**「初年度契約金額（または月額契約額の12ヶ月分）」**に対して報酬を得ます。

| 紹介者の区分 | 報酬率 | 備考 |
| :--- | :--- | :--- |
| **一般顧客・パートナー** | **契約金額の 10%** | 標準的な報酬率。現金またはポイント付与。 |
| **VIP顧客・戦略的パートナー** | **契約金額の 15%** | 長期的な関係性がある場合。または月間紹介数5件以上で適用。 |
| **内部社員（営業以外）** | **契約金額の 5%** | 社内報奨金制度として扱います。現金は不可の場合が多いので、商品券や休暇券など。 |

### 報酬支払条件
*   **支払時期:** 被紹介者の契約が**「初回請求から3ヶ月経過し、かつ解約されていない場合」**に支払います。（早期解約防止）
*   **支払方法:**
    *   個人: 銀行振込 または 指定されたポイントサイトへの付与
    *   法人: 請求書発行による現金払い
*   **上限:** 1人あたり年間の紹介報酬上限を設ける場合（例：100万円/年）。

---

## 3. 被紹介者特典（新規顧客優遇）

被紹介者は、紹介者からの紹介コードまたはリンクを使用して契約することで、以下の**「共通特典」と「事業固有特典」**の両方を受け取れます。

### A. 共通特典（全5事業共通）
*   **初月無料 または 10%割引:** 最初の契約期間（1ヶ月または1年）に対して適用。
*   **専用サポート窓口:** 紹介経由で契約した顧客は、優先的なサポート対応を受ける権利。
*   **紹介者との共有特典:** 紹介者がポイントを獲得した分と同額のポイントが被紹介者にも付与される（Win-Win構造）。

### B. 事業固有特典（例）
| 事業 | 被紹介者特典例 |
| :--- | :--- |
| **事業1: SaaS製品** | 基本プラン1ヶ月無料 + 導入サポート1回無料 |
| **事業2: コンサルティング** | 初回ヒアリング無料 + 提案書作成料50%オフ |
| **事業3: クラウドインフラ** | 初期設定料無料 + 3ヶ月間の無料トライアル期間延長 |
| **事業4: サポートサービス** | 緊急対応チケット3回分無料 |
| **事業5: 教育研修** | 受講料10%割引 + 教材デジタル版無料 |

## 4. 契約書テンプレート（紹介プログラム利用規約）

以下の内容は、紹介プログラムに参加する**紹介者**と**被紹介者**の双方が同意する必要がある「紹介プログラム利用規約」の骨子です。

---

### **紹介プログラム利用規約**

**第1条（目的）**
本規約は、[会社名]（以下「当社」という）が提供する主力5事業（以下「対象事業」という）の紹介プログラム（以下「本プログラム」という）の利用条件を定めるものです。

**第2条（紹介者の定義）**
本プログラムに参加し、被紹介者を対象事業に紹介する個人または法人を「紹介者」といいます。紹介者は、当社が定める登録手続きを完了している必要があります。

**第3条（報酬の算定基準）**
1. 紹介者の報酬は、被紹介者が対象事業と契約を締結し、かつ初回請求から3ヶ月間継続して契約が有効である場合に算定されます。
2. 報酬率は、紹介者の区分および対象事業に応じて、別紙「報酬率表」に定める割合とします。
3. 報酬は、現金、ポイント、または商品券のいずれかで支払います。支払いは被紹介者の契約が安定した後に、月次で実施します。

**第4条（被紹介者特典）**
被紹介者は、紹介者の紹介コードまたは専用リンクを使用して契約することで、本プログラムで定める特典（初月無料、割引など）を受けることができます。特典の適用は、紹介者が本プログラムに有効に参加している場合に限り効力を生じます。

**第5条（禁止行為）**
紹介者は、以下の行為を行ってはなりません。
1. 虚偽の情報や不正な手段による契約の誘導。
2. 自らのアカウントや関連するアカウントを用いた自己紹介。
3. 報酬の転売、譲渡、または第三者への委託。
4. 当社の商標、ロゴ、または本プログラムの内容の無断利用。

**第6条（規約の変更）**
当社は、必要に応じて本規約および報酬率を変更できるものとします。変更後は、当社ウェブサイトまたはメールにて通知します。

**第7条（準拠法及び管轄裁判所）**
本規約は日本法に準拠し、本規約に関する紛争は、[会社所在地]を管轄する裁判所を第一審の専属的合意管轄裁判所とします。

## 5. 実装・運用上の注意点

1. **追跡システムの構築**
   *   紹介コード、専用URL、Cookieによる追跡を徹底してください。
   *   どの事業で契約が成立したかを自動認識できるCRM（Salesforce, HubSpotなど）との連携が必要です。

2. **重複紹介の防止**
   *   複数の紹介者が同じ顧客を推薦した場合のルールを明確にします（例：「最初に登録された紹介者」または「契約直前に接触した紹介者」）。

3. **税務処理**
   *   紹介者への報酬は「雑所得」として扱われる場合が多く、源泉徴収が必要な場合があります。税理士と相談し、適切な報告書（源泉徴収票など）の発行体制を整えてください。

4. **透明性の確保**
   *   紹介ダッシュボードを用意し、紹介数、契約状況、報酬累計額をリアルタイムで確認できるようにしてください。

---

この設計案は、貴社の主力5事業の具体的な内容（B2B/B2C、単価、契約期間など）に応じて、報酬率や特典の内容を調整する必要があります。必要であれば、各事業の詳細に基づいてさらに具体的な数値設定をご提案できます。
