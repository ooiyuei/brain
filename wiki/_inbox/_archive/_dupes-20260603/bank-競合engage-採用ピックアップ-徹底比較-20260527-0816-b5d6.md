# AIpa Web 競合engage/採用ピックアップ 徹底比較

## 結論：明日使う1人

**「静岡県内の建設業・製造業で、採用担当が1人で回っている社長・人事課長」**

**理由**:
1. **engage（エン・ジャパン）** は「無料」を謳うが、**「応募者の受け皿（LP/サイト）」は作れない**。求人票のアップロード先がない。
2. **AirWORK（リクルート）** も同様。媒体への掲載は強いが、自社ブランドの採用サイト構築機能は弱い（または別料金/高機能版）。
3. **Wantedly** は「文化・ミスマッチ防止」には強いが、**「即戦力の現場職（建設/製造/物流）」には弱い**。
4. **AIpa Web** の価値は「媒体に出す前に、応募者が迷わない自社サイトと応募フォームを7日で作り、運用まで代行する」こと。

**明日のアクション**:
静岡の建設業10社リスト（OpenClaw生成済み）に、**「engageで求人を出しているが、自社サイトがない/古い」企業**を絞り、以下メールを送信。

```text
件名: 【ご提案】engage掲載前の「応募者受け皿」を7日で構築しませんか？

大井湧瑛です。
貴社のengage掲載、拝見しました。

現在、engageやAirWORKで求人を出されていても、
「応募者が詳細を確認できる自社サイト」が
お持ちでない、または古くなっている企業様が多いと伺っています。

AIpa Webでは、
「フォーム入力→AIが構成・文章・デザイン→7日公開→月額運用」
までを一括代行しています。

媒体に出す前に、応募者の迷いを減らす受け皿を整えるのが
最短の採用対策だと考えています。

もしよろしければ、サンプルサイトをご覧いただければ幸いです。
```

## 競合比較表：engage vs AIpa Web

| 比較軸 | **engage（エン・ジャパン）** | **AIpa Web（大井）** |
|------|--------------------------|------------------|
| **本質** | **求人媒体（広告枠）** | **採用サイト制作＋運用代行** |
| **価格** | 基本無料（掲載枠は有料） | 初期15万円（モニター）/ 月額2.98〜3.98万円 |
| **納期** | 即日〜数日（媒体側審査あり） | **7日**（自社サイトとして公開） |
| **応募者体験** | 媒体内のリストから遷移。ブランド感薄い。 | **自社ブランドのLP**。信頼感・伝わるコピー。 |
| **運用** | 媒体側が管理。自社更新は媒体側UI。 | **月額で更新代行**。募集要項変更もAIpa側が対応。 |

| **差別化** | 掲載数・認知度 | **「媒体に出す前の受け皿」の構築力** |

## なぜengageが「脅威」ではなく「味方」なのか

engageは**「集客の手段」**であり、**「集客先のコンテンツ」ではない**。

1. **engageで求人を出す** → 応募者が来る
2. **応募者が自社サイト（AIpa Web制作）を見る** → 信頼して応募
3. **AIpa Webのフォームで応募** → 応募データが一元化

**engageは「広告媒体」として使い、AIpa Webは「自社サイト」として使う**。
両立可能。むしろ、**「媒体に出す前に自社サイトがない」企業ほど、AIpa Webの価値が高い**。

## AIpa Webの勝ち筋：3つのポイント

### 1. 「媒体に出す前」の空白を狙う
多くの中小企業は「engageやAirWORKに載せれば応募が来る」と思っている。
しかし、**応募者がクリックして飛んだ先に、古いHPか、何も無いページがあれば、応募は離脱する**。
**「媒体に出す前に、応募者の受け皿を整える」**という文脈で営業する。

### 2. 地方・現場職に特化
Wantedlyやリクナビは「ホワイトカラー・新卒」が主戦場。
**建設・製造・物流・介護**などの現場職は、**「信頼できる会社かどうか」が重要**。
自社ブランドのサイトがあり、社長メッセージや社員インタビューがあれば、応募率は跳ね上がる。

### 3. 月額運用で「継続」させる
Web制作会社は「作って終わり」。
AIpa Webは**「月額で更新代行」**。
募集要項の変更、新卒/中途の切り替え、採用イベントの告知まで、**社長が手間をかけずにできる**。

- **静岡県建設業**は、採用に困っている。媒体に出す前に受け皿が必要。
