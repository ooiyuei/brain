# AIpaX 既存4社向けアップセル提案書面

## 共通の前提（営業トークの核）
*   **現状の価値確認**: 「既に構築された『会社別AI保存庫コア』は、貴社の知的財産です。この土台があるからこそ、新しいツールは短期間で稼働します。」
*   **アップセルの目的**: 単なる機能追加ではなく、「業務の自動化範囲の拡大」または「コストの最適化」を提供する。

---

## パターン1：追加ツール導入提案
**ターゲット**: 保存庫コアの活用が進み、別の業務課題（例：CS対応、新人教育、管理レポート）に気づいている顧客
**目的**: 月額単価の引き上げ（LTV向上）

### 提案書面アウトライン

#### 1. 現状の活用状況と次のステップ
*   **現状報告**: 現在構築中の「保存庫コア」に蓄積されたデータ（契約条件、FAQ、顧客情報）の活用状況。
*   **ボトルネックの可視化**: 「営業電話の回答支援」は解決しましたが、「商談後のフォロー漏れ」や「新人の質問対応」はまだ人手に依存していませんか？

#### 2. 提案：保存庫コアを活用した「第2の業務AIツール」
保存庫コアは「脳」です。脳に蓄えた知識を別の「手足」で動かすツールを追加提案します。

| 推奨ツール | 解決する課題 | 期待される効果 | 初期費用 | 月額費用 |
| :--- | :--- | :--- | :--- | :--- |
| **社内AIチャット / 新人教育AI** | 新人の質問対応、社内ルール確認 | 先輩の負担軽減、属人化防止 | 20〜40万円 | +2〜3万円 |
| **CS対応AI / 問い合わせ一次返信** | 問い合わせの属人化、対応遅延 | 24時間対応、即応性向上 | 30〜50万円 | +3〜5万円 |
| **管理レポートAI / 営業支援** | 管理者の現場確認、進捗把握 | 可視化による意思決定の高速化 | 20〜40万円 | +2〜3万円 |

#### 3. 追加導入のメリット（既存顧客特典）
*   **データ連携のスムーズさ**: 既存の保存庫コアと連携するため、新規構築時のような「資料整理」の工数が不要。
*   **初期費用の割引**: 既存コアとのセットアップ費用として、通常価格の**30%OFF**（または初期費用無料〜20万円）で提供。
*   **月額価格の段階値**: 2つ目のツール導入時点で、月額ベース料金を**10%OFF**（例：5万円→4.5万円/ツール）適用。

#### 4. 次のアクション
*   希望するツールのデモ（またはシミュレーション結果）のご提示
*   対象業務の選定とスケジュール調整

## パターン2：年契約提案
**ターゲット**: 月額利用に慣れ、継続的な価値を実感している顧客
**目的**: 解約リスクの排除、現金フローの安定、顧客ロイヤルティの向上

### 提案書面アウトライン

#### 1. 継続利用の安定性とコスト最適化
*   **月額契約の課題**: 毎月の請求確認、予算申請の手間、解約のハードル。
*   **年契約の提案**: 「12ヶ月間、価格変動なし・サポート体制固定」で運用を安定させます。

#### 2. 年契約プランの詳細
*   **割引率**: 月額料金の**2ヶ月分無料**（実質16.7%OFF）。
    *   例：月額5万円 → 年額50万円（通常60万円）
*   **固定価格保証**: 契約期間中、AIモデルの価格改定やインフラコスト上昇があっても、月額料金は据え置き。
*   **優先サポート**: 年契約顧客には、緊急対応や仕様変更時の優先対応枠を確保。

#### 3. 年契約締結のメリット
*   **予算計上の容易さ**: 年初に一括予算を確保できるため、毎月の経理処理が簡素化。
*   **継続ボーナス**: 年契約満了時に、次年度の更新価格をさらに**5%OFF**または**無料ツール1つ追加**の特典。

#### 4. 比較表：月額 vs 年契約

| 項目 | 月額契約 | **年契約（推奨）** |
| :--- | :--- | :--- |
| **初期費用** | なし | なし |
| **月額実質コスト** | 50,000円 | **41,667円** |
| **年間総額** | 600,000円 | **500,000円** |
| **解約手数料** | 30日以上前に通知 | なし（ただし早期解約は残額発生） |
| **価格保証** | なし | **あり** |
| **サポート優先度** | 標準 | **優先** |

#### 5. 次のアクション
*   年契約契約書のご提示
*   請求先口座・振込日設定

## パターン3：紹介報酬制度（リファラル）提案
**ターゲット**: AIpaXの成果に満足し、業界内での信頼関係がある顧客
**目的**: 低コストでの新規顧客獲得（5社目以降の通常価格顧客の確保）

### 提案書面アウトライン

#### 1. 成功事例の共有と感謝
*   **現状の成果**: 「貴社で導入したAIpaXが、〇〇の業務効率化に貢献していること、心より感謝申し上げます。」
*   **業界内の類似課題**: 「貴社と同じ業種・規模感の企業でも、同様のAI導入で効果を上げている事例があります。」

#### 2. 紹介報酬制度のご案内
貴社の成功体験を、同じ課題を持つ企業とつなぐことで、双方に利益をもたらす制度です。

*   **対象者**: 貴社の取引先、業界の知人、同業他社（競合ではない場合）など。
*   **報酬内容**:
    *   **成功報酬**: 紹介先が契約（初期費用支払い）した場合、**初期費用の10%** または **月額料金の3ヶ月分** を現金またはサービスクレジットとして還元。
    *   **例**: 紹介先が初期100万円・月額10万円の契約を結んだ場合、貴社には**10万円**の報酬（または月額10万円分3ヶ月無料クレジット）を支給。
*   **特典**: 紹介先にも**「紹介元特典」として初期費用5%OFF**を提供（Win-Win）。

#### 3. 紹介の進め方（お手伝いします）
*   **資料提供**: AIpaXの概要パンフレット、貴社の導入事例（匿名可）を提供。
*   **デモ同行**: 必要に応じて、AIpaX担当者が紹介先との初回ヒアリングやデモに同行。
*   **アフターフォロー**: 契約後のオンボーディングはAIpaXが担当するため、貴社の負担はありません。

#### 4. 制度の概要表

| 項目 | 内容 |
| :--- | :--- |
| **対象** | 既存顧客が紹介する新規企業 |
| **報酬条件** | 紹介先が契約成立時 |
| **報酬額** | 初期費用の10% **または** 月額料金の3ヶ月分（選択可） |
| **支払時期** | 紹介先の初期費用入金確認後、翌月末日 |
| **注意点** | 競合他社への紹介は対象外。紹介元の秘密保持義務あり |

#### 5. 次のアクション
*   紹介候補リストの共有（任意）
*   紹介承諾書（簡易版）へのサイン
*   紹介パンフレットの送付

## 営業担当者への指示（内部メモ）

1.  **順序の推奨**:
    *   まず**パターン1（追加ツール）**で月額単価を上げる。
    *   同時に**パターン2（年契約）**を提案し、解約リスクを低くする。
    *   関係性が良好であれば、**パターン3（紹介）**を最後に提案し、新規開拓のパートナーにする。
2.  **注意点**:
    *   既存4社は「初回事例価格」のため、通常価格とのギャップを感じさせる前に、**「保存庫コアの価値（資産化）」**を強調すること。
    *   追加ツール提案時は、必ず「既存の保存庫コアと連携できるため、工数が半分以下で済む」という点を強調し、導入ハードルを下げる。
3.  **資料作成**:
    *   各パターンごとに1ページ程度のPDF資料を作成し、メールまたは対面商談で提示する。
    *   紹介報酬制度については、口頭で親密な関係を築いた後に提案する。
