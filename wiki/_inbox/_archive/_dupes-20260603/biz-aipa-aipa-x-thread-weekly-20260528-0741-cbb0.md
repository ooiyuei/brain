# AIpa Web 事業戦略・実行計画（2026年5月27日時点）

---

## 🔍 概要

AIpa Web は「**AIを活用した採用サイト制作**」をコンセプトに、**顧客のフォーム入力**をもとに、**サイト構成・文章・デザインの初稿**を効率的に生成し、**最短7日で公開**するサービスです。

このサービスは「**地方中小企業**」を主なターゲットに、**採用サイトの整備**を通じて**求人広告の効果を高める**サポートを提供します。

---

## 🎯 ターゲット

- **地方中小企業**（静岡、浜松、沼津など）
- **採用に困っている企業**
- **業種**：建設・製造・物流・介護・飲食・美容・小売・地方サービス業
- **規模**：30〜300名規模

---

## 🧠 AIpa Web フロー

```plaintext
1. フォーム入力（顧客）
2. フォーム回答→AI用プロンプトに変換
3. Claudeで構成・コピー・文章作成
4. GPTimage2でデザイン方向性
5. Claudeで実装（Tailwind/Next.js/Astro/shadcn/ui/Vercel）
6. 人間が確認（事実関係/給与/権利/スマホ表示/応募導線/SEO）
7. 顧客確認（修正1回まで）
8. 公開（ドメイン/SSL/GA/応募通知）
9. 月額運用
```

## 📝 リスクと対策

| リスク | 対策 |
|--------|------|
| **安売りしすぎる** | 最低価格決定・修正回数制限・オプション明確化 |
| **修正地獄** | 修正1回・大幅変更は追加料金・初稿前に方向性確認 |
| **AI生成の品質事故** | 給与/待遇/条件は必ず顧客確認・AI生成文は人間確認 |
| **写真・ロゴの権利問題** | フォームで使用許可確認 |
| **競合に真似される** | 採用特化×業種別テンプレ×フォーム設計×月額運用×地方営業力 |
