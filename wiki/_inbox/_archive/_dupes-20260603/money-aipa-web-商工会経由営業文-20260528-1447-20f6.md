静岡県商工会議所

平素より、県内中小企業の経営支援にご尽力いただき、厚く御礼申し上げます。

このたび、県内商工会議所様経由で限定公開いたします、中小企業向け「AI採用サイト構築サービス」につきまして、会員企業様への特別モニター募集のご案内を差し上げました。

本サービスは、従来のWeb制作会社による高額・長期の受託制作を刷新し、**「フォーム入力とAIの活用」により、最短7日で採用サイトを作成・公開し、その後の更新運用まで一括で代行する**画期的なソリューションです。

### 1. サービス概要：AIpa Web（アイパ・ウェブ）

中小企業（従業員30〜300名規模）が直面する「人手不足」および「採用サイトの質の低下」という課題に対し、AIを活用して解決します。

*   **従来課題**: 制作費100〜150万円、納期1〜3ヶ月、頻繁な打ち合わせが必要
*   **AIpa Webの解決**: **制作費15万円（モニター価格）、納期7日間、フォーム入力のみで完了**

具体的には、顧客がフォームに回答するだけで、AIがサイト構成・コピーライティング・デザイン初稿を自動生成し、人間が事実確認を行った上で公開します。公開後も、募集要項の変更や新着情報の更新などを月額制で代行するため、**「社外のWeb担当者」**として機能します。

### 2. 静岡県商工会議所経由 特別モニタープラン

県内の産業構造（建設、製造、運輸、サービス業など）に特化し、地域貢献の一環として以下の特別価格で提供いたします。

| 項目 | 内容 |
| :--- | :--- |
| **サービス名** | AIpa Web（採用特化型AIサイト制作・運用代行） |
| **対象企業** | 静岡県内、従業員30名以上の中小企業（建設・製造・物流・サービス業など） |
| **モニター価格** | **初期費用：150,000円**（通常価格：1,000,000〜1,500,000円相当） |
| **月額プラン** | **100,000円/月**（サイト保守、募集要項更新、応募者対応AI設定含む） |
| **期間** | 3ヶ月間（事例掲載の条件付き） |
| **特典** | ・AIによる24時間応募者対応設定<br>・SEO最適化済みLPの公開<br>・IT導入補助金申請サポート |

### 3. 選定基準と狙い

本モニターは、単なる安価なWeb制作ではなく、**「AIを活用した新しい採用の標準」**を県内企業と共に作り上げることを目的としています。

*   **地域貢献**: 県内の人手不足解消に貢献するため、通常価格の10分の1以下の価格で提供いたします。
*   **事例創出**: 成功事例を県内各地で創出し、商工会議所経由で他の会員企業様へも展開していく予定です。
