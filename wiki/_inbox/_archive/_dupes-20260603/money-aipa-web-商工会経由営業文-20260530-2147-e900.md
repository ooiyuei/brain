現在、静岡県内でも建設、製造、物流、サービス業を中心に、現業職を中心とした人材不足が経営の足かせとなっています。従来の求人媒体への掲載だけでなく、応募者が企業を「知って」「応募したい」と思わせる「採用サイト（受け皿）」の整備が急務ですが、従来は制作コストが高く、納期も長期化しがちという課題がありました。

このたび、私たちはその課題を**「AIpa Web（アイパ・ウェブ）」**という独自のプロセスで解決するサービスを開発いたしました。
この度、地域経済の活性化と会員企業の採用支援に貢献するため、商工会議所様経由で、以下の通り**モニター企業（15社限定）**を募集しております。

#### 1. 本サービスの概要
**「AIpa Web」は、Web制作の面倒を「フォームとAIで消す」サービスです。**
打ち合わせだらけの受託制作ではなく、簡易なフォーム入力とAIの活用により、**最短7日間で高品質な採用サイト（LP）を公開**し、その後の更新まで月額で代行します。

*   **特徴1：圧倒的なスピードと低コスト**
    通常30〜50万円かかる採用サイト制作費を、モニター期間中は**初期費用15万円**（通常価格の約3〜5割OFF）で提供いたします。
*   **特徴2：AIによる24時間対応の採用サポート**
    公開後も、AIが応募者の質問に24時間対応し、採用担当者の負担を軽減します。
*   **特徴3：商工会議所連携による地域貢献**
    本プログラムは、単なるWeb制作ではなく、**「会員企業の採用安定化＝地域経済の安定」**を目指す取り組みです。

#### 2. モニタープラン内容
商工会議所会員企業様限定の特別プランとなります。

| 項目 | 内容 |
| :--- | :--- |
| **対象企業** | 静岡県内、従業員30〜300名の中小企業 |
| **初期費用** | **150,000円**（税込）<br>※通常価格 300,000〜500,000円相当 |
| **月額費用** | **100,000円**（税込）<br>※サイト更新・AI対応・アクセスレポート込み |
| **納期** | フォーム入力後、最短7日間での公開 |
| **条件** | 事例掲載の許可（匿名可） |

#### 3. 商工会議所様への貢献ポイント
本プログラムを通じて、以下の点で商工会議所の皆様のお役に立てると考えております。

1.  **会員企業への付加価値提供**: 会員企業に対して、最新のAI技術を活用した具体的な採用支援ツールを提供できます。
2.  **地域人材確保の基盤整備**: 地方中小企業の採用課題をテクノロジーで解決し、地域産業の持続可能性に貢献します。
3.  **IT導入支援の橋渡し**: 補助金（IT導入補助金等）の活用も含め、会員企業のデジタル化を促進します。

### 営業戦略上のポイント（大井へのメモ）

1.  **「15万→10万移行」の明確化**:
    *   初期費用15万円は「ハードルを下げる」ためのフック。
    *   月額10万円は「AI対応＋更新代行」の正当な価格設定。
    *   商工会議所に対しては、「通常価格の半額以下で会員に提供できる」という**割引率**を強調し、商工会議所への恩恵をアピールする。

2.  **「地域貢献」の文脈**:
    *   商工会議所は「地域の産業振興」が使命。単なる「Web制作会社」ではなく、「地域企業の採用課題を解決するパートナー」として振る舞う。
    *   「AIpa Web」は手段であり、目的は「会員企業の安定」であることを忘れない。

3.  **次のアクション**:
    *   商工会議所理事から「興味がある」という返答があれば、**「会員企業向け説明会」**の開催を提案する。
    *   その際、実際にAIpa Webで作成した架空の静岡建設会社のLPを見せ、品質の高さを示す。

4.  **リスク管理**:
    *   商工会議所経由で多数の問い合わせが殺到した場合、対応しきれないため、**「先着15社」**という制限を設け、希少性を演出する。
    *   契約時には「事例掲載の許可」を必ず取り、後の営業素材とする。
