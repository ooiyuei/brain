# IPAK-darts 什器コスト試算

## 結論：初期投資170万円の内訳と「明日使う1人」

[[entities/ipak-darts]] の事業計画書にある「初期170万円」は、**DARTSLIVE公式マシンのリース料込み**で算出された現実的な数字。
ただ、この170万が「床面積20坪・3レーン・完全自動化」の最低ラインであることを理解していないと、物件選びで死にます。

**明日使う1人**: 物件契約の「間取り図」を持って、DARTSLIVEの営業（または代理店）に「この配置で3台置けるか」を確認する人。
大井自身か、相川空輝。

## 1. 什器・設備コストの正体（170万円の内訳）

IPAK-dartsの核心は「ゲーセンの安物」ではなく「ダーツバーの高級感×無人の安さ」の中間。
内訳は以下の通り。

| 項目 | 金額（概算） | 備考 |
|--|--|--|
| **DARTSLIVEマシン（リース）** | 50,000円/月 | 3台分。初期費用は別途或いは0円（契約による） |
| **マシン設置・配線工事** | 300,000円 | 床配線・LAN工事。これが地味に高い |
| **什器（テーブル・椅子・棚）** | 200,000円 | 中古でも十分。Amazonやリサイクルショップ |
| **内装（看板・照明・QR設置）** | 300,000円 | 黒塗りの高級感を出すための照明が命 |
| **QR入退店システム** | 100,000円 | アプリ開発或いはSaaS利用料（初期） |
| **保証金・礼金・仲介手数料** | 500,000円 | 物件家賃15万の場合。これが最大リスク |
| **その他（備品・清掃機材）** | 50,000円 | トイレ清掃機材など |
| **合計** | **1,700,000円** | **※家賃は月次固定費として別途** |

### 重要ポイント
- **マシンの初期費用**: DARTSLIVE公式はリース契約が基本。初期費用0円の場合が多いが、保証金が必要。
- **配線工事**: 床に穴を開けてLANケーブルを通す必要がある。物件オーナーの許可が必須。
- **什器**: 初期は中古でOK。ダーツバー廃業時の出品を狙う。

## 2. 月次固定費（BEP37名の壁）

**BEP（損益分岐点）**:
月額4,980円の会員が37名いれば黒字。
→ **37名は少ない。安心するな。**

## 3. リスクと対策

### リスク1: 物件確保
- **対策**: 居抜き物件（前の飲食店など）を探す。配線工事が済んでいる場合、初期投資が50万円下がる。
- **行動**: 不動産屋に「ダーツバー用・床配線可・20坪・月15万円以下」で検索依頼。

### リスク2: chocoZAPの参入
- **対策**: chocoZAPはダーツを主力にしていない。IPAK-dartsは「ダーツ特化」で差別化。
- **行動**: DARTSLIVE検定公認の開催権を取得。コミュニティ形成。

### リスク3: マシン故障
- **対策**: 保守契約を締結。予備のダーツ板・スティックを在庫として持つ。
- **行動**: DARTSLIVEの保守担当者と連絡先を交換。

## 4. 3ヶ月以内に検証可能なアクション

1. **物件リストアップ**: 浜松・静岡・沼津の20坪以下物件を10件リスト化。
2. **DARTSLIVE営業との面談**: 「3台設置・無人運用」の条件で見積もり取得。
3. **QRアプリのプロトタイプ**: 入退店機能のみを簡易実装。テストユーザー5名で検証。

## 5. 証拠と明日使う人

- **証拠**: [[entities/ipak-darts]] の事業計画書。
- **明日使う人**: 物件契約の「間取り図」を持って、DARTSLIVEの営業に「この配置で3台置けるか」を確認する人。

## 6. 推測・要確認

- **DARTSLIVEのリース条件**: 契約内容により初期費用が変わる。[[要確認]]
- **物件の配線可否**: 物件オーナーの許可が必須。[[要確認]]
- **QRアプリの開発コスト**: 自社開発の場合、工数見積もりが必要。[[推測: 50万円程度]]
