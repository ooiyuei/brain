既存4社契約顧客に対するアップセル提案として、**「追加ツール（機能拡張）」**、**「年契約（コスト最適化・継続保障）」**、**「紹介報酬（顧客基盤の拡大）」**の3パターンを提案します。

既存顧客はすでに「会社別AI保存庫コア」の恩恵を受けており、信頼関係が構築されている状態です。そのため、単なる販売ではなく**「現在の成功事例をさらに拡大・安定させるための提案」**というスタンスで構成します。

---

# パターン1：追加ツール提案書（機能拡張型）

**目的**: 既存の「保存庫コア」を活かしつつ、別の業務課題を解決するツールを追加導入し、月額単価（LTV）を向上させる。
**ターゲット**: 営業支援ツール導入済みの顧客（例：リアルタイム電話支援）

### 提案の背景
> 「現在、営業現場でのAI支援が定着し、回答精度やスピードが向上していることと思います。その基盤である『保存庫コア』の強みを活かし、**『営業後のフォロー』や『新人教育』といった、次の工程の課題も同時に解消**することで、導入効果（ROI）を最大化しませんか？」

### 推奨パッケージ例：【営業×教育】セットアップ

| 項目 | 内容 | メリット |
| :--- | :--- | :--- |
| **既存** | 会社別AI保存庫コア + リアルタイム電話/営業回答支援 | 商談中の迷い解消、回答の標準化 |
| **追加提案** | **社内AIチャット / 新人教育AI** | 新人のオンボーディング期間短縮、属人化防止 |
| **追加提案** | **議事録整理AI** | 商談後の記録作業の自動化、保存庫への自動反映 |

### 価格提案（例）
*   **追加初期費用**: 30〜50万円（既存コアの流用により割安）
*   **月額追加**: +3〜5万円
*   **合計月額**: 8〜10万円（通常価格帯への移行準備）

### 営業トークのポイント
1.  **現状の肯定**: 「電話支援で商談の質が上がっているのは素晴らしい成果ですね。」
2.  **課題の提示**: 「一方で、新人がその知識をどう学ぶか、商談後の記録が手間で後回しになっていませんか？」
3.  **解決策の提示**: 「保存庫コアは共通なので、新人が質問すれば即座に『過去の成功商談の回答パターン』を提示できます。二度手間なしで教育と業務効率化が同時に実現します。」

# パターン2：年契約提案書（コスト最適化・継続保障型）

**目的**: 月払いから年払いへ移行させ、解約リスクを低減し、現金フローを安定させる。
**ターゲット**: 導入効果を実感し、継続利用を検討している顧客

### 提案の背景
> 「AIの精度は継続的な学習（データ蓄積）で向上します。月払いだと管理コストがかかるため、**年契約に切り替えることで、月額実質コストを最大20%割引**するとともに、**優先的なメンテナンス対応**と**データバックアップの強化**をご提供**します。」

### 年契約特典パッケージ

| 特典 | 内容 | 顧客価値 |
| :--- | :--- | :--- |
| **月額割引** | 月額実質 **15%〜20% OFF** | 予算計画の立てやすさ、コスト削減 |
| **優先サポート** | 緊急対応時のレスポンス優先 / 定期健康診断（四半期に1回） | 安心感、システム安定性の確保 |
| **データ保証** | 保存庫コアのデータバックアップ強化 / 新機能ベータテスト参加権 | 資産価値の保全、最新技術の早期利用 |

### 価格提案（例）
*   **従来**: 月額 5万円 × 12ヶ月 = 60万円/年
*   **年契約**: 月額 4万円相当 × 12ヶ月 = **48万円/年**
*   **初期費用**: 無料（または既存コアのメンテナンス費として10万円）

### 営業トークのポイント
1.  **継続の重要性**: 「AIは使えば使うほど賢くなります。月払いだと『次月どうしよう』と迷いが生じがちですが、年契約なら『使い続けること』に集中できます。」
2.  **コストメリット**: 「実質月額1万円以上の節約になります。また、年払いなので予算執行も一度で済み、事務処理が楽になります。」
3.  **安心感**: 「年契約のお客様には、システムの定期点検を無料で実施します。不具合が起きる前に予防できます。」

# パターン3：紹介報酬提案書（顧客基盤拡大型）

**目的**: 既存顧客をパートナーとして巻き込み、新規顧客（5社目以降）を獲得する。
**ターゲット**: AIpaXの成果に満足しており、業界内での影響力がある顧客

### 提案の背景
> 「AIpaXの『会社別AI保存庫コア』は、業界特有の知識を蓄積することで真価を発揮します。同じ業界や関連業界で、**『同じような課題を持つ仲間』**にこの価値を共有しませんか？ 成功事例を共有することで、お互いのAI活用レベルを底上げしましょう。」

### 紹介報酬制度（パートナープログラム）

| 項目 | 内容 | 報酬例 |
| :--- | :--- | :--- |
| **対象者** | 既存顧客が紹介した新規契約顧客 | - |
| **紹介者報酬** | **紹介成功時、月額報酬の3ヶ月分相当** または **現金10万円** | 例：月額10万円の契約なら30万円相当のクレジット or 現金 |
| **被紹介者特典** | **紹介者からの紹介枠として、初回事例価格（または特別割引）適用** | 通常価格100万円 → 70万円など（個別相談） |
| **継続報酬** | 紹介顧客が1年継続した場合、追加で月額10%のコミッション | 継続的な関係維持 |

### 営業トークのポイント
1.  **共感の誘発**: 「貴社がAI導入で得た成果は、業界の課題解決にもつながっています。同じ悩みを持つ他社様にも、この解決策を届けてみませんか？」
2.  **リスクの低減**: 「紹介される側にも『特別枠』をご用意していますので、安心してご提案いただけます。」
3.  **相互利益**: 「紹介成功時には、貴社のAIpaX利用料のクレジットとして還元します。実質的にシステム利用コストを下げるチャンスです。」

# 実施上の注意点・共通事項

1.  **個別最適化の徹底**:
    *   既存4社それぞれが「どのツール」で「どのような成果」を出しているかを把握し、パターン1の提案ではその成果をベースに「次のステップ」を提示する。
2.  **保存庫コアの価値強調**:
    *   どの提案でも、「単なるツール追加」ではなく、「**保存庫コアという資産がさらに豊かになる**」という文脈で伝える。
3.  **契約書の明確化**:
    *   年契約の場合：解約条件、データ持ち出し可否を明確化。
    *   紹介報酬の場合：報酬の支払いタイミング（新規契約入金後など）、対象期間を明確化。
4.  **次のアクションの提示**:
    *   パターン1: 「来週、新人教育AIのデモをお見せできますか？」
    *   パターン2: 「来月の請求から年契約に変更しませんか？ 書類作成をお手伝いします。」
    *   パターン3: 「今月、業界の交流会があります。一緒にAI活用事例を共有しませんか？」

この3パターンを組み合わせることで、既存顧客のLTV（顧客生涯価値）を最大化し、新規顧客獲得のエンジンとしても機能させます。
