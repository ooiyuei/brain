既存4社顧客に対するアップセル提案として、**「追加ツール導入」「年契約割引」「紹介報酬」**の3パターンを提案します。

既存顧客は「初回事例価格（初期50-80万／月額5万）」で契約済みであり、かつ「会社別AI保存庫コア（マザーボード）」が構築済みという状態です。
アップセルの核心は、**「基盤（保存庫）の価値を再確認させ、その上に高付加価値ツール（拡張機能）を載せる」**こと、および**「継続的な関係性（年契約・紹介）でロイヤルティを高める」**ことにあります。

以下に、各パターン別の説明書面（提案ポイント・メリット・リスクヘッジ）を記載します。

# AIpaX 既存4社向けアップセル提案書

## 共通の前提とメッセージ
**メッセージ：**
「すでに構築された『会社別AI保存庫コア』は、貴社のAI導入における『OS（オペレーティングシステム）』です。
今後はこのOSを安定運用しつつ、貴社の課題解決に特化した『アプリ（業務ツール）』を追加インストールすることで、ROI（投資対効果）を最大化します。」

---

## パターン1：追加ツール導入提案（拡張機能のインストール）

### 概要
既存の「保存庫コア」の上に、別の業務フローをカバーするツールを追加導入します。
最も推奨するのは、**「リアルタイム電話/営業回答支援」**（最有力）または**「社内AIチャット/新人教育AI」**です。

### 提案シナリオ
1.  **現状の棚卸し**: 既存ツールでカバーされていない「まだAI化できていない業務」をヒアリング。
2.  **ギャップ提示**: 「保存庫にはデータがありますが、それを『リアルタイム』で営業現場や新人が使える状態になっていません」と提示。
3.  **追加提案**: 特定ツール（例：新人教育AI）の初期構築費用と月額追加額を提示。

### 説明書面（提案ポイント）

| 項目 | 内容 |
|------|------|
| **提案タイトル** | **AIpaX 拡張機能「[ツール名]」導入のご提案** |
| **狙い** | 保存庫コアの活用範囲を広げ、月間利用価値を2倍〜3倍にする。 |
| **対象ツール例** | **社内AIチャット/新人教育AI**<br>・新人が「社内のルール・過去の商談事例」を24時間質問可能。<br>・既存保存庫の知識をそのまま検索エンジンとして活用。 |
| **費用感** | **初期費用**: 30〜50万円（既存コアとの連携コスト）<br>**月額追加**: 3〜5万円（既存月額5万＋合計8〜10万円） |
| **メリット** | 1. **ゼロからの構築不要**: 既存の保存庫（データ）をそのまま利用するため、学習コストが低い。<br>2. **即効性**: 既存のナレッジベースを「対話型」に変換。<br>3. **コスト効率**: 新規顧客価格（100万+）と比較し、既存顧客割引適用。 |
| **リスクヘッジ** | 「既存の保存庫と干渉しないか？」<br>→ **回答**: 保存庫コアは統合管理されますが、検索コンテキストを「新人向け」「営業向け」で分離可能。データは一元管理され、更新は一元で行われるため、矛盾が生じにくい。 |

## パターン2：年契約提案（継続安定化とコスト抑制）

### 概要
月額契約を年契約に変更し、顧客のランニングコストを抑制すると同時に、AIpaXの予測可能な収益基盤を強化します。
「通常価格移行ステージ」にある顧客にとって、**「月額単価の据え置きまたは微減」**と**「優先サポート権」**を提示します。

### 提案シナリオ
1.  **コスト比較**: 月額契約を12ヶ月継続した場合 vs 年契約の場合の比較表提示。
2.  **特典提示**: 年契約特典として「緊急対応の優先順位アップ」または「四半期に1回の深度ヒアリング」を追加。
3.  **契約更新**: 翌年以降の価格改定（通常価格移行）を見据えた「今の価格での長期ロックイン」を強調。

### 説明書面（提案ポイント）

| 項目 | 内容 |
|------|------|
| **提案タイトル** | **AIpaX 年間契約プランへの変更のご提案** |
| **狙い** | 顧客のAI運用定着率を高め、AIpaXの継続収益を安定させる。 |
| **費用感** | **月額**: 5万円 → **年間一括**: 54万円（2ヶ月分無料相当の割引）<br>※または月額据え置きで「サポート範囲の拡大」 |
| **メリット** | 1. **コスト削減**: 月額契約より約10〜15%の節約。<br>2. **価格ロック**: 5社目以降の通常価格（月額10万+）への移行前に、現在の低価格帯を1年間固定。<br>3. **安定運用**: 契約更新の手間がなくなり、運用に集中できる。 |
| **リスクヘッジ** | 「急に必要なくなった場合の解約は？」<br>→ **回答**: 年契約でも、月割り計算での中途解約は可能（ただし残月分の手数料あり）。ただし、最初の3ヶ月は解約不可（初期構築コスト回収のため）とするのが一般的。 |

## パターン3：紹介報酬提案（リファラルマーケティング）

### 概要
既存顧客に「AIpaXの成功事例」を共有させ、同業他社や取引先へ紹介してもらう仕組みです。
金銭的な報酬だけでなく、**「自社AI顧問としての地位向上」**や**「特典ツール無償提供」**をインセンティブとします。

### 提案シナリオ
1.  **成功事例の共有**: 貴社がAIpaXで得た成果（業務効率化、コスト削減率など）をデータ化して共有。
2.  **紹介のハードル低下**: 「紹介先との初回ヒアリングはAIpaX側が担当します」と伝え、顧客の負担をゼロに。
3.  **報酬提示**: 紹介契約成立時に、現金報酬またはサービスクレジットを提供。

### 説明書面（提案ポイント）

| 項目 | 内容 |
|------|------|
| **提案タイトル** | **AIpaX パートナーズプログラム：紹介特典のご案内** |
| **狙い** | 既存顧客を「サポーター」にし、低コストで新規顧客（5社目以降）を獲得。 |
| **報酬案** | **案A（現金）**: 紹介先契約額の20%（または固定額10万円）<br>**案B（サービス）**: 紹介先契約時、貴社に「追加ツール1つ無償導入」または「月額1ヶ月分無料」 |
| **メリット** | 1. **顧客の地位向上**: 「AI導入の先駆者」としての認知が社内で高まる。<br>2. **ネットワーク効果**: 業界内の標準化に貢献できる。<br>3. **低リスク**: AIpaX側が営業・提案・契約手続きを全て担当。 |
| **リスクヘッジ** | 「紹介先との関係性が悪化したら？」<br>→ **回答**: AIpaXが中立な第三者として契約を締結するため、貴社と紹介先の関係性に影響を与えない。また、紹介先とのトラブルはAIpaXが責任を持って対応。 |

## 実行ステップ（既存4社へのアプローチ順序）

1.  **Step 1: 現状の価値確認（Onboarding完了確認）**
    *   既存4社すべてに「保存庫コア」の正常稼働報告と、最初の成果データ（例：検索精度、回答速度の改善率）を共有。
    *   *目的*: 顧客に「投資対効果」を実感させ、アップセルの土台を作る。

2.  **Step 2: パターン1（追加ツール）の個別提案**
    *   各社の業務課題（営業、CS、新人教育など）に最も適した「追加ツール」を1つ提案。
    *   *優先度*: 最も課題が顕在化している社からアプローチ。

3.  **Step 3: パターン2（年契約）の提案**
    *   追加ツール導入が決定した社、または既存ツールの運用が安定している社へ年契約を提案。
    *   *ポイント*: 「通常価格移行前の最後のチャンス」という緊急性を演出。

4.  **Step 4: パターン3（紹介）の提案**
    *   上記のステップで満足度が得られた社へ、紹介プログラムを案内。
    *   *ポイント*: 「貴社の成功事例」を共有し、誇りを持って紹介できる環境を作る。

## 注意事項
*   **価格の透明性**: 初回事例価格（50-80万/月5万）は「β版価格」であることを明確にし、5社目以降の通常価格（100-150万/月10-15万）との差額を「既存顧客特典」として位置づける。
*   **保存庫コアの重要性**: アップセルの前提として、「追加ツールは保存庫コアがあってこそ価値がある」ことを常に強調し、コアの維持・更新の重要性を認識させる。
