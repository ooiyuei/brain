主力5事業を横断する「紹介プログラム（リファラルプログラム）」の設計案を作成しました。

このプログラムは、**「紹介者の動機付け（報酬）」**と**「被紹介者のハードル低下（特典）」**、そして**「法的リスクの回避（契約書）」**の3点に焦点を当てています。

※※注意：以下の内容は一般的なビジネスモデルに基づくテンプレート・指針です。実際の導入にあたっては、税務（源泉徴収など）、景品表示法、各事業の規約との整合性について、法務担当者および税理士と必ず協議してください。

---

# 主力5事業横断型 紹介プログラム設計書

## 1. プログラムの基本概念
*   **目的:** 既存顧客・パートナーからの信頼を活用し、新規顧客の獲得コスト（CAC）を抑制しつつ、顧客基盤を拡大する。
*   **対象事業:** 主力5事業（例：SaaS A, コンサルティング B, ECプラットフォーム C, 広告媒体 D, 教育サービス E ※仮称）
*   **横断のメリット:** 1つのアカウントで全事業の紹介・特典を管理できる「統合ポイント/報酬」を採用し、利便性を高める。

---

## 2. 紹介者報酬率（インセンティブ設計）

紹介者の動機を最大化するため、「現金報酬」と「サービス利用権（クレジット）」のハイブリッド型、または「高単価案件向け」の段階型を採用します。

### 案A：現金＋クレジット併用型（推奨：バランス型）
| 紹介対象事業 | 紹介者報酬（現金） | 紹介者特典（クレジット） | 備考 |
| :--- | :--- | :--- | :--- |
| **SaaS系（月額課金）** | 初月契約額の **20%** または **定額3,000円** | 全事業共通クレジット **5,000円分** | 継続課金にも10%の継続報酬を設定可 |
| **コンサルティング系（高額）** | 契約額の **5%～10%** | 全事業共通クレジット **10,000円分** | 高額ゆえに現金比率を高め、動機付け |
| **EC/プラットフォーム** | 販売額の **3%** | 全事業共通クレジット **1,000円分** | 低単価・高頻度のためクレジットで囲い込み |
| **広告媒体** | 初回投入額の **10%** | 全事業共通クレジット **2,000円分** | 広告効果の実感を得やすい金額設定 |
| **教育/研修** | 受講料の **15%** | 全事業共通クレジット **3,000円分** | 受講後の満足度向上にクレジットを活用 |

> **ポイント:** 「全事業共通クレジット」を採用することで、紹介者は「自社で使わなくても、他事業で使える」という柔軟性を感じ、紹介しやすくなります。

### 案B：ポイント制（長期囲い込み型）
*   **報酬:** 1円＝1ポイント（全事業共通）
*   **特典:** 紹介成功時に **1.5倍** のポイント付与
*   **還元:** ポイントは現金化不可（または一定額以上で現金化可）、サービス利用にのみ使用可。
*   **メリット:** 資金流出を抑え、自社エコシステム内での消費を促進。

---

## 3. 被紹介者特典（新規顧客のハードル低下）

被紹介者（新規顧客）が「すぐに始められる」「お得に始められる」仕掛けを作ります。

| 特典タイプ | 内容例 | 目的 |
| :--- | :--- | :--- |
| **初回割引** | 初月/初回契約額の **20% OFF** または **定額10,000円OFF** | 価格抵抗感を解消 |
| **無料トライアル延長** | 通常14日間 → **30日間** 無料延長 | 製品/サービスの価値を十分に体験させる |
| **初期費用免除** | 導入費・登録費 **0円** | 初期投資の障壁を撤去 |
| **特別サポート** | 専任コンサルタントによる **1対1オンボーディング**（1回） | 高付加価値事業での信頼構築 |
| **ギフトカード** | 1,000円分のAmazonギフトカードなど | 即効性のある魅力（SaaSなどでは効果的） |

> **注意点:** 景品表示法に抵触しないよう、「最大〇〇円OFF」ではなく「〇〇円OFF」と明確に表記するか、条件を明確にしてください。

## 4. 紹介フロー（UX設計）

1.  **紹介リンクの取得:** 会員ポータルから「紹介リンク」または「紹介コード」をコピー。
2.  **共有:** SNS、メール、LINEなどでリンクを共有。
3.  **新規登録:** 被紹介者がリンクから登録し、契約完了。
4.  **検証期間:** 契約から7日間のクーリングオフ期間または利用実績の確認（例：1ヶ月継続）。
5.  **報酬付与:** 検証期間終了後、紹介者アカウントに報酬（現金/クレジット）が自動付与。
6.  **通知:** メールで「紹介成功！報酬が付与されました」と通知。

## 5. 契約書テンプレート（紹介プログラム規約）

以下の条項を「紹介プログラム規約」として、会員規約とは別に、または追加条項として設けることを推奨します。

---

### **紹介プログラム利用規約**

**第1条（目的）**
本規約は、[会社名]（以下「当社」という）が提供する主力5事業（以下「対象事業」という）の新規顧客獲得を促進するため、既存顧客（以下「紹介者」という）が新規顧客（以下「被紹介者」という）を誘導し、その契約成立に応じて報酬を支給するプログラム（以下「本プログラム」という）の利用条件を定めるものです。

**第2条（紹介者の資格）**
紹介者となるには、当社が運営する会員ポータルに登録し、本プログラムへの参加申請が承認された個人または法人である必要があります。

**第3条（報酬の決定）**
1. 紹介報酬は、被紹介者が対象事業と新規契約を結び、かつ契約の履行が開始された時点で確定します。
2. 報酬の種類（現金またはサービス利用クレジット）および金額は、対象事業および契約内容に応じて、当社が別途定める「報酬基準表」に基づいて決定されます。
3. 紹介者は、報酬の決定について異議を申し立てることができません。

**第4条（報酬の支給方法）**
1. 現金報酬は、原則として月次で銀行振込により支払います。支払いは、紹介者が税務上の源泉徴収に必要な情報を当社に提出した後に実施します。
2. クレジット報酬は、紹介者の会員ポータルアカウントに自動付与され、対象事業の利用料金に充当できます。クレジットの有効期限は付与から1年間とします。
3. 報酬の支払いは、被紹介者の契約解除、返金、または詐欺行為が判明した場合に取り消すことができます。

**第5条（禁止行為）**
紹介者は、以下の行為を行ってはなりません。
1. 虚偽の情報や不正な手段による紹介。
2. 自らのアカウントや関連するアカウントを用いた自己紹介（セルフリファラル）。
3. 本プログラムを介して得た報酬を、転売または第三者への譲渡すること。
4. 当社または被紹介者の信用を毀損する行為。

**第6条（規約の変更）**
当社は、必要に応じて本プログラムおよび報酬基準を変更できるものとします。変更内容は会員ポータルおよびメールにて通知し、通知日から効力を生じます。

**第7条（準拠法および管轄裁判所）**
本プログラムに関する一切の紛争については、日本法を準拠法とし、[会社所在地]を管轄する裁判所を専属的合意管轄裁判所とします。

## 6. 実装・運用上の注意点

1.  **追跡技術の確立:**
    *   Cookieや referral code を用い、どの紹介者がどの新規顧客を誘導したかを正確に紐づけるシステムが必要です。
    *   クーリングオフ期間中の契約解除を防ぐため、検証期間（例：7日間）を設けることが一般的です。

2.  **税務処理:**
    *   **個人への現金報酬:** 原則として「雑所得」となり、紹介者が確定申告が必要です。ただし、源泉徴収義務がある場合（報酬額が一定以上など）や、報酬の種類（懸賞・当選金扱いなど）によって税務処理が異なります。必ず税理士と相談してください。
    *   **法人への現金報酬:** 経費として計上可能ですが、インボイス制度に対応した請求書発行が必要です。

3.  **景品表示法:**
    *   「最大10万円」といった曖昧な表現は避け、「10万円」または「10万円相当のクレジット」と明確にしてください。
    *   被紹介者への特典が「不当に高額」にならないよう注意（例：商品価格の20%を超える景品は規制対象になる可能性があります）。

4.  **社内周知:**
    *   営業チーム、カスタマーサポートチームに本プログラムを周知し、顧客からの問い合わせに対応できる体制を整えてください。
