既存契約4社に対するアップセル（追加売上・継続価値向上）提案として、以下の3パターンを作成しました。

AIpaXの核心である**「会社別AI保存庫コア（マザーボード）」**の価値を再認識させ、その上に**「業務AIツール（拡張機能）」**や**「年契約の安定性」「紹介報酬のインセンティブ」**を乗せる構成です。

# AIpaX 既存4社 アップセル提案3パターン

## 共通の前提と戦略
*   **現状**: 初期50-80万円 + 月額5万円の「初回事例価格」で契約済み。
*   **目標**: 月額単価の向上、契約期間の長期化、新規顧客への紹介（CAC削減）。
*   **基本スタンス**: 「保存庫コア」は既に完成しているため、追加コストは「新しいツールの実装」または「運用範囲の拡大」に起因すると説明し、正当性を担保する。

---

## パターン1：【機能拡張】「リアルタイム電話/営業回答支援」の追加導入
**ターゲット**: 営業部門の属人化や対応遅延に課題を感じている企業
**狙い**: 月額単価を5万円→10万円〜15万円へ引き上げ。

### 提案の論理構成
1.  **現状の可視化**: 既存の「保存庫コア」は構築済みですが、まだ「静的な知識」として眠っている状態です。
2.  **課題提示**: 営業現場では、保存庫にある情報を「その場で」引き出せていないため、回答のブレや遅延が発生しています。
3.  **解決策**: 保存庫コアに「リアルタイム電話/営業回答支援ツール」を接続し、通話中にAIがリアルタイムで回答候補を表示する。
4.  **価格提示**:
    *   **追加実装費**: 20〜30万円（既存コアへの接続作業費）
    *   **月額追加**: 5〜10万円（ツール利用料＋追加運用負荷）
    *   **合計月額**: 10〜15万円（通常価格帯への移行）

### 提案書面（概要）
```markdown
# 提案書：営業力向上のための「リアルタイムAI支援」追加導入

## 1. 背景
貴社では既に「会社別AI保存庫コア」が構築され、豊富な社内知識がAIで検索可能な状態にあります。
しかし、営業現場では「商談中にすぐ調べられない」「新人の対応に時間がかかる」といった課題が残っていませんか？

## 2. 提案内容
既存の保存庫コアを活用し、通話中にAIがリアルタイムで回答候補を提示する「リアルタイム電話/営業回答支援ツール」を追加導入します。

## 3. 期待効果
- 商談中の回答精度向上（属人化の解消）
- 新人営業のオンボーディング期間短縮
- 顧客への即応性向上による信頼度アップ

## 4. 費用感
- 追加実装費：20万円（既存インフラ活用のため低コスト）
- 月額料金：+5万円（合計月額10万円）
```

## パターン2：【契約形態】「年契約」への移行とコスト最適化
**ターゲット**: 月額払いに抵抗感がある、または予算管理を厳格に行いたい企業
**狙い**: 現金流入の安定化、解約リスクの低減、顧客ロイヤルティの向上。

### 提案の論理構成
1.  **柔軟性の提示**: 月々の予算変動に対応しやすい「月額制」から、より安価で安定した「年契約」への変更を提案。
2.  **インセンティブ**: 年契約にすることで、月額実質価格を10%〜15%割引、または「追加ツール1つの無料トライアル期間」を付与。
3.  **安心感の提供**: 年契約により、価格改定リスクから解放され、長期的なAI活用計画を立てやすくなることを強調。
4.  **価格提示**:
    *   **月額5万円 × 12ヶ月 = 60万円**
    *   **年契約一括払い**: **50〜55万円**（実質月額4.2〜4.6万円相当）
    *   **特典**: 年契約締結者限定で「社内AIチャット」の1ヶ月無料体験 or 「資料整理AI」の優先サポート

### 提案書面（概要）
```markdown
# 提案書：長期的なAI活用のための「年契約」プランへのご提案

## 1. ご提案の理由
現在ご利用いただいている月額プランは柔軟性が高い反面、長期的なコスト管理が難しい場合があります。
AIpaXでは、長期的なパートナーシップを築くため、年契約のお客様に特別割引と安定したサービス提供を行っています。

## 2. 年契約プランの特典
- **コスト削減**: 月額実質10%OFF（月額4.5万円相当）
- **価格固定**: 2年間の月額価格変動なし
- **特別特典**: 年契約締結時に「社内AIチャット」の1ヶ月無料利用券をプレゼント

## 3. 推奨理由
AI活用は「導入」で終わりではありません。年契約により、継続的なデータ蓄積とAIの学習効果を最大化し、ROI（投資対効果）を高めることができます。
```

## パターン3：【顧客獲得】「紹介報酬」制度の活用とネットワーク拡大
**ターゲット**: AIpaXの成果に満足しており、業界内で影響力を持つ経営者
**狙い**: 新規顧客（5社目以降）の獲得、既存顧客のエンゲージメント強化。

### 提案の論理構成
1.  **成功体験の共有**: 既存4社での成果（保存庫構築の完了、業務効率化の数値）を共有し、信頼性を再確認。
2.  **ネットワークの価値**: AIpaXは単なるツールではなく、「AI保存庫を持つ企業ネットワーク」の入り口であることを強調。
3.  **インセンティブ**: 紹介成功時に、現金報酬または「高額な追加ツール（例：CRM連携、外部API連携）の無料実装」を提供。
4.  **価格提示**:
    *   **現金報酬**: 紹介契約成立時、**10万円**の報酬
    *   **またはサービス報酬**: 紹介契約成立時、**「外部連携ツール」の実装費（通常30万円相当）を無料**で提供
    *   **双方のメリット**: 紹介先は「初回事例価格」で導入可能、紹介元はコスト削減または現金収入

### 提案書面（概要）
```markdown
# 提案書：AIpaXパートナープログラム「紹介報酬」のご案内

## 1. プラン概要
AIpaXの成果にご満足いただいているお客様に、業界内の同僚や取引先へAIpaXをご紹介いただくための特別プログラムです。

## 2. 紹介報酬（2つの選択肢から選べます）
### A. 現金報酬プラン
- 紹介先が契約成立時：**10万円**の現金報酬

### B. サービス充実プラン
- 紹介先が契約成立時：貴社向けに**「外部連携ツール（CRM/ERP連携）」**の実装費（通常30万円相当）を**無料**で提供

## 3. 紹介先の特典
- 紹介先は「初回事例価格（初期50-80万円/月額5万円）」で導入可能
- 貴社と同じ「会社別AI保存庫コア」基盤で、業界標準のAI活用が可能に

## 4. 次のステップ
ご紹介可能な先がいらっしゃいましたら、お気軽にご相談ください。
```

## 実施上の注意点・ポイント

1.  **順序の推奨**:
    *   まず**パターン1（機能拡張）**で月額単価を通常価格帯（10-15万円）へ引き上げる。
    *   その後、**パターン2（年契約）**へ移行を促し、現金流入を安定させる。
    *   関係性が良好な顧客に限り**パターン3（紹介）**を提案する。

2.  **「保存庫コア」の再評価**:
    *   アップセルの前提として、「既に構築された保存庫コアは資産であり、そこに新しいツールを乗せるのは非常に効率的」というロジックを一貫して伝える。

3.  **契約書面の更新**:
    *   追加ツール導入時は「別契約書」または「追加合意書」を作成し、既存の保存庫コアの所有権やデータ管理責任が変更されないことを明確にする。
    *   年契約時は「解約違約金」や「更新条件」を明確にする。
    *   紹介報酬時は「報酬支払い条件（契約成立・入金完了後など）」を明確にする。

4.  **既存4社への個別アプローチ**:
    *   各社の利用状況（どのツールを使っているか、データ入力は順調か）を確認し、最も効果的なパターンを1社ずつ選択して提案する。
