既存4社契約者に対するアップセル提案として、**「追加ツール導入」「年契約割引」「紹介報酬制度」**の3パターンを提案します。

既存顧客はすでに「会社別AI保存庫コア（マザーボード）」を構築済みであるため、**「基盤の活用深化」**と**「コスト最適化・収益化」**を軸に、それぞれの顧客の課題（営業、CS、新人教育など）に合わせた提案を行います。

---

# パターン1：追加ツール提案書
**テーマ：保存庫コアの活用領域を広げ、業務効率を最大化する**

既存顧客は「リアルタイム電話/営業回答支援」などの単一ツールで導入されているケースが多いですが、保存庫コアの真価は「複数ツールからのデータ蓄積と相互参照」にあります。既存の保存庫資産を活かしつつ、別の業務課題を解決するツールを追加提案します。

### 提案の背景
> 「すでに構築された『会社別AI保存庫コア』は、単なるデータ保管庫ではありません。営業、CS、新人教育など、異なる部門のAIツールが同じ『正解（知識）』を参照できる共通基盤です。既存ツールで得られた知見を、別の業務フローに展開することで、導入効果（ROI）をさらに高めます。」

### 推奨追加ツール候補（顧客の課題別）

| 顧客の現在の課題・希望 | 推奨追加ツール | 期待される効果 |
| :--- | :--- | :--- |
| **新人の教育負荷が高い**<br>（同じ質問を繰り返される） | **社内AIチャット / 新人教育AI** | 保存庫コア内のルール・FAQをリアルタイムで回答。新人の習熟期間を短縮。 |
| **商談後の記録・整理が手動**<br>（属人化・漏れがある） | **議事録整理AI / フォロー文生成AI** | 通話ログを自動で議事録化し、次のアクション（フォローメール等）を自動生成。 |
| **問い合わせ対応の遅延・ミス**<br>（営業が対応できない時間帯の対応） | **CS対応AI / 問い合わせ一次返信AI** | 保存庫コアの知識に基づき、正確な一次回答を自動生成。営業の負担軽減。 |
| **管理者が現場の状況を把握できない**<br>（レポート作成の手間） | **管理レポートAI** | 保存庫内のデータから、営業進捗やCS対応状況を自動集計・可視化。 |

### 価格・導入フロー
*   **初期費用**: 既存保存庫コアへの連携コストのため、通常価格の50%〜70%程度（または既存月額据え置きで初期費用のみ請求）。
*   **月額費用**: 追加ツール分の月額料金を加算（例：既存5万 → 追加ツール3万で合計8万円など）。
*   **メリット**: 新規構築不要のため、短期間（1〜2週間）での本格運用開始が可能。

# パターン2：年契約提案書
**テーマ：月額コストの最適化と、長期パートナーシップの確約**

既存顧客は月額制（通常5万円〜）で契約していますが、継続利用が安定している顧客に対して、年契約への変更を提案します。これにより、顧客の予算管理を容易にし、弊社も安定した収益基盤を確保します。

### 提案の背景
> 「AIpaXは『会社別AI保存庫コア』を継続的に更新・進化させるサービスです。月々の支払い管理の手間を省き、より安定的にコストを抑えるために、年契約プランをご用意しました。長期的なパートナーシップを重視する貴社に最適なプランです。」

### 年契約プランの詳細

| 項目 | 月額契約（現在） | **年契約提案** |
| :--- | :--- | :--- |
| **契約期間** | 月々更新 | **12ヶ月固定** |
| **月額単価** | 50,000円 | **45,000円**（10%割引） |
| **年間総額** | 600,000円 | **540,000円** |
| **特典** | - | **1ヶ月分無料**（実質13ヶ月分）<br>または **優先サポート窓口** |
| **更新通知** | 月末に自動請求 | **契約満了1ヶ月前に手動更新確認** |

### 提案のポイント
1.  **コスト削減**: 年間で6万円（10%）の節約を実現。
2.  **予算計画の立てやすさ**: 年間予算として一括で計上できるため、経理処理が容易。
3.  **継続保障**: 年契約期間中は、価格改定（通常価格への移行など）の影響を受けにくい、または価格据え置きの特典を提供（戦略に応じて調整）。
4.  **移行サポート**: 年契約への変更は、現在の月額契約から自動的に切り替え。解約の手間も不要。

# パターン3：紹介報酬制度提案書
**テーマ：既存顧客の成功事例を武器に、新規顧客を獲得する**

既存4社がAIpaXの価値を実感していることを前提に、同業他社や取引先企業への紹介を促します。紹介先が契約した場合、既存顧客に報酬（現金またはサービスクレジット）を提供します。

### 提案の背景
> 「AIpaXを導入された貴社が、他の企業でも同様の効果を得られることを、ぜひご友人や取引先にご紹介ください。貴社の成功事例は、他社にとって最も説得力のある『証』です。紹介いただいた企業との契約成立に応じて、感謝の報酬をご用意しています。」

### 紹介報酬制度の詳細

| 項目 | 内容 |
| :--- | :--- |
| **対象者** | 既存契約者（4社すべて） |
| **対象企業** | 紹介者から紹介された新規顧客企業 |
| **報酬条件** | 紹介先企業が**「保存庫コア＋ツール」の契約を締結**した場合 |
| **報酬内容** | **選択肢A**: 現金 50,000円<br>**選択肢B**: 月額利用料 1ヶ月分無料クレジット<br>**選択肢C**: 追加ツール 1つ無料導入権 |
| **支払い時期** | 紹介先企業の初回月額請求が完了した後、翌月末日に支払い/反映 |
| **制限** | 同一紹介者につき、年間で最大3社まで（または無制限、戦略に応じて） |

### 提案のポイント
1.  **成功事例の共有**: 既存顧客に「AIpaX導入前後の業務効率化データ」や「導入体験談」を共有し、紹介時の説得材料を提供。
2.  **低リスクな紹介**: 紹介先が決まらない限りコストは発生せず、決まった場合にのみ報酬が発生。
3.  **相互利益**: 紹介先は信頼できる知人の推薦で導入でき、紹介者は報酬またはサービス特典を得られる。
4.  **ネットワーク効果**: 業界横断的にAIpaXの認知を広げ、市場拡大につなげる。

# 実施上の注意点・次のステップ

1.  **顧客別カスタマイズ**:
    *   各社が現在どのツールを利用しているか、どの業務課題に最も関心があるかを確認し、パターン1の提案ツールを最適化してください。
    *   例：営業電話支援を使っている社 → 「議事録整理AI」を提案。CS対応を使っている社 → 「社内AIチャット」を提案。

2.  **年契約のタイミング**:
    *   導入から3〜6ヶ月経過し、安定して利用されている顧客に優先的に提案。
    *   「価格改定（通常価格へ）」の直前に年契約へ移行させることで、顧客の心理的ハードルを下げつつ、解約を防ぐ。

3.  **紹介報酬の周知**:
    *   既存顧客へのメールマガジンや、オンボーディング時の説明資料に紹介制度を組み込む。
    *   紹介リンクや紹介フォームを簡単に提供し、紹介のハードルを下げる。

4.  **法的・契約上の確認**:
    *   年契約の変更条項、紹介報酬の支払い条件、税務処理などを事前に法務・経理と確認してください。

これらの提案を、既存顧客との定期面談やサポート窓口を通じて、個別にアプローチすることを推奨します。
