# 特定技能拡大 構造的追い風活用戦略

## 1. 結論：なぜ今「特定技能」なのか

2026年5月現在、日本は**「労働力不足の深刻化」と「特定技能制度の拡大」**という構造的追い風の中にいる。

*   **需要側**: 建設、宿泊、飲食、介護、農業など14分野で人手不足が限界突破。特に地方の中小企業は採用が機能していない。
*   **供給側**: 日本への就労ビザ（特定技能1号/2号）の要件緩和や、母国での募集拡大。
*   **大井のポジション**: [[entities/aipax|AIpaX]]（会社別AI保存庫コア）と[[entities/aipa-web|AIpa Web]]（採用サイト制作）を組み合わせることで、**「外国人材の採用」から「定着・活用」までをAIで自動化する唯一の選択肢**になれる。

競合は「人材紹介会社（人件費高）」と「クラウドソーシング（質の担保難）」の二極化。その隙間を、**「AIによる管理コストの削減」**で埋める。

## 2. 市場の追い風（証拠ベース）

*   **制度拡大**: 2025年〜2026年にかけて、特定技能の活動範囲が拡大。例えば、飲食業での「調理業務」の範囲拡大や、建設業での「2号」への移行ルートの明確化。
*   **補助金**: [[entities/business-contests|ビジネスコンテスト]]関連の公的資金だけでなく、**「外国人材受入支援事業費補助金」**などが自治体レベルで拡充。
*   **社会問題化**: 「外国人材の離職」が社会問題化。企業は「採用」だけでなく「どうやって3年持つか」に困っている。ここが[[entities/aipax|AIpaX]]の得意分野（社内知識のAI保存・新人教育AI）と直結する。

## 3. 大井の事業との接続点（3本柱とのシナジー）

### 3.1 [[entities/aipax|AIpaX]]：「定着率」を売る

*   **課題**: 外国人材は言語・文化・業務ルールで即離職する。
*   **AIpaXの解決**: 社内の「暗黙知」を[[entities/aipax|AIpaX]]の保存庫コアに落とし、新人（外国人含む）がAIチャットで即座に業務ルールを引けるようにする。
*   **売り文句**: 「外国人材の離職率を30%下げるAI新人教育システム」。
*   **根拠**: [[entities/aipax|AIpaX]]は既に4社契約済み。その延長線上で「多言語対応」や「新人オンボーディング」モジュールを追加するだけで、既存顧客へのアップセルが可能。

### 3.2 [[entities/aipa-web|AIpa Web]]：「採用」を自動化

*   **課題**: 外国人材向けに多言語の採用サイトを作るのはコストがかかる。
*   **AIpa Webの解決**: [[entities/aipa-web|AIpa Web]]のフォーム入力→AI制作→公開フローで、**日本語・中国語・ベトナム語・ネパール語**の4言語サイトを7日で公開。
*   **売り文句**: 「多言語対応採用サイトを、月額3万円で更新し続ける」。
*   **根拠**: [[entities/aipa-web|AIpa Web]]はDeep Research済み。競合（Wix等）は「ツール」だが、大井は「運用代行」を売る。地方中小企業はITリテラシーが低いため、代行価値が高い。

### 3.3 [[entities/business-contests|ビジネスコンテスト]]：資金と認知

*   **戦略**: [[entities/business-contests|ビジネスコンテスト]]の「SDGs」や「地域創生」カテゴリで、**「外国人材受入支援プラットフォーム」**として応募。
*   **対象**: [[entities/business-contests|ビジネスコンテスト]]リストの「★★★高勝率」コンテスト（例：高校生みんなの夢AWARDの企業版、または地域活性化コンテスト）。
*   **狙い**: 賞金（資金調達）＋審査員経由での自治体・大手企業との接点。

## 4. 具体的なアクションプラン（3ヶ月以内）

### Phase 1: 検証（2026-05-27 〜 2026-06-10）

*   **ターゲット**: 静岡県内の建設業・介護業（人手不足が深刻な業種）。
*   **アクション**:
    1.  [[entities/aipa-web|AIpa Web]]のテンプレートを「外国人材採用特化版」に改造（4言語対応）。
    2.  [[entities/aipax|AIpaX]]のデモに「新人教育AI（多言語版）」モジュールを追加。
    3.  静岡県内の建設業5社にアプローチ。「外国人材の離職にAIで対策しませんか？」と持ちかける。
*   **KPI**: 3社とのヒアリング、1社のPOC（Proof of Concept）契約。

### Phase 2: 展開（2026-06-11 〜 2026-07-10）

*   **ターゲット**: 全国展開（オンライン営業）。
*   **アクション**:
    1.  [[entities/aipa-web|AIpa Web]]のLPを「特定技能対応」に特化。
    2.  [[entities/business-contests|ビジネスコンテスト]]の夏締切コンテストに「外国人材受入支援プラットフォーム」として応募。
    3.  人材紹介会社と提携（AIpaXを「定着支援ツール」として提供）。
*   **KPI**: 月額10万円の顧客3社、コンテスト応募完了。

### Phase 3: 拡大（2026-07-11 〜 2026-08-10）

*   **ターゲット**: 大手企業向けパッケージ化。
*   **アクション**:
    1.  [[entities/aipax|AIpaX]]の保存庫コアを「多言語HROS（Human Resource Operating System）」として再定義。
    2.  大手人材会社（パソナ、ロジックス等）とのAPI連携を検討。
    3.  [[entities/aipa-web|AIpa Web]]の月額顧客を10社に拡大。
*   **KPI**: 月収100万円達成（AIpaX 5社 + AIpa Web 10社 + スポット案件）。

## 5. リスクと対策

*   **リスク**: 外国人材の募集が競争激化でコスト増。
    *   **対策**: 大井は「募集」ではなく「定着」を売る。採用コストは他社に任せ、定着支援で収益を確保。
*   **リスク**: 法律・ビザの複雑さ。
    *   **対策**: 法律相談は専門家に委ね、大井は「AIツール」に集中。[[entities/aipax|AIpaX]]の保存庫に法務情報をインクルードするだけで、顧客は安心する。
*   **リスク**: 大井のフォーカス散漫。
    *   **対策**: [[entities/testall|Testall]]と[[entities/agents-of-flag|AOF]]は一旦休止。[[entities/aipax|AIpaX]]と[[entities/aipa-web|AIpa Web]]に全リソースを集中。

## 6. 明日から動くこと

1.  [[entities/aipa-web|AIpa Web]]のLPに「特定技能対応」セクションを追加。
2.  [[entities/aipax|AIpaX]]のデモに「新人教育AI（多言語版）」モジュールを追加。
3.  静岡県内の建設業5社にメール送信（[[entities/aipa-web|AIpa Web]]のLP付き）。

この戦略で、大井は「AIツール販売」から「社会課題解決プラットフォーム」へ移行できる。月収100万円も、この「構造的追い風」を使えば現実的。
