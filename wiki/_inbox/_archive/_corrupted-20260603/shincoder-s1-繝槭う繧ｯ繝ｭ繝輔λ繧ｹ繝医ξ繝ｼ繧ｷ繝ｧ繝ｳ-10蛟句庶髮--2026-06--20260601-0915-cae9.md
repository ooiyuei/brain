# 🚀 AIpaX school 第1期生募集戦略

## 1. 核心メッセージ（USP: 唯一無二の価値）

> **「AIを『道具』として使うのではなく、AIを『手足』のように操る起業家マインドセットを、17歳の現役起業家が直接教えます。」**

*   **競合との違い**:
    *   ❌ 既存スクール: 「コードの書き方」「ツールの操作方法」を教える（机上の学習）。
    *   ✅ AIpaX: 「どうやって世の中に価値を提供するか」「どうチームで合意形成するか」「どうAIで自動化するか」を教える（現場の学習）。
*   **信頼性の担保**:
    *   講師は17歳の大井（HR高等学院出身・ビジコン3連覇・現役AI起業家）。
    *   カリキュラムは「HR高等学院」で検証された7つのフレームワークを統合したv2。

## 2. ターゲット別ペインポイントと解決策

### A. 親御さん向け（購買決定者）
**不安**: 「AIに仕事が奪われる」「子供がAIに依存して思考力が低下する」「高額な教育投資の回収が不安」

| 親の不安 | AIpaX schoolの回答・価値 |
|----------|--------------------------|
| **「AIに負けないスキルが身につくのか？」** | AIを「手足」のように使いこなす「AIリテラシー」ではなく、「AIを活用して問題を解決する」実践力を身につけます。 |
| **「ただのゲームや遊びにならないか？」** | 卒業課題として「個人探究テーマ」や「サービス企画」を作成。単なる学習ではなく、**プロダクト制作**を通じた成長を実感できます。 |
| **「高額すぎて続けられるか？」** | 月額3万円は、大手プログラミングスクールの半額〜同等価格。しかし、**「起業家マインドセット」**という付加価値が含まれています。 |
| **「オンラインで本当に効果があるか？」** | HR高等学院で実証された「対話型・プロジェクト型」カリキュラム。動画視聴だけでなく、**大井自身とのメンタリング**が含まれます。 |

### B. 中高生本人向け（利用者）
**不安**: 「難しそう」「つまらない」「友達と違うことをして孤立したくない」「将来の選択肢を広げたい」

| 生徒のペイン | AIpaX schoolの魅力 |
|-------------|-------------------|
| **「学校の勉強は退屈」** | 「学校より現場で学ぶ」スタイル。実社会の課題（例：釣り場のDX、地元企業の課題）を題材にします。 |
| **「将来がわからない」** | 17歳の起業家が「今、ここで」何をしているかを共有。AI時代のキャリアパスをリアルに体験できます。 |
| **「一人でやるのは寂しい」** | チームでアイデアを練る「合意形成」モジュールあり。同じ志を持つ仲間と出会えます。 |
| **「すぐに成果が見たい」** | Module 1から「バイブコーディング体験」や「AIツールでの自動化」を実践。数週間で「作れた！」を実感。 |

## 3. 募集要項ドラフト（LP/DM用）

**タイトル**:
**【第1期生 5名限定】AIを「手足」のように使いこなすスキルを、17歳の現役起業家が直接教える**
〜HR高等学院のノウハウを凝縮した、実践的AIスクール〜

**サブタイトル**:
月3万円 / オンライン / 17歳の大井が直接指導 / 卒業後は「AIで何かを作れる」高校生に

**コンテンツ概要**:
*   **期間**: 1年間（5モジュール制）
*   **価格**: 月額30,000円（税込）
*   **対象**: 中学生・高校生（保護者同意必須）
*   **特徴**:
    1.  **現役起業家による直接指導**: 教科書通りの知識ではなく、現場で使える「思考のフレームワーク」を伝授。
    2.  **HR高等学院の知恵を統合**: ストーリーテリング、合意形成、ロジックモデルなど、社会で生きるための7つのフレームワークを習得。
    3.  **AI×実践プロジェクト**: ChatGPT/Claudeを活用し、自分だけのアプリやサービスを企画・開発。
    4.  **EEMUS連携チャンス**: 優秀な生徒は、静岡の企業インターン（EEMUS）への優先紹介枠あり。

**カリキュラムの抜粋**:
*   **Month 1-2**: AIマインドセット + バイブコーディング体験（AIと会話してコードを書く）
*   **Month 3-4**: 日常業務自動化 + チーム合意形成（AIで会議の議事録を自動生成し、合意形成の練習）
*   **Month 5-6**: エンジニアリング思考 + 個人探究テーマ設定（自分の興味をAIで深掘り）
*   **Month 7-8**: 本格アプリ開発 + サービス企画（Replit/Cursorを使って実際に形にする）
*   **Month 9-12**: データ分析 + マネタイズ + 卒業課題発表（作ったものを世に出す準備）

**応募条件**:
*   AIに興味がある方
*   1年間継続して学びたい方
*   保護者の方と週1回（または適宜）の進捗共有にご協力いただける方

## 4. 即座に動き出すためのアクションプラン

### Step 1: 認知獲得（今週中）
1.  **SNSでの発信（TikTok/YouTube/Instagram）**:
    *   **動画テーマ**: 「17歳起業家が教える、AIで月3万円稼ぐ方法の基礎」「HR高等学院で学んだ、AI時代の最強の勉強法」
    *   **ターゲット**: 静岡の中高生保護者層、都内の教育関心層
    *   **CTA**: 「第1期生5名を募集しています。詳細はDMまたはリンクから」
2.  **EEMUS経由のリーチ**:
    *   EEMUSの既存ネットワーク（静岡の高校生・企業）に「AIpaX schoolのプレスクールとして活用してほしい」と提案。
    *   「EEMUSインターン前のAIリテラシー講座」として位置づけ、EEMUS参加者から5名を募集。

### Step 2: 信頼構築（来週〜再来週）
1.  **無料ウェビナー/体験会開催**:
    *   **テーマ**: 「AIで1時間でプロダクトを作る体験」
    *   **内容**: 実際のAIツール（Cursor等）を使って、簡単なアプリや自動化スクリプトを作るデモ。
    *   **目的**: 「AIpaX school」のリアルな指導スタイルと、大井の熱量を伝える。
2.  **保護者向け説明会**:
    *   親御さん向けに「AI時代の教育の必要性」と「AIpaX schoolのカリキュラムの意義」を説明。
    *   大井自身が「なぜ17歳で起業したのか」「AI教育の重要性」を語ります。

### Step 3: 募集完了（来月頭）
1.  **5名の選考**:
    *   応募者から「学びへの意欲」「継続性」を確認し、5名を選出。
2.  **開講準備**:
    *   カリキュラムv2の詳細スライド作成。
    *   個別メンタリングのスケジュール調整。

## 5. 競合優位性の強化ポイント

*   **「17歳」という年齢の活用**:
    *   親御さんには「若さゆりの新鮮な視点と、現場のリアルさ」をアピール。
    *   生徒には「自分と同じ年齢の人が、すでに社会で活躍している」というロールモデル提示。
*   **「HR高等学院」のノウハウ統合**:
    *   単なるAIツール講座ではなく、「社会で生きるための思考法」が含まれていることを強調。
    *   これは既存のプログラミングスクールにはない、**「人間力×AI力」**の差別化ポイントです。
*   **「EEMUS」の連携**:
    *   静岡ローカルでは、EEMUSとの連携が最大の強み。「AIスキル＋地元企業インターン」というキャリアパスを提示。
