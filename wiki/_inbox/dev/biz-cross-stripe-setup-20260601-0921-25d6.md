大井グループの主力5事業（AIpa Web / AIpaX / AIpaX school / EEMUS / Testall）におけるStripe決済設定のロードマップを、各事業ごとに商品設定、価格モデル（定期 vs 単発）、支払い方法、法的考慮、導入工数を軸に整理します。以下は、それぞれの事業についての詳細なロードマップです。

## 1. AIpa Web

### 1.1 商品設定
- **AIpa Web**は、AIチャットボットの作成・運用を提供するSaaS製品です。
- **商品設定**は、以下のような構成で進めます：
  - **AIpa Web Standard**：月額10,000円（定期）のプラン
  - **AIpa Web Pro**：月額20,000円（定期）のプラン
  - **AIpa Web One-Time**：単発で50,000円（初期導入費用）

### 1.2 価格モデル
- **定期**：月額プラン（Standard / Pro）
- **単発**：初期導入費用（One-Time）
- **価格モデルの選択肢**は、顧客の利用頻度や規模に応じて柔軟に設定可能。

### 1.3 支払い方法
- **Stripeでの支払い方法**は、以下をサポートします：
  - クレジットカード（Visa / MasterCard / JCB / American Express）
  - デビットカード
  - クレジットカードの自動更新（定期プラン）
  - クレジットカードの手動入力（単発プラン）

### 1.4 法的考慮
- **Stripeの契約**は、日本国内での利用に適応しており、税務処理や取引履歴の記録が可能。
- **個人情報保護法（APPI）**の観点から、顧客の支払い情報は暗号化され、セキュリティ対策が施されています。
- **契約書**は、Stripeの利用規約と大井グループの利用規約を組み合わせた形で作成。

### 1.5 導入工数
- **導入工数**は、以下の通りです：
  - **商品設定**：2人日（商品の登録・価格設定）
  - **価格モデル設定**：1人日（定期・単発の選択肢の設定）
  - **支払い方法設定**：1人日（Stripeの支払い方法の設定）
  - **法的考慮の準備**：3人日（契約書の作成、法的リスクの確認）
  - **全体導入**：7人日（総合的な導入作業）
