## 1. AIpa Web

### 商品設定
AIpa Webは、AIを活用したWebサービスを提供する事業です。主な商品は「AIpa Webサービス」であり、顧客がAIを活用したWebアプリケーションの構築や運用を依頼する形態です。商品設定としては、以下の3種類を想定しています。

- **AIpa Web Standard（月額）**：月額定額制で、基本的なAI機能を提供。
- **AIpa Web Pro（月額）**：月額定額制で、高度なAI機能を提供。
- **AIpa Web One-off（単発）**：プロジェクトごとの単発契約。

### 価格モデル
- **定期（月額）**：月額定額制で、継続的なサービス利用を想定。
- **単発**：プロジェクトごとの単発契約で、一括払い。

### 支払い方法
- **クレジットカード**：Stripeの標準的な支払い方法。
- **銀行振込**：単発契約において、プロジェクト終了後に振込を想定。
- **PayPal**：一部の国際顧客向けに導入を検討。

### 法的考慮
- **契約書の作成**：定期契約では、サービス内容、利用範囲、契約期間、支払い条件を明記。
- **データ保護法（GDPR）**：EU圏の顧客を対象に、データの取り扱いに関する条項を含む。
- **税務処理**：国内および海外の顧客に応じて、税額計算を明確にする。

### 導入工数
- **初期設定**：商品設定、価格モデル、支払い方法の設定に約10人日。
- **法的設定**：契約書の作成、GDPR対応、税務処理の設定に約8人日。
- **テスト・導入**：Stripe APIとの連携、テスト環境の構築に約5人日。
- **合計**：約23人日。
