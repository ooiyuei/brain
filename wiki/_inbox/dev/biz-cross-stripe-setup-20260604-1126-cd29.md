## 1. AIpa Web

### 1.1 商品設定  
AIpa Webは、AIを活用したWebアプリケーションの提供サービスであり、主に企業向けのSaaS（Software as a Service）として販売されています。  
- **商品名**：AIpa Web（月額プラン）  
- **商品説明**：AIを活用したWebアプリケーションの開発・運用サポートサービス  
- **商品カテゴリ**：SaaS / サービス / AI関連  
- **商品属性**：月額定額 / サポート / チーム向け  

### 1.2 価格モデル（定期 vs 単発）  
- **定期**：月額定額（例：10万円 / 月）  
- **単発**：なし（SaaSとしての提供形態のため）  
- **価格帯**：10万円〜30万円 / 月（プランごとに異なります）  
- **割引制度**：年払い割引（例：年額120万円で月額10万円）  

### 1.3 支払い方法  
- **Stripeでの支払い方法**：  
  - クレジットカード（VISA / MasterCard / JCB / American Express）  
  - PayPal（一部国での利用）  
  - クレジットカードの自動更新（月額プラン）  
- **支払いタイミング**：月次（月初日）  
- **支払い確認**：StripeのWebhookで自動通知  

### 1.4 法的考慮  
- **契約書**：SaaS契約書（サービス利用規約 / 保守契約書 / 連絡先情報）  
- **データ保護**：GDPR / PIPEDA / JIS Q 7000（日本国内）  
- **税務処理**：VAT（EU） / 消費税（日本） / 事業税（海外）  
- **責任範囲**：サービスの停止・遅延時の補償範囲（契約書に明記）  
- **個人情報保護**：Stripeのデータ保護ポリシーとの整合性確認  

### 1.5 導入工数  
- **初期設定**：  
  - Stripeアカウントの作成（1日）  
  - 商品設定（1〜2日）  
  - 価格モデルの設定（1日）  
  - 支払い方法の設定（1日）  
- **テスト環境構築**：  
  - Stripeテストカードの利用（1日）  
  - サーバー側のStripe SDK導入（2日）  
- **本番環境導入**：  
  - StripeのWebhook設定（1日）  
  - サーバー側の支払い処理の実装（3〜5日）  
- **合計工数**：約8〜10人日
