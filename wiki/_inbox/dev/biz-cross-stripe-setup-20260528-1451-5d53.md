## 1. AIpa Web

### 商品設定
AIpa Webは、AIを活用したWebアプリケーションの提供サービスです。主に、企業向けのAIチャットボットやカスタマーサポートツールとして利用されます。商品設定としては、以下の3種類を想定しています。

- **AIpa Web Standard**：月額定額プラン（10ユーザーまで）
- **AIpa Web Pro**：月額定額プラン（50ユーザーまで）
- **AIpa Web Custom**：カスタマイズプラン（要見積もり）

### 価格モデル
- **定期**：月額定額（Standard / Pro）  
- **単発**：カスタマイズプラン（要見積もり）  
- **追加料金**：ユーザー数増加時の追加料金（例：10ユーザーごとに10,000円）

### 支払い方法
- **クレジットカード**：Stripe標準のカード払い
- **銀行振込**：月次決済のオプション
- **企業向け**：法人口座での支払い（Stripe Connectを活用）

### 法的考慮
- **契約書**：サービス利用契約書の作成（個人利用と企業利用の区分）
- **データ保護**：GDPRや日本国内の個人情報保護法（APPI）への対応
- **課税**：日本の法人税・所得税の計算対応
- **支払い条件**：支払い期限（例：月末締め、翌月支払い）の明記

### 導入工数
- **商品設定**：1人日（Stripe APIの設定と商品登録）
- **価格モデル**：1人日（定期・単発の設定と料金の設定）
- **支払い方法**：1人日（支払い方法の選択肢の設定）
- **法的考慮**：2人日（契約書の作成と法的リスクの評価）
- **導入全体**：約5人日（開発・運用・法務の協業）

## 総括

- **共通点**：
  - すべての事業で「クレジットカード」が基本の支払い方法
  - すべての事業で「定期プラン」が中心
  - すべての事業で「法的考慮」（契約書・データ保護・課税・支払い条件）が必須

  - AIpa Webは「単発プラン」も対応可能
