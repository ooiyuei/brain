主力5事業を横断した紹介プログラム（リファラルプログラム）の設計案を作成しました。
「事業横断」であることが最大の強みとなるよう、**共通ルール**と**事業固有の特性**をバランスよく組み合わせた構成にしています。

※具体的な数値（報酬額など）は業界標準や貴社の利益率に合わせて調整してください。

---

# 主力5事業横断型 紹介プログラム設計書

## 1. プログラムの基本概念
*   **プログラム名案:** 「Cross-Referral Program（クロス・リファラル・プログラム）」
*   **目的:** 既存顧客・関係者からの紹介により、新規顧客獲得コスト（CAC）を抑制しつつ、5事業間のシナジー（クロスセル）を促進する。
*   **対象者:**
    *   **紹介者:** 既存顧客、取引先、社員、パートナー企業など
    *   **被紹介者:** 新規顧客（5事業いずれかのサービスを利用する可能性のある個人・法人）

## 2. 紹介者報酬率（インセンティブ設計）

「事業横断」を活かし、**「紹介された事業の単価・利益率」**と**「紹介者のステータス」**で段階的に報酬を設定します。

| 区分 | 報酬タイプ | 報酬内容（例） | 備考 |
| :--- | :--- | :--- | :--- |
| **基本報酬** | 成功報酬 | 契約金額の **5%〜10%** | 事業によって粗利が異なるため、上限を設定 |
| **固定報酬** | 案件タイプ別 | 小規模: 1万円<br>中規模: 3万円<br>大規模: 5万円以上 | 契約金額が不明な場合や、固定費回収型のサービス向け |
| **ボーナス** | クロスセル促進 | 紹介先が**別の事業**も契約した場合、追加 **20%** | 横断シナジーを強化するためのインセンティブ |
| **特別枠** | VIP紹介者 | 累計紹介数に応じて、ギフト券や特別サービス付与 | 常連紹介者へのロイヤルティ向上 |

> **ポイント:** 報酬は「契約成立時」に支払うのではなく、「初回決済完了後」または「契約成立後3ヶ月（クーリングオフ期間経過後）」に支払うことで、解約リスクを回避します。

## 3. 被紹介者特典（新規顧客獲得促進）

被紹介者が「紹介されたから入る」という心理的ハードルを下げるため、**紹介者との連動特典**と**単独特典**の2層構造にします。

| 特典タイプ | 内容（例） | 目的 |
| :--- | :--- | :--- |
| **共通特典** | **初月無料** または **10%割引** | 5事業共通で適用。導入の敷居を下げる |
| **紹介者連動特典** | 紹介者からの「特別アドバイスセッション」1回無料 | 信頼関係を活用したサービス提供 |
| **事業固有特典** | 例：SaaSなら追加ストレージ、コンサルティングなら診断レポート | 各事業の魅力（USP）をアピール |
| **早期契約特典** | 紹介月内に契約すると、追加で1ヶ月延長 | 成約スピードを上げる |

## 4. 契約書テンプレート（主要条項）

紹介プログラム参加者（紹介者）と貴社との間で締結する「紹介委託契約書」の主要条項です。

---

### **紹介委託契約書（主要条項抜粋）**

**第1条（目的）**
本契約は、株式会社[貴社名]（以下「甲」という）が、[紹介者名]（以下「乙」という）に新規顧客の紹介を委託し、その成功に応じて報酬を支払うに関する事項を定める。

**第2条（紹介の定義）**
乙が甲に対して紹介する新規顧客（以下「紹介先」という）が、甲の主力5事業（[事業A]、[事業B]、[事業C]、[事業D]、[事業E]）のいずれかにて契約を締結し、かつ初回決済が完了した場合を「紹介成功」と定義する。

**第3条（紹介者の義務）**
1. 乙は、紹介先に対して甲のサービス内容、料金体系、契約条件などを正確かつ虚偽のない範囲で説明するものとする。
2. 乙は、甲のブランドイメージを毀損するような行為、または違法・反社会的な行為を行ってはならない。
3. 乙は、紹介先に対して「甲からの直接連絡」を促すものとし、甲と紹介先の間に甲のスタッフ以外の仲介行為を行ってはならない。

**第4条（報酬の計算と支払）**
1. 紹介成功に伴う報酬は、甲が定める「紹介プログラム規程」に基づき計算する。
2. 報酬の支払いは、紹介先の契約が成立し、かつクーリングオフ期間等が経過した月の末日に、甲の指定する口座へ振込む。
3. 紹介先が契約を解約した場合、既に支払った報酬の返還を求めることはできないが、甲は必要に応じて報酬の調整を行う権利を有する。

**第5条（排他性）**
乙は、本契約期間中、甲の競合他社に対して同様の紹介活動を行ってはならない。

**第6条（秘密保持）**
乙は、本契約に関連して知り得た甲の営業秘密、顧客情報等を第三者に漏洩してはならない。

**第7条（契約期間）**
本契約は、[開始日]から[終了日]まで有効とする。期間満了後、双方が書面により同意した場合は自動的に更新される。

## 5. 運用上の注意点とリスク管理

1.  **重複紹介の防止:**
    *   複数の紹介者が同じ顧客を紹介した場合、**「最初に登録された紹介者」**を優先するか、**「成約に貢献した割合」**で分配するかを事前にルール化します。
    *   管理システム（CRM）での追跡が必須です。

2.  **不正防止:**
    *   自己紹介（自分自身を他人として紹介）の防止。
    *   紹介者・被紹介者の関係性（親族、社内関係など）の制限。
    *   解約率が高い場合の報酬返還条項。

3.  **税務処理:**
    *   紹介者への報酬は「雑所得」として扱われる場合が多く、源泉徴収が必要な場合があります。
    *   法人の場合、経費として計上できる範囲を確認してください。
    *   被紹介者への特典も、課税対象となる可能性があるため、税理士と相談してください。

4.  **事業横断の課題:**
    *   各事業の営業担当者が「自分の事業しか紹介しない」傾向を防ぐため、**「他事業紹介ボーナス」**を設けるなど、組織内のインセンティブ調整が必要です。

## 6. 次のステップ

1.  **詳細な数値設定:** 各事業の粗利率に基づき、報酬率の上限を決定。
2.  **システム構築:** 紹介リンクの発行、成約追跡、報酬計算の自動化。
3.  **社内周知:** 営業チームへの説明会実施（彼らが紹介者になるため）。
4.  **パイロット運用:** 1〜2事業でテスト実施し、フィードバックを反映。

この設計案をベースに、貴社の具体的な事業内容に合わせて調整してください。必要であれば、各事業別の詳細なインセンティブ設計も作成可能です。
