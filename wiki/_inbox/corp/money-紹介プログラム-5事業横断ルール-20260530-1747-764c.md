主力5事業を横断する「紹介プログラム（リファラルプログラム）」の設計案を作成しました。
事業領域が異なる場合でも、**「共通のルール」**と**「事業固有の特典」**を組み合わせることで、運用の複雑さを避けつつ、各事業の魅力を引き出す構成にしています。

---

# 主力5事業横断型 紹介プログラム設計書

## 1. プログラムの基本概念
*   **名称案:** 「Cross-Referral Program（クロス・リファラル・プログラム）」
*   **目的:** 既存顧客の信頼を活用した新規顧客獲得（CAC削減）と、他事業へのクロスセル促進。
*   **対象:** 全5事業の既存顧客（B2B/B2Cに応じて調整必要）

## 2. 紹介者報酬率（インセンティブ設計）

報酬は「現金」か「サービス利用券（クレジット）」かを選択できるハイブリッド型を推奨します。
※ 税務処理の簡素化と、自社サービスへのリテンション効果を狙います。

| 区分 | 報酬タイプ | 報酬内容 | 条件・上限 |
| :--- | :--- | :--- | :--- |
| **基本報酬** | サービス利用券 | **契約金額の 10%** または **固定額（例: 1万円）** | 月間上限 5万円まで |
| **高額契約ボーナス** | 現金 or 高額クレジット | 契約金額が基準額（例: 100万円）を超えた場合、**追加で 5%** | 年額上限 20万円まで |
| **継続報酬** | 継続ボーナス | 契約が1年継続した場合、**月額の 5%** をクレジット付与 | 最大12回まで |

> **ポイント:**
> *   **現金支給:** 税務処理（源泉徴収等）が複雑になるため、B2C向けは「商品券」や「ポイント」を推奨。
> *   **クレジット支給:** B2B向けは、自社サービスの利用促進に直結するため「利用クレジット」を推奨。
> *   **横断効果:** 事業Aの顧客が事業Bを紹介した場合も、同じ報酬体系で適用し、社内連携を促進。

## 3. 被紹介者特典（新規顧客向け）

紹介された側（被紹介者）にも魅力的な特典を用意し、紹介者の信頼を損なわないようにします。

| 特典タイプ | 内容例 | 目的 |
| :--- | :--- | :--- |
| **初回割引** | 初月料金 **50%OFF** または **固定額割引（例: 1万円OFF）** | 契約ハードルを下げる |
| **無料トライアル延長** | 通常30日 → **60日間** 無料トライアル | 製品/サービスの価値を十分に体験させる |
| **プレミアム機能解放** | 通常有料の上位プランを **1ヶ月間無料** で開放 | 高単価プランへのアップセル機会 |
| **コンサルティング支援** | 初回導入サポート会議 **1回無料** | B2Bにおける安心感の提供 |

> **ポイント:**
> *   各事業によって「割引」が有効な範囲が異なるため、**「紹介リンク経由で契約すると自動適用」**される仕組みが必須。
> *   被紹介者が「誰に紹介されたか」を気にしないよう、紹介者の名前はあえて伏せるか、感謝状として後日送付する形が好ましい。

## 4. 契約書テンプレート（紹介プログラム特約条項）

既存のサービス利用契約書とは別に、または主契約書に追加する「紹介プログラム参加規約」の主要条項を記載します。

---

### 【別紙】紹介プログラム参加規約（抜粋）

**第1条（目的）**
本規約は、（会社名）が提供する主力5事業（以下「本サービス」という）の紹介プログラム（以下「本プログラム」という）に参加する際、紹介者（以下「A」という）と被紹介者（以下「B」という）の権利義務を定める。

**第2条（紹介の成立）**
Aが本プログラムに登録し、発行された専用紹介リンクまたは紹介コードをBに提供し、Bがそのリンク/コード経由で本サービスの契約を締結した時点で、紹介が成立する。

**第3条（紹介者報酬）**
1. Aは、Bが本サービスとの契約を締結し、かつ初回決済が完了した時点で、以下の報酬を取得する権利を得る。
   - 報酬額: 契約金額の10%（税別）
   - 支払方法: Aの選択により、現金振込 または 本サービス利用クレジット
2. 報酬の支払いは、Bの契約が有効であること、およびAの口座情報/クレジット残高登録が完了していることを条件とする。
3. 報酬の上限は、個人は月間5万円、法人は月間20万円とする。

**第4条（被紹介者特典）**
Bは、Aの紹介により契約した場合、以下の特典を享受する。
1. 初回契約料金の50%割引（または固定額割引）
2. 無料トライアル期間の延長（通常30日→60日）
※ 特典は重複適用不可。

**第5条（不正行為の禁止）**
以下の行為を行った場合、報酬の支払いを停止し、既に支払われた報酬の返還を求めるものとする。
1. 自己紹介、または身内・関係会社間での不正な紹介
2. 機械的なクリックやボットによる契約の偽装
3. 報酬を得ることを目的とした、虚偽情報の提供

**第6条（終了）**
本プログラムは、（会社名）が事前に通知することなく終了または変更できるものとする。終了後も、終了前に成立した紹介については本規約に従う。

## 5. 運用上の注意点とベストプラクティス

### ① トラッキングの徹底
*   **ツール選定:** HubSpot, Salesforce, または専用リファラルプラットフォーム（ReferralCandy, GrowSurf等）を導入。
*   **Cookie/ID連携:** 5事業間で顧客IDを共通化し、A事業の顧客がB事業を閲覧・契約した際に正しく紐づくようにする。

### ② 税務・法務対応
*   **源泉徴収:** 個人への現金支給の場合、報酬額に応じて源泉徴収義務が発生する可能性がある。税理士と相談し、「報酬」ではなく「懸賞・当選金」として扱うか、適切な経費処理とするか決定。
*   **景品表示法:** 被紹介者への割引額が「不当に高額」とみなされないよう、適正範囲内（通常は実勢価格の10%以内の割引が安全圏）に設定。

### ③ 社内周知とインセンティブ
*   **社内販売員への報酬:** 自社営業員が他事業をクロスセルした場合にも、別途インセンティブ（例: 契約額の2%）を用意し、社内横断を促進。
*   **成功事例の共有:** 月次で「最も紹介された事業」「高額報酬を獲得した顧客」を社内チャネルで共有し、競争心を刺激。

### ④ 各事業固有のカスタマイズ例
| 事業 | 推奨報酬タイプ | 被紹介者特典の工夫 |
| :--- | :--- | :--- |
| **SaaS系** | 利用クレジット（月額割引） | 上位プランの無料トライアル |
| **コンサルティング** | 現金（成功報酬型） | 初回診断の無料実施 |
| **EC/商品販売** | ポイント（10%還元） | 送料無料券 + 初回10%OFF |
| **プラットフォーム** | 手数料免除期間 | 出品/登録料の永久免除 |
| **サブスクリプション** | 月額料金の半額免除（1ヶ月） | 期間限定プレミアム機能開放 |

## 6. 次のステップ

1.  **現状分析:** 各事業の平均契約単価、顧客獲得コスト（CAC）、既存顧客のLTVを算出。
2.  **報酬シミュレーション:** 上記案に基づき、100件紹介された場合のコストとROIを試算。
3.  **パイロット実施:** 1つの事業でテスト運用し、紹介率、契約转化率、不正事例の有無を確認。
4.  **全事業展開:** 結果を基に報酬率を調整し、全5事業へ展開。

この設計案を基に、各事業の特性に合わせて微調整してください。必要であれば、具体的な事業内容（例：ITサービス、小売、金融など）を教えていただければ、より詳細な数値設定や条項を提案できます。
