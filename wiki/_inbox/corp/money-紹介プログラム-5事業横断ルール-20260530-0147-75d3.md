ご提示いただいた「主力5事業横断ルール」に基づく紹介プログラム（リファラルプログラム）の設計案を作成しました。

複数の事業を横断して運用するため、**「統一されたプラットフォーム感」**と**「各事業固有の特性への対応」**のバランスが重要です。以下に、報酬率、特典、契約書の骨子案を提案します。

---

# 主力5事業横断型 紹介プログラム設計案

## 1. 基本方針
*   **名称案:** 「[会社名]パートナーズプログラム」または「Cross-Side Referral Program」
*   **目的:** 既存顧客・業界関係者からの信頼を活用し、5つの主力事業間のクロスセリングを促進し、顧客生涯価値（LTV）を最大化する。
*   **対象者:** 一般顧客、業界専門家、エージェント、法人顧客（B2B/B2C混合想定）

---

## 2. 紹介者報酬率（インセンティブ設計）

横断型であるため、**「一律の基準」**と**「事業固有の調整」**の2層構造が推奨されます。

### A. 基本ルール（横断共通）
| 項目 | 内容 | 備考 |
| :--- | :--- | :--- |
| **報酬形態** | 現金（銀行振込）または 商品券/ポイント | 法人向けは現金、個人向けはポイントが好まれる傾向 |
| **計算基準** | 紹介された新規契約の**初回売上（または月額課金額）**の〇% | 継続課金型の場合は、継続期間中（例：6ヶ月間）毎月支給 |
| **支払条件** | 紹介先顧客の契約成立かつ**支払い完了後** | 解約返金時の返還条項を明記 |
| **上限額** | 年間〇万円まで / 無制限 | 税務処理の簡素化のため上限を設ける場合が多い |

### B. 事業別報酬率の推奨レンジ（例）
主力5事業が「高利益率・低単価」か「低利益率・高単価」かによって異なります。以下は一般的なSaaS・サービス業を想定した例です。

| 事業カテゴリ | 推奨報酬率 | 理由・特徴 |
| :--- | :---: | :--- |
| **事業1: 高単価・長期契約型**<br>(例: 基幹システム導入) | **3% 〜 5%** | 単価が高いため絶対額が大きくなる。契約期間が長いため、リテncy（継続率）重視。 |
| **事業2: 月額サブスクリプション型**<br>(例: クラウドツール) | **20% 〜 30%**<br>(初回) | 業界標準。継続課金の場合は月々20%など。 |
| **事業3: 低単価・大量販売型**<br>(例: 電子書籍/アプリ) | **10% 〜 15%** | 単価が低いため、高いパーセンテージが必要だが、絶対額は抑える。 |
| **事業4: 中間商材・代理店型**<br>(例: 保険/金融商品) | **5% 〜 10%** | 競合他社との差別化のため、業界平均より少し高めに設定。 |
| **事業5: 新規開拓困難な専門事業**<br>(例: 特殊コンサルティング) | **10% 〜 15%** | 紹介者の専門知識が必要なため、高いインセンティブで動機付け。 |

> **💡 横断型特有の工夫:**
> *   **クロスボーナス:** 紹介者が「事業A」を介して「事業B」の顧客を獲得した場合、追加で〇%のボーナスを支給。
> *   **ランク制度:** 年間紹介実績に応じて、報酬率が1.2倍になる「VIPパートナー」制度を設ける。

## 3. 被紹介者特典（新規顧客向けオファー）

紹介者だけでなく、紹介された側（新規顧客）にも魅力があることで、紹介者が躊躇せずに紹介できるようになります。

### A. 共通特典（全5事業共通）
*   **初回割引:** 初回契約額の **10% 〜 20% OFF**
*   **無料トライアル延長:** 通常14日間 → **30日間** 延長
*   **専用サポート窓口:** 紹介経由で契約した顧客は、優先対応または専任アカウントマネージャー付き

### B. 事業別特典の例
| 事業 | 被紹介者特典案 |
| :--- | :--- |
| **事業1** | 導入支援セミナー（通常有料）を無料提供 / 初期設定サポート1ヶ月無料 |
| **事業2** | 上位プランへの無料アップグレード（3ヶ月間） |
| **事業3** | 限定コンテンツ/グッズのプレゼント |
| **事業4** | 契約書類作成代行サービス無料 / 初回審査料免除 |
| **事業5** | 初回コンサルティング料の半額割引 |

## 4. 契約書テンプレ（紹介契約書）の骨子

紹介者と会社間の法的関係を明確にするため、以下の条項を含む簡易契約書（またはWeb上の同意画面）が必要です。

### 【紹介契約書 主要条項】

#### 第1条（目的）
本契約は、紹介者（以下「A」という）が、会社（以下「B」という）の5つの主力事業（以下「対象事業」という）の新規顧客をBに紹介し、その紹介による契約成立に対して報酬を支給する関係を定める。

#### 第2条（紹介の定義）
1. 紹介とは、AがBの指定するフォーム、メール、または専用リンクを通じて、見込み顧客（以下「紹介先」という）の情報をBに提供することをいう。
2. 紹介先が既にBの顧客である場合、またはBが既に接触済みの場合、報酬は支払わない。

#### 第3条（報酬の計算と支払）
1. 報酬額は、紹介先が対象事業と契約し、かつ代金の支払いが完了したことを条件とする。
2. 報酬率は、各事業の別表（報酬率表）に定める割合による。
3. 支払いは、毎月末日締めで翌月末日頃、指定口座へ振込を行う。

#### 第4条（解約・キャンセル時の処理）
1. 紹介先の契約が解約され、代金が返還された場合、既に支払った報酬の全額または一部を返還させることができる。
2. その場合、AはBに対し、返還額を請求されるか、または次回支払うべき報酬から控除されることに同意する。

#### 第5条（不正行為の禁止）
Aは、以下の行為を行ってはならない。違反した場合、報酬を支払わないほか、損害賠償を負う。
1. 自己または関係者名義での偽装紹介。
2. 報酬を得る目的での自らの顧客情報の不正利用。
3. Bの商標や評判を毀損する行為。

#### 第6条（秘密保持）
Aは、業務上知り得たBの営業秘密、顧客情報、報酬率の詳細などを外部に漏洩してはならない。

#### 第7条（契約期間と終了）
1. 本契約は、Aが本契約に同意した日より効力を生じ、原則として継続する。
2. いずれかの当事者は、30日前までに書面（メール含む）で通知することで、本契約を終了できる。
3. 契約終了後、終了前に紹介された顧客の契約が成立した場合でも、報酬は支払われる。

## 5. 運用上のアドバイス（横断型特有の注意点）

1.  **一元管理プラットフォームの導入**
    *   5事業それぞれで別々の紹介フォームを作ると、紹介者が混乱します。
    *   **推奨:** 専用ランディングページ（LP）と紹介ダッシュボードを1つに統合し、ログイン一つで全事業の紹介状況・報酬を確認できるようにしてください。
2.  **クロスセリングの促進**
    *   「事業Aの顧客に事業Bを勧めた場合」のルールを明確にしてください。同じ顧客に対して重複して報酬を支払うとコストが膨らみます。「顧客IDの重複チェック」を厳格に行う必要があります。
3.  **税務処理の簡素化**
    *   個人紹介者への支払いは「雑所得」となり、源泉徴収（20.315%）が必要になる場合があります。
    *   対策: 報酬を「商品券」や「ポイント」で支払う、または法人紹介者（エージェント）として扱えるようにし、請求書による支払い（経費扱い）にするなどの設計が必要です。税理士と相談してください。
4.  **定期的なレポート**
    *   月次で「どの事業が最も紹介されているか」「報酬総額はどれくらいか」を分析し、施策を調整してください。
