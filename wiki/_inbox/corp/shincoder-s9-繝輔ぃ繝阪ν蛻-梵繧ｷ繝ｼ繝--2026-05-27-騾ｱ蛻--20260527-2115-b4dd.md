# AIpaX school 戦略統合メモ

## 🎯 核心戦略：EEMUS × AIpaX の「相乗効果」設計

AIpaX school（月3万円）を単なるスクールではなく、**EEMUS（静岡ローカルキャリアマッチング）の「プレ・キャリア・エリート」認定プログラム**として位置づける。

### 1. 差別化ポイント：なぜ月3万円なのか？
- **競合との価格帯の明確な分離**:
    - QUREO/Tech Kids: 月1〜3万円（プログラミング基礎）
    - AIpaX: **月3万円**（「AIを手足のように使う」実務スキル + 「17歳現役AI起業家」からの直接指導）
- **価値提案**:
    - 「プログラミングの書き方」ではなく、「AIを使ってどう社会課題を解決し、どう起業するか」の**思考回路と実行力**を教える。
    - HR高等学院のノウハウ（ストーリーテリング、合意形成、ロジックモデル）を統合し、**「現場で使えるビジネススキル」**として提供。
    - **大井（17歳現役AI起業家）の直接指導**は、既存スクールにはない「リアルなロールモデル」であり、中高生・親にとっての最大のフック。

### 2. ターゲットとペインの再定義
- **中高生本人**:
    - 「学校より現場で学びたい」「AIを使って何か作りたい」「将来の選択肢を広げたい」
    - EEMUSと連動することで、「静岡の企業でインターン→就職」への具体的なパスが見える。
- **親（子のAI時代不安）**:
    - 「AIに仕事を奪われるのではないか」「子供がAIを正しく使えるか不安」
    - **解決策**: AIを「道具」として使いこなすスキルを身につけさせることで、不安を「競争力」に変える。
    - **安心材料**: HR高等学院のノウハウ統合（教育の質の担保）+ 大井の実績（17歳で起業・月収100万円目標の実現可能性）。

### 3. カリキュラム v2 の強み（HR高等学院ノウハウ統合）
- **Module 1 (Month 1-2)**: AIマインドセット + バイブコーディング体験
    - **F1 ストーリーテリング 3軸**: 普通の思考×武器×提案 → 自己PR・アイデア出しの基礎。
- **Module 2 (Month 3-4)**: 日常業務自動化 + チーム合意形成
    - **F2 合意形成 4ステップ**: チームプロジェクトでの対人スキル。
- **Module 3 (Month 5-6)**: エンジニアリング思考 + 個人探究テーマ設定
    - **F3 11項目ロジックモデル**: 卒業課題（個人探究）の設計。
- **Module 4 (Month 7-8)**: 本格アプリ開発 + サービス企画
    - **F4 CHINTAI型 5項目企画**: 具体的なサービス企画のフレームワーク。
- **Module 5 (Month 9-12)**: データ分析 + マネタイズ + 卒業課題発表
    - **F5 産業課題×AI 4項目**: 業界分析とビジネスモデルの構築。

### 4. EEMUSとの連動による付加価値
- **EEMUS プレスクール**: 企業インターン前に「AIリテラシー」を身につけることで、企業へのアピール力が大幅アップ。
- **企業側へのメリット**: 「AI使える高校生」は、企業のDX推進や新規事業開発の視点を持てるため、採用時の選考が有利。
- **シナジー**: AIpaXで学んだスキルをEEMUSで実践 → 企業からの評価 → 就職/インターン継続 → AIpaXの卒業生ネットワークへ。

### 5. 次の一手（アクションプラン）
1. **カリキュラム1本目の設計**: 「AIツール入門」30分コース（無料体験）を作成。
2. **EEMUS連携の具体化**: 静岡の高校1〜2校、企業1〜2社と連携し、ベータテスト（10人）を実施。
3. **講師・メンター候補のリストアップ**: HR高等学院の社会人ネットワークを活用。
4. **マーケティング**: SNS（TikTok/YouTube）で「17歳現役AI起業家・大井」のリアルな日常と指導風景を発信。

### 6. 収益化ロードマップ
- **Phase 1 (現在〜)**: 無料コンテンツ配信 → 認知獲得。
- **Phase 2 (1〜3ヶ月)**: β版スクール（10〜20人）でコース検証。
- **Phase 3 (4〜6ヶ月)**: 有料化（月3万円）正式ローンチ。
- **Phase 4 (6ヶ月後〜)**: 法人研修（企業の新卒向けAI研修パッケージ）へ展開。

この戦略により、AIpaX schoolは「単なるプログラミングスクール」ではなく、「AI時代のリーダーを育成するエリート・キャリア・プレスクール」としてのポジションを確立できる。
