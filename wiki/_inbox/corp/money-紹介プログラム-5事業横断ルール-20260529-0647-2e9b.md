主力5事業を横断する「紹介プログラム（リファラルプログラム）」の設計案を作成しました。
事業領域が異なる場合でも、**「共通のルール」**で運用し、管理コストを抑えつつ、各事業の特性を尊重するハイブリッド型のアプローチを提案します。

---

# 主力5事業横断型 紹介プログラム設計書

## 1. 基本方針
*   **横断性:** 1つのアカウント（紹介者）で5事業すべてをカバー。報酬は「事業単位」ではなく「紹介成功単位」で統一または柔軟に選択可能。
*   **簡素化:** 契約書は「マスター契約書」1通で完結し、個別事業の細則はWebページまたは別添で管理。
*   **インセンティブ:** 紹介者・被紹介者双方に魅力的な特典を設定し、ネットワーク効果を最大化。

---

## 2. 紹介者報酬率（インセンティブ設計）

報酬体系は、**「定額型（成功報酬）」**と**「継続型（リベート）」**の2パターンを併用し、事業特性に合わせて選択できるようにします。

| 区分 | 報酬タイプ | 内容 | 適用例 |
| :--- | :--- | :--- | :--- |
| **紹介者** | **A. 定額成功報酬** | 契約成立時に一括支給 | 新規顧客獲得を最優先したい場合<br>（例：SaaS導入、コンサルティング） |
| | **B. 継続リベート** | 契約期間中、月額売上の○%を継続支給 | 長期的なリレーション維持が重要<br>（例：クラウドサービス、定期メンテナンス） |
| | **C. 混合型** | 初月定額 + 翌月以降リベート | 初期導入ハードルを下げつつ継続を促す |

### 推奨報酬率の目安（業界平均ベース）
※各事業の粗利構造に合わせて調整してください。

| 事業カテゴリ | 推奨報酬率（定額の場合） | 推奨リベート率（継続の場合） | 備考 |
| :--- | :--- | :--- | :--- |
| **事業A: 高額B2Bソリューション** | 契約額の **5%〜10%** | 2%〜3%（最大12ヶ月） | 成約単価が高いため、定額が有効 |
| **事業B: 定期課金SaaS** | 月額単価の **100%〜150%** | 10%〜15%（最大12ヶ月） | 解約防止が鍵。継続報酬が重要 |
| **事業C: 人材・コンサルティング** | 1件あたり **固定額（例: 5万円〜）** | なし | 業務の複雑さにより定額が管理しやすい |
| **事業D: 小売・EC関連** | 売上額の **3%〜5%** | なし | 高額な定額報酬は利益を圧迫するため低め |
| **事業E: 保険・金融商品** | 初回手数料の **20%〜30%** | なし | 業界慣習に準拠。厳格なコンプライアンス必要 |

> **💡 横断運用のポイント:**
> 紹介者は「どの事業で紹介したか」をアプリまたはWebフォームで選択して報告。報酬は**「共通ポイント」**として付与し、後で現金化または各事業のサービス利用券と交換できるようにすると、柔軟性が高まります。

---

## 3. 被紹介者特典（新規顧客優遇）

被紹介者（新規顧客）にも魅力的な特典を用意し、紹介者の背中を押します。

| 特典タイプ | 内容例 | 効果 |
| :--- | :--- | :--- |
| **割引特典** | 初回契約額の **10%〜20%OFF** | 即効性のある誘因 |
| **無料期間延長** | 無料トライアル期間を **1ヶ月延長** | 検討期間の拡大 |
| **付加サービス** | 初期設定サポート無料 / 追加モジュール無料 | 手間とコストの軽減 |
| **ギフト券** | Amazonギフト券 **3,000円〜5,000円** | 汎用性が高く、心理的ハードル低下 |

> **💡 横断運用のポイント:**
> 「5事業共通のギフト券」または「ポイント」を提供することで、被紹介者がどの事業を選んでもメリットを感じられます。

---

## 4. 契約書テンプレート（マスター契約書）

個別事業ごとに契約書を作成せず、**「リファラル紹介プログラム参加契約書」**を1通に統一します。各事業の詳細は「別紙1：事業別特約事項」で管理します。

---

### 【契約書テンプレート】リファラル紹介プログラム参加契約書

**契約日:** 202_年__月__日

**甲:** [会社名]（以下「甲」という）
**乙:** [紹介者氏名/会社名]（以下「乙」という）

甲と乙は、甲が運営する主力5事業（以下「対象事業」という）の新規顧客獲得を目的とする紹介プログラム（以下「本プログラム」という）に関する以下の通り契約を締結する。

#### 第1条（目的）
乙は、甲の対象事業を自らのネットワークを通じて紹介し、甲は乙に対して本プログラムに基づき紹介報酬を支払うものとする。

#### 第2条（紹介の定義）
本プログラムにおける「紹介成功」とは、乙が指定するフォーム等を通じて甲に新規顧客（以下「被紹介者」という）を登録させ、被紹介者が甲の対象事業と契約を締結し、かつ契約が有効に存続している状態をいう。

#### 第3条（紹介報酬）
1. 甲は、紹介成功ごとに乙に対し、以下の通り紹介報酬を支払う。
   - **報酬額:** 別紙1「事業別報酬率表」に定める金額または割合による。
   - **支払時期:** 被紹介者の契約代金が甲に全額入金された月の翌月末日。
2. 報酬は、原則として銀行振込により支払う。

#### 第4条（被紹介者特典）
甲は、乙の紹介により新規契約を締結した被紹介者に対し、別紙1「被紹介者特典表」に定める特典を提供する。この特典の提供に伴う費用は甲が負担する。

#### 第5条（禁止行為）
乙は、以下の行為を行ってはならない。
1. 虚偽の情報による紹介。
2. 自らの利益のために被紹介者を欺く行為。
3. 甲の業務を妨害する行為。
4. 本プログラムを第三者に譲渡または転貸すること。

#### 第6条（契約の解除）
乙が前条の禁止行為を行った場合、甲は事前通知なく本プログラムからの退出を命じ、未払いの報酬を支払わない権利を有する。

#### 第7条（準拠法及び管轄裁判所）
本契約の解釈に起因する紛議については、日本法を準用し、甲の本店所在地を管轄する裁判所を第一審の専属的合意管轄裁判所とする。

## 5. 運用上のアドバイス

1.  **管理プラットフォームの選定:**
    *   手動で管理すると事務負荷が高いです。ReferralCandy、PartnerStack、または自社開発のポータルサイトを活用し、「紹介リンクの自動発行」「成約の自動追跡」「報酬の自動計算」を実現してください。
2.  **コンプライアンス対応:**
    *   金融・医療・保険など規制の強い事業が含まれる場合は、各業界のガイドライン（景品表示法、業法など）に準拠した文言を別紙に追加する必要があります。
3.  **紹介者のエンゲージメント向上:**
    *   「紹介ランキング」や「達成率に応じたボーナス」を導入し、競争心を刺激します。
    *   四半期ごとに紹介実績優秀者に特別イベントへの招待などを行うと、ロイヤルティが高まります。

この設計案をベースに、各事業の具体的な数字やブランドカラーに合わせて調整してください。必要に応じて、個別事業の詳細な報酬率表を作成することをお勧めします。
