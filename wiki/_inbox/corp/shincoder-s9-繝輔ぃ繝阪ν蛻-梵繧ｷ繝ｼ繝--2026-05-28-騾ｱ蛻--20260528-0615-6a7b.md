# AIpaX school 戦略統合メモ

## 🎯 核心戦略：EEMUS × AIpaX のシナジー

**「AIpaX school」は単なるスクールではなく、EEMUSの「プレ・キャリア・イネーブラー」として位置づける。**

### 1. 問題の再定義
- **現状**: 中高生のAI不安は「スキル不足」ではなく「活用の場（キャリア）との接続不全」にある。
- **解決**: AIスキル習得 → EEMUS経由で地元企業インターン → 即戦力としての評価 → 就職/起業への橋渡し

### 2. 価格設定の正当化（月3万円）
- **従来スクールとの差別化**:
  - QUREO/Tech Kids: 月1〜3万円（プログラミング基礎）
  - AIpaX: 月3万円（**AI実践 + キャリア接続 + 17歳現役起業家メンタリング**）
- **価値提案**:
  - 「AIを手足のように使う」実践スキル
  - EEMUS経由の地元企業インターンへの優先アクセス
  - 大井（17歳現役AI起業家）の直接指導・HR高等学院ノウハウ統合

### 3. カリキュラムv2の強み
- **HR高等学院ノウハウ統合**:
  - F1 ストーリーテリング 3軸 → 自己PR・プレゼン力
  - F2 合意形成 4ステップ → チームワーク・リーダーシップ
  - F3 11項目ロジックモデル → 課題解決力
  - F4 CHINTAI型 5項目企画 → サービス企画力
  - F5 産業課題×AI 4項目 → 業界分析力
- **5モジュール構成**:
  - Month 1-2: AIマインドセット + バイブコーディング体験
  - Month 3-4: 日常業務自動化 + チーム合意形成
  - Month 5-6: エンジニアリング思考 + 個人探究テーマ設定
  - Month 7-8: 本格アプリ開発 + サービス企画
  - Month 9-12: データ分析 + マネタイズ + 卒業課題発表

### 4. ターゲット層の明確化
- **高校生**: EEMUS連動（インターン×AI教育）。大学受験「情報」科目対策にも有効
- **中学生**: 中学プログラミング必修化の上位版として
- **小学生（高学年）**: 「AIで何か作れる」体験を先に与える

### 5. 収益化ロードマップ
1. **無料コンテンツ配信**: SNS（TikTok/YouTube）でAIスキル解説 → 認知獲得

3. **有料化**: 月額制スクール正式ローンチ
4. **法人研修**: 企業の新卒向けAI研修パッケージ
