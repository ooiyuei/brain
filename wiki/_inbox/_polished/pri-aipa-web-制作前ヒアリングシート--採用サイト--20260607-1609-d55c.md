# AIpa Web ヒアリングシート（採用サイト・制作前）

このシートにご記入いただくと、打ち合わせを増やさずに制作を進められます。
回答をもとにAIが構成・採用コピー・初稿を作成し、最終確認の上で**最短1週間で公開**します。

- わからない項目は **「未定」「相談したい」「提案してほしい」** で構いません。
- 給与・待遇・勤務条件など事実に関わる項目は、公開前に必ずもう一度確認いたします。

《要記入: 提出方法・期限・問い合わせ先（メール/電話）を確定してから配布》

## ご記入後の流れ

```
①フォーム回答 → ②AIが構成・コピー・初稿を作成 → ③当社が事実確認
→ ④御社確認（修正は1回まで） → ⑤公開（ドメイン/SSL/応募通知設定）→ ⑥月額運用
```

> 修正は1回までを基本とし、大幅な変更・回数追加はオプションで承ります。
> 給与・待遇・応募条件はトラブル防止のため、公開前に必ず御社の最終確認をいただきます。
