# AIpa Web 発注前FAQ（採用サイト制作）

> 想定: 商談前後に送る「発注前のよくある質問」。価格・プラン・条件は entity 実データに準拠。料金幅と数字の確定は《要記入》で大井判断待ち。

## Q1. 料金はいくら？

3プランあります。

- **ライト（採用LP・1ページ/1職種）**: 初期 9.8〜19.8万 / 月額 9,800〜19,800円
- **スタンダード（採用サイト・複数ページ/複数職種）**: 初期 29.8〜49.8万 / 月額 29,800〜49,800円
- **プレミアム（採用ブランディング・撮影/動画/広報込み）**: 初期 69.8〜120万 / 月額 79,800〜150,000円

月額には制作後の運用・更新が含まれます。

**モニター枠**: いま3社限定で、通常30〜50万円相当の採用LPを **初期15万円** でやってます（事例掲載に協力いただける条件）。

## Q2. どれくらいで公開できる？

最短7日です。フォームに入力 → AIが構成・コピー・文章の初稿を作る → こちらが事実確認 → お客様確認（修正1回まで）→ 公開、という流れ。打ち合わせを何度も重ねる受託とは進み方が違います。

## Q3. 「AI応募者対応」って何？

応募者が抱く質問（給与/勤務地/応募条件など）に、サイト上でAIが24時間答える機能です。求人広告から来た人が「ここで聞けば分かる」状態を作る。採用サイトでここまでやってるところはまだ少ないので、応募の取りこぼしを減らせます。

《要記入: 実装方式（チャットUIか/FAQ自動生成か）と、提供がどのプランからかを大井確定》

## Q4. ふつうのWeb制作と何が違う？

「AIで作る」サービスではなく、**フォームでヒアリングを標準化 → AIが初稿 → 人間が事実確認 → 月額で運用** というオペレーション全体を商品にしてます。だから安い・早い・やりとりが少ない。位置づけは制作会社というより「社外のWeb担当者」です。

## Q5. 解約はできる？

《要記入: 契約期間と中途解約の扱いを大井確定（entityにモニター契約テンプレあり=`05_corp/legal/contracts/2026-05-26-aipa-web-monitor-10man-template.md` を参照して記入）》

ドラフトには「契約1年・残月数分を支払い」とありましたが、entity に裏付けがないため確定前です。

## Q6. こちらで用意するものは？

社員インタビュー・社長メッセージ・福利厚生・募集要項など、会社の事実情報をご提供ください。AIがそれをもとに文章化します。原稿そのものは不要です（ゼロから作成はオプション）。写真・ロゴは使用許可の確認をフォームでお願いします。

## Q7. 効果は測れる？

応募数・ページビュー・応募導線の通過率などをレポートします。数字を見て、募集要項や導線の改善提案もします（提案の範囲はプランによって変わります）。

## Q8. 公開後のサポートは？

運用CSが対応します。募集要項の変更・写真差し替え・お知らせ更新などは月額に含まれます。更新の回数・対応範囲はプランごと（ライト=月1回軽微修正 / スタンダード=月2〜4回更新+改善提案 / プレミアム=月次レポート+定例MTG）。
