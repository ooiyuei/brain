# Testall ユーザーペルソナ 5種

> テスト結果を、次の45分でやるべき勉強に変える。受験戦略OS。
> ターゲット = 高校1〜3年の独学受験生（塾なし層 ≈70万人）+ その親。

**製品前提（entities/testall.md より）**
- コアバリュー: テスト返却 → 弱点特定 → **今日の45分タスクに自動変換**（Claude AI診断）
- 機能: テスト追加 / AI診断 / 集中モード（45分タイマー） / 週計画 / マイページ（連続学習日数）
- 料金: Free（テスト3回・基本診断）/ スタンダード 500円/月（無制限・AI診断・集中モード）/ プレミアム 1,500円/月（詳細分析・優先サポート）
- スマホメイン（下部5タブUI）

ペルソナは「誰が払い、誰が使い、どの機能でどう刺さるか」で切り分けた。属性のバリエーションではなく**課題と決済者のバリエーション**で5種を立てる。

---

## ペルソナ1 — 自走したい高2「仕組みで上げたい」層（中核ターゲット）

| 項目 | 内容 |
|------|------|
| プロフィール | 高2・男子・16歳。塾なし。地方公立。スマホは1日4時間以上。 |
| 学力帯 | 中堅国公立を狙える偏差値55前後。やる気はあるが計画が続かない。 |
| コア課題 | テストは受けるが「次に何を復習すべきか」を自分で設計できない。市販の学習管理アプリは「何を勉強するか」を自分で決めさせるので、結局続かない。 |
| 利用シーン | 定期テスト・模試が返却された夜。点数と分野別ミスを入力 → AI診断 → その場で「今日の45分」を着手。 |
| 刺さる機能 | AI診断 → 集中モード（45分タイマー）→ 連続学習日数。 |
| 決済者 | **本人**（おこづかい / バイト代）。価格感度が高い → スタンダード500円が刺さる。 |
| 解決の瞬間 | 「やる気」ではなく「仕組み」で動けた、と本人が実感する初日。テスト入力から着手まで一気通貫で迷いが消える。 |

| 項目 | 内容 |

| 利用シーン | 模試返却後と週初め。週計画で全体配分を見つつ、日々はAI診断が出した優先度マトリクスで45分を回す。 |
| 刺さる機能 | AI診断（弱点優先度マトリクス）→ 週計画 → 集中モード。 |

| 解決の瞬間 | 「捨てる科目・伸ばす科目」が数字で示され、迷いの時間がゼロになる瞬間。 |

## ペルソナ4 — 子の勉強を支えたい親（決済者ペルソナ・親）

| 項目 | 内容 |
|------|------|
| プロフィール | 高1〜高2の子を持つ40代の親。塾は高い／合わなかったので入れていない。教育熱はあるが勉強を教える時間も知識もない。 |
| コア課題 | 子に「勉強しなさい」しか言えず、何をどう支えればいいか分からない。塾代（月数万円）は重いが、放置も不安。 |
| 利用シーン | 子のテスト結果を見て、塾の代わりになる仕組みを探している時。月500〜1,500円なら塾より圧倒的に安く、AIが診断まで自動でやる点に価値を感じる。 |
| 刺さる訴求 | 「塾の10分の1以下のコストで、テスト結果から今日やるべき勉強が決まる」「連続学習日数で続いているかが親にも見える」。 |
| 決済者 | **親**（本人は使うが払わない）。価格より「子が続けられるか」「成果が見えるか」を重視 → プレミアム1,500円も許容。 |
| 解決の瞬間 | 「勉強しなさい」と言わなくても子が机に向かう仕組みができ、課金が「安い」と納得できた時。 |

## ペルソナ横断サマリー

| # | 学年 | 中核課題 | 主決済者 | 想定プラン | 接点 |
|---|------|----------|----------|-----------|------|
| 1 | 高2 | 仕組みで自走したい | 本人 | スタンダード | オーガニック / 紹介 |
| 2 | 高3 | 受験戦略の逆算 | 本人 or 親 | プレミアム | 受験期検索 |

| 5 | 高2 | 継続できない / 映え起点 | 本人 | スタンダード | TikTok GTM |

**設計上の含意**
- 決済者は「本人」と「親」の二系統。LP・課金導線は**子向け（自走・映え・継続）**と**親向け（塾代比較・成果可視化）**で文面を分ける。
- 全ペルソナに共通するキラー機能は **AI診断 → 集中モード（45分）** の一気通貫。ここが体験のコア。離脱を防ぐのは**連続学習日数**。
- 受験期（夏〜秋）はペルソナ2と親（4）の課金意欲が跳ねる。TikTok投下と課金訴求をこの時期に集中。
