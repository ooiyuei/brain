## 🎯 30秒でわかるAIpaXとは？

AIpaXは、**中小企業向けのAI導入支援プラットフォーム**です。  
**「会社別AI保存庫コア」と「業務AIツール」**を組み合わせることで、  
**AIをゼロから構築するのではなく、AIを「使いやすく」「導入しやすく」**する仕組みを提供します。

- **対象**: 中小企業の経営者（AI導入意欲あり・予算50万出せる）
- **価格**: 初回50〜80万円 + 月額5万円（初回事例価格）
- **状態**: 既に4社契約済み・5社目営業中
- **目的**: AI導入の「5%問題」を解決し、中小企業のAI活用を実現

## 🔍 AI導入の「5%問題」とその解決

中小企業のAI導入は、**「5%問題」**と呼ばれる課題があります。  
つまり、**導入企業の5%しか成功していない**という現状です。

### なぜ5%しか成功していないのか？

1. **AIの導入が難解**  
   AIは技術的なハードルが高く、中小企業が自社で導入するのは困難です。

2. **AIの使い方が分からない**  
   AIを導入しても、使い方が分からないと、導入した意味がありません。

3. **導入コストが高すぎる**  
   多くの中小企業は、AI導入の初期費用や月額コストを負担できません。

4. **導入後の運用が困難**  
   AIを導入しても、運用がうまくいかないと、導入した意味がありません。

---

## ✅ AIpaXが解決する「5%問題」

AIpaXは、**中小企業のAI導入を「ゼロから」ではなく、「AI保存庫コア＋業務ツール」で実現**します。

### 1. AI保存庫コアで「AIの基盤を整える」

AI保存庫コアとは、  
**会社ごとの資料・契約・料金・FAQ・顧客情報・例外条件・やり取り履歴**などを、  
AIが検索・参照・再利用できる形に整理する**共通基盤**です。

- **AIが勝手に情報を作らない**  
  顧客の資料は顧客が管理し、AIはそれを参照・利用します。
- **AIが勝手に正式情報を作らない**  
  AIは情報の検索・整理をしますが、正式情報は顧客が決定します。
- **AIが勝手に情報を更新しない**  
  顧客が更新し、AIはそれを反映します。

### 2. 業務AIツールで「AIを実際の業務に応用」

AI保存庫コアの上に、**業務に応じたAIツール**を追加します。

| 顧客の課題 | 最初に提案するツール |
|----------|------------------|
| 営業電話で回答がズレる | リアルタイム電話/営業回答支援 |
| 新人がルールを何度も聞く | 社内AIチャット/新人教育AI |
| 商談後の整理が遅い | 議事録整理AI |
| 問い合わせ属人化 | CS対応AI |
| フォロー漏れ多い | フォロー文生成AI |
| 管理者が現場見たい | 管理レポートAI |

## 💰 価格の根拠と特典

### 価格設定の根拠

AIpaXの価格は、**「AI導入のコストを最小限に」**と**「AIの実用性を最大限に」**を両立させるため、  
**「AI保存庫コア＋最初の業務ツール」**をベースに設定されています。

| 段階 | 初期 | 月額 |
|------|------|------|
| 初回事例価格 | 50〜80万円 | 5万円前後 |
| 通常価格 | 100〜150万円 | 10〜15万円以上 |
| 本格導入 | 200〜500万円 | 20〜50万円 |

### 価格決定要素

- 管理対象領域 / 更新頻度 / ツール数 / 利用量・実費 / 外部連携 / 運用対応速度
- 紙資料・OCR / 大量旧資料整理 / 自社側か顧客側か

→ **対象領域内なら、顧客別条件や例外条件は削らない**（そこまで扱えるのが保存庫の価値）
