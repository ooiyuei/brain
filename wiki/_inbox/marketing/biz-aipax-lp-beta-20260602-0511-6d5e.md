## 🎯 30秒でわかるAIpaXとは？

AIpaXは、**中小企業向けのAI導入支援サービス**です。  
**「会社別AI保存庫コア」**と**「業務AIツール」**を組み合わせることで、AIを実際の業務に応用できる環境を構築します。  
**中小企業の経営者**を対象に、**AI導入意欲あり**かつ**予算50万円出せる**企業を対象に、**β顧客5社募集**を開始します。

---

## 🔍 AI導入の現状とAIpaXの解決策

### 中小企業のAI導入課題：「5%の壁」

中小企業において、AI導入の成功は**「5%の壁」**に阻まれています。  
その理由は、**AIの導入コストが高すぎる**、**導入後の運用が複雑**、**AIの成果が見えにくい**などです。

AIpaXは、**中小企業の経営者にとって実現可能なAI導入**を実現するため、**初期費用50〜80万円**、**月額5万円**で、**AI保存庫コア＋業務ツール**を提供します。

## 💰 価格の根拠：50〜80万円（初期）＋月額5万円

### 価格の決定要素

| 要素 | 説明 |
|------|------|
| 管理対象領域 | 対象領域の広さが価格に影響 |
| 更新頻度 | 頻繁に更新される資料はコストが高め |
| ツール数 | 業務ツールの数が価格に影響 |
| 利用量・実費 | 使用量や実費の負担が価格に影響 |
| 外部連携 | 外部システムとの連携が必要な場合は価格が高め |
| 運用対応速度 | 高速対応が必要な場合は価格が高め |
| 紙資料・OCR | 紙資料のOCR処理が必要な場合は価格が高め |
| 大量旧資料整理 | 大量の旧資料を整理する必要がある場合は価命が高め |
| 自社側か顧客側か | 自社側で管理するか顧客側で管理するかが価格に影響 |

### 価格の根拠：中小企業のAI導入コストを下げた

AI導入の初期費用は、**「AIツールの単体導入」**が主流ですが、**AIpaXは「AI保存庫コア＋業務ツール」**を組み合わせることで、**AI導入のコストを大幅に下げました**。

- **AIツール単体導入**：100〜150万円（初期）＋月額10〜15万円
- **AIpaX**：50〜80万円（初期）＋月額5万円

**AIpaXは、AI導入の初期費用を50〜80万円に抑え、月額も5万円と低コスト**で、中小企業がAIを導入できる環境を提供しています。
