# 主力5事業 SEO記事戦略マトリックス

## 前提と方針
月収100万期限（2026-06-21）まで残り1ヶ月。SEOは長期戦だが、**「検索流入」ではなく「検索エンジン最適化された営業資料・LP」**として機能させる。
つまり、SEO記事そのものを収益化せず、**「信頼の証」として商談で使う**のが正解。

対象5事業：[[entities/aipax-school]] / [[entities/testall]] / [[entities/aipax]] / [[entities/eemus]] / [[entities/agents-of-flag]]

## 各事業の詳細戦略

### 1. [[entities/aipax-school]]：親の「将来不安」を解消する記事
**ターゲット**: 中高生の保護者（30-40代、情報収集力が高い）
**核心**: 「AIに仕事を奪われる」不安を「AIを操る側になる」希望に変える。

- **記事タイトル案**: 「2030年、AIに仕事を奪われないために。中高生が今やるべき3つのこと」
- **構成**:
    1. 現状：AIの進化と雇用への影響（一般論）
    2. 問題：従来の教育（暗記）では勝てない
    3. 解決：[[entities/aipax-school]]のカリキュラム（HR高等学院のノウハウ統合）
    4. 証拠：大井自身の実績（17歳で月収100万の道筋）
    5. CTA: 無料体験申込（親向け説明会）
- **明日使う1人**: 大井。保護者向け説明会でこの記事をスクリーン共有し、「僕もこうやって来た」と語る。

### 2. [[entities/testall]]：受験生の「孤独」をAIが埋める
**ターゲット**: 高校1-3年、浪人生（スマホ依存層）
**核心**: 「勉強は才能よりマネジメント」。テスト結果を次のアクションに変える。

- **記事タイトル案**: 「テストの点数が上がらない理由。それは『復習』ではなく『戦略』が足りないから」
- **構成**:
    1. 問題：テストを受けても何が悪いかわからない
    2. 解決：Testallの仕組み（テスト入力→AI診断→45分タスク）
    3. 比較：Studyplusとの違い（計画 vs 実行）
    4. 証拠：大井の受験経験とFounder-Market Fit
    5. CTA: ベータ登録（先行特典）
- **明日使う1人**: 大井。受験生向けSNS（X/TikTok）で記事の抜粋を配信し、LPへ誘導。

### 3. [[entities/aipax]]：中小企業の「AI導入」を具体化する
**ターゲット**: 中小企業経営者（50-200名規模）
**核心**: 「AIは怖い」ではなく「AIは会社別保存庫で安全に使える」。

- **記事タイトル案**: 「中小企業がAIを導入する際、最も失敗する3つのポイントと回避策」
- **構成**:
    1. 問題：AIツールは多いが、自社のデータと紐つかない
    2. 解決：[[entities/aipax]]の「会社別AI保存庫コア」概念
    3. 事例：4社契約済みの実績（匿名で構造を示す）
    4. 証拠：Notion AIとの違い（実行追跡 vs 記録）
    5. CTA: 無料ヒアリング（保存庫設計図の提供）
- **明日使う1人**: 大井。既存顧客4社へのフォローアップメールに記事へのリンクを添付し、信頼を深める。

### 4. [[entities/eemus]]：静岡の「人材流出」を止める
**ターゲット**: 静岡の地元企業、高校
**核心**: 「東京に行かなくても、静岡で未来がある」。

- **記事タイトル案**: 「静岡県外進学率71%の現実。地元企業が見落としている『高校生との接点』」
- **構成**:
    1. 問題：静岡の高校生が東京へ流出し、地元企業が人手不足
    2. 解決：[[entities/eemus]]のマッチングモデル
    3. 証拠：夢AWARD 2026応募予定の企画書（公開可能な部分）
    4. 比較：Wantedlyとの違い（ローカル特化）
    5. CTA: 企業向け説明会申込
- **明日使う1人**: 相川。静岡の商工会議所や地元企業への営業資料として使用。

### 5. [[entities/agents-of-flag]]：リアルeスポーツの「夢」を可視化
**ターゲット**: ゲーマー、スポンサー、施設オーナー
**核心**: 「サバゲーの次は、これだ」。競技OSとしての革新性。

- **記事タイトル案**: 「サバゲーの興奮を、ゲームのように。リアルFPSスポーツ『Agents of Flag』の構想」
- **構成**:
    1. 問題：既存サバゲーの痛み・装備差・自己申告
    2. 解決：AOFの仕組み（A/Bフラッグ、ヘッドショット音、観戦UI）
    3. 証拠：PoC準備中の詳細仕様（デバイス、ルール）
    4. 比較：既存光線銃施設との違い
    5. CTA: 面談申込（ホットスプリングス等）
- **明日使う1人**: 大井。スポンサー候補への提案資料として使用。

## 検証アクション（3ヶ月以内）

1. **記事作成**: 上記5記事のドラフトを1週間で作成（大井が書く）
2. **LP化**: 記事をLP化し、CTA（申込フォーム）を設置
3. **配信**: SNS（X/TikTok）で記事へのリンクを配信
4. **検証**: 1ヶ月後に申込数・商談数を確認

## 注意点
- **証拠**: 大井の実績（月収100万の道筋、4社契約）を必ず記載
- **競合**: 既存サービスとの違いを明確に（価格、機能、ターゲット）
- **明日使う1人**: 各事業で明確に指定し、即時活用

## 結論
SEO記事は「検索流入」ではなく「営業の武器」として機能させる。
各事業の核心価値を1記事に凝縮し、明日から営業で使い倒す。
