## 🧠 AIpaXとは？（論理重視・信頼性訴求）

AIpaXは、**中小企業のAI導入を支援するプラットフォーム**です。  
**単発のAIツールを作ることではなく、まず顧客企業ごとに「会社別AI保存庫コア」を作り、その上に必要な業務AIツールを追加していく**という、**長期的なAI導入戦略**を提供します。

---

## 📌 AI導入の現状と課題（中小企業のAI導入5%問題）

中小企業のAI導入率は**5%未満**とされています。  
その理由は、**AI導入のハードルが高い**ことが主な原因です。

- **AI導入のコストが高すぎる**
- **AIの用途が不明確で、導入の価値が分からない**
- **AIの導入が業務に合致しない**
- **AI導入後の運用が困難**

---

## 🎯 AIpaXの解決策

AIpaXは、**中小企業のAI導入を低コストで実現**するためのプラットフォームです。

### ✅ 3つの解決ポイント

1. **AI導入の初期コストを抑える**  
   AIpaXは、**初期導入費用を50-80万円**と設定し、中小企業でも導入可能な価格にしています。

2. **AI導入の目的を明確にする**  
   AIpaXは、**顧客企業の業務に合わせたAIツールを提供**し、導入の価値を明確にします。

3. **AI導入後の運用をサポートする**  
   AIpaXは、**月額5万円**でAIツールの運用をサポートし、長期的な導入を可能にします。

## 💰 価格の根拠（50-80万円）

AIpaXの価格は、**中小企業のAI導入を実現するためのコスト**を考慮し、**50-80万円**と設定されています。

### 📌 価格決定要素

| 要素 | 説明 |
|------|------|
| 管理対象領域 | 顧客企業の業務に応じたAI保存庫の構築 |
| 更新頻度 | 顧客企業の資料更新頻度に応じたAIの学習 |
| ツール数 | 必要な業務AIツールの数 |
| 利用量・実費 | AIの利用頻度と実費に応じた価格 |
| 外部連携 | 他システムとの連携が必要な場合 |
| 運用対応速度 | AIの運用対応速度に応じた価格 |
| 紙資料・OCR | 紙資料のOCR処理が必要な場合 |
| 大量旧資料整理 | 大量の旧資料の整理が必要な場合 |
| 自社側か顧客側か | 自社側での運用か、顧客側での運用か |

## 📌 5原則（信頼性訴求）

1. **保存庫コアとツール出力を混ぜない**
2. **顧客資料の正しさは顧客側が決める**
3. **AIが勝手に正式情報を作らない**
4. **不足・矛盾・最新版不明を検出して提示する**
5. **対象領域内の顧客別条件・例外条件は削らない**
