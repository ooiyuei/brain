# AIpa Web 静岡県商工会議所経由営業文

---

## 📌 タスク概要

- **目的**: 静岡県の中小企業向けに「AIpa Web」を商工会議所経由で営業
- **対象**: 建設・製造・運輸・サービス業（従業員30〜300名）
- **価格**: 初期15万円（モニター限定）→ 月額10万円（通常プラン）
- **強み**: AIを活用した採用サイト制作・運用代行、7日で公開可能
- **キーメッセージ**: 「フォーム入力だけで、採用サイトを最短7日公開」

---

### 📩 件名  
**AIpa Web：中小企業向けAI採用サイト制作・運用代行サービス（静岡県商工会議所経由）**

---

### 📄 本文

静岡県商工会議所 拝啓

いつもお世話になっております。

本日は、中小企業の採用課題を解決するための新サービス「AIpa Web」について、ご案内させていただきます。

---

### 📌 1. なぜ「AIpa Web」が必要なのか？

静岡県の中小企業様には、以下のような課題がございます。

- **人材不足**：建設・製造・運輸・サービス業では、現場の崩壊が深刻
- **採用の手間**：Webサイトの制作・運用に時間がかかる
- **コストの負担**：高額な制作費と長期的な納期

---

### 📌 2. AIpa Web とは？

**AIpa Web** は、中小企業向けの「AI採用サイト制作・運用代行サービス」です。

- **AIを活用したWeb制作**：フォーム入力だけで、採用サイトを最短7日で公開
- **月額運用サポート**：社外Web担当者として、求人票・応募導線・更新まで代行
- **価格**: 初期15万円（モニター限定）→ 月額10万円（通常プラン）

---

### 📌 3. AIpa Web の特長

| 特長 | 内容 |
|------|------|
| **AIで制作** | フォーム入力だけで、AIが構成・文章・デザインを生成 |
| **7日で公開** | 通常の制作に比べて、大幅に短縮 |
| **月額運用** | 求人票・応募導線・更新まで、月額で対応 |
| **低コスト** | 通常の制作費（30〜50万円）と比較して、大幅に低コスト |
| **地域貢献** | 地方の中小企業を支援し、地域経済の活性化に貢献 |

---

### 📌 4. 静岡県商工会議所との連携

静岡県商工会議所は、中小企業の支援と地域経済の活性化に貢献されています。  
AIpa Web は、そのような「地域貢献」の一環として、中小企業の採用課題を解決し、  
地域の雇用と経済の活性化に貢献します。

---

### 📌 5. モニター契約（初期15万円）

- **対象**: 静岡県内の中小企業（建設・製造・運輸・サービス業）
- **価格**: 初期15万円（事例掲載条件付き）
- **特典**: 月額運用サポートが無料で利用可能

---

### 📌 6. 今後の予定

- **5月28日**: 静岡県商工会議所3カ所（静岡・浜松・沼津）に、営業メールをBcc送信
- **6月上旬**: モニター企業3社を獲得し、事例掲載
- **6月中旬**: IT導入補助金の申請サポート体制を整える
