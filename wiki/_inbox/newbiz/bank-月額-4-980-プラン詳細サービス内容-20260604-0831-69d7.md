# 月額¥4,980 プラン詳細サービス内容

## 月額¥4,980 プランのサービス内容

**月額¥4,980**は、**スタンダードプラン**として提供される。このプランでは、**専用施設でのダーツプレイが通い放題**となる。QRコードによる入退店が可能で、**人と話す必要は一切ない**。**お酒は一切提供しない**ため、**未成年でも利用可能**。**全年齢層**が利用できる仕組みを実現している。

## サービス内容の詳細

| サービス内容 | 詳細 |
|--------------|------|
| **専用施設** | ダーツ専用の施設で、照明・マシンが完備されている |
| **QR入退店** | QRコードで入退店を自動化し、人と話す必要がない |
| **お酒なし** | お酒は一切提供しない。未成年でも利用可能 |
| **年齢制限なし** | 全年齢層が利用できる |
| **通い放題** | 月額¥4,980で、ダーツプレイが通い放題 |
| **マシンリース** | DARTSLIVEというマシンをリースし、利用可能 |
| **イベント参加** | 月1回のイベントに参加可能（DARTSLIVE検定/大会） |
