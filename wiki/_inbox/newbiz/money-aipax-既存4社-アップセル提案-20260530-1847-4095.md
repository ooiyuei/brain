# パターン1：追加ツール・機能拡張提案書
**目的：** 既存の「保存庫コア」の上に、別の業務フローのAIツールを載せ、月額単価を5万円→10万円以上へ引き上げる。
**ターゲット：** 営業支援ツール導入後、「新人教育」や「CS対応」に課題を感じている顧客

## 1. 提案の背景：基盤の活用率向上
現在、貴社には「会社別AI保存庫コア」と「営業回答支援ツール」が導入されています。
このコアは、営業データだけでなく、**社内ルール、契約条件、技術資料、顧客情報**など、あらゆる業務知識を保存・検索できる「AIの脳」です。

現状の営業支援だけでなく、この「脳」を他の部署や業務でも活用することで、**AI導入のROI（投資対効果）を最大化**できます。

## 2. 推奨追加ツール3選
保存庫コアへの追加コストは低く、短期間で効果を実感できるメニューです。

| 追加ツール名 | 解決する課題 | 月額目安 | 期待される効果 |
| :--- | :--- | :--- | :--- |
| **社内AIチャット / 新人教育AI** | 新人の質問対応負荷、ルール確認の遅れ | +3〜5万円 | 先輩の負担軽減、新人の早期戦力化 |
| **CS対応AI / 問い合わせ一次返信** | 問い合わせの属人化、返信遅延 | +3〜5万円 | 24時間対応可能、顧客満足度向上 |
| **議事録・フォロー文生成AI** | 商談後の整理漏れ、手作業の時間 | +2〜3万円 | 営業時間の確保、正確な記録の維持 |

## 3. 提案メリット
*   **データ連携の即時性：** 既存の保存庫コアと連携するため、新たなデータ構築工数が不要。営業で蓄積された知識が、CSや新人教育でも即座に活用可能。
*   **月額コストの最適化：** 個別にツールを購入するより、コア基盤共用のため割安。
*   **標準価格への移行：** 追加ツール導入を機に、月額を通常価格（10-15万円）へ調整し、より安定した運用体制を構築。

## 4. 次のアクション
*   現在最も困っている業務（例：新人の質問対応）を1つ選んでいただき、デモ環境での動作確認を実施します。
*   対象領域の定義を行い、追加ツールの初期構築費用（通常価格相当：100-150万円相当の価値を、既存顧客特典として割引提案）をご提示します。

# パターン2：年契約・長期利用割引提案書
**目的：** 月額収入の安定化、解約リスクの低減、現金フローの改善。
**ターゲット：** 導入効果を実感しており、継続利用を検討している顧客

## 1. 提案の背景：継続利用の安定化
AIツールは「使えば使うほど賢くなる」性質があります。月次契約では、利用頻度の波や予算の都合で利用が途切れるリスクがあります。
貴社の「会社別AI保存庫コア」は、長期的にデータを蓄積することで、より高精度な回答と業務効率化を実現します。

## 2. 年契約プランのご案内
月次契約を継続していただく代わりに、**年契約（12ヶ月一括払い）**をご提案いたします。

### 比較表

| 項目 | 月次契約（現在） | 年契約（提案） |
| :--- | :--- | :--- |
| **月額単価** | 5万円（初回事例価格） | **4万円**（20%割引） |
| **年間総額** | 60万円 | **48万円** |
| **支払い回数** | 毎月 | 一括（または分割2回） |
| **特典** | - | **初期構築費用の全額免除**<br>（次回追加ツール導入時の優先権） |

## 3. 年契約のメリット
*   **コスト削減：** 年間12万円の節約。
*   **予算計上の容易さ：** 年間予算として固定できるため、経理処理が簡素化。
*   **継続的な最適化：** 12ヶ月間、安定してデータを蓄積できるため、AIの精度が飛躍的に向上。
*   **優先サポート：** 年契約顧客には、緊急対応や新機能の優先アクセス権を提供。

## 4. 次のアクション
*   年契約締結の場合、現在適用中の「初回事例価格」を維持しつつ、月額をさらに割引。
*   契約更新日は、貴社の会計年度に合わせて柔軟に設定可能。
*   来週中に契約書案を送付し、ご確認いただけますか？

# パターン3：紹介報酬・パートナー制度提案書
**目的：** 新規顧客獲得（MRR増加）、既存顧客との関係性深化。
**ターゲット：** AIpaXの成果を実感し、業界内でのネットワークを持つ経営者

## 1. 提案の背景：成功体験の共有
貴社で「会社別AI保存庫コア」を活用し、業務効率化や売上向上に成功されたことを心より感謝申し上げます。
AIpaXは、同じような課題を持つ他社とも連携し、業界全体のAI化を推進したいと考えています。

## 2. 紹介報酬制度のご案内
貴社が信頼する取引先や業界の知人企業に、AIpaXをご紹介いただいた場合、以下の報酬を支給いたします。

### 報酬体系
| 紹介対象 | 報酬内容 | 条件 |
| :--- | :--- | :--- |
| **新規契約社** | **初期費用の20%** または **月額1ヶ月分** | 契約成立後、初回請求時に支払 |
| **追加ツール導入社** | **追加ツールの初期費用の10%** | 既存顧客が追加ツールを導入した場合 |
| **年契約締結社** | **月額1ヶ月分のボーナス** | 年契約締結時に追加支給 |

### 具体例
*   貴社の取引先A社が、通常価格（初期150万円/月額10万円）で契約した場合：
    *   報酬：**15万円（初期費用の20%）** または **10万円（月額1ヶ月分）** のいずれかを選択。
*   貴社がさらにB社を推薦し、B社が追加ツール（月額+5万円）を導入した場合：
    *   報酬：**5万円** の追加ボーナス。

## 3. 紹介の進め方
*   **デモ環境の提供：** 貴社の成功事例（匿名化）を交えたデモ資料を提供。
*   **共同商談：** 必要に応じて、AIpaX担当者が貴社とともにお伺いし、技術的な質問に対応。
*   **紹介後のフォロー：** 紹介先企業のオンボーディングを優先的に行い、貴社の評判を損なわないよう徹底管理。

## 4. 次のアクション
*   現在、AI導入を検討している取引先や知人がいらっしゃいましたら、お名前をお知らせください。
*   紹介先企業との初回ヒアリング日程調整のお手伝いをいたします。
*   報酬の詳細な規定書（PDF）を送付いたしますので、ご一読ください。

# 共通の次のステップ（営業フロー）

1.  **現状の棚卸し：** 既存4社それぞれに対し、現在の保存庫コアの利用状況と、満足度の高いポイントをヒアリング。
2.  **個別提案の選択：** 各社の状況に応じて、上記3パターンのうち最も刺さるパターンを1つ選定し、提案資料を作成。
3.  **面談実施：** 提案資料に基づき、具体的な導入イメージやコストメリットを説明。
4.  **契約締結：** 追加ツール導入、年契約、または紹介契約の締結。

**注意点：**
*   既存顧客へのアップセルは、**「既存サービスの品質維持」**を前提に行うこと。
*   追加ツール導入時は、既存の保存庫コアとの連携テストを丁寧に行い、信頼を損なわないこと。
*   紹介報酬制度は、**「成功体験の共有」**というポジティブな文脈で提示し、営業圧力を感じさせないこと。
