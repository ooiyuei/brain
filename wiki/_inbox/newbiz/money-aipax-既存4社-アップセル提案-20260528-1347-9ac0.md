既存4社顧客向けに、**「AIpaXの価値をさらに高めるアップセル」**を提案するための3つのパターンを作成しました。

既存顧客は「初回事例価格（初期50-80万／月額5万）」という優遇価格で契約しているため、**「その恩恵を継続させるための正当な理由」**と**「業務効率化の具体的な成果」**をセットで提示する必要があります。

以下に、3つの提案パターンと、それぞれの説明書面（アウトライン）を記載します。

---

### アップセル提案の前提条件（営業トークの骨子）
*   **現状の感謝と確認**: 「現在構築中の『会社別AI保存庫コア』が貴社の基盤として安定しています」
*   **次のステップの提示**: 「基盤ができた今、より高度な活用（または継続的な最適化）をご提案します」
*   **価格の正当化**: 「単なるツール追加ではなく、貴社の『知の資産』を強化し、離脱リスクを下げます」

## パターン1：【継続・安定化】年契約割引＆保守範囲の拡大
**ターゲット**: 保守・運用への不安がある、またはコスト管理を重視する顧客
**狙い**: 月額5万→月額4.5万（年払い時）など、継続契約を促し、LTV（顧客生涯価値）を安定させる。

### 提案の概要
現在個別契約となっている月額運用を**「年契約」**に変更し、以下のメリットを提供します。
1.  **月額料金の割引**: 年一括払いで月額実質10%OFF（例：5万→4.5万/月）。
2.  **保守範囲の明確化と拡大**: 「軽微な不具合修正」の定義を緩和し、月1回の「保存庫コアの健康診断」を無償で実施。
3.  **価格ロックイン**: 次回の更新時、通常価格（10-15万）への移行を1年間猶予（または段階的移行）とする特典。

### 説明書面：年契約・保守強化プラン

```markdown
# AIpaX アップセル提案書：年契約・保守強化プラン

## 1. 提案の背景
現在、貴社の「会社別AI保存庫コア」は安定して稼働しております。
AIの精度は「データの質」と「継続的な学習」に依存します。
より高精度な回答を得るためには、月次での微調整だけでなく、定期的な構造最適化が必要です。

## 2. 年契約プランのメリット
月額契約を「年契約（一括払い）」に変更いただくことで、以下の3つの特典を提供します。

### ① コスト削減（月額実質10%OFF）
*   通常月額：50,000円
*   **年契約月額換算：45,000円**
*   年間節約額：60,000円

### ② 保守範囲の拡大（安心のサポート）
月額運用に含まれる「軽微な修正」に加え、以下のサービスを無償で追加します。
*   **四半期ごとの保存庫コア健康診断**: 検索精度の分析とインデックス再構築
*   **緊急対応優先枠**: 通常2営業日→24時間以内の対応
*   **新機能ベータテスト参加権**: 新ツールの早期導入とフィードバック反映

### ③ 価格保護（プライスロック）
*   契約期間中、月額料金の値上げはありません。
*   契約更新時、通常価格（10-15万円）への移行を**1年間猶予**します（段階的移行の猶予期間として活用可能）。

## 3. 推奨される理由
貴社の業務フローにAIpaXが深く組み込まれるほど、突然のサービス停止や精度低下はビジネスリスクとなります。
年契約により「継続的な最適化」を担保し、貴社のAI活用を安定させます。

## 4. 契約条件
*   契約期間：1年
*   支払方法：年一括払い（または分割2回）
*   開始日：2026年6月1日
```

## パターン2：【機能拡張】追加ツール（リアルタイム電話支援など）のセット導入
**ターゲット**: 営業効率化やCS対応に課題がある、または既存の「保存庫」をより活用したい顧客
**狙い**: 初期費用を回収しつつ、月額収入を2-3倍に増やす。

### 提案の概要
既存の「保存庫コア」の上に、**「リアルタイム電話/営業回答支援ツール」**または**「社内AIチャット」**を追加導入します。
*   **初期費用**: 30-50万円（既存顧客向け特別価格）
*   **月額追加**: 5-8万円（合計月額10-13万円へ）
*   **特典**: 既存の保存庫データを活用した「即座の精度向上」を保証。

### 説明書面：追加ツール導入提案書

```markdown
# AIpaX アップセル提案書：業務特化ツール追加導入

## 1. 提案の背景
貴社の「会社別AI保存庫コア」が完成した今、その知識を「リアルタイムで業務に活用」する段階へ移行しませんか？
既存の保存庫は「記憶」ですが、追加ツールは「行動」を支援します。

## 2. 追加ツール候補（より効果的なものを選択）

### A. リアルタイム電話/営業回答支援ツール
*   **課題**: 電話対応中に資料を探したり、正確な回答を思い出せない。
*   **解決**: 通話中にAIが保存庫を参照し、画面に回答候補をリアルタイム表示。
*   **効果**: 対応時間の30%短縮、顧客満足度の向上。

### B. 社内AIチャット/新人教育AI
*   **課題**: 新人が同じ質問を繰り返す、ルール確認に時間がかかる。
*   **解決**: 保存庫の知識を基に、社内FAQとして24時間対応。
*   **効果**: 管理者の負担軽減、新人の習熟期間の短縮。

## 3. 導入プラン（既存顧客特別価格）

| 項目 | 内容 | 金額 |
|------|------|------|
| **初期構築費** | 既存保存庫との連携設定、ツールカスタマイズ | **300,000円** (通常50万円相当) |
| **月額運用費** | ツール利用料 + 既存コアの維持管理 | **追加 50,000円** |
| **合計月額** | 既存(5万) + 追加(5万) | **100,000円** |

## 4. 導入スケジュール
1.  **第1週**: 要件定義（既存保存庫との連携ポイント確認）
2.  **第2-3週**: 環境構築・テストデータ反映
3.  **第4週**: 社内テスト・トレーニング
4.  **第5週**: 本格運用開始

## 5. 特典
*   既存の保存庫データを活用するため、**初期学習コストゼロ**で高精度な回答が可能です。
*   導入後3ヶ月間は、ツール精度の微調整を無償で実施します。
```

## パターン3：【紹介・パートナー】紹介報酬制度の開始
**ターゲット**: AIpaXの成果に満足しており、業界内での信頼がある顧客
**狙い**: 新規顧客獲得コストを下げ、既存顧客との絆を深める（エンゲージメント向上）。

### 提案の概要
AIpaXの成功事例を共有いただき、同業他社や関連業界への紹介を依頼します。
*   **報酬**: 紹介契約成立時、**初期費用の10%** または **月額費用の3ヶ月分** をキャッシュバックまたはサービスクレジットとして提供。
*   **目的**: 既存顧客を「パートナー」として位置づけ、ネットワーク効果を生む。

### 説明書面：紹介パートナー制度のご案内

```markdown
# AIpaX 紹介パートナー制度のご案内

## 1. 制度の趣旨
AIpaXは、貴社の成功事例を基に、より多くの企業様のAI活用を支援していきたいと考えております。
このたび、既存顧客様向けに「紹介パートナー制度」を開始しました。

## 2. 対象者
AIpaXの導入効果にご満足いただいている既存顧客様

## 3. 紹介報酬
紹介いただいた企業様が、AIpaXと契約を締結した場合、以下のいずれかを選択して報酬を提供します。

### 選択肢A：キャッシュバック
*   契約初期費用の **10%** を現金で還元
    *   例：初期費用100万円の場合 → 10万円還元

### 選択肢B：サービスクレジット（月額割引）
*   月額費用の **3ヶ月分** を無料クレジットとして付与
    *   例：月額10万円の場合 → 3ヶ月間無料（合計30万円分）

## 4. 進め方
1.  **紹介**: 貴社から、AIpaXの導入効果についてご紹介します。
2.  **接触**: 弊社担当者から、貴社の成功事例を共有し、詳細をご提案します。
3.  **契約**: 紹介先企業様と契約が成立します。
4.  **報酬**: 契約成立後、翌月分の請求書にクレジットとして反映、または現金送金いたします。

## 5. 注意事項
*   紹介先企業様との直接的な金銭授受は行わず、すべて弊社と貴社間で処理いたします。
*   紹介先企業様の機密情報は厳重に管理いたします。

## 6. お問い合わせ
ご興味のある方は、担当者までお気軽にご相談ください。
貴社の成功事例を共有いただくことで、業界全体のAI活用が進むと信じております。
```
