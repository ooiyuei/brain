# AIpaX 既存4社向け アップセル提案3パターン

## 前提条件
- **対象**: 既に契約済みの4社（初期50-80万/月額5万の事例価格）
- **目的**: 月額10-15万円への価格改定、または追加ツール導入によるLTV向上
- **共通メッセージ**: 「保存庫コア」の完成度が高まったため、より高度な業務自動化が可能になった。

---

## パターン1：【月額改定】「保存庫コア」の標準化による高付加価値化提案
**狙い**: 月額を5万円 → **10〜15万円**へ引き上げる。
**対象**: 現在、保存庫コアの活用が進んでおり、検索精度や回答の質に満足している顧客。

### 提案の論理構成
1.  **現状の棚卸し**: 過去3ヶ月で蓄積された「会社別AI保存庫コア」のデータ量と検索精度の向上を報告。
2.  **価値の転換**: 「単なる資料保存」から「リアルタイムな意思決定支援」へ進化していることを示す。
3.  **価格改定の理由**:
    -   保存庫コアのメンテナンス負荷が標準化され、安定した高品質なサービス提供が可能になった。
    -   従来は「β版・事例価格」だったが、今後は「標準製品」としての価格体系へ移行する。
4.  **特典**: 月額改定に伴い、月1回の「保存庫最適化レポート」を無償提供。

### 説明書面（アジェンダ案）

```markdown
# 提案書：AIpaX 月額サービス改定のご提案

## 1. 現状の成果報告（過去3ヶ月）
- 保存庫コアへのデータ入庫件数：〇〇件
- 検索精度の向上率：〇〇%
- 営業現場での活用事例：「〇〇の質問に〇〇秒で回答可能に」

## 2. サービスの進化：保存庫コアの標準化
- 従来：個別調整中心のβ版
- 今後：全社共通の「高品質AI保存庫コア」として安定運用
- 結果：回答の正確性、更新速度、セキュリティが大幅に向上

## 3. 月額料金改定のご提案
- 現行月額：50,000円
- 新月額：**100,000円 〜 150,000円**（対象領域・データ量による）
- 理由：
  1. AIモデルの推論コスト増（高精度化に伴う）
  2. 保存庫コアの高度なメンテナンス体制の確立
  3. 他社導入事例との価格バランス調整

## 4. 改定後の特典
- 月1回「保存庫最適化レポート」の提供
- 緊急対応優先枠の確保
- 新機能（例：CRM連携プレビュー版）の優先アクセス権

## 5. 合意事項
- 改定有効日：2026年〇月〇日
- 契約更新：年契約割引適用（10%OFF）
```

## パターン2：【追加ツール】「リアルタイム電話/営業回答支援」のフル導入提案
**狙い**: 既存の保存庫コアに、**リアルタイム電話支援ツール**を追加導入し、初期費用＋月額を追加徴収。
**対象**: 営業電話での対応に課題があり、かつ保存庫コアのデータを電話で使いたいと考えている顧客。

### 提案の論理構成
1.  **課題の再確認**: 電話対応での「即答の難しさ」「属人化」の問題を再提示。
2.  **解決策**: 保存庫コアと連携した「リアルタイム電話支援ツール」の完全版導入。
3.  **価格提示**:
    -   初期費用：30-50万円（既存コアとの連携設定費含む）
    -   月額：+5-10万円（ツール利用料＋推論コスト）
4.  **ROI（投資対効果）提示**: 電話対応時間の短縮率、成約率向上予測を提示。

### 説明書面（アジェンダ案）

```markdown
# 提案書：リアルタイム電話支援ツールの追加導入について

## 1. 課題：電話対応のボトルネック
- 顧客からの質問に、保存庫コアの情報を「即時」に反映できていない
- 新人営業の電話対応品質にばつきがある

## 2. 解決策：保存庫コア連携型「リアルタイム電話支援」
- 通話中にAIが保存庫コアをリアルタイム検索
- 回答候補を画面に表示、または音声で案内
- 通話終了後、自動的に議事録・フォロー文を生成

## 3. 導入費用・月額
- 初期費用：300,000円 〜 500,000円
  （既存コアとのAPI連携設定、電話回線設定費含む）
- 月額追加：50,000円 〜 100,000円
  （推論コスト、リアルタイム処理負荷費）

## 4. 期待される効果
- 電話対応時間の30%短縮
- 新人営業の初月フォローアップ完了率 100%達成
- 顧客満足度向上（即答性）

## 5. 導入スケジュール
- 第1週：要件定義・電話回線設定
- 第2週：保存庫コアとの連携テスト
- 第3週：本番環境デプロイ
- 第4週：営業チーム研修・運用開始
```

## パターン3：【年契約・紹介報酬】長期契約と紹介によるコスト削減提案
**狙い**: 現金フローの安定化と、既存顧客からの新規顧客獲得（紹介）。
**対象**: 価格改定に抵抗感がある顧客、または紹介ネットワークが広い顧客。

### 提案の論理構成
1.  **年契約割引の提案**: 月額を年払いすることで、**10-15%の割引**を提供。
2.  **紹介報酬制度の開始**:
    -   紹介成功時：紹介元顧客に**月額1ヶ月分相当のクレジット**を付与。
    -   または、紹介先の新規導入費用から**10%のキャッシュバック**。
3.  **メリットの強調**:
    -   顧客側：コスト削減、新規顧客獲得による収益化。
    -   AIpaX側：安定した収益基盤、顧客基盤の拡大。

### 説明書面（アジェンダ案）

```markdown
# 提案書：年契約割引および紹介報酬制度のご案内

## 1. 年契約割引のご提案
- 月額払い：100,000円/月
- **年払い（12ヶ月）：1,080,000円（10%OFF）**
  - 月額実質：90,000円/月
  - 特典：年2回の「保存庫コア大規模最適化」を無償で実施

## 2. AIpaX紹介報酬制度の開始
- 対象：既存顧客
- 報酬内容：
  - 紹介先が契約成立時：**月額1ヶ月分相当のクレジット**を付与
  - または、紹介先の新規導入費用から**10%のキャッシュバック**
- 条件：紹介先は「保存庫コア」の導入を必須とする

## 3. 導入のメリット
- 顧客側：コスト削減、新規顧客獲得による収益化
- AIpaX側：安定した収益基盤、顧客基盤の拡大

## 4. 今後のロードマップ
- 2026年Q3：紹介報酬制度の詳細ルール公開
- 2026年Q4：年契約顧客限定の「プレミアムサポート」開始

## 5. 合意事項
- 年契約の有効期間：2026年〇月〜2027年〇月
- 紹介報酬の支払い方法：クレジットまたはキャッシュバック
```
