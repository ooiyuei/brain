## 🎯 **SFC AO志望テーマ**

**「AIによる思考の外部化が進む社会において、人間の主体性や意思決定能力はどのように再定義されるべきか」**

### 🧠 **問題意識**
- AIを使えばすぐ答えが出る → 考える前に答えを得てしまう
- AIの出力を吟味せず受け入れる危険
- 人間が「思考する主体」から「AIの出力を承認するだけの存在」になる可能性
- 「思考力」より「問いを立てる力」の格差が生まれる

### 📚 **自分の経験で語れる点**
- Gymee / AIアプリ開発 / AI活用 / AI導入支援
- スタートアップ6社での実務経験
- 起業家育成プログラムでの学び
- 独立後の事業開発

---

## 📚 **SFC AO志望理由書のポイント**

- **内申4.7**（強い）
- **英語は高1レベル**（最終目標：英検準1級）
- **AOでは「優等生的な志望理由書」より「面白い・問いを持っている」と思わせる方が刺さる**
- **主戦場は英語より、実績・問い・思考・言語化**

## 📖 **HR高等学院インタビュー（2025-10-07公開）**

### 🎯 **ダーツ部部長としての活動**
- 高1で地元進学校を退学→東京上京
- 静岡の**クリエイターズプロジェクト**と**FUJI（起業家育成プログラム）**に偶然参加→起業意識覚醒
- 入学の決め手：RePlayceスタッフ・**小野さん**の熱量と親身さ
- **カフェ事業**：2025年5月に代々木ベース内でオープン。豆から挽くコーヒーを提供、好評
- **ダーツ機械製造会社**と企画進行中（→ [[ipak-darts]]に直結）
- 構想：通信制高校同士の連携で影響力を創る。普通校の生徒の価値観を変えたい

### 🧠 **自己分析（インタビューより）**
- 「一人でなんでもやろうとしちゃう、一人でやった方が早くね？って考えちゃうタイプ」→HR高で変わり始めた
- 「学歴とか本当にどうでもいいですし、大事なのは自分が何をしたいかを軸に、自分軸で生きていくこと」

## 🧠 **思考スタイルと話し方の癖**

| 強み | 弱み |
|------|------|
| 熱量が伝わる | 背景から入りすぎる |
| ストーリーで話せる | 長くなりやすい |
| 相手の感情を見る | 言い直しが多い |
| 抽象化ができる | 枝分かれしやすい |
| ビジネス文脈に接続できる | 整理前の思考まで口に出る |

### 🗣 **改善方針**
- **結論 → 具体例 → 学び** の順に話す
- **口癖**：「えっと」「まー」「なんだろうな」「っていう」「みたいな」「そこだよね」「かなって感じ」

## 🧬 **占術メモ（重要箇所のみ）**

- **西洋占星術**：太陽・月・水星が9ハウス（獅子座）にコンジャンクション。金星・火星・土星が10ハウス。スコルピオ上昇宮
- **インド占星術**：天秤座ラグナ。太陽・月・水星・金星・ケートゥが第10ハウス（蟹座）に集中 → キャリア・公的認知が強烈に強調された命式。現在の主期（マハーダシャー）は水星期（2015〜2032）
- **22〜32歳**（壬戌大運）が飛躍期。情報・ネットワーク・移動が活性化
- **18歳（2026年）**：正財・絶。土台づくりの年、エネルギーは発散期
- 西洋/インド/四柱すべてが「社会の中で目立つ役割・人とのつながりで成果」を示す

## 🧠 **強み**

1. **相手に伝わる話し方** — 感情・状況に合わせた伝え方。納得して動いてもらうことを意識
2. **短期間で結果を出す力** — 何が成果につながるかを見極めて集中するスプリント型
3. **課題を提案に落とし込む力** — 引き出し → 整理 → 提案 → 実行まで入れる
4. **丁寧さと人間関係を大切にする姿勢** — 責めるよりまずフォロー。安心感・信頼も成果の前提

---

## 📌 **SFC AO志望テーマの補足**

- **AIによる思考の外部化**：AIが思考を補助するが、人間の主体性が低下するリスク
- **AIの出力を吟味する力**：AIの結果を批判的に見る能力
- **問いを立てる力**：AIが答えを出す前に、なぜその答えが正しいのかを問う力
- **人間の価値**：AIができない「問いを立てる力」が人間の価値
