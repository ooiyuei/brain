# カスハラノート法的証拠保全弁護士監修プラン

## 1. 結論：なぜ「法的証拠」なのか

[[entities/kasunote]] の核心は「記録」ではなく「**証拠能力**」にある。
月980円→2,980円への値上げと並行し、**「裁判で使えるデータ」としての差別化**を打ち出す。

- **現状の弱点**: 単なるログ。改ざん可能なら法的効力は薄い。
- **解決策**: 改ざん検知 + 第三者検証 + 弁護士監修の3点セット。
- **明日使う人**: 労務トラブルで泣いている店長、または人事。

## 2. 技術的証拠保全の実装（2週間）

弁護士に「この仕組みなら証拠能力が高い」と言わせるための要件定義。

### 2.1 改ざん不可能なタイムスタンプ（TSA）
- **実装**: RFC 3161準拠のTSA（Time Stamping Authority）連携。
- **方法**: ユーザーが記録したJSONデータハッシュ値を、外部TSAサーバーに送信し、検証可能なタイムスタンプトークンを取得。
- **ツール**: `openssl` コマンドラインツールまたは `tsa-client` ライブラリ。
- **証拠**: 「いつ」「誰が」「何を見たか」が後から変えられない。

### 2.2 第三者検証（Trusted Third Party）
- **実装**: AWS CloudTrail または Azure Monitor Logs に、改ざん防止ログをリアルタイム送信。
- **方法**: ユーザーのローカルDBだけでなく、クラウド側の不変ログに同期。
- **証拠**: 「サーバー管理者（大井含む）でさえ後からデータを書き換えられない」構造。

### 2.3 弁護士監修による「証拠性」の定義
- **依頼**: 労働問題に強い弁護士（例：**弁護士法人ALG&Associates** や **弁護士法人北星法律事務所** の労働部門）に監修を依頼。
- **成果物**: 「本サービスが出力するデータは、民事訴訟において**書証としての証拠能力**を有すると考えられる」という意見書。
- **価格感**: 初回監修で50-100万円（月額保守込みなら20-30万円/月）。
- **[要確認]**: 大井の予算（AIpaXの収益から捻出可能か）。

## 5. リスクと対策

- **リスク**: 弁護士費用が高額。
    - **対策**: 月額保守契約で抑える。または、成果報酬型で交渉。
- **リスク**: 技術的実装が複雑。
    - **対策**: 既存のTSAサービス（例：`DigiCert TSA`）を利用し、自前実装を最小限に。
- **リスク**: 法的証拠としての効力が認められない。
    - **対策**: 弁護士監修を徹底し、裁判例を調査。
