## アプリ名・1行説明  
**学習計画メーカー**  
高校生が1日30分で学習計画を自動作成するアプリ

## ターゲット  
**明日使う1人**  
偏差値60台の高校2年生。模試で点数が伸び悩んで、学習計画を立てられない。  
「勉強する時間はあるけど、どう始めるか分からない」「何を優先すべきか分からない」

## 功能リスト v0.1 (MVP・3機能のみ)  
1. **学習科目選択**  
   - 選択肢: 数学・国語・英語・理科・社会  
   - ユーザーが3科目を選択（例: 数学・英語・理科）  

2. **学習時間割作成**  
   - 1日30分を3科目に振り分ける  
   - 毎日同じ時間割を自動生成（例: 数学10分・英語10分・理科10分）  

3. **学習内容提案**  
   - 毎日の時間割に合わせて、具体的な学習内容を提案（例: 数学→平方根の計算問題）  
   - 毎日1問ずつ出題（例: 平方根の計算問題1問）  

## 画面構成 (3画面)  
1. **ホーム画面**  
   - タイトル: 学習計画メーカー  
   - ボタン: 「科目選択」  
   - ボタン: 「学習時間割」  
   - ボタン: 「学習内容」  

2. **科目選択画面**  
   - テキスト: 「3科目を選択してください」  
   - ボタン: 数学 / 国語 / 英語 / 理科 / 社会  
   - ボタン: 「確定」  

3. **学習内容画面**  
   - テキスト: 「今日は何を勉強しますか？」  
   - テキスト: 「数学→平方根の計算問題」  
   - ボタン: 「次」  

## データモデル  
- **ユーザー情報**  
  - ID: string  
  - 名前: string  
  - 科目: array<string>  
  - 学習日: date  

- **学習内容**  
  - ID: string  
  - 科目: string  
  - 内容: string  
  - 日付: date  

## 技術スタック  
- **Frontend**: React Native + Expo  
- **Backend**: Supabase（無料プラン）  
- **広告**: AdMob  
- **決済**: Stripe（無料プラン）  

## 月3万円達成のロジック  
- **DAU x 広告単価**  
  - DAU（日間アクティブユーザー）: 100人  
  - 広告単価: 300円  
  - 月収入: 100人 × 300円 × 30日 = 900,000円  
  - 課金率: 1%（月収入 × 1% = 9,000円）  
  - 合計月収入: 99,000円
