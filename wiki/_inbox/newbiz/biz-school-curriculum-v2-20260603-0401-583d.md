# AIpaX school 1年カリキュラム（5モジュール完成版）

## 🎯 概要

AIpaX schoolは、中高生向けのAI教育スクールで、AIを手足のように使えるスキルを実践的に身につけることを目的としています。カリキュラムは、**AIマインドセット → 日常業務自動化 → エンジニアリング思考 + 外部連携 → 本格アプリ開発 + リリース → データ分析 + マネタイズ**の5つのモジュールに分かれ、それぞれ2ヶ月（合計1年）で学習を進めます。各モジュールには学習目標、週次トピック、課題、必要ツール、評価方法が明記されています。

---

## 📌 モジュール1: AIマインドセット（Month 1-2）

### ✅ 学習目標
- AIの基本的な概念を理解し、AIを「手足」のように使えるマインドセットを構築する。
- AIツールの使い方を学び、簡単なAIプロンプト設計を実践する。
- AIの可能性と限界を理解し、AIを活用するための倫理観を養う。

### 📅 週次トピック（8週）

| 週 | トピック | 内容 |
|----|----------|------|
| 1 | AIとは？ | AIの定義、種類（機械学習、自然言語処理など） |
| 2 | AIの歴史 | AIの進化と現在の技術の進歩 |
| 3 | AIツールの紹介 | ChatGPT、Claude、Canva AIなどの使い方 |
| 4 | プロンプト設計 | AIと対話するためのプロンプトの設計法 |
| 5 | AIと倫理 | AIの倫理的課題と社会への影響 |
| 6 | AIと未来 | AIがもたらす未来の社会像と個人の可能性 |
| 7 | AIと人間 | AIと人間の協働の重要性 |
| 8 | AIマインドセットのまとめ | 学んだことを振り返り、自己評価 |

### 📝 課題
- AIツール（例：ChatGPT）を使って、自分の生活に役立つAIプロンプトを3つ作成し、実行結果を報告。
- AIの倫理的課題についての短いエッセイを提出。

### 🛠 必要ツール
- ChatGPT / Claude / Canva AI / Google Colab / Notion（プレゼン作成用）

### 🧾 評価方法
- 週次課題の提出状況
- プロンプト設計の実践成果
- エッセイの内容と論理性
- グループでのディスカッションへの参加度

## 📌 モジュール2: 日常業務自動化（Month 3-4）

### ✅ 学習目標
- 日常業務をAIで自動化するスキルを習得し、効率的な作業方法を学ぶ。
- チームでの合意形成と協働のスキルを身につける。
- AIツールを活用して、業務プロセスを改善する実践を経験する。

### 📅 週次トピック（8週）

| 週 | トピック | 内容 |
|----|----------|------|
| 1 | 日常業務の自動化とは？ | 自動化の概念とAIの役割 |
| 2 | ツールの選定と導入 | 自動化ツールの選び方と導入プロセス |
| 3 | プロセスの設計 | 自動化する業務の流れを設計 |
| 4 | プロトタイピング | サンプルの作成とテスト |
| 5 | チームでの合意形成 | チームでアイデアを共有し、合意を形成 |
| 6 | チームワークの実践 | グループでプロジェクトを進める |
| 7 | プロジェクトの改善 | フィードバックを元に改善 |
| 8 | 自動化のまとめ | 学んだことを振り返り、自己評価 |

### 📝 課題
- 自分の日常業務をAIで自動化するプロトタイプを作成し、グループ内で発表。
- チームで合意形成し、プロジェクトを進める。

### 🛠 必要ツール
- Google Apps（スプレッドシート、ドキュメントなど） / Notion / Zapier / Airtable / チーム協働ツール（Slack、Discordなど）

### 🧾 評価方法
