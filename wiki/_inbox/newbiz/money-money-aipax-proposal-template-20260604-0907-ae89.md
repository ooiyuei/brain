# AIpaX β顧客向け提案書

## 1. 顧客課題の言語化

中小企業のAI導入率は5%未満とされており、その理由は以下の3つが主因です。

- **AI導入のハードルが高い**：技術的な知識やコストが原因で導入が進まない
- **導入後の成果が見えない**：初期投資に対するROIが明確でない
- **業務に即したAIツールが不足**：汎用AIに依存し、業務に即した支援が得られない

特に中小企業では、経営者がAI導入を検討する際、**「導入後の成果が明確でない」「導入コストが高い」「導入後の運用が複雑」**という課題が深刻です。

---

## 2. AIpaX が解決すること

AIpaXは、**「会社別AI保存庫コア + 業務AIツール」**の組み合わせで、中小企業のAI導入課題を解決します。

### 会社別AI保存庫コア

- 会社の資料・契約・料金・FAQ・顧客情報・例外条件・やり取り履歴をAIが検索・参照・再利用できる形に整理
- 顧客ごとの条件や例外を保存し、AIがその条件を理解して対応
- 保存庫は共通基盤として、業務ツールと連携して動作

### 業務AIツール

- リアルタイム電話 / 営業回答支援
- 社内AIチャット / 新人教育AI
- 議事録整理AI
- フォロー文生成AI
- CS対応AI
- 管理レポートAI
- CRM補助ツール

---

## 3. 導入後のBefore/After（具体的な業務シーン3つ）

### Before: 営業電話で回答がズレる

- 営業担当者が電話で顧客に回答する際、ルールや条件が曖昧で、回答がズレる
- 顧客に不満が生じ、信頼が低下

### After: リアルタイム電話 / 営業回答支援

- AIが顧客の質問を分析し、正しい回答を提示
- 営業担当者がAIの回答を参考に、正確な回答を提供
- 顧客満足度が向上

### Before: 新人がルールを何度も聞く

- 新人営業がルールを確認する際、上司に何度も尋ねる
- 業務効率が低下し、新人の成長が遅れる

### After: 社内AIチャット / 新人教育AI

- 新人営業がAIチャットに質問を投げると、ルールや手順が即座に提示
- 新人営業がルールを理解し、業務効率が向上

### Before: 商談後の整理が遅い

- 商談後の資料整理が遅く、次の商談準備が遅れる
- 業務の連続性が低下

### After: 議事録整理AI

- AIが商談の議事録を自動で整理し、次の商談準備をサポート
- 業務の連続性が向上

## 4. 価格構造の根拠

### 初回価格: 50〜80万円

- **AI保存庫コア構築**：会社の資料・契約・料金・FAQ・顧客情報・例外条件・やり取り履歴をAIが検索・参照・再利用できる形に整理
- **最初の業務ツール導入**：リアルタイム電話 / 営業回答支援ツールの導入
- **初期設定・カスタマイズ**：顧客の業務フローに合わせたカスタマイズ

### 通常月額: 10〜15万円

- **AI保存庫コアの運用**：資料更新 / 入庫前処理 / 旧情報の非アクティブ化 / 検索条件の軽微調整
- **業務ツールの運用**：回答カードの軽微改善 / 利用ログ確認 / 回答できなかった質問の整理
- **不具合修正**：軽微な不具合修正

### 通常コンサル200万円との比較

| 項目 | AIpaX（初回50万円） | 通常コンサル（200万円） |
|------|---------------------|--------------------------|
| AI保存庫コア構築 | 30万円 | 100万円 |
| 業務ツール導入 | 20万円 | 50万円 |
| 初期設定・カスタマイズ | 0万円 | 50万円 |
| 月額運用 | 10万円 | 100万円 |
| 合計 | 50万円 | 200万円 |

## 6. 大井+チーム体制

- **大井湧瑛**（CEO）：AI導入の戦略立案・顧客対応

## 8. リスクと保証

### リスク

- **導入後の運用が複雑**：初期設定やカスタマイズが複雑で、導入後の運用が困難
- **導入後の成果が見えない**：初期投資に対するROIが明確でない

### 保証

- **導入後のサポート**：月額運用でサポートを提供
- **導入後の成果確認**：導入後の成果を確認し、必要に応じて改善

## おわりに

AIpaXは、中小企業のAI導入課題を解決するための**「会社別AI保存庫コア + 業務AIツール」**の組み合わせです。導入後の成果が明確で、導入コストが低く、運用が簡単なため、中小企業のAI導入を推進します。
