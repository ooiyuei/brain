# 親向け説明ペライチコピー

## Testallとは？  
Testallは、受験生向けのAI学習マネージャー。テスト結果を入力するだけで、AIが弱点を分析し、45分のタスクに自動変換する仕組み。  
**「勉強は才能よりマネジメント」**がコンセプト。  
**「テスト結果→弱点分析→45分タスク」**を一貫して自動化。  
**月収100万円期限: 2026-06-21（残33日）**。  
**AIpaX 4社契約済・月20万**。  
**3本柱: Testall(名)/AIpaX(金)/Agents of Flag(夢)**。  

## なぜTestallが必要？  
**高校1-3年の独学受験生（塾なし層70万人）**がターゲット。  
**「テストを受けても何を復習すべきか分からない」「勉強計画が続かない」**という課題を解決。  
**既存の学習アプリは「何を勉強するか」を自分で決める必要がある**が、Testallは**テスト結果入力→AI診断→実行まで一貫して自動化**。  
**「やる気ではなく「仕組み」で成績を上げたい層」**にリーチ。  

## Testallの特徴  
- **AI診断**: Claude AIが弱点分析・優先度付け  
- **集中モード**: 今日やるべき45分タスクを提示 + タイマー  
- **週計画**: 週単位の学習スケジュール管理  
- **マイページ**: 成長記録・連続学習日数  
- **料金設計**: Free（テスト3回まで・基本診断） / スタンダード（500円/月 / 無制限テスト・AI診断・集中モード） / プレミアム（1,500円/月 / 全機能・詳細分析・優先サポート）  

## Testallの実装状況  
- **開発ステータス**: MVP実装済・教育者連携準備中  
- **技術スタック**: Next.js + Supabase + Claude AI  
- **JUDGE評価**: 75点 / Tier A  
- **開発タスク**: テスト追加フォーム実体化（/app/test/new） / 集中モード本体（/app/focus/run） / AI診断結果ページ（/app/test/[id]） / データ永続化（Supabase接続） / Stripe課金  

## Testallの市場規模  
- **TAM**: 2,160億円（受験市場全体）  
- **SAM**: スマホで勉強管理したい高校生・浪人生層  
- **SOM**: 先行登録からのバイラル獲得  

## TestallのGTM戦略

2. 受験期（夏〜秋）に集中投下  
3. 先行登録 → 無料体験 → 有料転換  

## Testallの競合分析  
| 競合 | 弱点 | Testallの優位 |
|------|------|-------------|
| Studyplus Planning | **最大脅威**。計画機能を強化中 | テスト結果起点の自動診断 |
| atama+（塾向けAI） | 塾専用・高価格 | 個人・低価格・スマホ完結 |

| 既存学習管理アプリ | 手動計画・AI診断なし | AI自動化で差別化 |  

## TestallのAI設計  
- **モデル**: `claude-sonnet-4-6`（速度重視）→ 深い分析は Opus  
- **エンドポイント**: `/api/diagnose`  
- **プロンプト**: `src/lib/prompts/` に集約  
- **診断ロジック**: テスト結果（科目・点数・分野別ミス）→ 弱点優先度マトリクス → 45分タスク生成
