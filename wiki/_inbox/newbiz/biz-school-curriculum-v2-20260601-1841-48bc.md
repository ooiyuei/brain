# AIpaX school 1年カリキュラム（5モジュール完成版）

AIpaX schoolは、中高生向けのAI教育スクールで、AIを手足のように使えるスキルを実践で身につけることを目的としています。このカリキュラムは、AIマインドセットから始まり、日常業務自動化、エンジニアリング思考、本格アプリ開発、データ分析とマネタイズまでをカバーし、1年間の学習を5つのモジュールに分けて設計しています。各モジュールは2ヶ月間の学習期間を想定しており、学習目標、週次トピック、課題、必要ツール、評価方法を明確に設定しています。

---

## 🎯 モジュール概要

| モジュール | 学習期間 | 主な内容 | キーポイント |
|----------|----------|----------|-------------|
| Module 1 | 1〜2ヶ月 | AIマインドセット + バイブコーディング体験 | AIの基本概念、AIツールの使い方、プログラミングの基礎 |
| Module 2 | 3〜4ヶ月 | 日常業務自動化 + チーム合意形成 | 自動化ツールの活用、チームでのアイデア共有、合意形成 |
| Module 3 | 5〜6ヶ月 | エンジニアリング思考 + 個人探究テーマ設定 | プログラミングの実践、個人的なAIプロジェクトの設定 |
| Module 4 | 7〜8ヶ月 | 本格アプリ開発 + サービス企画 | アプリの開発、サービスの企画と設計 |
| Module 5 | 9〜12ヶ月 | データ分析 + マネタイズ + 卒業課題発表 | データの分析、収益化戦略、卒業プロジェクトの発表 |
