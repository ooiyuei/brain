既存契約4社に対するアップセル（追加売上）提案として、**「追加ツール導入」「年契約への移行」「紹介報酬制度」**の3パターンを提案します。

これらは、AIpaXの核心である「会社別AI保存庫コア（マザーボード）」の価値をさらに深め、顧客の離脱を防ぎつつLTV（顧客生涯価値）を最大化するための戦略です。

---

# パターン1：追加ツール提案（「マザーボード」への拡張機能追加）

**目的:** 既存の「保存庫コア」の活用範囲を広げ、月額単価の引き上げと定着率の強化を図る。
**対象:** 現在「リアルタイム電話/営業回答支援」など単一ツールを利用中のお客様

### 提案の論理構成
1.  **現状の課題提示:** 「保存庫コア」は既に構築済みであり、そこに蓄積されたデータ（顧客情報、契約条件、FAQ）を別の業務フローでも活用しないのはもったいない。
2.  **ソリューション:** 既存の保存庫を再利用し、**「追加ツール」**をマザーボードに接続する。
3.  **メリット:** 初期構築コスト（保存庫作成）が不要なため、通常価格の1/3〜1/2程度の初期費用で導入可能。

### 推奨される追加ツール候補（優先度高）

| 追加ツール | 想定される顧客のペイン | 保存庫コアとの相性 |
| :--- | :--- | :--- |
| **社内AIチャット / 新人教育AI** | 「同じ質問を新人が繰り返す」「マニュアルが古い」 | 保存庫内のFAQや契約条件をリアルタイム参照可能 |
| **議事録整理AI** | 「商談後の整理に時間がかかる」「共有が遅れる」 | 保存庫内の顧客別条件を自動反映させ、正確な議事録を生成 |
| **CS対応AI** | 「問い合わせの属人化」「夜間の対応漏れ」 | 保存庫の知識を基に、一貫性のある回答を自動生成 |

### 提案書面（アウトライン）

**タイトル：【既存顧客限定】AIpaX保存庫コアの活用範囲拡大プラン**

1.  **現状の成果確認**
    *   現在構築済みの「会社別AI保存庫コア」には、貴社の重要なビジネスナレッジが蓄積されています。
    *   これまで「営業電話支援」で活用いただき、感謝申し上げます。

2.  **次のステップ：保存庫の「多角化」**
    *   保存庫は「電話支援」だけでなく、社内業務全体を効率化する基盤です。
    *   初期構築コスト（50-80万円相当）を免除し、**追加ツールのみ**の導入を提案します。

3.  **推奨プラン：「新人教育・社内共有」セット**
    *   **内容:** 社内AIチャットツールの追加
    *   **初期費用:** 10万円（保存庫再利用のため実質無料に近い価格）
    *   **月額追加:** 3万円〜5万円
    *   **効果:** 新人のオンボーディング期間の短縮、属人化の解消

4.  **オファー**
    *   今月中に追加ツールを導入いただければ、次回の月額料金の改定を3ヶ月間凍結（または10%割引）いたします。

# パターン2：年契約提案（継続収益の安定化と割引誘導）

**目的:** 月額契約から年契約へ移行させ、現金フローの安定化と解約リスクの低減を図る。
**対象:** 導入効果を実感し、継続利用を希望しているお客様

### 提案の論理構成
1.  **コストの透明化:** 月額契約の場合、1年間で支払う総額がいくらになるかを可視化。
2.  **インセンティブ提示:** 年契約にすることで、月額実質価格を下げたり、追加機能を無料開放したりする。
3.  **安心感の提供:** 年契約＝「長期的なパートナーシップ」の証であり、優先的なサポートを受けられることを強調。

### 価格比較シミュレーション（例：月額5万円の場合）

| 契約形態 | 初期費用 | 月額費用 | 1年間の総支払額 | メリット |
| :--- | :--- | :--- | :--- | :--- |
| **月額契約** | 0円 | 5万円 | **60万円** | 柔軟性が高い |
| **年契約** | 10万円 | **3.5万円** | **52万円** | **年間8万円節約**<br>追加ツール1つ無料 |

※注：初期費用は「保存庫の維持管理費」として位置づけ、月額割引と相殺する形で見せる。

### 提案書面（アウトライン）

**タイトル：【長期利用特典】AIpaX年契約プランへの変更のご案内**

1.  **ご利用状況のご報告**
    *   貴社ではAIpaXを〇ヶ月間ご利用いただき、〇〇の効率化に貢献いただいております。

2.  **年契約への変更メリット**
    *   **コスト削減:** 月額実質価格を**30%OFF**（月額5万円→3.5万円）に改定。
    *   **機能追加:** 「管理レポートAI」または「CRM補助ツール」を**6ヶ月間無料**で提供。
    *   **優先サポート:** 年契約顧客専用チャネルでの緊急対応優先権。

3.  **ご提案内容**
    *   **契約期間:** 12ヶ月
    *   **初期費用:** 10万円（システム維持管理費）
    *   **月額費用:** 3.5万円（税込）
    *   **合計支払額:** 52万円（月額契約の場合の60万円と比較し、8万円の節約）

4.  **締切**
    *   本特典は〇月〇日までの申込者に限ります。
    *   年契約への変更により、次回の月額請求額から適用されます。

# パターン3：紹介報酬提案（顧客からの営業力活用）

**目的:** 既存顧客を「営業パートナー」として位置づけ、低コストで新規顧客（5社目以降）を獲得する。
**対象:** AIpaXの成果を実感し、信頼関係が構築できているお客様

### 提案の論理構成
1.  **感謝の表明:** 現在の成果に対して感謝し、パートナーとしての関係を強化。
2.  **Win-Winの構造:** 紹介成功時に、紹介者（既存顧客）と被紹介者（新規顧客）の両方にメリットがあることを提示。
3.  **ハードルの低下:** 「営業活動」ではなく「つなぎ役」に徹してもらうため、負担感を軽減。

### 報酬体系案

| 紹介成功時の報酬 | 内容 |
| :--- | :--- |
| **紹介者（既存顧客）へ** | **月額料金の3ヶ月分** または **現金30万円** |
| **被紹介者（新規顧客）へ** | **初期費用10万円割引** または **追加ツール1ヶ月無料** |

※新規顧客の通常価格（初期100-150万/月額10-15万）を見据え、紹介者は高額な報酬を支払えるが、新規顧客の導入ハードルを下げることで成約率を上げる。

### 提案書面（アウトライン）

**タイトル：【パートナーズプログラム】AIpaX紹介キャンペーンのご案内**

1.  **パートナーシップの提案**
    *   AIpaXは、導入に成功した顧客様を「パートナー」として位置づけ、共に成長する仕組みを作りました。
    *   貴社の成功事例は、他社様にとっても大きな参考になります。

2.  **紹介キャンペーン内容**
    *   貴社から他社様へAIpaXをご紹介いただき、契約が成立した場合、以下の報酬をお支払いします。
    *   **報酬:** 月額料金の3ヶ月分（例：月額5万円の場合、15万円相当のクレジットまたは現金）
    *   **特典:** ご紹介いただいた先様には、初期費用10万円の割引特典をご用意しております。

3.  **簡単な進め方**
    1.  貴社が「AIpaXに興味がありそうな先」を1名ご紹介ください。
    2.  私たちから資料を送付し、ヒアリングを実施します。
    3.  契約成立次第、報酬を請求書発行後、翌月月末に支払います。

4.  **お問い合わせ**
    *   ご興味のある方は、担当までお気軽にお申し付けください。
    *   紹介事例の共有（匿名可）も歓迎しております。

# 実施上の注意点・共通事項

1.  **保存庫コアの価値強調:**
    *   どの提案でも、「保存庫コア（マザーボード）」が既に完成していることを前提に話す。これにより、追加ツールは「ゼロからの開発」ではなく「既存資産の拡張」として捉えられ、抵抗感が減る。
2.  **価格の明確化:**
    *   5社目以降は通常価格（初期100-150万/月額10-15万）へ移行することを前提としているため、既存4社へのアップセルは「β価格（初回事例価格）の延長」ではなく、「本格導入へのステップアップ」として位置づける。
3.  **個別カスタマイズ:**
    *   各社の業種、現在の課題、導入ツールに応じて、パターン1の追加ツール候補を最適化する。
4.  **タイミング:**
    *   オンボーディング完了後、または導入効果（KPI）が明確に出た時点で提案するのが最も効果的。
