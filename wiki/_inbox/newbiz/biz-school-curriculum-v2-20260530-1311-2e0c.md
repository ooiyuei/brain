# AIpaX school 1年カリキュラム（5モジュール完成版）

---

## 🎯 はじめに：AIpaX schoolとは？

AIpaX schoolは、**中高生向けのAI教育スクール**で、AIを手足のように使えるスキルを実践で身につけることを目的としています。  
**価格は月額3万円**で、**1年間のカリキュラム**は5つのモジュールに分かれ、それぞれ2ヶ月間の学習期間を設けています。  
このカリキュラムは、**AIマインドセット → 日常業務自動化 → エンジニアリング思考 + 外部連携 → 本格アプリ開発 + リリース → データ分析 + マネタイズ**の流れで構成されており、**実践型のAI教育**を実現します。

## 📌 モジュール1：AIマインドセット（Month 1-2）

### 🎯 学習目標
- AIの基本的な概念とその社会への影響を理解する
- AIを「手足」として使えるマインドセットを構築する
- バイブコーディング体験を通じてAIの実用性を体感する

### 📅 週次トピック（8週間）

| 週 | トピック | 説明 |
|----|----------|------|
| 1 | AIとは？ | AIの定義、歴史、技術の進化 |
| 2 | AIの社会への影響 | AIがもたらす社会変化と課題 |
| 3 | AIの倫理と責任 | AIの倫理的使用と責任の所在 |
| 4 | AIツールの紹介 | ChatGPT、Claude、Canva AIなど |
| 5 | プロンプト設計入門 | プロンプトの作り方と効果的な表現 |
| 6 | バイブコーディング体験 | AIツールを使って簡単なアプリを作成 |
| 7 | AIと未来の仕事 | AIが変える職種と新しい働き方 |
| 8 | AIマインドセットのまとめ | 学んだことを振り返り、自己評価 |

### 📝 課題
- AIツールを使って、**自分の生活に役立つAIアプリのアイデア**を1つ提案
- プロンプト設計の練習として、**AIに質問を投げて、その返答を分析**する

### 🛠 必要ツール
- **ChatGPT / Claude / Canva AI** などのAIツール
- **Replit / Cursor**（コーディング用）
- **Notion / Google Drive**（資料作成用）

### 📊 評価方法
- **AIツールの使い方**と**プロンプト設計の理解度**
- **バイブコーディング体験の成果**（アプリの完成度）
- **自己評価と振り返りの質**
