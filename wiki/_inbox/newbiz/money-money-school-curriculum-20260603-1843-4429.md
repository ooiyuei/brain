# AIpaX school 1年カリキュラム (5モジュール構造)

## 🎯 5モジュール構成 (1年間 × 2ヶ月 = 10ヶ月)

### Module 1: AIマインドセット + バイブ体験 (Month 1-2)
- **学習目標**: AIを手足のように使えるマインドセットを完成
- **週次トピック**:
  1. AIの現状と未来 (2026年AI市場動向)
  2. AIと人間の関係性 (AIが解決できる課題)
  3. AIツールの選定と使い方 (ChatGPT / Claude / Canva AI)
  4. プロンプト設計の基礎 (AIと対話する技術)
  5. AIと人間の協働 (AIが補助する仕事)
  6. AIの倫理と責任 (AIの限界とリスク)
  7. AIで何か作れる体験 (簡単なコード・デザイン作成)
  8. AIと人間の未来 (AI時代の生き方)
- **課題と発表**: AIツールで作成した作品を発表 (例: AIで作成したデザイン・文章)
- **必要ツール**: Antigravity / Cursor / Claude / Figma
- **評価方法**: 作品の完成度 + プロンプト設計の質 + チームでの発表

### Module 2: 日常業務自動化 (Month 3-4)
- **学習目標**: 日常業務をAIで自動化するスキルを習得
- **週次トピック**:
  1. 日常業務のAI化 (例: メール自動作成・スケジュール管理)
  2. AIツールの選定と使い方 (Replit / Cursor / Notion AI)
  3. プロンプト設計の応用 (業務に特化したプロンプト)
  4. AIで自動化する仕事 (例: データ整理・情報収集)
  5. AIと人間の協働 (AIが補助する仕事)
  6. AIの限界とリスク (AIができない業務)
  7. AIで自動化する仕事の実践 (例: メール自動作成)
  8. AIと人間の未来 (AI時代の生き方)
- **課題と発表**: AIで自動化した業務を発表 (例: 自動メール作成・データ整理)
- **必要ツール**: Antigravity / Cursor / Notion AI / Figma
- **評価方法**: 作品の完成度 + プロンプト設計の質 + チームでの発表

  5. AIと人間の協働 (AIが補助する仕事)
  6. AIの限界とリスク (AIができない業務)
  7. AIと人間の未来 (AI時代の生き方)
  8. AIと人間の協働の実践 (例: AIと人間の協働)
- **課題と発表**: AIと人間の協働で作成した作品を発表 (例: AIと人間の協働で作成したデザイン)
- **必要ツール**: Antigravity / Cursor / Figma / Notion AI
