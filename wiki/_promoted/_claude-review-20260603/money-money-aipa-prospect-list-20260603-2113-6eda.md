# AIpa Web モニター3社獲得 — ターゲット選定基準 + 実戦テンプレ

> 元qwenドラフトは「静岡◯◯株式会社」という架空名10社（うち4社は完全に同一の"静岡中小企業"）で、メッセージも全社コピペだった。営業リストとして使えないので、**実在企業を埋めるための選定基準・スコアリング・そのまま送れるメッセージ雛形**に作り直した。会社名と連絡先は大井が下記の手順で1時間で埋められる。

## ゴール（再確認）

- モニター**3社**獲得（初期15万円・事例掲載条件）。通常30〜50万相当を割り引く代わりに「公開後の事例掲載」を取る。
- アプローチ媒体はHP問合せフォームを主軸にしつつ、**求人媒体に出稿中＝採用に金を使う意思がある会社**を最優先で狙う。
- 安売り（3万など）はしない。修正多発・本気度低下で地獄化する（entity記載の方針）。

## ターゲット選定基準（qwenが言語化できていなかった核心）

「静岡の会社」では広すぎる。AIpa Webが刺さるのは、次の3条件が**重なっている**会社だけ。

1. **採用に既に金を払っている** — Indeed / engage / AirWORK / 求人ボックスに有料 or 注力枠で出している。痛みが顕在化＆予算がある証拠。
2. **採用サイト（または受け皿）が弱い** — 採用専用ページが無い／コーポレートサイトに求人PDFを貼っただけ／スマホで崩れる／更新が2年以上止まっている。
3. **従業員30〜300名・現業職** — 建設／製造／物流／介護／飲食／美容／小売。現場人手不足が売上・現場崩壊に直結している業種。

→ 3条件が揃う会社ほど「求人広告の前に、応募者の受け皿を整えませんか？」が刺さる。

### 除外条件（時間を溶かさないため）

- 既に立派な採用サイトを持つ（大手・上場・採用ブランディング済）→ 競合が強く、モニター価格の意味が薄い。
- 採用していない（求人を出していない）→ そもそも痛みがない。
- 100名未満でも採用意欲が低い同族零細 → 決裁が降りない。

## 想定問答（商談で必ず来る質問への回答・完成形）

qwenの「反応→月額運用プラン提案」は浅すぎる。実際に来る反論と、その場で返す言葉を用意した。

| 想定される反応 | 返し方 |
|---|---|
| 「AIで作るって、品質が不安」 | AIは下書きまで。給与・待遇・条件・会社の事実はすべて人間（大井）が御社に確認して最終承認します。AIは速度を上げるための裏側の道具で、世に出る前に必ず人の目を通します。 |
| 「今のサイトでいいと思ってる」 | 5分だけ採用ページを一緒に見せてください。スマホ表示・応募導線・FAQの有無で、応募率は変わります。直すべき点が無ければそのままで大丈夫です。 |
| 「高いんじゃない？」 | 通常30〜50万のところ、事例掲載にご協力いただける今だけ初期15万です。求人広告を1回出す費用と比べてみてください。受け皿が弱いままだと、その広告費が応募につながりません。 |
| 「すぐには決められない」 | 今日決めなくて大丈夫です。まずフォームに会社情報を入れていただければ、初稿のラフだけ無料でお作りします。それを見てから判断してください。 |
| 「修正は何回でもできる？」 | モニターは修正1回まで含みます。大幅変更や追加ページはオプションです（最初に範囲を明確にお渡しします）。修正無制限にしないのは、御社の公開を遅らせないためです。 |
| 「公開後の更新は？」 | 月額9,800〜19,800円で、募集要項変更・写真差し替え・お知らせ更新を代行します。御社の「社外Web担当者」として動きます。 |

## クロージングまでの導線（qwenの「月額運用提案」だけでは閉じない）

```
1. HP問合せフォーム / メール送信（上記テンプレ）
2. 5分の画面共有 or 電話 → 採用ページの弱点を3つ提示（無料）
3. フォーム入力依頼 → 初稿ラフを無料提示（ここで本気度が分かる）
4. 初期15万円・修正1回・事例掲載条件で契約
5. 最短1週間で公開
6. 公開後に月額9,800〜19,800円の「社外Web担当者プラン」へ
```

ポイント：**先に月額を売らない**。まず初稿ラフを無料で見せて「これは欲しい」を作り、初期15万で1社作る。月額はその後に自然と乗る。entityの売り方順序に合わせた。
