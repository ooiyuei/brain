# AIpaX 導入のご提案

会社別AI保存庫コア ＋ 業務AIツール

---

## はじめに

このたびは AIpaX にご関心をお寄せいただき、ありがとうございます。

中小企業のAI導入は、まだ全体の5%ほどしか進んでいません。理由はシンプルで、「コストが高い」「使い方が分からない」「結局うちの業務には合わない」の3つに集約されます。多くのAI導入は、汎用のチャットボットを置いて終わり。自社のルールも、料金表も、お客様ごとの例外条件も知らないAIは、現場では使えません。

AIpaX は、ここを逆から作ります。まず**御社専用の「AI保存庫コア」**を構築し、御社の資料・契約・料金・FAQ・顧客情報・例外条件をAIが正しく参照できる状態にする。その上に、御社が一番困っている業務のツールを1つ載せる。これがAIpaXの導入の形です。

本提案では、初回導入の内容・価格・スケジュールをまとめています。

## なぜ「保存庫コア」から作るのか

一般的なAI導入は、ツールを1つ作って納品して終わりです。次の業務でまた必要になったら、またゼロから作り直す。これだと費用も時間もかさみ、AIが御社のことを覚えてくれません。

AIpaX は構造が違います。

```
御社専用 AI保存庫コア（土台）
        ＋
業務AIツール（必要な分だけ載せる）
```

**保存庫コア**は、御社の資料・契約・料金・FAQ・顧客情報・お客様ごとの条件・例外条件・やり取りの履歴を、AIが検索・参照・再利用できる形に整理した共通基盤です。一度ここを作っておけば、次に別の業務ツールを足すときも、土台はそのまま使えます。毎回作り直さない。これがAIpaXの一番の価値です。

保存庫コアがあることで、

- AIが御社のルールや例外条件を理解した上で、業務に即した回答を返せる
- 社内の知識が一箇所にまとまり、新人教育や引き継ぎが楽になる
- AIが勝手に正式情報を作らないため、回答を信頼して使える

---

## 業務ツールは、御社の一番の困りごとから1つ

初回は、御社が今いちばん困っている業務を1つ選んでツール化します。

| 御社のお困りごと | 最初に載せるツール |
|----------|------------------|
| 営業電話で回答がブレる・担当で差が出る | リアルタイム電話／営業回答支援 |
| 新人が同じルールを何度も聞いてくる | 社内AIチャット／新人教育AI |
| 商談後の議事録・整理が後回しになる | 議事録整理AI |
| 問い合わせ対応が特定の人に偏っている | CS対応AI |
| フォローの抜け漏れが起きる | フォロー文生成AI |
| 管理者が現場の状況を把握しづらい | 管理レポートAI |

最も導入効果が見えやすい初手は、**保存庫コア ＋ リアルタイム電話／営業回答支援**の組み合わせです。営業現場で「この条件のときどう答えるか」を、その場でAIが御社のルールに沿って提示します。

## AIpaX の5つの約束

保存庫コアは、御社の情報を扱う土台です。だからこそ運用には明確なルールを設けています。

1. 保存庫コアの情報と、ツールが生成した出力を混ぜません
2. 御社資料の正しさを決めるのは、あくまで御社側です
3. AIが勝手に正式情報を作りません
4. 情報の不足・矛盾・最新版が不明な点は、検出してお知らせします
5. 対象領域内のお客様別条件・例外条件は、勝手に削りません

加えて、立ち上げ期の3ヶ月は運用サポートに含めて伴走し、その間に見つかった不具合は無償で修正します。

## よくあるご質問

**Q. 本当に中小企業でも使えますか？**
はい。AIpaX はもともと中小企業向けに設計しています。大がかりなシステム導入ではなく、御社の一番困っている業務から小さく始め、効果を見ながら広げられます。

**Q. 導入したあとの運用が不安です。**
立ち上げ期の3ヶ月は運用サポートに含めて伴走します。保存庫の更新・回答改善・不具合修正まで一緒に進めるので、社内にAI担当がいなくても運用できます。

**Q. なぜ「AIが勝手に正式情報を作らない」ことが大事なのですか？**
AIが推測で料金や条件を答えてしまうと、現場で誤案内につながります。AIpaX は保存庫にある御社の正しい情報だけを参照し、情報が足りないときは「足りない」と返します。だから安心して現場で使えます。

**Q. 4ヶ月目以降はどうなりますか？**
月額5万円で運用を継続します。対象領域内の資料更新・回答改善・軽微な不具合修正などが含まれます。新しいツールの追加や対象領域の拡張は、別途ご相談のうえお見積もりします。
