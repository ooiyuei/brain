# AIpa Web モニター獲得用 営業メール 5本

> 用途: AIpa Web の事例モニター3社を獲得するためのコールドメール。
> 価格の正本は [[entities/aipa-web]]: **採用LPを初期15万円（通常30〜50万相当）・事例掲載条件**。
> ※元ドラフトの「月額15万円」は誤り（モニターは初期15万・事例掲載条件）。月額運用は別途9,800〜19,800円〜。

## 共通の前提（5本に効く設計）

- **ターゲット**: 地方の建設・製造・運輸・サービス業、従業員30〜300名、採用に困っている経営者・採用担当
- **痛み**: 求人広告にお金を払っても、応募者が見る「会社の受け皿（採用サイト）」が古い/無い → 応募が逃げる
- **オファー**: フォーム入力 → AI制作 → 人間が事実確認 → 最短7日公開 → 初期15万（事例掲載条件・3社限定）
- **NG表現**: 「AIでWebサイトを作ります」（安く聞こえる）。売るのは「採用の受け皿を、安く早く整える社外Web担当者」
- **CTA統一**: 「15分だけオンラインで現状を見せてください。今の求人ページを一緒に見ながら、改善点を3つお伝えします」
- 1社1社、件名と冒頭1文だけ相手の社名・業種・地域に差し替えて送る。テンプレ感を消すのはここだけで十分

差出人の固定ブロック（全メール末尾に付ける）:

```
――――――――――
AIpa Web ／ 大井湧瑛
採用サイトの制作・更新代行（AIを活用した「社外Web担当者」）
Mail: 《要記入: 送信元メールアドレス》
Tel : 《要記入: 電話番号（無ければ削除）》
Web : 《要記入: サービスページURL（無ければ削除）》
※今だけ事例協力いただける3社限定で、通常30〜50万円相当の採用LPを初期15万円でご用意します。
```

## パターン1: 数字訴求（建設業向け）

**件名**: 採用サイト、7日で。今だけ事例協力3社限定で初期15万円

**本文**:

```
《要記入: 会社名》 採用ご担当者さま

突然のご連絡失礼します。採用サイトの制作・更新を代行している、AIpa Webの大井と申します。
御社の求人を拝見し、ご連絡しました。

求人広告にお金をかけても、応募者が最後に見る「会社の採用ページ」が
古かったり無かったりすると、そこで応募が止まってしまいます。
建設業は人手の確保が売上に直結するぶん、ここの取りこぼしがもったいないと感じています。

AIpa Webは、フォームに沿ってお答えいただくだけで、
・採用サイトの構成
・仕事内容や求める人物像の文章
・スマホで見やすいデザイン
までをこちらで用意し、最短7日で公開します。給与や条件などの事実は必ず御社にご確認いただきます。

今、事例として実績に掲載させていただける3社限定で、
通常30〜50万円相当の採用LPを初期15万円でご用意しています。

15分だけオンラインで、今の求人ページを一緒に見ながら、
改善点を3つお伝えするところから始めませんか。来週前半でご都合いかがでしょうか。
```

**狙い**: 「7日」「初期15万円」「3社限定」の3つの数字で、判断材料を最初に渡す。
**この型が効く相手**: 価格と納期で意思決定する現場系・即断型の経営者。

## パターン2: 質問型（製造業向け）

**件名**: 御社の求人、応募者は「会社の中」を見られていますか？

**本文**:

```
《要記入: 会社名》 採用ご担当者さま

はじめてご連絡します。採用サイトの制作・更新代行をしている、AIpa Webの大井です。

ひとつだけ伺わせてください。
求人広告から興味を持った人が、御社のことをもっと知りたくなったとき、
仕事の中身・働く人・1日の流れが伝わるページは、今ありますか。

製造業の採用では「危なそう」「きつそう」という先入観を、
現場のリアルな写真と言葉で先に解いておけるかどうかで、応募数が変わります。
ここを整える採用サイトを、こちらでご用意します。

AIpa Webは、フォームにお答えいただくだけで、構成・文章・デザインの初稿をこちらで作成し、
御社に事実確認いただいたうえで最短7日で公開します。

今は事例にご協力いただける3社限定で、通常30〜50万円相当を初期15万円でご案内しています。

まずは15分、今の求人ページを一緒に見ながら改善点を3つお伝えします。
ご都合のよい日を2〜3いただけますか。
```

**狙い**: 質問で「自社にそのページが無い」と気づかせてから解決策を出す。
**この型が効く相手**: 課題を言語化できていない、説得より納得で動く担当者。

## パターン3: 事例風（建設業向け・社会的証明）

**件名**: 採用サイトに手が回らない建設会社が、7日で受け皿を作った話

**本文**:

```
《要記入: 会社名》 採用ご担当者さま

採用サイトの制作・更新を代行している、AIpa Webの大井と申します。

最近こんなご相談が増えています。
「求人は出している。でも採用サイトまで手が回らず、自社の魅力が伝えきれていない」
特に建設業は、本業が忙しいほど採用の見せ方が後回しになりがちです。

AIpa Webでは、フォームにお答えいただくだけで、
構成・採用コピー・デザインの初稿をこちらで用意し、
御社に事実確認いただいたうえで、最短7日で採用サイトを公開します。
公開後の募集要項変更や写真差し替えといった更新も代行します。

《要記入: 既存の納品事例があれば1社の要約（業種・公開までの日数・反応）を1〜2行で記載。
無ければこの段落は削除する。捏造はしない》

今は事例にご協力いただける3社限定で、
通常30〜50万円相当の採用LPを初期15万円でご用意しています。御社もこの枠でいかがでしょうか。

15分のオンラインで、今の求人ページの改善点を3つお伝えします。来週でご都合のよい時間はありますか。
```

**狙い**: 「同じ立場の会社が解決した」で心理的ハードルを下げる。
**注意**: 実績が無いうちは捏造禁止。事例段落は《要記入》のまま空欄か削除で運用する。

## パターン4: 緊急性（製造業向け・機会損失）

**件名**: 求人広告の費用、受け皿が無いと半分こぼれているかもしれません

**本文**:

```
《要記入: 会社名》 採用ご担当者さま

採用サイトの制作・更新代行をしている、AIpa Webの大井です。

求人広告は、応募者を「入口」まで連れてくる費用です。
でも、入口の先にある採用ページが弱いと、せっかく払った広告費の一部が、
そこで応募に至らずこぼれてしまいます。製造業の採用が厳しい今ほど、この取りこぼしは痛いはずです。

AIpa Webは、フォームにお答えいただくだけで構成・文章・デザインの初稿を用意し、
御社に事実確認いただいて最短7日で公開します。次の求人募集に間に合わせることもできます。

今は事例にご協力いただける3社限定で、通常30〜50万円相当を初期15万円でご用意しています。
枠が埋まり次第この価格は終了しますので、検討中の場合はお早めにお声がけください。

15分のオンラインで、今の求人ページを見ながら改善点を3つお伝えします。
今週か来週で、ご都合のよい時間をいただけますか。
```

**狙い**: 「広告費がこぼれている」という具体的な損失と、3社限定の枠で締切感を作る。
**注意**: 煽りすぎない。事実（受け皿の重要性・枠の限定）の範囲で緊急性を出す。

## パターン5: 同地域・共感（建設業向け・地元軸）

**件名**: 《要記入: 地域名（例: 静岡）》の建設会社の採用、受け皿づくりを手伝います

**本文**:

```
《要記入: 会社名》 採用ご担当者さま

はじめまして。採用サイトの制作・更新を代行している、AIpa Webの大井と申します。

《要記入: 地域名》の中小企業の採用課題を見ていて、共通して感じることがあります。
求人にはお金をかけているのに、「会社の受け皿」である採用サイトだけが
手薄なまま、という会社がとても多いことです。本業が忙しい建設業ほど、ここまで手が回りません。

私たちは、その手の回らない部分を引き受ける「社外のWeb担当者」のような立場です。
フォームにお答えいただくだけで、構成・文章・デザインの初稿をこちらで作り、
御社の事実確認のうえ最短7日で公開、その後の更新も代行します。

今は事例にご協力いただける3社限定で、通常30〜50万円相当の採用LPを初期15万円でご案内しています。
同じ地域の会社の採用を、まず1社しっかり形にしたいと思っています。

15分だけ、今の求人ページを一緒に見る時間をいただけませんか。
来週前半でご都合のよい時間を2〜3いただけると助かります。
```

**狙い**: 「地元の会社を助けたい」という距離の近さで、コールドメールの壁を下げる。
**この型が効く相手**: 地縁・人間関係で動く地方の経営者。紹介・商工会議所経由とも相性が良い。
