# AIpaX β導入 業務委託契約書（テンプレート）

> 用途: AIpaX の β（初回事例価格）顧客と締結する業務委託契約のひな型。
> 締結前に弁護士レビューを通すこと（このテンプレは商談を止めないための土台。最終版ではない）。
> 角括弧 `[ ]` は締結ごとに埋める箇所。`《要記入: 》` は大井本人の確認が要る箇所。

---

委託者 [顧客企業名]（以下「甲」）と、受託者 大井湧瑛 株式会社（屋号: AIpaX、以下「乙」）は、乙が甲に提供する「会社別AI保存庫コア」および業務AIツールに関し、次のとおり業務委託契約（以下「本契約」）を締結する。

---

## 第1条（目的）

本契約は、乙が甲に対し、甲の業務データを AI が検索・参照・再利用できる形に整理した基盤「会社別AI保存庫コア」（以下「保存庫コア」）を構築・運用し、その上に甲の業務を支援する業務AIツールを提供することを目的とする。

本契約は β（初回事例）導入を前提とし、乙のサービス品質向上のための運用データ取得・改善を相互に織り込んだ条件とする。

## 第2条（業務範囲）

乙は甲に対し、別紙1「対象領域定義書」で合意した範囲について、次の業務を提供する。

**(1) 保存庫コアの構築**
- 甲の資料・契約・料金・FAQ・顧客情報・顧客別条件・例外条件・やり取り履歴等のうち、別紙1で定めた対象領域のデータを整理し、AI が検索・参照できる形に反映する。
- 対象領域内の顧客別条件・例外条件は削らずに扱う。

**(2) 業務AIツールの提供**
本契約で導入するツールは次のとおりとする（別紙2で確定）。

- 導入ツール: [ リアルタイム電話/営業回答支援 等、別紙2で指定 ]

選択可能なツール群: リアルタイム電話/営業回答支援、社内AIチャット/新人教育AI、議事録整理AI、フォロー文生成AI、CS対応AI、管理レポートAI、CRM補助ツール。追加ツールは第3条の追加業務として別途見積もる。

**(3) 月額運用に含む業務**
対象領域内の資料更新、入庫前処理、甲の確認後の保存庫反映、旧情報の非アクティブ化、検索条件の軽微調整、回答カードの軽微改善、利用ログ確認、回答できなかった質問の整理、軽微な不具合修正。

## 第3条（範囲外業務・追加料金）

次の業務は本契約の月額運用に含まれず、実施する場合は事前見積もりのうえ別途合意する。

新ツール作成、対象領域外データの追加、外部システム連携（電話回線・CRM・Zoom/Meet 等）、紙資料・手書き資料の OCR、大量旧資料の整理、FAQ・契約条件・社内ルールの新規作成代行、緊急対応、別部署・別商品ラインへの展開、顧客側アカウント構築、乙側から甲側への運用移管、現地作業、法務・契約条件整理代行。

## 第4条（役割分担・5原則）

乙は次の原則に従って業務を行い、甲はこれを理解のうえ協力する。

1. 保存庫コアとツール出力を混在させない。
2. 甲の資料の正しさは甲が最終決定する。乙は甲の確認を経て保存庫へ反映する。
3. AI は正式情報を勝手に生成しない。
4. 乙は資料の不足・矛盾・最新版不明を検出し甲に提示する。
5. 対象領域内の顧客別条件・例外条件は削らない。

甲は、保存庫コアに反映する資料の提供・内容確認、および AI 出力の業務利用判断について責任を負う。

## 第5条（委託料）

| 区分 | 金額（税別） | 支払時期 |
|---|---|---|
| 初期費用（保存庫コア構築 + ツール1種初期設定） | [ 50万〜80万 ] 円 | 本契約締結後 [ ] 日以内 |
| 月額運用費用 | [ 5万 ] 円 | 毎月末締め翌月末払い |

実費（OpenAI API 等の利用量課金）の取扱いは別紙3の管理方式に従う。

> 注: 上記は β（初回事例）価格。通常価格は初期100〜150万・月額10〜15万（5社目以降適用）。本契約の価格は《要記入: 甲に提示した実際の金額を確定》。

## 第6条（OpenAI / Supabase 管理方式）

API・データベースの管理方式は、別紙3で次のいずれかを選択する。

- A. 乙側管理（標準）: 導入が早く修正しやすい。実費は乙が管理し甲へ請求。
- B. 甲側管理: 機密性が高い。情報システム部門のある企業向け。追加費用オプション。
- C. 乙側から甲側への移管: 初期は乙側で早く導入し、後に甲側へ移管。別料金。

本契約では [ A / B / C ] を採用する。

## 第7条（通話ログ・利用ログの保存）

リアルタイム電話/営業回答支援ツールを導入する場合、通話ログの保存方式は別紙3で次のいずれかを選択する。

- A. 保存しない。
- B. 文字起こし・検知質問・回答候補・検索ログのみ保存（標準。改善に利用）。
- C. 音声と文字起こしの両方を保存（追加設計が必要）。

本契約では [ A / B / C ] を採用する。

## 第8条（データの取扱い・秘密保持）

1. 甲が乙に提供する業務データおよび本契約に関して知り得た相手方の秘密は、相手方の書面承諾なく第三者に開示・提供しない。本義務は契約終了後 [ 3 ] 年間存続する。
2. 乙は甲のデータの漏洩・紛失・改ざんを防ぐため、合理的なセキュリティ対策を講じる。
3. 乙は、サービス改善および AI の精度向上を目的として、第7条で甲が同意した範囲で利用ログを利用できる。甲の個別の業務データを、甲以外の保存庫コアに転用しない。
4. 個人情報を含むデータは、個人情報保護法その他関係法令に従って取り扱う。

## 第9条（知的財産権）

1. 乙が提供する保存庫コアおよび業務AIツールの仕組み・プログラム・汎用テンプレートに関する知的財産権は乙に帰属する。甲は本契約の範囲で利用する権利を持つ。
2. 甲が乙に提供した業務データ、および当該データそのものに対する権利は甲に帰属する。
3. 甲は、乙の事前承諾なく、乙の提供物を第三者へ再販売・再配布・改変しない。

## 第10条（契約期間・解約）

1. 本契約の期間は締結日から [ 1 ] 年間とし、いずれからも期間満了 [ 1 ] か月前までに書面の申し出がなければ同一条件で1年間更新する。
2. 甲または乙は、[ 30 ] 日前までに書面で通知することにより本契約を解約できる。
3. 既に発生した初期費用および解約日までの月額費用は返金しない。

## 第11条（解約後のデータ）

1. 解約後、乙は保存庫コアの運用を停止する。
2. 甲は、解約日から [ 30 ] 日以内に申し出ることにより、甲が提供した業務データおよび保存庫コアに反映された甲のデータの返却（[ CSV / JSON 等 ] の形式でのエクスポート）を受けられる。
3. 返却完了後、乙は甲のデータを [ 30 ] 日以内に削除する。削除の完了は甲の求めに応じて書面で報告する。

## 第12条（一般条項）

1. 本契約は日本法に準拠し、日本語を正文とする。
2. 本契約に定めのない事項、または解釈に疑義が生じた事項は、甲乙が誠実に協議して解決する。
3. 本契約の変更は、書面による甲乙双方の合意による。
4. 本契約に関する紛争は《要記入: 第一審の合意管轄裁判所（例: 東京地方裁判所 / 静岡地方裁判所）》を第一審の専属的合意管轄裁判所とする。
