# dinii時代の人脈 6月接触リスト

> 大井が dinii で働いていた頃の人脈を、AIpa Web / AIpaX の最初の顧客・紹介ルートに変える。
> 「久しぶりの連絡」を、押し売りにせず、自然に商談の入口へつなげる5本のシナリオ。

## 前提（このリストの考え方）

- **売り込みから入らない。** 1通目は近況報告と相手への質問だけ。商品の話は2通目以降。
- **dinii で得た共通言語（飲食店オペレーション・店舗CS・配信施策）を入口にする。** ここが大井の他の起業家にない武器。
- **相手によって出す事業を変える。**
  - 飲食店オーナー・店舗側 → **AIpa Web**（採用サイト/店舗Web）または AIpaX のCS対応AI
  - dinii社員・業界仲間（CS/営業）→ **紹介ルート**として扱う（自分で買う相手ではない）
- 連絡手段は LINE / 個人DM を基本にする。フォーマルなメールは元上司など一部のみ。

## 1. 元同僚（現役 dinii 社員 / CS）

- **狙い**: 紹介ルート化。dinii 導入先の飲食店で「採用サイトが弱い」「電話対応が属人化してる」店を紹介してもらう。
- **接触タイミング**: 6月第2週。相手の繁忙を避け、平日の夜か週末の昼に LINE。
- **1通目（売り込みなし）**:

  > 《名前》、お疲れさま。久しぶり、dinii の頃ぶりだね。
  > いま自分で会社やってて、飲食店向けに採用サイトとAI業務ツールを作る仕事をしてる。
  > diniiの導入先まわってる感覚って今も近いと思うんだけど、最近の店舗側の「人が採れない」「電話・問い合わせ対応がきつい」みたいな空気ってどう？
  > 久しぶりに少し話せたら嬉しい。

- **2通目（反応があったら）**: 自分の事業を1〜2行で説明 → 「もし“採用に困ってる店”“CS属人化で困ってる店”が頭に浮かんだら、1社だけ紹介してくれない? 無理な売り込みはしないから安心して」
- **想定リターン**: dinii 導入先飲食店の紹介 1〜3件 / 飲食業界の最新の困りごと一次情報。

## 2. 元上司

- **狙い**: 信頼の借り。事業の壁打ち相手 + 上司の人脈（飲食チェーン本部・SaaS界隈）への紹介。
- **接触タイミング**: 6月第2〜3週。きちんとしたメールで。
- **1通目（メール）**:

  > 件名: ご無沙汰しております（大井です）
  >
  > 《元上司の名前》さん、ご無沙汰しております。dinii でお世話になった大井です。
  > 退職後、飲食店向けに「採用サイト + AI業務支援」の事業を立ち上げて動いています。dinii で店舗オペレーションとCSを間近で見られた経験が、そのまま今の事業の土台になっています。
  > 直接お礼をお伝えしたいのと、もしお時間をいただけるなら、いま作っているものを一度見ていただき、率直なご意見をいただけたら嬉しいです。
  > オンライン15分でも構いません。ご都合いかがでしょうか。

- **想定リターン**: 事業フィードバック / 飲食チェーン本部・SaaS人脈への紹介 / リファレンス（実績紹介の後ろ盾）。

## 3. 担当していた飲食店オーナー（dinii 顧客）

- **狙い**: AIpa Web の **モニター1社目**候補。すでに大井を知っている=信頼ゼロからじゃない。
- **接触タイミング**: 6月第3週。LINE か直接電話。
- **1通目**:

  > 《オーナー名》さん、ご無沙汰してます、dinii でお世話になった大井です。
  > いま独立して、飲食店さんの採用サイトを作る仕事をしてます。求人出しても人が来ない、という相談をよくもらうので。
  > 《店名》さんの採用、今どんな感じですか? もし困りごとがあれば、お礼も兼ねて初回モニター価格で一度お手伝いできればと思ってます。久しぶりに顔出してもいいですか?

- **2通目（前向きなら）**: AIpa Web モニター条件を提示 → **初期15万円・事例掲載条件・最短7日公開**（[[entities/aipa-web]] のモニター価格に準拠）。
- **想定リターン**: モニター契約1件 / 事例として使える実績（地方飲食店の採用LP）。

## 4. 業界仲間（他社 CS）

- **狙い**: 横のつながりからの紹介 + 情報交換。買う相手ではなく、つなぐ相手。
- **接触タイミング**: 6月第3〜4週。DM。
- **1通目**:

  > お疲れさま。久しぶり。
  > いま飲食店向けにAI業務ツールと採用サイトの事業やってて、CS現場の「これ毎回同じ質問されてしんどい」みたいな話をめちゃくちゃ集めたくて。
  > 最近そっちの現場どう? 今度オンラインで近況交換しない?

- **想定リターン**: CS現場のリアルな課題（AIpaX のCS対応AIの仕様に反映）/ 別の困ってる店・企業の紹介。

## 5. 紹介ハブ人材（顔が広い人）

- **狙い**: 一番のレバレッジ。この人に事業を正しく理解してもらえば、紹介が連鎖する。
- **接触タイミング**: 6月最終週。まずカジュアルに会う約束。
- **1通目**:

  > 《名前》さん、ご無沙汰してます。
  > 独立して飲食店向けのAI・Web事業を始めました。《名前》さんは顔が広いので、近いうちに一度、いまやってることを聞いてもらえたら嬉しいです。
  > 「こういう人/店を紹介できるよ」が一番ありがたいので、軽くランチかオンラインでどうですか?

- **2通目（会えたら）**: 「紹介してほしい人物像」を1枚で渡す → ①採用に困ってる地方の飲食/サービス業オーナー ②CS属人化で困ってる中小企業。
- **想定リターン**: 紹介の連鎖 / 大井の事業を語ってくれる“伝道者”1人。

## 6月の進め方（順番）

1. **週2** 元同僚・元上司に着手（信頼が厚い順から。ここで事業の言語化を固める）
2. **週3** 飲食店オーナー＝モニター狙い + 業界仲間で情報収集
3. **週4** 紹介ハブで横展開を仕込む
4. 返信が来たら **48時間以内に2通目**。熱があるうちに会う約束まで取る。

## 想定問答（共通・聞かれたら答える用）

- **Q. dinii 辞めて何してるの?** → 「飲食・中小向けに、採用サイトとAI業務ツールを作る会社をやってる。diniiで見てきた店舗の現場感がそのまま武器になってる」
- **Q. それ高いんでしょ?** → 「採用サイトはモニターなら初期15万から。AIツールは事業ごとに見積もる。まず1社、ちゃんと結果出すのが先だと思ってる」
- **Q. dinii と競合しないの?** → 「全然違う。diniiはモバイルオーダー/POS。こっちは採用と、店舗の問い合わせ・電話対応のAI。むしろ併用できる」
- **Q. なんで自分に連絡を?** → 「dinii時代に一緒に現場見てた人にこそ、最初に話したかった。信頼してる人から始めたい」

## このリストの欠落（大井が埋めないと進まない）

- dinii 在籍期間と担当領域（プロフィールに何を書くかに直結）
