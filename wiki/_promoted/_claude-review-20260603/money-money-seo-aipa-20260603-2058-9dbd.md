# 中小企業の採用サイト制作費用はいくら？相場と「7日で公開」する新しい選び方

## メタ情報（公開時に使う）

- **タイトル**: 中小企業の採用サイト制作費用｜相場と、月額15万円で7日公開する方法
- **ディスクリプション**: 中小企業の採用サイト制作費用の相場は初期30〜100万円が中心。AIpa Webは打ち合わせ中心の受託をやめ、フォーム入力→AI初稿→人が確認の流れで最短7日公開。モニター価格・運用込みプランまで具体的に解説します。
- **想定読者**: 従業員30〜300名・地方の建設／製造／運輸／サービス業で、採用に困っていて、採用サイトを作りたいが「高い・遅い・面倒」で止まっている経営者・採用担当

## この記事の結論（先に言う）

中小企業の採用サイト制作費用は、制作会社に頼むと初期30〜100万円、納期1〜5ヶ月が中心です。一方、無料の採用ツールやAIサイトビルダーは0円〜数千円で作れますが、「結局、自分で構成も文章も考える」必要が残ります。

AIpa Webは、その中間で抜け落ちている層を取りに行くサービスです。フォームに答えるだけで、サイトの構成・採用コピー・文章・デザイン初稿までこちらで作り、最短7日で公開します。公開後の更新も月額で代行します。

> 売っているのは「Webサイト」ではありません。**「中小企業が採用サイトを作るときの面倒を、全部こちらで引き受けること」**です。

## 1. 採用サイト制作費用の相場（4つの選択肢を比較）

採用サイトの作り方は、ざっくり4タイプに分かれます。費用・納期・「自分でやる量」で並べると、どこに何が抜けているかが見えます。

| 作り方 | 代表例 | 費用の目安 | 納期 | 弱点 |
|---|---|---|---|---|
| 無料採用ツール | Airワーク、engage | 0円 | 10〜15分 | 無料だが差別化できない・型が決まりすぎ |
| AIサイトビルダー | STUDIO / Wix / Framer | 月590円〜／$10〜 | 即日 | 速いが「セルフ前提」。構成も文章も自分で考える |
| 制作会社 | 受託のWeb制作会社 | 初期40〜100万円 | 1〜5ヶ月 | 品質は出るが、高い・遅い・打ち合わせが多い |
| **AIpa Web** | — | **モニター初期15万円／運用込み** | **最短7日** | 撮影・動画など重い演出は別オプション |

ポイントは2つです。

1. **無料ツールとAIビルダーは「ツールを渡されるだけ」**。結局、誰が構成を決め、誰が文章を書くのかという問題が残ります。中小企業の現場には、それを考える時間も人もいないことが多い。
2. **制作会社は高くて遅い**。採用は「今すぐ人がほしい」課題なのに、公開まで数ヶ月かかると、その間の採用機会を逃します。

AIpa Webは、この「ツールは渡されたが自分で考える時間がない」「制作会社は高くて遅い」の両方の隙間を埋めます。

## 2. なぜ中小企業は採用サイト制作で詰まるのか

### 痛みが大きい業種ほど、後回しになっている

採用の痛みは、業種によって深さが違います。特に建設・物流・製造・介護・飲食・美容のような**現場仕事を抱える地方中小（20〜300名）**では、人手不足がそのまま売上減・現場崩壊に直結します。

ところが、こうした会社ほど採用サイトの優先順位が下がりがちです。理由はシンプルで、「作るのが面倒だから」。求人媒体には毎月お金を払っているのに、応募者が最初に見る「受け皿」である自社の採用ページが整っていない、という状態が起きています。

### 採用サイトは、実は「型」が決まっている

一方で、採用サイトの中身はかなり定型化できます。実務で必要なブロックは、ほぼ次の構成に収まります。

- ファーストビュー（一番上の印象）
- 採用キャッチコピー
- 会社紹介
- 数字で見る会社（離職率・平均年齢・売上など）
- 仕事内容
- 求める人物像
- 1日の流れ
- 社長メッセージ
- 社員インタビュー
- 福利厚生
- 募集要項
- よくある質問（FAQ）
- 応募導線

「痛みは大きいのに、構成は型化できる」。この組み合わせこそ、AIで効率化しながら品質を保てる理由です。ゼロから毎回考えるのではなく、型に沿ってヒアリングし、AIが初稿を作る。だから速くて安い。

## 3. AIpa Webの差別化ポイント

よくある誤解が「AIで安く作るサービスでしょ？」というものです。違います。安さは結果であって、売りではありません。

差別化の本質は、**作業を商品にしているのではなく、オペレーション全体を商品にしている**ことです。

```
フォームでヒアリングを標準化
  → AIが構成・コピー・文章を作成
  → 人が事実関係を確認（給与・待遇・権利・スマホ表示）
  → 7日で公開
  → 月額で更新を代行
```

- **AIサイトビルダー（Wix／STUDIO／Framer）との違い**：あちらは「ツールを渡す」だけ。構成も文章も顧客が自分で考えます。AIpa Webは構成・コピー・文章まで作って渡します。
- **制作会社との違い**：あちらは打ち合わせを重ねて数ヶ月かけます。AIpa Webはフォーム入力中心で、最短7日。
- **採用支援会社との違い**：あちらはWeb制作になると高く・遅くなりがち。AIpa Webは「応募者の受け皿」を安く早く用意します。

> 立ち位置は「AI Web制作会社」ではなく、**「社外のWeb担当者」**です。作って終わりではなく、公開後の更新まで担当します。

## 4. 制作フロー（フォーム入力から公開、運用まで）

実際の流れは次の通りです。顧客がやることは「フォームに答える」と「最後に内容を確認する」の2つだけです。

| ステップ | 内容 | 主担当 |
|---|---|---|
| 1 | 16セクションのフォームに入力 | 顧客 |
| 2 | フォーム回答をAI用の指示に変換 | AIpa Web |
| 3 | Claudeで構成・採用コピー・文章を作成 | AIpa Web |
| 4 | GPTimage2でデザインの方向性・FVビジュアル案 | AIpa Web |
| 5 | Claudeで実装（Tailwind / Next.js / Astro / shadcn/ui） | AIpa Web |
| 6 | 人が事実確認（事実関係・給与・権利・スマホ表示・応募導線・SEO） | AIpa Web |
| 7 | 顧客が内容を確認（修正は1回まで） | 顧客 |
| 8 | 公開（独自ドメイン・SSL・アクセス解析・応募通知） | AIpa Web |
| 9 | 月額で更新・改善を代行 | AIpa Web |

フォームを基本選択式にしているのは、入力のハードルを下げるためです。「未定でOK」「相談したい」「提案してほしい」という逃げ道も用意してあるので、完璧に答えられなくても進みます。

### モニター価格（事例掲載に協力いただける会社限定）

通常30〜50万円相当の採用LPを、**初期15万円**で提供します。条件は「公開後、事例として紹介させていただくこと」です。

なお、「3万円で作ります」のような極端な安売りはしません。安すぎると修正が多発し、本気度も下がって、結局どちらも消耗するからです。価格には「最後まで責任を持って公開・運用する」という意味が含まれています。

## 7. よくある質問

**Q. 本当に7日で公開できますか？**
A. フォームの回答が揃い、修正のやり取りがスムーズであれば、最短7日で公開できます。撮影や動画など重い制作が入る場合は別途お時間をいただきます。

**Q. AIが作った文章をそのまま載せるのは不安です。**
A. AIが作るのは初稿までです。給与・待遇・条件など事実に関わる部分は、必ず人が確認し、最後は顧客に確認していただいてから公開します。

**Q. 写真やロゴがありません。**
A. デザインの方向性はこちらで提案できます。撮影が必要な場合はオプションで対応します。素材の使用許可はフォームで確認します。

**Q. 公開した後、自分で更新できますか？**
A. 更新は月額プランでこちらが代行します。「採用ページの担当者を外注する」イメージです。
