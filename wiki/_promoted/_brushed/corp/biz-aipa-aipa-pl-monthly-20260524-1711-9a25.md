# AIpa Web 月次PL試算書（月売上45万円想定）

## 1. 前提条件と事業概要

本試算は、AIpa Webの初期フェーズ（β顧客獲得〜安定稼働直前）における、**「モニター価格（月15万円）× 3社」**という売上想定に基づく月次損益計算書（PL）である。

**【基本前提】**
*   **対象顧客**: 地方中小企業（建設/製造/運輸/サービス業）
*   **契約形態**: 月額サブスクリプション（運用代行含む）
*   **単価**: 月額150,000円（モニター価格。通常価格100〜150万円の初期導入支援枠）
*   **顧客数**: 3社
*   **売上総額**: 450,000円/月
*   **期間**: 開業後3ヶ月目〜6ヶ月目あたりの安定運用期を想定
*   **人件費**: 代表者（大井）の給与は「営業利益」算出前の固定費として処理（※利益剰余金への配当は含まず、事業継続のための最低生活保障として計上）

## 3. 各項目の数字根拠と詳細説明

### 3.1 売上高：450,000円
*   **根拠**: モニター契約3社分の月額料金。
*   **補足**: 通常価格（100〜150万円）ではなく、事例作成を条件としたモニター価格（15万円）を採用しているため、単価は低めだが、LTV（顧客生涯価値）を重視した戦略。3社目は既存顧客からの紹介または求人媒体経由で獲得することを想定。

### 3.2 原価（COGS）：120,000円（売上高比率：26.7%）
AIpa Webの原価構造は、従来のWeb制作会社（人件費が原価）とは異なり、**「AIによる自動化効率」と「外注の最小化」**が鍵となる。

1.  **AI API費用：15,000円**
    *   **根拠**: Claude Pro/API + GPT-4o APIの使用料。
    *   **計算**: 月3社分。各社でフォーム入力→AI構成案生成→コピー生成→実装コード生成を行う。1社あたりのプロンプト量・推論量は約5,000円相当。
    *   **内訳**: Claude Sonnet 3.5（長文生成）、GPT-4o（画像生成・コード補完）、Embedding（検索用）等の組み合わせ。

2.  **ホスティング：5,000円**
    *   **根拠**: Vercel Hobby/Proプランの分割負担、ドメイン維持費（1社あたり1,500円/年）、SSL証明書、バックアップサービス。
    *   **計算**: 3社分で月額約5,000円。Next.js/Astroの静的サイト生成（SSG）により、サーバーコストは極めて低い。

3.  **外注費：80,000円**
    *   **根拠**: AIが苦手とする「写真撮影」「動画編集」「複雑なCMS連携」を外部委託。
    *   **内訳**:
        *   写真撮影・編集：1社あたり20,000円（フリーランスカメラマン）。3社中1社で実施（月2社目は自社撮影で対応）。
        *   特殊実装：1社あたり20,000円（React/Next.jsの微調整）。
    *   **重要**: 通常、Web制作の原価の大部分は「制作工数」だが、AIpa Webでは「フォーム→AI→人間確認」のフローにより、制作工数を極小化。外注費は「品質担保」のための最小限の支出。

4.  **ソフトウェア：20,000円**
    *   **根拠**: Figma（チームプラン）、Notion（ビジネスプラン）、CRM（HubSpot無料〜スターター）、メール配信ツール（Mailchimp等）の月額費。

### 3.3 販管費：100,000円（売上高比率：22.2%）
営業活動と事務管理にかかる費用。

1.  **広告宣伝費：50,000円**
    *   **根拠**: B2B2B戦略（求人代理店・士業経由）を主軸とするが、直接アプローチのためのデジタル広告費。
    *   **内訳**: LinkedIn広告（ターゲット：人事担当、経営者）、Indeed/engageの採用サイト構築キャンペーンへの参加費、Webメディアへの寄稿料。

2.  **旅費交通費：20,000円**
    *   **根拠**: 地方顧客への現地訪問。
    *   **計算**: 月2回、新幹線・レンタカー代。地方顧客との信頼構築には「顔見せ」が不可欠。

3.  **通信費・光熱費：10,000円**
    *   **根拠**: 自宅オフィス兼用時の増加分。

4.  **保険・租税公課：10,000円**
    *   **根拠**: 国民健康保険料の月額平均、法人税（法人化した場合）の予備費。個人事業主の場合は社会保険料の自己負担分を含む。

5.  **事務用品・その他：10,000円**
    *   **根拠**: 契約書印刷、備品消耗品、雑費。

### 3.4 大井人件費：150,000円
代表者（大井）の給与。

*   **根拠**: 初期フェーズでは、代表者が営業・PM・制作・運用を兼任するため、最低生活保障としての給与を設定。
*   **計算**: 月額15万円。これは「手取り」ではなく、事業からの支出として計上。
*   **注意点**: 営業利益が8万円と低い状態では、代表者の手取りは実質的に「利益剰余金」に依存することになる。つまり、**「月15万円の給与は保証されていない」**状態。これはスタートアップの初期段階では一般的だが、リスク管理が必要。

## 4. 損益分岐点の計算

### 4.1 固定費の合計
*   原価（固定部分）：AI API 15,000 + ホスティング 5,000 + ソフトウェア 20,000 = **40,000円**
*   販管費（固定部分）：広告 50,000 + 通信費 10,000 + 保険 10,000 + 事務用品 10,000 = **80,000円**
*   人件費： **150,000円**
*   **総固定費：270,000円**

### 4.2 変動費率
*   外注費は顧客数に比例するため変動費とみなす。
*   1社あたりの変動費（外注費）：80,000円 / 3社 = **26,667円**
*   **変動費率：26,667円 / 150,000円 = 17.8%**

### 4.3 損益分岐点売上高
*   **損益分岐点売上高 = 固定費 / (1 - 変動費率)**
*   **損益分岐点売上高 = 270,000 / (1 - 0.178) = 270,000 / 0.822 ≈ 328,467円**
*   **損益分岐点顧客数 = 328,467円 / 150,000円 ≈ 2.19社**

**結論：顧客が3社あれば、黒字（営業利益8万円）を確保できる。**

## 5. 戦略的考察とリスク

### 5.1 黒字の質
*   営業利益8万円は、代表者の「時間的余裕」を生む。
*   3社目の顧客獲得に注力できる状態。
*   4社目（月額15万円）を追加すれば、営業利益は23万円となり、代表者の給与を20万円に引き上げられる。

### 5.2 リスク
1.  **顧客離脱**: 1社でも解約すると、売上30万円となり、固定費27万円を差し引くと営業利益3万円。人件費15万円を賄えず、赤字転落。
    *   **対策**: 解約防止策（月額運用の付加価値向上、定期MTG）を徹底。
2.  **外注費の増大**: 写真撮影や動画編集の需要が増えると、外注費が原価の大部分を占めるようになり、粗利が圧迫される。
    *   **対策**: AIによる画像生成の精度向上、顧客自身による写真撮影の指導、動画はテンプレート化。
3.  **人件費の未払**: 月8万円の営業利益では、代表者の生活費を賄えない。
    *   **対策**: 初期は個人事業主として節税を図るか、法人化して給与を控除対象とする。または、補助金（IT導入補助金等）を活用し、初期投資を回収。

### 5.3 今後の展開
*   **顧客数5社**: 売上75万円、営業利益18万円。人件費20万円に引き上げ可能。
*   **顧客数10社**: 売上150万円、営業利益108万円。人件費30万円、外注費80万円（2名雇用）を計上しても黒字。
*   **顧客数20社**: 売上300万円、営業利益252万円。法人化し、代表者給与50万円、制作担当2名（各30万円）、営業担当1名（30万円）を雇用可能。

## 6. 結論

月売上45万円（顧客3社）の想定では、**営業利益8万円**を確保できる。これは代表者の最低生活保障としては不十分だが、**損益分岐点（2.19社）を上回っており、事業として成立している**ことを示す。

今後の成長のためには、**「顧客数10社」への拡大**が最重要KPIとなる。そのためには、AIpa Webの「フォーム→AI→公開」の効率性をさらに高め、外注費を抑制しつつ、営業チャネル（求人代理店・士業）を強化する必要がある。
