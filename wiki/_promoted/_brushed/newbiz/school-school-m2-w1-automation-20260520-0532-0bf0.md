# AIpaX School Module 2 Week 1: 「だるい」を消去するAIの力

## 1. 授業の概要と目的

**タイトル**: 『明日から「だるい」がなくなる：AIを活用した日常の自動化入門』
**対象**: 中高生（13-18歳）
**時間**: 90分（講義15分＋ワーク60分＋振り返り15分）
**講師**: AIpaX School 講師

### 学習の目的
この授業では、AIを「魔法の道具」ではなく「作業を代行してくれる部下」として捉え直すことを目的とします。中高生が抱える日常的な「だるさ」（宿題の準備、スケジュール調整、SNSの投稿準備、英語の添削など）を特定し、プロンプトエンジニアリングの基礎を通じて、それらをAIに自動化・効率化するスキルを習得します。

### 核心概念：AIは「思考のアウトソーシング」である
多くの生徒はAIを「答えを教えてくれる検索エンジン」として使います。しかし、真のパワーは「自分の頭で考えたくない、あるいは時間がかかるプロセス」をAIに委ね、その結果を「検証・修正」する点にあります。本授業では、この**「指示（プロンプト）→ 実行（AI）→ 検証（人間）」**のサイクルを体験し、日常の負担を減らす方法を学びます。

---

## 2. 導入：なぜ「だるい」をなくすのか？（15分）

### 2.1. 「だるさ」の正体とAIの役割
講師はまず、生徒たちに以下の質問を投げかけます。
「毎日の生活で、『めんどくさい』『時間がかかる』『やりたくない』と感じることは何ですか？」

生徒からの回答（例：朝の支度、宿題のまとめ、友達への返事、英語の作文）をホワイトボードに書き出します。ここで重要なのは、講師が「それはAIで解決できるかもしれない」と即断せず、まず生徒の感情に共感することです。

**講師の解説ポイント**:
> 「みんなが感じているその『だるさ』、実は大半が『定型作業』か『情報の整理』です。AIは、定型作業や情報整理が得意な『超優秀な部下』です。例えば、料理のレシピを考えるのは大変ですが、冷蔵庫にある食材から『今夜の献立案』を提案させるのは簡単です。AIはあなたに代わって『下準備』をしてくれるのです。」

### 2.2. 4つのシナリオ紹介
本授業で扱う4つの具体的なシナリオを提示します。

1.  **宿題タスク管理AI**: 教科書やプリントの情報を整理し、期限別のタスクリストを作成する。
2.  **勉強スケジュール自動化**: 定期テストの日程と自分の部活動・趣味を考慮し、最適な勉強時間を提案する。
3.  **SNS下書きAI**: 投稿したい気持ちや写真の情報を基に、ハッシュタグ付きの投稿文を生成する。
4.  **英語添削AI**: 書いた英文を添削し、より自然な表現や文法修正を提案する。

## 3. 実践ワーク：各15分×4つのシナリオ（60分）

生徒は、自分の興味や課題に応じて、以下の4つのワークから**1つ**を選び、集中して取り組みます。講師は巡回指導を行い、プロンプトの改善を助言します。

### ワーク1: 宿題タスク管理AI（15分）

**課題**: 手元の教科書、プリント、または学校のポータルサイトから得た情報を、AIに整理させ、実行可能なタスクリストを作成する。

**ステップ**:
1.  **情報収集**: 今週やるべき宿題やテスト範囲をメモする（例：「数学P10-15」「英語単語テスト」「歴史のレポート」）。
2.  **プロンプト作成**: 以下のテンプレートを使ってAIに指示を出す。
    ```text
    君は優秀な学習アシスタントです。以下の宿題情報を基に、効率的に終わらせるためのタスクリストを作成してください。
    条件：
    - 優先順位（重要度と期限）をつけてください。
    - 各タスクに「予想所要時間」を記載してください。
    - 休憩時間を含めた全体のスケジュール案も提示してください。

    【入力情報】
    - 数学：教科書P10-15、期限：金曜日
    - 英語：単語テスト、期限：水曜日
    - 歴史：レポート（500字）、期限：来週月曜日
    ```
3.  **検証・修正**: AIの出力を受け取り、「これならできそうか？」「時間は適切か？」を自分でチェックする。必要に応じてプロンプトを修正する（例：「数学は苦手なので、解説付きの勉強法も提案して」）。

**学習ポイント**: AIは「指示された情報」しか知りません。正確な入力が、正確な出力を生みます。

### ワーク2: 勉強スケジュール自動化（15分）

**課題**: 定期テストまでの期間と、自分の生活リズムを考慮し、崩れない勉強スケジュールを作成する。

**ステップ**:
1.  **状況把握**: 残り日数、教科数、苦手科目、部活動の時間、睡眠時間を把握する。
2.  **プロンプト作成**:
    ```text
    君はプロの学習コーチです。以下の条件で、定期テストまでの勉強スケジュールを作成してください。
    条件：
    - 1日あたりの勉強時間は最大3時間まで（長時間は集中力が持たないため）。
    - 苦手科目には多めの時間を割り当てる。
    - 部活動がある日は、帰宅後の勉強時間を短くする。
    - 週末は復習と予習に充てる。

    【私の状況】
    - 残り：5日
    - 教科：数学（苦手）、英語（普通）、理科（得意）、社会（普通）
    - 部活動：火・木・土の18:00-20:00
    - 就寝時間：23:00
    - 起床時間：6:30
    ```
3.  **検証・修正**: スケジュールが現実的か確認する。「土曜日の朝が空いていない」といったフィードバックがあれば、プロンプトを修正する。

**学習ポイント**: AIは「制約条件」を与えられると、それに応じた最適解を出します。自分の生活リズムを正確に伝えることが重要です。

### ワーク3: SNS下書きAI（15分）

**課題**: 投稿したい写真や気持ちに基づき、SNS（Instagram/TikTok等）用の投稿文を生成する。

**ステップ**:
1.  **素材準備**: 投稿したい写真の内容、伝えたいこと、ターゲット層（友達向けか、広く知ってもらいたいか）を整理する。
2.  **プロンプト作成**:
    ```text
    君はSNSマーケティングの専門家です。以下の情報を基に、Instagram用の投稿文を3パターン作成してください。
    条件：
    - パターン1：カジュアルで友達向け
    - パターン2：おしゃれで雰囲気重視
    - パターン3：情報提供重視（ハッシュタグ多め）
    - 絵文字を適度に使って、読みやすくしてください。

    【素材】
    - 写真：カフェで読書をしている写真
    - 伝えたいこと：新しい本を読んだ感想、カフェの雰囲気が好き
    - ターゲット：同じく読書好きな友人
    ```
3.  **検証・修正**: 生成された文を見て、「自分らしくない」「長すぎる」と感じた場合は、プロンプトを修正する（例：「もっと短く、箇条書きで」）。

**学習ポイント**: AIは「トーン＆マナー」を指定することで、様々なスタイルの文章を生成できます。自分の声を見つけるための素材として使いましょう。

### ワーク4: 英語添削AI（15分）

**課題**: 自分で書いた英文を添削し、より自然な表現を学ぶ。

**ステップ**:
1.  **下書き作成**: 英語でメッセージや作文を書く（例：「友達への招待メール」や「旅行の計画」）。
2.  **プロンプト作成**:
    ```text
    君は英語の家庭教師です。以下の英文を添削してください。
    条件：
    - 文法ミスや不自然な表現を指摘してください。
    - より自然な表現に書き換えた修正案を提示してください。
    - なぜその修正が必要なのか、簡単に理由を日本語で説明してください。

    【私の英文】
    "I want to go to the park with you tomorrow. It is sunny. Let's play."
    ```
3.  **検証・修正**: 添削結果を理解し、修正された英文を自分の言葉として覚え直す。さらに、「より丁寧な表現は？」と追加質問する。

**学習ポイント**: AIは「正解」だけでなく「理由」も教えてくれます。これを学習機会として活用しましょう。

## 4. 振り返りと共有（15分）

### 4.1. ワークの共有
希望する生徒に、自分のワーク結果と、プロンプトの工夫点を発表してもらいます。
例：
- 「スケジュールAIで、部活動の時間を正確に入れたら、現実的な案が出た」
- 「SNS下書きで、3つのパターンが出たので、自分の好みに近いものを選んだ」

### 4.2. 講師のまとめ
講師は、生徒の発表を踏まえ、以下のポイントを強調します。

1.  **AIは「鏡」である**: 入力が雑なら出力も雑です。自分の考えを整理してAIに入力することが、自分自身の思考を深めることにつながります。
2.  **検証が重要**: AIの出力は「下書き」です。必ず自分で読み、修正、追加を行うことで、初めて自分のものになります。
3.  **継続的な改善**: 一度のプロンプトで完璧は求めません。フィードバックを繰り返すことで、より良い結果が得られます。

### 4.3. 次のステップへの提案
- **宿題**: 今週、実際にAIを使って「だるい」作業を1つ自動化し、その効果を記録する。
- **次回予告**: Module 2 Week 2では、AIを活用した「情報収集と整理」について学びます。

## 5. 講師用メモ：指導上の注意点

- **プロンプトの具体性**: 生徒が「宿題をまとめて」とだけ指示すると、曖昧な結果になります。「教科書P10の計算問題3つ」といった具体性を促します。
- **批判的思考の育成**: AIの出力を盲目的に受け入れないよう、「これは正しいか？」「自分ならどうするか？」を常に問いかけます。
- **個別対応**: 生徒のレベルに合わせて、プロンプトの難易度を調整します。初心者にはテンプレートを提供し、上級者には自由記述を促します。
- **倫理観の強調**: AIの使用において、著作権やプライバシーに注意すること、またAIの出力をそのまま提出しないことを明確に伝えます。

## 6. 参考：プロンプトエンジニアリングの基礎フレーム

生徒に以下のフレームを配布し、プロンプト作成の指針とします。

```
[役割設定] + [背景情報] + [具体的な指示] + [出力形式の指定] + [制約条件]

例:
[役割設定]: 学習アシスタント
[背景情報]: 数学が苦手、残り5日
[具体的な指示]: 勉強スケジュールを作成
[出力形式の指定]: 表形式、時間別
[制約条件]: 1日3時間以内、部活動考慮
```

このフレームを意識することで、生徒はより効果的にAIと対話できるようになります。
