# EEMUS（エームス）向け 静岡県中堅・成長企業候補20社リスト

## 概要と選定基準

EEMUSのミッションである「静岡の高校生と地元企業をつなぐ」を実現するため、以下の基準で20社を選定しました。

1.  **業種**: 静岡県が抱える深刻な人手不足（建設、製造、物流、農業・食品、観光）に直結する業種。
2.  **規模**: 従業員数100〜500名程度の「中堅・成長企業」。大企業は採用プロセスが硬直しやすく、中小企業はリソース不足でインターン受入が困難なため、EEMUSのようなプラットフォーム経由でのマッチングが最も効果を発揮しやすい層。
3.  **地域密着性**: 静岡県内（静岡市、浜松市、沼津市、富士市など）に本社または主要事業所を構え、地域貢献への意識が高い企業。

---

## 1. 建設・インフラ業（人手不足率79.5%の最前線）

静岡県は土木・建築分野の担い手不足が深刻です。高校生に「建設の面白さ」をリアルに伝えることで、将来の採用パイプラインを確保する企業群。

### 1. 株式会社清水建設 静岡支店 / 静岡地区関連会社
- **業種規模**: 建設業 / 従業員数：数百名（支店規模）
- **インターン受入メリット仮説**: 大手ブランド力と地域密着の両立。高校生に「大企業の安定性」と「現場のダイナミズム」を同時に体験させ、将来のエンジニアリング人材を早期に獲得。
- **接点**: 静岡県建設業協会、県内高校土木科・普通科との連携。

### 2. 株式会社富士建設
- **業種規模**: 建設業 / 従業員数：約300名
- **インターン受入メリット仮説**: 富士山麓や静岡平野でのインフラ整備実績が豊富。地域に根ざした中堅企業としての信頼性が、高校生の「地元で働きたい」という心理に訴求。
- **接点**: 富士市役所、地元高校のキャリア教育担当教員。

### 3. 株式会社東海建設
- **業種規模**: 建設業 / 従業員数：約400名
- **インターン受入メリット仮説**: 静岡を基盤とする強豪企業。プロジェクトごとのチームワークを体験させることで、若者の「仲間と crete する仕事」の価値を伝達。
- **接点**: 浜松市商工会議所、静岡県建設業組合。

### 4. 株式会社アイシン・エィ・ダブリュ 関連建設部門 / 関連請負会社
- **業種規模**: 製造業関連インフラ / 従業員数：数十〜百数名
- **インターン受入メリット仮説**: 愛知に本拠を置くが静岡にも巨大な存在感。Aisinグループのブランド力を背景に、精密な施工技術や安全管理の重要性を学ばせ、高品質な人材を育成。
- **接点**: 愛知県の採用ネットワーク、静岡県内の高校OBネットワーク。

### 5. 株式会社フジタ 静岡支店 / 関連会社
- **業種規模**: 建設業 / 従業員数：数百名
- **インターン受入メリット仮説**: 大規模プロジェクトへの参画経験があるため、高校生に「社会インフラの裏側」を見せることで、職業への尊敬と興味を喚起。
- **接点**: 静岡県建築士会、県内高校の技術科。

## 2. 製造業・ものづくり（静岡県経済の柱）

- **インターン受入メリット仮説**: 自動車産業のサプライチェーンの要。自動化された生産ラインと、それを支えるエンジニアリングの両方を体験させ、ものづくりの奥深さを伝える。
- **接点**: 静岡県自動車工業会、県内高校の工業科。

### 7. 株式会社アイシンAW（現：アイシン）関連の中小サプライヤー企業

- **インターン受入メリット仮説**: 大企業ではなく、地域に根ざした中堅サプライヤーの魅力（柔軟性、専門性）をアピール。愛知・静岡の産業クラスターの一員としての自負を共有。
- **接点**: 静岡県自動車工業会、地元商工会議所。

### 8. 株式会社富士重工業（SUBARU）関連の地域協力会社

- **接点**: 群馬・長野の採用ネットワークとは別に、静岡支店や関連工場での地域採用枠を活用。

### 9. 株式会社小松製作所 静岡関連事業所

- **インターン受入メリット仮説**: 大型機械の設計・製造プロセスを見学。物理的なスケール感と技術の融合を体験させ、エンジニア志望の高校生を惹きつける。

## 活用アドバイス

1.  **業種別のアプローチ**:
    *   **建設・製造業**: 「技術の面白さ」「ものづくりの誇り」を強調。
    *   **食品・観光業**: 「地域貢献」「SDGs」「地元愛」を強調。
    *   **メディア・サービス業**: 「情報発信」「コミュニケーション」を強調。

2.  **接点の強化**:
    *   各社との直接連絡だけでなく、**静岡県商工会議所連合会**、**静岡県経済連合会**、**静岡県建設業協会**などの業界団体を介した紹介を有効活用してください。
    *   県内高校の**キャリア教育担当教員**とのネットワークも重要です。EEMUSが「地域の未来を担う人材育成プラットフォーム」として位置づけられることで、学校の後押しを得やすくなります。

3.  **EEMUSの価値提案**:
    *   中堅・成長企業は、大手のような大規模な採用イベント予算がない場合が多いです。EEMUSが「低コスト・高効率」で「リアルな職場体験」を提供することで、採用コストの削減と採用品質の向上という両方のメリットを提示してください。

このリストを基に、まずは5〜10社程度からパイロットプログラムを実施し、成功事例（ケーススタディ）を作成することが、今後の拡大に不可欠です。
