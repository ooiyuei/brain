# HR高等学院式「合意形成アドベンチャー」ワークシート
**テーマ：中高生のためのチームビルディング・プロトタイプ**

## 1. 導入：なぜ「合意形成」が必要なのか？

中高生にとって、最も難しい課題の一つは「自分たちの意見がバラバラで、何を決められない」ことです。
「やりたいこと」は人それぞれ。でも、チームで動く以上、**「全員が納得できる答え」**を見つける必要があります。

このワークでは、HR高等学院で実践されている**「合意形成の4ステップ」**を使います。
これは、議論を「対立」ではなく「創造」に変えるための強力なフレームワークです。

---

## 2. ワークシート本体

### 【共通ルール】
- **時間厳守**: タイマーを必ず使う。
- **批判禁止**: 「それはダメ」「無理」という言葉は禁止。「でも」「そして」で繋ぐ。
- **全員参加**: 沈黙しているメンバーには積極的に声をかける。

---

### ステップ1：意見を分解（10分）
**目的**: 「何が好きか」を抽象的な感情ではなく、具体的な要素にバラバラにする。

**【ワーク】**
以下の質問に、各自が自分の希望を「要素」に分解して書き出してください。
例：「海に行きたい」→「波乗りしたい」「新鮮な海鮮が食べたい」「夕日を見たい」

| 項目 | 書き出し例（ヒント） | あなたの希望（要素分解） |
| :--- | :--- | :--- |
| **① 体験したいこと** | アトラクション、スポーツ、観光、ショッピング、食事 | |
| **② 重視する価値** | スピード、贅沢、節約、思い出、写真映え、リラックス | |
| **③ 避けたいこと** | 移動の疲れ、高額、待ち時間、暑さ/寒さ、空腹 | |
| **④ 誰と何をしたいか** | 友達と、先生と、グループで、個人で、食事、ゲーム | |

*(各メンバーが各自のシートに記入。その後、5分間で各自の希望を30秒ずつ発表し、他のメンバーは「なるほど」と頷きながらメモを取る。)*

---

### ステップ2：共通点を探す（5分）
**目的**: 「違い」ではなく「同じ」を見つける。ここが合意形成の鍵。

**【ワーク】**
メンバー全員が書き出した「要素」の中から、**3人以上が興味を示した項目**を探してください。

| 分類 | 共通点として浮上した要素（例：「食事が大事」「写真に残したい」など） |
| :--- | :--- |
| **価値観の共通点** | （例：「思い出を大切にしたい」「予算内で楽しみたい」） |
| **行動の共通点** | （例：「移動は短くしたい」「アクティブに動きたい」） |
| **条件の共通点** | （例：「WiFi環境がある」「トイレが綺麗」） |

**講師への指示**:
「『全員が完全に一致しなくてもいい』と伝えてください。『3人中2人が興味がある』でも立派な共通点です。『誰も興味がない』項目は一旦棚上げにしましょう。」

---

### ステップ3：共通点から再考（5分）
**目的**: 見つかった共通点を使って、新しい視点で問題を再定義する。

**【ワーク】**
ステップ2で見つけた共通点を使って、以下の文を完成させてください。

> **「私たちが本当に求めているのは、〇〇（共通点）を満たすことだ。」**

例：
- 「私たちが本当に求めているのは、**『移動の疲れを最小限にして、食と自然を楽しむこと』**だ。」
- 「私たちが本当に求めているのは、**『全員が写真に収められるような、インパクトのある体験』**だ。」

**講師への指示**:
「この文が『チームの憲法』になります。この文から外れるアイデアは却下、この文に沿うアイデアは採用、という基準を作ります。」

---

### ステップ4：アイデア出し合う（10分）
**目的**: 再定義した目標に向かって、具体的なプランを複数出す。

**【ワーク】**
ステップ3の文を満たすプランを、3つ以上書き出してください。
*「A案」「B案」「C案」のように、異なる方向性の案を出し合います。*

| 案名 | 具体的な内容 | ステップ3の目標との整合性 |
| :--- | :--- | :--- |
| **A案** | | |
| **B案** | | |
| **C案** | | |

**【最終決定】**
3つの案の中から、**「最も全員がワクワクする案」**を1つ選び、その理由を一言で書いてください。

**決定したプラン**: __________________________

## 3. 題材別30分ワーク手順＆講師ガイド

以下の3つの題材から、いずれか1つを選んでワークを進めます。

### 題材A：修学旅行プラン
**【シナリオ】**
クラス全員で京都・奈良へ行く。予算は1人8万円。移動手段は新幹線。
**【講師ガイド】**
- **ステップ1**: 「京都に行きたい」ではなく「伏見稲荷の鳥居を撮りたい」「抹茶スイーツを食べたい」「奈良の鹿と触れ合いたい」など、具体的な体験に分解させます。
- **ステップ2**: 「歴史に興味がある人」と「グルメに興味がある人」の共通点は？→「食と文化をセットで楽しみたい」という共通点が見つかる可能性があります。
- **ステップ3**: 「移動の疲れを減らしつつ、食と文化を深く体験する」ことが目標になります。
- **ステップ4**: 
    - A案：京都中心部でホテルを取り、徒歩で観光（移動最小化）
    - B案：奈良と京都を結ぶバスツアー（効率重視）
    - C案：宇治と奈良を組み合わせ、抹茶と鹿をセットで（テーマ特化）
- **ポイント**: 予算オーバーにならないよう、ステップ2で「予算内」を共通点として組み込むことを促します。

### 題材B：学園祭企画
**【シナリオ】**
クラスで学園祭の出し物を決める。準備期間は2週間。人数は30人。
**【講師ガイド】**
- **ステップ1**: 「模擬店をやりたい」ではなく「売上が欲しい」「多くの人に来てほしい」「自分たちが楽しいものが作りたい」に分解します。
- **ステップ2**: 「売上が欲しい」と「自分たちが楽しい」の共通点は？→「売れるけど、作り手も楽しめるもの」が見つかります。
- **ステップ3**: 「準備期間2週間」という制約を共通点として組み込みます。「短期間で完成し、多くの人と関われる企画」が目標になります。
- **ステップ4**: 
    - A案：手作りポップコーン（準備簡単、匂いで集客）
    - B案：クラス対抗ゲームコーナー（参加型、盛り上がる）
    - C案：インスタ映えするフォトスポット（SNS拡散、準備は簡単）
- **ポイント**: 「準備期間」というリソース制約を、ステップ2で明確に共通点として位置づけることが重要です。

### 題材C：部活動の改革
**【シナリオ】**
部活の顧問の先生が代わり、新しい活動方針を決める。現在の課題は「練習が退屈」「部員同士の仲が悪い」。
**【講師ガイド】**
- **ステップ1**: 「練習が退屈」→「戦略を考えたい」「体力作りより技術向上」「ゲーム形式が欲しい」に分解。
- **ステップ2**: 「仲が悪い」→「コミュニケーション不足」「役割分担が不明確」に分解。共通点として「互いを理解し合える時間が必要」が見つかります。
- **ステップ3**: 「技術向上」と「コミュニケーション」を両立する活動が目標になります。
- **ステップ4**: 
    - A案：スポーツ×戦略会議（練習前に10分ミーティング）
    - B案：ペアリング制度（先輩後輩のペアで課題を設定）
    - C案：外部コーチ招致（新しい視点を取り入れ、刺激を得る）
- **ポイント**: 部活動は「人間関係」が課題になりやすいため、ステップ2で「信頼構築」を共通点として抽出させることが鍵です。

## 4. 講師用：合意形成の核心を教えるポイント

### 1. 「反対」を「素材」に変える
生徒は「自分の案が通らない」と不満を持ちます。
ここで重要なのは、**「反対意見こそが、より良い案を作るための素材」**だと伝えることです。
- 「A案に反対？じゃあ、A案のどこがダメで、B案のどこが欲しいのか？それを分解しよう。」
- こうすることで、対立が「分析」に変わります。

### 2. 「共通点」は必ずしも「妥協点」ではない
多くの生徒が「共通点＝折衷案（どっちつかず）」だと誤解します。
- 「共通点は『方向性』であって、『答え』ではない。」
- 「『移動を少なくしたい』という共通点があっても、それは『電車で行く』ことだけを意味しない。『移動を少なく』という条件を満たすなら、バスでも、現地集合でもOK。」
- こうして、**制約条件を共有し、自由度を高める**ことを説明します。

### 3. 「再考」の重要性
ステップ3で「私たちが本当に求めているのは～」という文を作ることは、**問題の再定義**です。
- 例：「修学旅行でUSJに行きたい」という要望があった場合、表面の要望は「アトラクション」ですが、本質的な要望は「非日常体験」かもしれません。
- もし本質が「非日常」なら、USJでなくても、普段行けない場所へのキャンプでも、本質は満たされます。
- この「本質の抽出」が、画期的なアイデアを生みます。

### 4. 時間配分の厳格さ
- ステップ1（分解）で時間をかけすぎると、ステップ4（創造）で時間がなくなります。
- タイマーを鳴らしたら、必ず次のステップへ移ります。
- 「完璧な答え」ではなく「十分な答え」を見つける練習をさせます。

## 5. 事後フィードバック（5分）

ワーク終了後、以下の3点について全員で振り返ります。

1. **どのステップが最も難しかったか？**
   - 意見の分解？共通点の発見？再考？アイデア出し？
2. **共通点を見つけることで、視点がどう変わったか？**
   - 「自分の希望だけ」から「チームの目標」へ意識が移りましたか？
3. **今回のフレームワークを、他の場面で使えそうか？**
   - 友達との約束、部活の決定、将来の進路選びなど。

**講師の結びの言葉**:
「合意形成とは、全員が同じ意見になることではありません。『それぞれの違いを認めつつ、共通の目標に向かって進む方法』を見つけることです。この4ステップは、未来のビジネスでも、人生の重要な決断でも必ず役に立ちます。ぜひ、次の機会にも使ってみてください。」

**このワークシートの意図**:
このワークは、単に「計画を立てる」ことを目的としていません。
**「対立をどう解決し、創造的な合意に至るか」**というプロセスを体験させることが主眼です。
中高生は、自分の意見が通らないことを「負け」だと捉えがちですが、このワークを通じて、「相手の意見を取り入れることで、より良い答えが生まれる」という**協働の快感**を味わってもらうことが、HR高等学院の教育哲学です。
