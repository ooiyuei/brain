# 思考フレームワーク集 (実戦投入レディ)

> HR高等学院の授業ノウハウを、大井の事業 (AIpa Web / AIpaX / AIpaX school / Testall / EEMUS) に即適用できる形に整理。
> 営業・コンテンツ・PRD・ヒアリング・スクール教材 すべてで使う。

---

## F1: ストーリーテリング 3軸フレーム

```
[普通の思考]  ×  [武器・素材]  ×  [提案した思考]
```

### いつ使うか
- 営業提案書 (顧客の常識を提示→当社の武器→新しい考え方の提案)
- LP/SEO記事 (読者の固定観念→自社の差別化→Before/After)
- X投稿/note (「普通こう思うよね」から逆張りで結論)
- 投資家ピッチ (既存業界の常識→AI×〇〇の武器→新市場の提案)

### 大井事業への適用テンプレ
**AIpa Web 営業文用**:
```
普通: 採用サイトは100万で制作会社に頼む → 数ヶ月後完成 → 応募来ない
武器: AI応募者対応24h + 業界特化制作 + 月運用込み
提案: 月15万で「採用サイト+AIで応募育成」がワンセット → 求人サイト代を半減できる
```

**AIpaX 営業文用**:
```
普通: AI導入したいけど何から始めればいいか分からない (5%しか使えてない)
武器: 会社別保存庫コア + 業務AIツール
提案: 50-80万で「自社専用AI基盤」が3ヶ月で動く → コンサル200万の1/4
```

## F2: 合意形成 4ステップ「見つける」

```
1. 意見を分解
2. 共通点を探す
3. 共通点から再考する
4. アイデアを出し合う
```

### いつ使うか
- 商談ヒアリング (顧客の言いたいこと/隠れニーズ/予算/期待 を分解)
- チーム内議論 (新人や外注先との認識合わせ)
- 共同創業者との方向性合意
- 顧客の意見対立の解消

### 大井事業への適用
- AIpa Web の **16セクションヒアリングフォーム** がこの4ステップを内蔵
- AIpaX のβ顧客との **対象領域定義書** もこの構造
- 「商談時に出た意見をその場で分解→共通点抽出→再提案」 の3分テクで成約率UP

## F3: 個人テーマ探究 11項目ロジックモデル

```
①テーマ概要 ②Why you? ③価値(自分/社会) ④現状 ⑤ありたい姿
⑥問題 ⑦原因 ⑧課題 ⑨解決方法 ⑩仮説検証 ⑪検証結果
```

### いつ使うか
- **新規事業のPRD化** (4-5ステージ分の構造化済)
- **投資家向け事業説明** (10-15分プレゼンの自然な構成)
- **ビジコン応募書類** (夢AWARD等)
- **AIpaX schoolの卒業課題テンプレ**

### 大井事業の現在地マッピング
| 事業 | ①-③ | ④-⑤ | ⑥-⑨ | ⑩-⑪ |
|---|---|---|---|---|
| AIpa Web | 済 | 済 | 済 | モニター3社で検証中 |
| AIpaX | 済 | 済 | 済 | **4社契約済 = ⑪検証結果あり** |
| AIpaX school | 済 | 済 | 済 | 第1期生体験会で⑩⑪取得予定 |
| EEMUS | 済 | 済 | 設計中 | 夢AWARDで⑩⑪を一気に得る |
| Testall | 済 | 済 | 済 | 教育者連携で⑩実施中 |

→ **5事業すべてのステータスを11項目で可視化**して `/departments` や Notion に出すと一目で進捗わかる

## F4: CHINTAI型 サービス企画 5項目

```
①誰のためのもの? (具体的ペルソナ1人)
②どんな暮らしの課題・ニーズに注目?
③どんなサービス・仕組みで叶える?
④アイデア実現に必要なモノ・コト
⑤ビジネスモデル (支払う主体)
```

### 審査基準 (大井の判断軸そのまま)
- 自分が使いたい・友達に勧めたいか? (N=1の視点)
- その会社がやるべきと言えるか? (「どこでもできる」じゃない)
- N=1の視点 × 社会潮流 = 説得力

### いつ使うか
- 新規事業案の即評価 (5分で5項目埋まらない案はNG)
- AIpa Web の各セクションヒアリングの裏フレーム
- AIpaX schoolの「中高生が事業案を作る」課題

## F5: 産業課題×AI解決 4項目フレーム

```
①この産業の今 (現状・規模感)
②課題は何?
③解決策 (AI/技術革新含めて)
④笑顔になるのは誰?
```

### いつ使うか
- AIpaX のβ顧客向け **業界別ヒアリング初期** (3分で4項目埋める)
- 競合分析・市場調査の標準フォーマット
- AIpa Web の **業種別アプローチ文** の構造

### 大井事業への即時適用
| 業界 | ① 現状 | ② 課題 | ③ AI解決 | ④ 笑顔 |
|---|---|---|---|---|
| 中小製造業 | DX率10% | 採用×AI未活用 | AIpa Web (採用)+AIpaX (運用) | 経営者+応募者+現場 |
| 介護業界 | 慢性人手不足 | 採用ミスマッチ | AIpa Web で現場リアル発信 | 介護福祉士+施設+利用者 |
| 中高生教育 | 受験勉強偏重 | AI時代対応皆無 | AIpaX school でAI使い育成 | 親+生徒+講師 |

## F6: メディア戦略フレーム (熱海復活モデル)

```
官民連携 × ブランド再構築 × 年代別ターゲット × メディア媒体シフト
→ 結果可視化 (図表で証明)
```

### 大井の発信戦略に転用
- **テレビ・YouTube** → 上の世代へリーチ
- **X/Instagram** → 若者へリーチ
- **note** → 検索流入+深いストーリー
- **TikTok** → Z世代の認知獲得

### 年代別ターゲット (AIpaX school 親向け×中高生)
- **親(40-50代)** → Yahoo広告/Facebook/LINE/note記事
- **中高生(13-18)** → TikTok/Instagram/X
- **両方** → 体験会で対面接触

## OpenClawの量産プロンプトへの組み込み

> - 普通の思考: 読者がそのテーマで持っている固定観念

> 例 (ジャパネット): 高画質→大きくプリント→贈り物になる」
