# ソロプレナー 11ステップ完全ロードマップ

> シンコード「クロードコードスタートアップの教科書」フレームワーク。
> アプリ開発・SaaS・サース全般で再現性高い。

---

## ステップ一覧

### 1. アイデア発掘
- 自分の痛みから (マイクロフラストレーション・小さなイライラを毎日メモ)
- 競合いてOK = 需要証明 + 1機能特化で隙間狙う
- 海外バズアプリの日本ローカライズ も有効

### 2. 要件定義 + LP作成
- ChatGPT 4.5 or Claude 4.7 と壁打ち
- 要件定義 → Claude Design で LP生成
- ウェイティングリスト機能 (Form Spree) で需要収集

### 3. 広告で需要検証
- X広告 + Meta広告 各1万円ずつ
- ランディングページに誘導 → ウェイトリスト登録数

### 4. CPA計算で需要判定
**CPA = 広告費 ÷ ウェイトリスト人数**
- **100-300円 → 需要あり・開発GO**
- **300円超 → ピボット検討**

### 5. クロードコードで開発スタート
- ツール何でもいい (Claude Code / Codex / Cursor)
- MVP重視 (最小限機能・最低限セキュリティ)
- 課金機能は MVPに含める (バズった時のため)

### 6. SNS運用 (開発と並行)
- ビルドインパブリック (X)
- TikTok / Note / YouTube
- いいね回り・コメント・関係性構築

### 7. MVP ロンチ
- 「完璧主義 = 現実逃避」
- Google認証だけでOK
- 4-5日で出す
- Google Analytics 必須

### 8. 分析
- 課金されたか? (最重要指標)
- 100人ダウンロード→1人課金 = CVR 1% (合格)
- 200人で0人 = ピボット検討

### 9. アップデート (フィードバックループ)
- バケツの穴 (ファネル分析) 探す
- AB テスト (マイクロコピー1個変えるだけで CVR 1.3倍)
- 「明日使う1人」が変わる瞬間を作る

### 10. PMF達成シグナル
**定量**:
- 継続率15%+/30日
- CVR 3%+
- オーガニック成長 (広告止めても増える)

**訂正**:
- 「なくなったら残念」40%+
- 紹介口コミ自然発生

**経済**:
- LTV ≧ CAC × 3

### 11. ユニットエコノミクス → スケール or バイアウト

#### ユニットエコノミクス チェック
```
LTV (Life Time Value: 1ユーザー平均総額)
÷ CAC (Customer Acquisition Cost: 1顧客獲得コスト)
≧ 1 (最低限) ・3 (理想)
```

AI APIコストは LTVから引く (粗利で計算)

#### スケール戦略
- PMF達成後 = 集客拡大期
- 「乾いた燃料で焚き火」イメージ
- 広告・インフルエンサー・紹介プログラム
- 自然拡散 (スクショ映え・1秒で理解できる1画面)

#### バイアウト or 続行
- 連続起業家 = バイアウトして次へ
- 大井タイプ = 「01から11やる側が好き」(本人発言)

## 達成期間目安 (大井向け)

| ステップ | 期間 | 主な作業 |
|---|---|---|
| 1-2. アイデア+LP | 1日 | 痛みリスト→LP生成 |
| 3-4. 広告検証 | 1-2週間 | 1万円投資・CPA測定 |
| 5-7. 開発+リリース | 1-3週間 | MVP・課金実装 |
| 8-9. 分析+改善 | **継続** | 毎週改善 |
| 10. PMF達成 | 1-6ヶ月 | 最大の壁 |
| 11. スケール | PMF後 | 集客拡大 |

→ **「月10万円のアプリ」を1個作る = 平均2-3ヶ月**

## 「PMF達成までは8割ピボット覚悟」

- 1番難しいフェーズ
- 個人開発有利点 = やめやすい (チーム抱えてないから)
- 失敗 = スキル蓄積・無駄じゃない
- 1個目で当てる人 < 5-10個出して1個当てる人

→ **大井の17事業ポートフォリオは正しい戦略**
