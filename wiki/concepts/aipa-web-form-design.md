# AIpa Web フォーム設計（16セクション）

> AIpa Webの中核。顧客がフォーム入力するだけで、AIが採用サイトの構成・コピー・文章・デザイン初稿を作れるレベルの情報を回収する設計。

## 設計方針

**目的**：AIが初稿を作るための材料を集めること（顧客に全部考えさせない）

| 原則 | 内容 |
|------|------|
| 選択式優先 | 自由記述は最小限に |
| 提案希望ボタン | 「相談したい」「未定でOK」を入れる |
| 必須項目を絞る | 詰まらず進める |
| セクション分岐 | 必要な人だけ詳細回答 |
| 添付は最後にまとめる | 入力疲れ防止 |

## 16セクション

### 1. 基本情報
- 会社名
- ご担当者名
- メールアドレス
- 電話番号
- 現在の会社ホームページURL
- 会社の所在地・主な対応エリア

### 2. 今回作りたいサイトについて
- 作りたいサイトの種類
- サイト制作で実現したいこと
- 希望納期
- 公開希望日

### 3. ロゴ・色・写真素材
- ロゴデータの有無
- ロゴデータ添付
- 使いたい色
- 避けたい色
- 写真素材の有無
- 使用できる写真の種類
- 写真・パンフレット・求人票などの添付

### 4. 採用ターゲット
- 採用したい人材のターゲット
- 募集職種
- 採用人数
- 求める人物像
- 求める人物像の補足

### 5. サイト冒頭のコピー・メインビジュアル
- 採用サイトタイトル案
- **求職者に一番伝えたいメッセージ**
- サイト冒頭の説明文
- メインビジュアルの方向性

### 6. 会社紹介・会社の魅力
- 会社紹介文の有無
- 会社紹介・理念・ビジョン・行動指針
- 求職者に伝えたい会社の魅力
- その他の魅力

### 7. 数字で知る会社
- 掲載できる数字情報
- 具体的な数字

### 8. 事業内容・会社の強み
- 事業内容
- 会社の強み・他社との違い
- 取引先・実績・受賞歴

### 9. 仕事内容・職種紹介
- 掲載したい職種数
- 職種ごとの仕事内容
- 仕事の魅力・やりがい
- 仕事で大変なこと
- 1日の流れを掲載するか
- 1日の流れ

### 10. 人・メッセージ
- 社長メッセージ
- 採用担当メッセージ
- メッセージ原稿
- 社員インタビューの有無
- 掲載できる社員数
- 社員インタビューで載せたい内容

### 11. 募集要項・福利厚生
- 募集要項の有無
- 募集要項の内容
- 掲載したい条件
- 福利厚生・制度
- 研修・教育制度・キャリアステップ

### 12. デザインの方向性
- 希望するデザインの雰囲気
- 参考にしたいサイトURL
- 避けたいデザイン・苦手な雰囲気

### 13. 応募導線
- 応募方法
- 誘導したいURL
- 応募後の流れ
- 選考フロー

### 14. 公開後の更新・月額運用
- 公開後に更新したい内容
- 月額運用サポートへの興味
- 更新頻度の希望

### 15. AI利用・権利確認
- AI生成テキストの使用可否
- AI生成画像・ストックフォトの使用可否
- 写真・ロゴ・文章の使用許可確認

### 16. その他
- 事前に伝えておきたいこと

## 自然な日本語表現への変換

制作用語のままだと顧客が固まる → 顧客向け表現に直す。

| 制作用語（NG） | 顧客向け（OK） |
|--------------|--------------|
| 採用キャッチコピーを入力してください | 求職者に一番伝えたいメッセージを教えてください |
| 採用サイトタイトル | 採用サイトのタイトル案があればご記入ください |
| 会社紹介・理念・ビジョンを入力 | 会社紹介・理念・ビジョン・行動指針など、掲載したい内容があればご記入ください |
| メインビジュアル等の写真・素材について | サイトで使える写真素材はありますか？ |

## 必須項目 vs 任意項目

### 必須
会社名 / 担当者名 / メールアドレス / 作りたいサイトの種類 / 制作目的 / 採用ターゲット / 募集職種 / 会社の魅力 / 希望デザイン / 写真素材の有無 / 応募方法 / AI利用可否 / 素材の使用許可

### 任意（AIで初稿作れるから）
採用サイトタイトル / 採用キャッチコピー / 導入文 / 会社紹介文 / 理念・ビジョン / 数字で知る会社 / 事業内容 / 取引先・実績 / 社長メッセージ / 社員インタビュー / 1日の流れ / キャリアステップ / 参考サイトURL

## 検証フロー

1. Googleフォームで16セクション実装
2. 架空企業で回答サンプル作成（静岡建設・物流・美容・製造・飲食）
3. Claudeに投げて初稿出力
4. 実際に1サイト完成させる
5. 検証ポイント：制作時間 / どこで詰まるか / AI精度 / 人間の修正ポイント / フォーム項目の不足
