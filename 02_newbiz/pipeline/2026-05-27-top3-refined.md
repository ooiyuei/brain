## 推奨着手順 (Claude-CEO の判断)

### 🥇 即実行: A1 tobira-log
**理由**: kasunote の LP A/Bテストで1週間以内に CPA測定可能。コード 80% 流用。リスク最小・実行最速。
**6/2 までにやること**:
1. kasunote LP 末尾に「配達・訪問業者向け版を準備中」リンク追加
2. tobira-log 専用 LP (Form Spree ウェイトリスト) 別途用意
3. kasunote 既存ユーザーリストに 1通メール「配達・訪問業者の方は別ツール準備中、ベータ参加しませんか」
4. X広告 1万円投化 (運送・配達・引越し業界ターゲット) → CPA測定

### 🥈 並列: A2 aisho-check (B2B 検証)
**理由**: AIpaX 既存4社にヒアリングだけで広告費ゼロで需要検証可能。
**5/30 までにやること**:
1. AIpaX 4社の経営者面談時に「AI利用ポリシーの悩み」を質問 (3社中2社が「困ってる」なら GO)
2. 並列でLP案を Claude Design で作成 (yakki-check のテンプレ流用)
3. ISO/IEC 42001 関連の検索ボリュームを SEO ツールで確認

### 🥉 7-9月の本番に向けて準備: B1 senbatsu-lp
**理由**: 総合型選抜の受験シーズン (7-9月) に合わせて準備。今は LP と添削エンジンの精度作りに集中。
**6/15 までにやること**:
1. 大井自身の AO 出願書類を few-shot プロンプトに整形
2. Testall ユーザーに「総合型選抜の対策してる?」アンケート (告知 = 広告費ゼロ)
3. 受験シーズン前の 7月 X広告本格投化に向けて LP整備

## 詳細: A2 aisho-check (AI利用ポリシー自動草案)

### 絶対に買う1人: 田中義雄さん 57歳 金属部品加工35名社長
- 息子(28歳)が ChatGPT 使い始めた、社員5名が個人アカウントで使用、顧客図面を貼り付けた疑い
- 顧問弁護士「ポリシー作れ」見積もり50万円→保留中
- このサービス見つける→「月4,980円でたたき台」即決

### LPコピー (最強案)
> キャッチ: 「社員がAIに顧客データを流している。あなたの会社にルールはありますか?」
> サブ: 弁護士費用30万円不要。15分の質問に答えるだけでAI利用規程が完成します。
> CTA: 無料で草案を作る(15分)

### MVP コア機能
業種・規模・利用AIツール・取引先機密性レベル入力(10問) → AI利用規程ドラフトPDF出力。弁護士監修テンプレ + GPT-4o 穴埋め生成。

### 失敗シグナル
- 3ヶ月後 AIpaX4社+紹介で10社未到達 → 自覚なし、コンテンツSEOへ転換
- 解約理由「一度作ったら使わない」 → 都度課金(1草案3,980円)へ
- 2026末GL改定骨抜き → タイミング依存崩壊、ISO単独で再評価

## 詳細: B1 senbatsu-lp (AO志望理由書AI添削)

### 絶対に買う1人: 佐藤麻里さん 17歳 公立高2 (推測)
- バドミントン インターハイ・部長経験あり、一般入試苦手
- 先生「総合型受けてみれば?」→志望理由書書き方分からず
- 学校進路面談月1回15分、地方塾なし、東京予備校費用無理
- 月980円「Netflixより安い」→母クレカ→使い始める

### LPコピー (最強案)
> キャッチ: 「総合型専門塾は月5万円。うちは980円です。」
> サブ: Testall 9,000人が使う学習AIが、志望理由書を4軸でスコアリング。書くたびに上がる合格ライン。
> CTA: 無料で添削してみる

### MVP コア機能
志望理由書テキスト貼付 → 4軸(一貫性・独自性・熱量・論理性)スコアリング + 段落ごと改善コメント。大井のAO出願書類を few-shot に。

### 失敗シグナル
- 無料→有料転換 30%未満 → 都度課金(1添削198円)へ
- 添削回数 平均3回/月以下 → エンゲージメント施策追加
- 2-3月にユーザー全消失 → 季節ビジネス確定、学校B2Bライセンスへピボット

## 詳細: A1 tobira-log (配達・訪問カスハラ記録)

### 絶対に買う1人: 中村浩二さん 42歳 引越下請12名社長
- ヤマト・佐川下請、宅配ドライバー8名抱える
- 去年「荷物床置き」で老人から電話30回 → ドライバー3ヶ月後辞職 → 採用費40万損失
- 別ケース「訪問時傷」証拠なく5万示談 → 「記録あれば」と後悔
- 2026/10カスハラ義務化ニュース見た
- 月980×8=7,840円「採用費40万 vs 月7,840円」即決

### LPコピー (最強案)
> キャッチ: 「ドライバーが怒鳴られた。証拠がなくて5万円払った。あの日に戻れたら。」
> サブ: 玄関先カスハラをワンタップで記録。GPS・タイムスタンプ付きPDF即時出力。2026年10月の義務化前に備える。
> CTA: 14日間無料で試す

### MVP コア機能
スマホアプリ ワンタップ録音 + GPS/タイムスタンプ自動 + 管理者ダッシュボードリアルタイム送信 + PDF即時出力。

### 失敗シグナル
- kasunote LP A/Bテスト 1週間内 クリック率既存の50%未満 → 訪問業者向け独立LP化
- 導入10社のドライバー月間記録ゼロ → 実運用未使用、UI ウォークスルー必要
- 2026/10後 競合大手(勤怠SaaS)が類似機能 → kasunote+tobira-log+yakki-check 統合ダッシュボード化へ
