## 新機能・新製品リリース

### Anthropic — Claude 課金体系の刷新（6/15 施行）
- **内容:** Agent SDK と `claude -p`（ヘッドレス実行）をサブスクプールから分離し、API 標準レートの月次ドルクレジットに移行。従来の 15〜30 倍の補助が終了。
- **対象:** プログラマティックに Claude を回す開発者・自動化ユーザー（＝大井の OpenClaw / 定期タスク）。
- **料金変化:** Pro=$20/月（〜50タスク）、Max 5×=$100/月（〜250タスク）、Max 20×=$200/月（〜500タスク）。Enterprise Standard 席は $0 クレジット＝Premium 席か専用席へ要アップグレード。
- **API 変更:** 対話型 Claude Code TUI・Cowork・claude.ai は明示的に対象外（影響なし）。Web Search ツールが SEC filing データを返すよう強化。
- **要対応:** ①直近30日の Agent SDK 使用量を監査 ②サブスククレジット vs 直接 API 課金を比較 ③プロンプトキャッシュ有効化（キャッシュヒットで入力 0.1 倍）④6/15 前にクレジット opt-in。
- **ソース:** https://www.digitalapplied.com/blog/anthropic-claude-credit-overhaul-june-15-2026 / https://www.infoworld.com/article/4171274/anthropic-puts-claude-agents-on-a-meter-across-its-subscriptions.html
- **信頼度:** 高（複数メディアで一致、InfoWorld 含む）

### OpenAI — GPT-5.5 Instant が ChatGPT 新デフォルト
- **内容:** 5/5 に GPT-5.5 Instant をリリース、高難度プロンプトで幻覚 52.5% 減。5/7 に推論・翻訳・文字起こし可能なリアルタイム音声モデルを API 公開。
- **対象:** 全 ChatGPT ユーザー（デフォルト切替）＋音声 API 利用開発者。
- **API 変更:** **GPT-4.5 は 6/27 に ChatGPT から引退**（30日サンセット後）。GPT-5.6 はバックエンドにシグナルあるがモデルカード・エンドポイント・ベンチ未公開＝未確認。
- **ソース:** https://techcrunch.com/2026/05/05/openai-releases-gpt-5-5-instant-a-new-default-model-for-chatgpt/ / https://openai.com/index/gpt-5-5-with-trusted-access-for-cyber/
- **信頼度:** 高（TechCrunch・OpenAI 公式）

### Google — Gemini CLI を Antigravity CLI へ統合、Gemini Intelligence 展開
- **内容:** **6/18 から Gemini CLI / Code Assist IDE 拡張が AI Pro・Ultra・無料枠でリクエスト提供を停止**、マルチエージェント基盤 Antigravity CLI に一本化。Gemini 3 Deep Think を AI Ultra 向けに展開。6月下旬から Gemini in Chrome / Personal Intelligence を Android に。
- **対象:** Gemini CLI 利用者（要 Antigravity 移行）、Android（Galaxy S26/Pixel 10 先行）。
- **ソース:** https://developers.googleblog.com/an-important-update-transitioning-gemini-cli-to-antigravity-cli/ / https://blog.google/products-and-platforms/platforms/android/gemini-intelligence/

- **信頼度:** 高
