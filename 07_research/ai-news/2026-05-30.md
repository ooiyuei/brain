## 新機能・新製品リリース

### Claude Opus 4.8 / Claude Code / Claude Design (Anthropic)
- **内容:** Opus 4.8 (判断・正直さ・長時間自走の強化)、Claude Code に dynamic workflows・Fast mode 高速化・agent/browser/plugin/MCP の拡張、Anthropic Labs から Claude Design (デザイン・プロトタイプ・スライド・1ページャー共同制作) を新発表。法務向けに MCP コネクタ20+・実務分野プラグイン12本を追加。
- **対象:** Claude API ユーザー全員 / claude.ai / Claude Code / Legal teams
- **料金変化:** Opus 4.7 と同価格。Fast mode は値下げ
- **API 変更:** Opus 4.8 即時利用可
- **ソース:** [9to5Mac - Opus 4.8](https://9to5mac.com/2026/05/28/anthropic-upgrades-claude-with-new-opus-4-8-model-heres-whats-new/), [Anthropic News](https://www.anthropic.com/news)
- **信頼度:** 高

### GPT-5.5 Instant (OpenAI)

- **対象:** 全 ChatGPT ユーザー、API利用者
- **料金変化:** 公表なし (既定モデル切替)
- **API 変更:** `chat-latest` 経由で利用、GPT-5.3 Instant は 3ヶ月併存後に廃止
- **ソース:** [TechCrunch](https://techcrunch.com/2026/05/05/openai-releases-gpt-5-5-instant-a-new-default-model-for-chatgpt/), [OpenAI 公式](https://openai.com/index/gpt-5-5-instant/)
- **信頼度:** 高
