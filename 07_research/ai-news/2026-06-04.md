### Anthropic — サブスクのプログラム利用課金を分離（2026-06-15 施行）
- **内容:** Agent SDK・`claude -p`・Claude Code GitHub Actions・Agent SDK 認証のサードパーティアプリが、サブスク定額枠から外れ「月次クレジットプール」へ移行。**標準 API 定価で従量課金、未使用分の繰越なし**。ターミナルの対話 Claude Code と web チャットは従来どおりサブスク内。
- **クレジット枠:** Pro $20 / Max 5x $100 / Max 20x $200（Team/Enterprise は席単位・従量の別変種）。
- **オーバーフロー挙動:** 「usage credits」トグル ON＝枠超過後も API 単価で課金継続／OFF＝枠消尽後はリクエスト拒否。
- **対象:** **OpenClaw（24h 自走・headless）が直撃対象**。CI/自動化パイプラインの予算が月中で効くようになる。
- **料金変化:** 補助価格 → API 実価格へ。利用量次第で実質 12〜150 倍。
- **API 変更:** 別件で 6/15 に Sonnet 4 / Opus 4 を API 退役（既報・継続フォロー）。
- **ソース:** https://codersera.com/blog/anthropic-june-2026-billing-change-claude-code/ , https://devtoolpicks.com/blog/anthropic-splits-claude-subscriptions-agent-sdk-credit-june-2026 , https://www.techtimes.com/articles/317625/20260602/anthropic-ends-subscription-subsidy-agents-june-15-credit-pool-replaces-flat-rate-access.htm
