## 2. 受賞者の年齢分布

| 年齢層 | 傾向 |
|---|---|
| 10代（高校生） | 高校生特化コンテストで有力。一般コンテストでは希少性が強みになる |
| 20代前半（大学生） | 最も多い層。大学発スタートアップが多数 |
| 20代後半〜30代 | 一般コンテスト（ICC/IVS/B Dash）は社会人スタートアップが主流 |
| 10代で法人代表 | 全コンテスト通じて非常に希少（ほぼ前例少ない）→ストーリー価値が最大化 |

**大井への含意**: 17歳・法人代表・実際の事業（Testall/EEMUS開発）という属性は、あらゆるコンテストで「最若手起業家」としての希少性を持つ。**審査員心理として「この子が本物なら応援したい」という感情が動きやすい。**

## 3. 事業フェーズ別傾向

| フェーズ | 向いているコンテスト | 注意点 |
|---|---|---|
| アイデア段階 | 高校生グランプリ、夢AWARD、TSG、日本アントレプレナー大賞 | ストーリーの強さが勝負 |

**現時点の大井**: Testall（開発中・売上なし）、EEMUS（企画中）→ プロトタイプ段階。**アイデア〜プロトタイプ対応コンテストが最適。**

## 4. スライド・プレゼン形式の傾向

調査した受賞者のプレゼン共通パターン（複数コンテスト公開資料・審査員コメントより）：

### 共通構造（勝ちスライド5枚）

1. **課題（Who/Pain）**: 具体的な人物像と痛みを1枚で
2. **解決策（How/Why Us）**: 自分にしかできない理由
3. **ビジネスモデル（Money）**: 誰が何にお金を払うか
4. **牽引力（Traction）**: 実績・数字・検証した証拠
5. **チーム・ビジョン（Dream）**: なぜ自分がやるのか

### 高校生向けコンテストでの追加ポイント
- 「社会課題との接続」を必ず入れる（SDGs枠・共感軸）
- 「自分の体験」を起点にするプランが評価されやすい
- 審査員が「応援したくなる」物語性

## 5. 受賞事業の特徴キーワード（出現頻度順）

1. **社会課題 × 新技術** の掛け算（最多）
2. **地方 × グローバル** のギャップ解消
3. **弱者支援**（障害者・高齢者・シングルマザー・不登校）
4. **既存産業のDX**（農業・製造業・医療・教育）
5. **創業者の原体験**（自分が困った→解決した）

---

## 6. 大井湧瑛 勝ちパターン仮説

### 強みとして使えるカード
- 17歳・高校生・法人代表 → ストーリー希少性 最大
- 清水（静岡）出身 → 地方コンテスト・地方創生枠で有利
- AIコンサル（Almeo） → 生成AI領域が評価される文脈と合致
- Testall開発経験 → プロダクト思考・技術力を証明できる
- EEMUS（介護DX） → 社会課題（高齢化）との直結

### 弱みと対処
| 弱み | 対処法 |
|---|---|
| 売上ゼロ | 「顧客ヒアリング件数・パイロット検討企業数」で代替 |
| チームが薄い | ソロ起業家の強みとして「自律・スピード」を訴求 |
| AIコンサルは抽象的 | 具体案件1本を事例として提示 |
| 法人設立経緯が説明難 | 「高校生なのに法人設立した理由」をストーリーの核心に |

### 最適プレゼン戦略
```
高校3年（17歳）、清水出身。
介護施設でのアルバイトで、現場のDXの遅れに衝撃を受けた。
でも、ITを説明できる人間がいない。
「通訳者がいれば解決できる」と思い、Almeoを創業。
今、EEMUSで介護AI支援SaaSを開発中。
静岡の介護施設5社とのヒアリングから始まった。
```

※上記はサンプル。実際の体験に基づいて修正すること。
