# Almeo ロゴ コンセプト案 5案

> Claude (俺) は実際の画像ファイルは作れない。
> このドキュメントは Figma / Canva / 外注デザイナーに渡す **「コンセプト指示書」**。
> 5案あって、大井がどれを実装するか決める。

---

## 共通前提

- ブランドステートメント: 「静岡の中小企業と、一緒に走る。」
- パーソナリティ: 誠実・静か・本気・若い・地に足
- 色: `--almeo-black` (#1a1a1a) on white がメイン
- フォント方向: Inter / DM Sans / Manrope のいずれか (geometric sans-serif)

---

## 案1. **シンプル・ワードマーク** ⭐ Day 1 推奨

### コンセプト
「ALMEO」5文字をピュアに置く。装飾なし。Apple / MUJI 系の最小主義。

### 視覚仕様
```
┌─────────────────────────────┐
│                             │
│         ALMEO               │
│                             │
└─────────────────────────────┘

フォント: Inter Display Bold (or DM Sans Bold)
ウェイト: 700 (Bold)
字間 (letter-spacing): -0.02em (緊密)
カラー: #1a1a1a
背景: white or #f5f1ea (beige)
```

### 強み
- **Day 1 で即実装可能** (Figma 5分・CSS のみで実現)
- ブランド成熟度ゼロでも違和感ない
- どんな媒体でも読める (favicon・名刺・LP)
- 拡張性最大 (将来シンボル追加可)

### 弱み
- 「らしさ」が薄い (他社と被る)
- 印象に残りにくい (3秒で覚えられない)

### 適性
- **Day 1 公開時の暫定ロゴ**
- 法人登記前で「シンプルさが武器」のフェーズ

---

## 案2. **アンダーライン付き** (動詞性)

### コンセプト
「ALMEO」の下に細い水平線。「下から支える」「進んでいく」「地に足ついてる」を視覚化。
タグライン「一緒に走る」と意味的に呼応。

### 視覚仕様
```
┌─────────────────────────────┐
│                             │
│         ALMEO               │
│         ─────               │
│                             │
└─────────────────────────────┘

- ワードマーク: Inter Display Bold #1a1a1a
- アンダーライン: 1.5px solid #2d5f3f (forest green)
- アンダーラインの幅: ALMEO の幅と同じ
- 文字とラインの間: 6px
```

### バリエーション
- ライン色を `--almeo-forest` にしてアクセント (推奨)
- ライン色を `--almeo-black` にしてシック
- ライン代わりに「↗」矢印にして「上昇」表現 (推測:過剰演出かも)

### 強み
- 案1 より「動きと意志」を含む
- アンダーライン1本で「静岡」「自然」「進む」3つの意味
- 名刺・印刷でも美しい

### 弱み
- 装飾増えるので、面によって使い分け必要

### 適性
- **Week 1-2 で本実装する案**
- LP の Header・名刺・ピッチ表紙

## 案3. **A の山モチーフ** (静岡らしさ)

### コンセプト
「ALMEO」の **A** の三角形を「山」「テント」に見立てる。
静岡 = 富士山・南アルプス・天竜川流域。地に根ざした象徴。

### 視覚仕様
```
┌─────────────────────────────┐
│                             │
│    ▲ L M E O                │
│                             │
└─────────────────────────────┘

- A の代わりに、より三角的なシンボル「▲」
- 残り4文字 (LMEO) は Inter Bold で揃える
- 三角形は #1a1a1a or #2d5f3f
- 三角形の底辺は他文字と揃える
```

### バリエーション
- 三角形を中空にして「山輪郭」だけ
- 三角形の中に細い水平線を入れて「山道」「川」表現
- 「▲」を傾けて「動き」追加 (推測:バランス難)

### 強み
- 静岡 / 地方密着の物語が視覚化
- 「A」の独自性で他社と差別化
- 拡張時にシンボル単独使用しやすい

### 弱み
- 実装難度 中-高 (フォントカスタマイズ必要)
- 「山」と「テント」を読み取られない可能性
- 抽象度高くて「Almeo らしさ」に直結しないかも

### 適性
- 中長期 (Phase 2) で本格採用するなら
- 大井が「静岡の物語性」を強く出したい時

## 案4. **2文字シンボル「Ao」** (拡張用)

### コンセプト
- フルマーク = 「ALMEO」
- 短縮シンボル = 「Ao」 (Almeo の最初と最後の英文字)
- favicon・アイコン・小サイズで使う

### 視覚仕様
```
┌─────────────────────────────┐
│                             │
│    ┌──┐                     │
│    │Ao│                     │
│    └──┘                     │
│                             │
└─────────────────────────────┘

- 円形 or 角丸正方形 (12px) の中に「Ao」
- A は大文字、o は小文字 (大小コントラスト)
- 背景: #1a1a1a / 文字: #ffffff (反転版あり)
- サイズ: favicon 32px / app icon 1024px まで対応
```

### バリエーション
- 「Ao」を Bold Italic にして「動き」追加
- 円形・角丸・正方形の選択
- インバース版 (白背景・黒文字)

### 強み
- favicon / アプリアイコン / SNS プロフィール画像で使える
- 案1〜3 のフルマークと組み合わせ可能
- スケーラブル (12px〜1024px)

### 弱み
- フルマークと独立しすぎる可能性 (ブランド統一性に注意)
- 「Ao」が「青」と誤読される可能性 (日本人読者)

### 適性
- favicon・アプリアイコン・SNS プロフィール
- 案1 or 案2 のフルマークの **補完シンボル** として

## 案5. **走る人モチーフ** (タグライン直接表現)

### コンセプト
「一緒に走る」を visualize。
ALMEO の文字の「O」を「ランナーが進む足跡」 or 「O」自体が「走る人の頭」に見える形状に。

### 視覚仕様
```
┌─────────────────────────────┐
│                             │
│    ALME◯                    │
│         ➜                    │
│                             │
└─────────────────────────────┘

- O を強調 (他文字より太く・大きく)
- O の右上に小さな矢印「➜」(動きの方向)
- カラー: #1a1a1a + #2d5f3f (forest accent for the arrow)
```

### 強み
- タグラインと完全に呼応
- 動的・進行性
- 他社と被らない独自性

### 弱み
- やりすぎ感 (装飾過多のリスク)
- 解釈が読者依存 (「ランナー?」と分からないかも)
- 印刷時に矢印が消えやすい

### 適性
- Phase 2 以降のチャレンジ案 (Day 1 不採用)
- X バナー・キャンペーンビジュアル

## 推奨ロードマップ

### Day 1 (今夜〜明日朝)
- **案1 (シンプルワードマーク)** を CSS で実装
- favicon は **案4 の「Ao」**を 32x32px の SVG で
- ヒーローセクションでロゴを大きく見せる

### Week 1 (Day 2-7)
- **案2 (アンダーライン付き)** を Figma で作って正式版に
- 名刺デザインも案2 で

### Month 2-3
- **案3 (A の山モチーフ)** を試作 (大井判断次第)
- メディア露出時のビジュアル強化

### 永久に保留
- 案5 (走る人) は採用しない可能性高い (装飾過多)
- ただし「キャンペーンビジュアル」として一時利用は OK

## ロゴ実装の発注先 (大井判断)

### 自分で作る (推奨)
- **Figma** で 1-2時間 (大井・無料)
- 案1 〜 案4 は技術的に Figma で完結
- 大井が直接いじれる = 修正即時可能

### Canva
- テンプレ豊富 (有料 ¥1,500/月)
- ただしオリジナリティ低い・「Canva 感」出る

### 外注 (Lancers / クラウドワークス)
- ¥3万-10万
- 期間 1-2週間
- 修正回数限定
- → **Phase 2 (Week 2 以降) で検討**

### 外注 (デザイン事務所)
- ¥30-100万
- 期間 1-2ヶ月
- 高品質
- → **法人化後・売上安定後**
