# Almeo ロゴ v3 — 大井美意識フル反映

> **brain 抽出した大井美意識**:
> - 「美しい構造 × 弱者の違和感 × 主人公感」で脳汁
> - 「ダサい」が最大の忌避 (平凡・地味・安っぽい NG)
> - 比喩好き: PCスペック / OS / ゲーム / **武器・刀・焼け野原・ぶち抜く**
> - メタルギア・恐るべき子供達計画への憧れ
> - ストイック + ハードコア + サイバー感
> - 黒・白・グレー硬質 + アクセント1色
> - インテリ × ストリート の混合
> - 「俺っぽい・主人公感」がブランドアセットに必要
>
> → **「Apple/MUJIっぽい品」だけじゃ大井は燃えない。武器・OS・サイバー要素を盛る。**

---

## §1. 大井美意識マトリクス (デザイン要件への翻訳)

| 大井の感性 | デザイン要件 |
|---|---|
| 「美しい構造」 | 幾何学的・グリッド・ジオメトリック |
| 「主人公感」 | ヒロイック・主役性・中心性 |
| 「ダサくない」 | 装飾過多 NG・余白多め・厳選1要素 |
| 「武器・刀」 | エッジ立つ・シャープ・直線優位 |
| 「OS・ターミナル」 | モノスペース・コード感・カーソル |
| 「ゲーム」 | ロゴアートの強度・記号性 |
| 「メタルギア」 | 角張った装甲感・サイバー金属 |
| 「焼け野原・ぶち抜く」 | 強い動詞性・破壊性・ストロークの太さ |
| 「主人公・俺」 | 一人称的・断言型・「俺が」の宣言 |

### 🥇 v3-1. **`> almeo_` ターミナルカーソル風** ⭐ 一番俺っぽい

#### コンセプト
- 行頭プロンプト `>` + 社名 + カーソル `_` (点滅)
- **コードを書く17歳起業家** をそのまま記号化
- メタルギア OS / ハッキング / プログラマー世界観
- カーソル `_` = 「次のコマンド待ち」= 「次に何やる」を問い続ける大井の哲学

#### 視覚仕様
```
┌──────────────────────────┐
│                          │
│   > almeo_               │
│                          │
└──────────────────────────┘

フォント: JetBrains Mono (or IBM Plex Mono) Bold
> (プロンプト): 緑#2d5f3f or 白#fff
almeo: 黒 #1a1a1a (ライト) / 白 #fff (ダーク)
_ (カーソル): 点滅アニメーション or 静的下線
背景: 黒 #1a1a1a (ダークモード推奨)
```

#### バリエーション
- a) 通常: `> almeo_`
- b) コマンド実行中: `> almeo --run`
- c) 静止 (ロゴ): `> almeo_` (カーソル下線静的)
- d) アニメ (Web): カーソル点滅 1Hz

#### 強み (大井刺さり)
- ✅ **「俺は技術者で経営者」** のシンボル直撃
- ✅ メタルギア OS / ハッカー世界観 ぴったり
- ✅ 「ダサくない・サイバー」両立
- ✅ AI 文字列 (codestral / claude code / gpt-cli 等) と同じ系譜
- ✅ 17歳代表 + ハードコア = アンダードッグ的主人公感
- ✅ favicon は `_` 単独 (カーソルだけ) で覚えやすい
- ✅ 「Almeo を起動する」感

#### 弱み
- ⚠ 商工会のおじさん 60代には「何これ」感あり (説明必要)
- ⚠ 完全フォーマル提案 (税理士/弁護士向け) では浮く可能性

#### 適性
- **Day 1 メインロゴ** (Web Hero / X bio / 名刺裏)
- **大井の発信ロゴ** (note・X)
- フォーマル提案では **§v3-2 (装甲枠)** との併用

#### 実装コード (Day 1 即動く)

```tsx
// components/ui/LogoTerminal.tsx
export function LogoTerminal({ blink = true }: { blink?: boolean }) {
  return (
    <div className="font-mono text-2xl font-bold tracking-tight flex items-center">

      <span>almeo</span>
      <span className={blink ? 'animate-pulse' : ''}>_</span>
    </div>
  );
}
```

```css

        font-weight="700" font-size="22" fill="#fff"
```

### 🥈 v3-2. **`▌ALMEO▐` 装甲ステンシル** メタルギア直撃版

#### コンセプト
- ALMEO の左右に **装甲ブラケット `▌▐`** (or ステンシル枠)
- 軍用車両・武器箱・メタルギア風
- ハードコア度マックス
- 商工会向けには「強さ・骨太」と説明可能

#### 視覚仕様
```
┌──────────────────────────┐
│                          │
│   ▌ALMEO▐                │
│                          │
└──────────────────────────┘

フォント: Inter Display Black (or Eurostile Heavy)
▌ ▐: 黒い太いバー (ALMEO の高さ × 1.2倍)
ALMEO: 黒 #1a1a1a, ウェイト 900, 字間 +0.05em (ステンシル感)
背景: 白 #fff or グレー #f5f1ea
```

#### バリエーション
- a) 通常: `▌ALMEO▐`
- b) シンプル: `ALMEO` 単独 (バー外す)
- c) フル装甲: `[▌ALMEO▐]` (角括弧追加)
- d) 数字付き: `▌ALMEO-01▐` (型番風・ロボ感)

#### 強み (大井刺さり)
- ✅ **メタルギア / 武器箱 / 装甲** 直撃
- ✅ 「ぶち抜く」「焼け野原」イメージ
- ✅ シールド感 = 中小企業を守る右腕
- ✅ 強烈な視覚インパクト (X バナーで効く)
- ✅ ストイック・ハードコア

#### 弱み
- ⚠ favicon サイズ (32px) ではバーが消える → favicon は ALMEO 単独
- ⚠ ライト系 (MUJI 系) との両立が難しい

#### 適性
- **X バナー・OG画像・キャンペーンビジュアル**
- 「俺が前線に立つ」発信ロゴ
- フォーマル提案: バー外して `ALMEO` 単独で

### 🥉 v3-3. **`Λ|MEO` 電脳ジオメトリック** AI 隠し符号化

#### コンセプト
- A をギリシャ文字 **Λ** (ラムダ・関数型プログラマー記号)
- L を **|** (パイプ・Unix のパイプライン)
- 「AI = ラムダ計算 + パイプライン」のオタク的暗号
- AL の AI 化を **強制的に視覚化**

#### 視覚仕様
```
┌──────────────────────────┐
│                          │
│   Λ|MEO                  │
│                          │
└──────────────────────────┘

フォント: IBM Plex Sans (or Inter) Bold
Λ (ラムダ): A の代わり、底辺なし
|: L の代わり、垂直線のみ
MEO: 通常のセリフ無しフォント
カラー: 黒 #1a1a1a / アクセント #2d5f3f
```

#### バリエーション
- a) 通常: `Λ|MEO`
- b) 装飾: `Λ.|.M.E.O` (区切り点)
- c) アクセント: `Λ|MEO` (Λ だけ forest green)

#### 強み (大井刺さり)
- ✅ **オタク的・関数型・サイバー** 三拍子
- ✅ AI = ラムダ計算 (理論計算機科学) との暗合
- ✅ Unix パイプ = 「OS の右腕」哲学
- ✅ 「読める人にしか読めない」秘密記号性
- ✅ 大井のインテリ感を出す

#### 弱み
- ⚠ 一般人には読みにくい (`Λ|MEO` を ALMEO と読ませる難)
- ⚠ Web URL バーに表示時、ブラウザフォントで化ける可能性
- ⚠ favicon 不向き

#### 適性
- **Phase 2** のテックメディア向けロゴ (note / Qiita / Zenn)
- 開発者向けピッチ・社内資料・GitHub Org アイコン

### v3-4. **`almeo/` フォルダパス風** OS 構造化

#### コンセプト
- 末尾にスラッシュ `/` = ディレクトリ表現
- OS の「中に入っていく」感
- 「Almeo の中には全部ある」哲学
- ターミナルで `cd almeo/` を打つ感

#### 視覚仕様
```
┌──────────────────────────┐
│                          │
│   almeo/                 │
│                          │
└──────────────────────────┘

フォント: JetBrains Mono Bold
カラー: 黒
/: 細い (1.5px stroke)
```

#### 強み
- ✅ OS 比喩直撃
- ✅ シンプル + 拡張性
- ✅ Easter egg として `almeo/web/` `almeo/school/` 展開可

#### 弱み
- ⚠ パッと見「URL」っぽい (「これクリックできるの?」感)

#### 適性
- 開発者向けセクション
- GitHub Org

#### コンセプト

- 「デジタルから滲み出す」感
- レトロサイバー (TRON / Cyberpunk 2077)

#### 視覚仕様
```
┌──────────────────────────┐
│                          │

│                          │
└──────────────────────────┘

フォント: JetBrains Mono
░ 文字: グレー #999 → 透明にフェード
```

#### 強み
- ✅ レトロサイバー
- ✅ 「アナログから AI へ移行する」物語

#### 弱み
- ⚠ ピクセル文字は印刷で再現困難
- ⚠ 解像度依存

#### 適性
- 限定キャンペーン用 (Phase 3)

### v3-6. **`A.` モノグラム** ストイック極致

#### コンセプト
- 「A」+ ピリオド単独
- MUJI / Acne Studios / Aesop 系の超ミニマル
- 「説明しない強さ」
- favicon と表ロゴ統一可能

#### 視覚仕様
```
┌──────────────────────────┐
│                          │
│   A.                     │
│                          │
└──────────────────────────┘

フォント: Inter Display Black (or Times Bold)
A: 縦長 / ピリオド: A の右下に密着
カラー: 黒
```

#### 強み
- ✅ 超ミニマル
- ✅ 「俺は説明しない」強さ
- ✅ favicon 即対応
- ✅ Acne / Apple のような風格

#### 弱み
- ⚠ ブランド認知ゼロでこれは早すぎる (3年後採用候補)
- ⚠ Almeo であることが伝わらない

#### 適性
- **Phase 3 以降** のブランド成熟時の最終形
- Day 1 は不採用

### v3-7. **`{ almeo }` ブレース囲み** プログラマー文化

#### コンセプト
- 中括弧 `{ }` で囲む = JSON / オブジェクト / クラス定義
- 「Almeo オブジェクトを起動する」感
- TypeScript / React の文化を持つ起業家のロゴ

#### 視覚仕様
```
┌──────────────────────────┐
│                          │
│   { almeo }              │
│                          │
└──────────────────────────┘

フォント: JetBrains Mono Bold
中括弧: 細め (1.5px stroke)
内側 padding: 8px
```

#### 強み
- ✅ プログラマー直撃
- ✅ JS/TS 開発者から愛される
- ✅ コードと組み合わせ可能 (`{ almeo: 'jp' }`)

#### 弱み
- ⚠ favicon サイズで括弧が消える

#### 適性
- 開発者向けセクション
- GitHub Org

## §5. ロゴと色の組合せ (大井好み)

- 黒 + 白 + アクセント1色 が好み
- スピリチュアル色 (パステル・紫) NG
- ビビッド色 NG
- 静岡 = forest green 採用済
